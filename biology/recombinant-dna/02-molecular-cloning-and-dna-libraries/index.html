2. Molecular Cloning and DNA Libraries

Molecular cloning combines the tools from Chapter 1 into a complete workflow for isolating and propagating a specific piece of DNA - and, scaled up, for capturing an organism's entire genetic content as a searchable library.

The Basic Cloning Workflow

Steps in Molecular Cloning

A DNA fragment of interest and a vector are cut with the same restriction enzyme (Chapter 1), mixed together so complementary sticky ends anneal, and sealed by DNA ligase - forming a recombinant molecule that is then introduced into a host cell (transformation, Chapter 6 of Genetics), where the vector's origin of replication allows it to be copied every time the host divides, producing many identical copies (clones) of the original insert.

Selecting for Successfully Transformed Cells

Because transformation is inefficient, vectors carry a selectable marker (commonly antibiotic resistance) so that only cells that successfully took up the vector survive subsequent antibiotic selection; additional screening (such as blue-white colony screening) can further distinguish cells carrying a vector with an actual insert from those carrying an empty, re-ligated vector.

Genomic and cDNA Libraries

Definition - Genomic Library and cDNA Library

A genomic library is a collection of clones collectively representing an organism's entire genome, generated by cutting the whole genome into fragments and cloning each into a vector. A cDNA library instead clones complementary DNA reverse-transcribed from mRNA (Chapter 3 of Molecular Biology), representing only the genes actively transcribed in the tissue or condition sampled - and, since it lacks introns, is often easier to express functionally in a heterologous host (Chapter 3).

Screening a Library

A specific gene of interest is identified within a library by screening - historically using a labelled probe that hybridizes to complementary sequence, and increasingly by PCR (Chapter 4) directly, given how much faster and more specific PCR-based screening has become compared to older hybridization-based methods.
