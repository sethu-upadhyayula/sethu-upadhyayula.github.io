5. Neural Networks and Deep Learning

A neural network is a model built from layers of simple connected units, loosely inspired by biological neurons — this chapter surveys their basic structure and why stacking many such layers ("deep learning") has become so effective across a wide range of problems.

Neurons and Layers

Definition — Artificial Neuron

An artificial neuron computes a weighted sum of its inputs, adds a bias term, and passes the result through a nonlinear activation function — networks arrange many such neurons into layers, with an input layer, one or more hidden layers, and an output layer, each layer's neurons typically taking the previous layer's outputs as their inputs.

Why Nonlinearity Matters

Without a nonlinear activation function, stacking any number of layers would collapse mathematically to a single linear transformation — no more expressive than the linear regression covered in Chapter 2. Nonlinear activation functions are what let a multi-layer network represent far more complex, curved decision boundaries and relationships than a single linear layer ever could.

Backpropagation

Definition — Backpropagation

Backpropagation is the algorithm used to compute how a neural network's loss (Chapter 4) changes with respect to every individual weight in the network, working backward from the output layer to the input layer using the chain rule — providing exactly the gradient information that gradient descent (Chapter 4) needs to update every weight in the network on each training step.

Specialized Architectures

Common Deep Learning Architectures
  • Convolutional neural networks (CNNs) — apply small, shared filters across an input, well suited to image data where local spatial patterns (edges, textures) matter.
  • Recurrent neural networks (RNNs) — process sequences one element at a time while carrying forward a hidden state, well suited to sequential data such as text or time series.
  • Transformers — process an entire sequence at once using an attention mechanism (Chapter 6) that lets every position directly weigh every other position, now the dominant architecture behind large language models.

Why Depth Helps

Remark — What "Deep" Buys You

Successive hidden layers tend to build increasingly abstract representations on top of the layers before them — in an image classifier, early layers might respond to edges, middle layers to shapes built from those edges, and later layers to whole objects built from those shapes — this hierarchical feature learning, largely automatic rather than hand-engineered, is a major reason deep learning has displaced hand-crafted features (Chapter 1) across many domains.
