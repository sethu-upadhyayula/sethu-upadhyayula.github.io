2. Supervised Learning

Supervised learning is the most common form of machine learning in practice — given a training set of examples, each paired with a known correct output, a model learns a mapping general enough to predict the correct output for new examples it hasn't seen.

Regression versus Classification

Definition — Regression and Classification
  • Regression — the output to predict is a continuous numeric value (a house's price).
  • Classification — the output to predict is one of a fixed, discrete set of categories (whether an email is spam or not).

Linear and Logistic Regression

Linear regression predicts a numeric output as a weighted sum of the input features, with the weights chosen (Chapter 4) to minimize the difference between predicted and actual outputs across the training set. Logistic regression adapts this same idea to classification — computing a weighted sum of features, then squashing it through a function that outputs a probability between 0 and 1, despite its name, one of the most common baseline techniques for binary classification.

Decision Trees and Ensembles

Tree-Based Methods

A decision tree predicts an output by asking a sequence of simple questions about the input's features (e.g., "is square footage over 2000?"), branching to a different further question or a final prediction based on the answer. A random forest trains many such trees on different random subsets of the training data and features, then averages their predictions — an example of an ensemble method, generally more accurate and less prone to overfitting (Chapter 4) than any single tree alone.

Support Vector Machines

A support vector machine (SVM) is a classification technique that seeks the separating boundary between classes with the maximum possible margin (distance) to the nearest training example of either class — a well-regarded technique from the era before deep learning (Chapter 5) became dominant for many classification tasks, still useful today on smaller datasets or where interpretability matters.

Evaluating a Supervised Model

A model's quality is assessed on a held-out test set the model never saw during training, using a metric matched to the task — mean squared error for regression, or accuracy, precision, and recall for classification — with the test-set gap from training-set performance being one of the key signals used to detect overfitting, discussed further in Chapter 4.
