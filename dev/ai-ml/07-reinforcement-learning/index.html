7. Reinforcement Learning

Unlike supervised learning (Chapter 2), where a model learns from examples already labeled with the correct answer, reinforcement learning learns by an agent taking actions in an environment and receiving reward feedback — learning, over many attempts, which actions tend to lead to better outcomes.

Agents, Environments, and Rewards

Definition — The Reinforcement Learning Setup

An agent observes some state of its environment, chooses an action, and receives a numeric reward along with a new state — repeating this loop over time. The agent's goal is to learn a policy (a strategy for choosing actions given a state) that maximizes its total accumulated reward over time, not just its immediate reward at any single step.

Exploration versus Exploitation

An agent faces a fundamental tradeoff between exploitation (taking the action currently believed to be best, based on experience so far) and exploration (trying a different, less certain action to potentially discover something even better) — too little exploration risks settling for a mediocre policy, while too much wastes time on actions already known to be poor.

Value Functions and Q-Learning

Definition — Value Function

A value function estimates the total future reward an agent can expect from a given state (or state-action pair), following its current policy — Q-learning, a foundational reinforcement learning algorithm, learns exactly such an estimate for each state-action pair directly from experience, then derives a policy by simply choosing, at each state, whichever action has the highest learned estimate.

Deep Reinforcement Learning

Deep reinforcement learning replaces a table of per-state value estimates (impractical once the state space is too large to enumerate, such as raw pixels from a game screen) with a neural network (Chapter 5) that learns to estimate those same values directly from raw, high-dimensional state input — combining reinforcement learning's sequential-decision framing with deep learning's capacity to learn directly from complex raw inputs, notably behind some well-publicized results in learning to play complex games at a high level.
