Compilers

A compiler translates source code written in a high-level language into instructions a processor (Computer Architecture) can actually execute — a translation carried out through a well-defined sequence of phases, each covered in this subject. Along the way, this subject also covers interpreters and just-in-time compilation as alternative or complementary execution strategies, and the runtime support (memory management, calling conventions) compiled code depends on.
