Operating Systems

An operating system manages a computer's hardware resources and provides the abstractions - processes, files, virtual memory - that everything else in this section is ultimately built on top of. This subject covers how an OS schedules competing processes, coordinates them safely, manages memory, and organizes persistent storage.
