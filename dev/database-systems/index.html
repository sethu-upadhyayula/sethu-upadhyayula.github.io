Database Systems

Database systems manage persistent, structured data reliably and efficiently, even under concurrent access and hardware failure. This subject covers the relational model that still underlies most production databases, how a schema is designed to avoid data anomalies, how transactions provide correctness guarantees under concurrency and failure, how databases answer queries efficiently at scale, and how NoSQL systems relax some of these guarantees in exchange for different tradeoffs.
