2. What Changes from the Twomover

On the surface, a threemover looks like "a twomover with one extra move tacked on." In practice, that single extra ply on each side changes the genre so fundamentally that threemover composition is often treated as a separate discipline with its own literature, its own award sections, and its own community of specialists. This chapter surveys the main structural differences.

Continuations Replace Immediate Mates

In a twomover, the key move itself must already create a position in which every Black reply walks into an immediate mate. There is no room for a second White move - the key does all the work.

In a threemover, the key typically does something gentler: it sets up a threat - a mating sequence that White intends to play on moves 2-3 if Black does nothing to stop it. Black's first defence then forces White to choose a continuation, White's second move, which itself sets up the actual mate on move 3. The continuation is where much of the threemover's "content" lives: it is the move that responds intelligently to what Black has just done.

Because the continuation can be tailored to each individual defence, a single key can branch into many different two-move "mini-twomovers," each triggered by a different Black reply. This is the combinatorial engine that makes threemovers so much richer than twomovers despite using "only" one extra move.

Strategic Content: Plans, Not Just Tactics

A twomover is fundamentally a single tactical idea compressed into one move. A threemover, by contrast, can express a genuine plan - a sequence of moves that only makes sense when viewed as a whole. Typical plans include:

  • Battery preparation: the key arranges a battery (two aligned pieces, one masking the other) that does not yet fire; the continuation fires it.
  • Piece transfer: a piece moves to a square from which it will deliver mate on move 3, but only after Black's first reply has fixed the geometry needed (for example, by blocking an escape square).
  • Provoking a weakness: the key creates a threat so strong that Black is forced to make a structural concession (blocking one of the king's own escape squares, for instance), which White's continuation then exploits.
  • Decoy and clearance: the key lures a Black piece to a square where it will get in its own side's way (interference), or vacates a White square that the mating piece will need to pass through.

This is the sense in which threemovers admit "more positional, strategic ideas alongside the tactical fireworks" - the extra move is exactly the space needed for a plan with a setup phase and a payoff phase.

Changed Play Across Three Phases

In twomover theory, "changed play" usually refers to the contrast between set play (what would happen if it were Black to move in the diagram) and post-key play (what actually happens after White's key). A threemover has room for a third phase:

  • Set play: the analysis of the diagram position as if Black were to move first, extended (where relevant) across Black's first and White's hypothetical reply.
  • Try play: one or more plausible keys that fail - sometimes only deep into a variation, after a continuation that looks promising but ultimately allows Black an escape or a stalemate.
  • Post-key (actual) play: the real solution, following the unique key.

A composer can now arrange for the same Black defence to be met by three different White responses depending on which phase we're in - a different "mate" in the set play, a different (failing) continuation in the try, and the actual continuation in the real solution. When these three responses are thematically related (for example, all three involve the same piece moving to different squares along the same line), the result is a tightly woven thematic complex that rewards very close study. This kind of three-phase changed play is essentially unavailable to the twomover, which has at most two phases to play with.

Secondary Threats and the "Threat Within the Threat"

When the key establishes a threat (a planned mate on move 3 if Black does nothing relevant), Black's defensive moves are not played in a vacuum - many defences carry their own follow-up ideas. A move that blocks White's threatened mate might, in doing so, also open a check against the White king, create a new flight square, or set up a counter-threat of its own (for instance, threatening to promote a pawn, or unpinning a piece).

A well-constructed threemover anticipates this. The continuation chosen for each defence must not only deal with the immediate defensive try but also neutralise any secondary threat that the defence introduces. This double duty - "answer the defence and answer what the defence threatens" - is one of the main sources of the surprising, multi-purpose continuations that threemover connoisseurs prize. A continuation that merely re-establishes the original mating threat, ignoring a new danger Black has created, is likely to be refuted (the problem would then be unsound, or the line would simply not work as a genuine variation).

A Side-by-Side Comparison

Twomover (#2):
1. key (creates threat AND covers all defences with immediate mates)
  1...any defence 2. mate

Threemover (#3):
1. key (creates a threat for move 3)
  1...defence A 2. continuation A (deals with A and any secondary threat) any 3. mate
  1...defence B 2. continuation B (different plan!) any 3. mate
  1...weak defence C 2. short mate

The single extra row in the threemover's tree - the continuation row - is where almost all of the genre's extra richness is concentrated. Everything discussed in this chapter (plans, three-phase changed play, secondary threats) is really about what can happen in that one additional row. With this structural picture in mind, Chapter 3 turns to the specific named themes - recurring patterns that composers use to fill that row with memorable, surprising content.
