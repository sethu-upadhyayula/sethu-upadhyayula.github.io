5. Algebraic Notation

Algebraic notation is the standard method for recording chess moves. Every serious chess player should learn to read and write it - it allows you to study games, solve puzzles, and communicate with other players worldwide.

Piece Identifiers

K = King    Q = Queen    R = Rook    B = Bishop    N = Knight    (Pawn has no letter)

Basic Move Notation

  • Pawn moves: just the destination square - e4, d5, c6
  • Piece moves: piece letter + destination square - Nf3 (knight to f3), Bb5 (bishop to b5)
  • Captures: insert x before the destination - Nxe5 (knight captures on e5), dxe5 (d-pawn captures on e5)
  • Check: append + - Qh5+
  • Checkmate: append # - Qf7#
  • Kingside castling: 0-0
  • Queenside castling: 0-0-0
  • Pawn promotion: destination square + piece letter - e8=Q (pawn promotes to queen on e8)
  • En passant: notated as a regular capture - exd6 (en passant implied by context)

Disambiguation

When two identical pieces can move to the same square, the notation must specify which piece moves:

  • If they are on different files: add the file letter - Rab1 (rook on the a-file moves to b1)
  • If they are on the same file but different ranks: add the rank number - R1b3
  • If both are needed: add both file and rank - Qd1e2

Annotation Symbols

SymbolMeaning
!Good move
!!Brilliant move (often a sacrifice or surprising idea)
?Mistake
??Blunder (a serious error, often losing material or the game)
!?Interesting or speculative move
?!Dubious move (questionable, probably not best)
⩲ or +/=White has a slight advantage
⩱ or =/+Black has a slight advantage
±White has a clear advantage
∓Black has a clear advantage
+-White is winning
-+Black is winning
=Equal position / draw

Sample Game Notation

1.e4 e5 2.Nf3 Nc6 3.Bb5 a6 4.Ba4 Nf6 5.0-0 Be7 6.Re1 b5 7.Bb3 d6 8.c3 0-0
- The Ruy López (Spanish Opening), a classic sequence reaching the main line

In this notation, White's move and Black's reply share the same move number (e.g., 1.e4 = White plays e4, e5 = Black replies with e5).
