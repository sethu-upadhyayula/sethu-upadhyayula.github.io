5.2 Gaussian Copula Model

The Gaussian copula model, introduced by David Li (2000), became the industry standard for pricing synthetic CDO tranches. It provides a tractable way to model joint default times with a single correlation parameter, enabling fast tranche pricing via semi-analytic methods. Its limitations — exposed catastrophically in the 2007–2009 financial crisis — are as instructive as its successes.

The One-Factor Gaussian Copula

Definition — One-Factor Gaussian Copula (Li Model)

For each obligor $i = 1, \ldots, n$, define a latent variable: $$X_i = \sqrt{\rho}\, M + \sqrt{1-\rho}\, Z_i$$ where $M \sim N(0,1)$ is the common (systematic) factor, $Z_i \sim N(0,1)$ are independent idiosyncratic factors, and $\rho \in [0,1]$ is the asset correlation. Default of name $i$ by time $T$ occurs when $X_i \leq \Phi^{-1}(Q_i(T))$, where $Q_i(T)$ is the marginal survival probability calibrated to the CDS curve of name $i$.

Conditional Default Probability

Theorem — Conditional Independence

Conditional on the systematic factor $M = m$, defaults are independent. The conditional default probability of name $i$ by time $T$ is: $$p_i(T \mid m) = P(X_i \leq \Phi^{-1}(Q_i(T)) \mid M = m) = \Phi\!\left(\frac{\Phi^{-1}(Q_i(T)) - \sqrt{\rho}\,m}{\sqrt{1-\rho}}\right)$$ For a homogeneous portfolio ($Q_i = Q$ for all $i$), this simplifies to $p(T \mid m) = \Phi\!\left(\frac{\Phi^{-1}(1-Q(T)) - \sqrt{\rho}\,m}{\sqrt{1-\rho}}\right)$.

Portfolio Loss Distribution

Definition — Loss Distribution via Gaussian Quadrature

For a homogeneous portfolio of $n$ names each with notional $1/n$ and LGD $(1-R)$, the conditional portfolio loss fraction is binomially distributed given $M$: $$L(T) \mid M \sim \frac{(1-R)}{n} \cdot \text{Binomial}(n,\, p(T \mid M))$$ The unconditional loss distribution is obtained by integrating out $M$: $$F_L(x) = P(L(T) \leq x) = \int_{-\infty}^{\infty} P(L(T) \leq x \mid M=m)\, \phi(m)\, dm$$ This integral is evaluated numerically via Gauss–Hermite quadrature (typically 32–128 quadrature points). The expected tranche loss $e(D) - e(A)$ follows directly.

Base Correlation

Different tranches of the same CDO imply different correlations — the correlation smile. Base correlation is the market-standard way to quote this smile in a consistent framework.

Definition — Base Correlation

The compound correlation of a tranche $[A,D]$ is the single $\rho$ that prices the tranche to its market spread. However, compound correlation is non-unique (multiple solutions can exist). Base correlation resolves this by quoting the correlation of the base tranche $[0, D]$ for each detachment $D$: $$\rho_{\text{base}}(D) = \text{such that } \text{Fair Spread}_{[0,D]}(\rho) = \text{Model Price}_{[0,D]}$$ The $[A,D]$ tranche is priced as the difference of two base tranches: $$\text{EL}_{[A,D]} = \text{EL}_{[0,D]}(\rho_{\text{base}}(D)) - \text{EL}_{[0,A]}(\rho_{\text{base}}(A))$$ Base correlations increase monotonically with attachment — a "correlation smile" or "skew" — reflecting that senior tranches require higher correlation to match market prices.

Remark — Limitations of the Gaussian Copula

The Gaussian copula has thin tails: it underestimates the probability of simultaneous large losses. During the 2007–09 crisis, systemic defaults far exceeded model predictions calibrated in benign periods. The model also assumes a constant correlation regardless of the scenario, which fails during stress. Alternative copulas (Student-$t$, Clayton, Marshall–Olkin) provide heavier tails and better tail dependence but at the cost of tractability. Despite its well-known flaws, base correlation remains the quoting convention for index tranche markets.
