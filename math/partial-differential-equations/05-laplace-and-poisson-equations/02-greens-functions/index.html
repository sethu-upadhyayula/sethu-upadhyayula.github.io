5.2 Green's Functions

The Green's function of a domain $\Omega$ is a two-point kernel $G(x,y)$ that encodes the solution of the Dirichlet problem $-\Delta u = f$ in $\Omega$, $u = 0$ on $\partial\Omega$, via a simple integral formula. It is constructed by subtracting from the fundamental solution $\Phi(x-y)$ a corrector harmonic in $y$ that cancels the boundary singularity.

Fundamental Solution

Definition 5.2.1 — Fundamental Solution of the Laplacian

The fundamental solution of $-\Delta$ in $\mathbb{R}^n$ is

$$\Phi(x) = \begin{cases} -\dfrac{1}{2\pi}\log|x| & n = 2, \\[6pt] \dfrac{1}{n(n-2)\omega_n}|x|^{2-n} & n \geq 3, \end{cases}$$

where $\omega_n$ is the volume of the unit ball in $\mathbb{R}^n$. One verifies $-\Delta \Phi = \delta_0$ in the distributional sense: for any $\varphi \in C_0^\infty(\mathbb{R}^n)$, $-\int \Phi \Delta\varphi\, dx = \varphi(0)$.

Green's Function for a Domain

Definition 5.2.2 — Green's Function

For a bounded domain $\Omega \subset \mathbb{R}^n$ and $x \in \Omega$, the Green's function $G(x,\cdot): \Omega \setminus\{x\} \to \mathbb{R}$ is defined by

$$G(x,y) = \Phi(y-x) - \varphi^x(y),$$

where $\varphi^x$ is the unique harmonic corrector satisfying

$$\Delta_y \varphi^x(y) = 0 \text{ in } \Omega, \quad \varphi^x(y) = \Phi(y-x) \text{ on } \partial\Omega.$$

Thus $G(x,y) = 0$ for $y \in \partial\Omega$, and $G(x,\cdot)$ is harmonic in $\Omega \setminus \{x\}$ with the same singularity as $\Phi(\cdot - x)$ at $y = x$. The Green's function is symmetric: $G(x,y) = G(y,x)$.

Theorem 5.2.3 — Green's Representation Formula

Let $u \in C^2(\Omega) \cap C^1(\overline\Omega)$ and $-\Delta u = f$ in $\Omega$. Then for $x \in \Omega$:

$$u(x) = \int_\Omega G(x,y) f(y)\, dy - \int_{\partial\Omega} u(y) \frac{\partial G}{\partial \nu_y}(x,y)\, dS(y).$$

In particular, if $-\Delta u = f$ in $\Omega$ and $u = g$ on $\partial\Omega$, then $u(x) = \int_\Omega G(x,y) f(y)\, dy + \int_{\partial\Omega} g(y) K(x,y)\, dS(y)$, where $K(x,y) = -\frac{\partial G}{\partial \nu_y}(x,y)$ is the Poisson kernel.

Proof Sketch

Apply Green's second identity $\int_\Omega (u\Delta v - v\Delta u)\, dy = \int_{\partial\Omega}(u\frac{\partial v}{\partial\nu} - v\frac{\partial u}{\partial\nu})\, dS$ with $v(y) = G(x,y)$. Remove a small ball $B(x,\varepsilon)$ to avoid the singularity, use $G = 0$ on $\partial\Omega$, $-\Delta_y G = \delta_x$, and send $\varepsilon \to 0$. The ball integral contributes $u(x)$ by the mean value property. $\square$

Poisson Kernel for the Half-Space

Theorem 5.2.4 — Poisson Kernel for $\mathbb{R}^n_+$

For the upper half-space $\mathbb{R}^n_+ = \{x \in \mathbb{R}^n : x_n > 0\}$, the Poisson kernel is

$$P(x, y) = \frac{2 x_n}{n\omega_n |x - y|^n}, \quad y \in \partial \mathbb{R}^n_+ = \mathbb{R}^{n-1} \times \{0\},$$

and the solution to $\Delta u = 0$ in $\mathbb{R}^n_+$, $u = g$ on $\partial\mathbb{R}^n_+$ (with $g$ bounded and continuous) is $u(x) = \int_{\mathbb{R}^{n-1}} P(x,y) g(y)\, dy'$.

Green's Function and Poisson Kernel for the Unit Ball

Theorem 5.2.5 — Green's Function for the Unit Ball

For $\Omega = B(0,1) \subset \mathbb{R}^n$ and $x \in B(0,1)$, define the reflection $x^* = x/|x|^2$. The Green's function is

$$G(x,y) = \Phi(y-x) - |x|^{2-n}\Phi\!\left(y - x^*\right), \quad n \geq 3,$$

and the corresponding Poisson kernel (for boundary data on $\partial B(0,1)$) is

$$K(x,y) = -\frac{\partial G}{\partial \nu_y}(x,y) = \frac{1 - |x|^2}{n\omega_n |x - y|^n}, \quad |y| = 1.$$

The Poisson integral formula for the ball is: if $\Delta u = 0$ in $B(0,1)$ and $u = g$ on $\partial B(0,1)$, then $u(x) = \frac{1-|x|^2}{n\omega_n}\int_{\partial B(0,1)} \frac{g(y)}{|x-y|^n}\, dS(y)$.

Example 5.2.6 — Poisson Integral in 2D

In $\mathbb{R}^2$, the Poisson kernel for the unit disk is $K(r,\theta; \varphi) = \frac{1-r^2}{2\pi(1 - 2r\cos(\theta-\varphi) + r^2)}$ (the Poisson kernel for the disk in polar coordinates), and

$$u(r,\theta) = \frac{1}{2\pi}\int_0^{2\pi} \frac{(1-r^2) g(\varphi)}{1 - 2r\cos(\theta - \varphi) + r^2}\, d\varphi.$$

This formula solves the Dirichlet problem for the Laplacian on the disk and is the basis for the Schwarz-Poisson formula in complex analysis.
