2.1 Method of Characteristics

The method of characteristics transforms a first-order PDE into a system of ODEs along special curves called characteristics. Along these curves, the PDE reduces to an ODE that can be solved explicitly.

Definition 2.1.1 — Linear First-Order PDE

Consider the linear first-order PDE in two variables:

$$a(x,y)\, u_x + b(x,y)\, u_y = c(x,y)$$

with initial data specified along a curve $\Gamma$ in the $xy$-plane (not tangent to the characteristics).

Theorem 2.1.1 — Characteristic ODEs

The characteristic curves of $a u_x + b u_y = c$ are the solutions of the system of ODEs:

$$\frac{dx}{ds} = a(x,y), \quad \frac{dy}{ds} = b(x,y), \quad \frac{du}{ds} = c(x,y)$$

Along each characteristic, $u$ satisfies the ODE $du/ds = c$. By solving this ODE system with initial conditions on $\Gamma$, we obtain $u$ on the entire region swept by characteristics from $\Gamma$.

The Transport Equation

Example 2.1.1 — Transport Equation $u_t + c\,u_x = 0$

Here $a = c$ (constant wave speed), $b = 1$ in $(x,t)$ coordinates, and the right-hand side is zero. The characteristic ODEs are:

$$\frac{dx}{ds} = c, \quad \frac{dt}{ds} = 1, \quad \frac{du}{ds} = 0$$

From $dt/ds = 1$ we get $t = s + t_0$. From $dx/ds = c$: $x = cs + x_0 = ct + x_0$. So characteristics are straight lines $x - ct = x_0 = \text{const}$.

From $du/ds = 0$: $u$ is constant along characteristics. With initial data $u(x,0) = u_0(x)$:

$$\boxed{u(x,t) = u_0(x - ct)}$$

The initial profile $u_0$ propagates unchanged at speed $c$ to the right. No information is created or destroyed — just translated.

Example 2.1.2 — Variable Coefficient $y\,u_x + u_y = 0$

Characteristic ODEs: $\dot x = y$, $\dot y = 1$, $\dot u = 0$.

From $\dot y = 1$: $y = s$. From $\dot x = y = s$: $x = s^2/2 + x_0$. So $x - y^2/2 = x_0 = \text{const}$ are the characteristics (parabolas).

Since $u$ is constant along characteristics: $u(x,y) = f(x - y^2/2)$ for arbitrary function $f$, determined by the initial curve.

Domain of Dependence

Definition 2.1.2 — Domain of Dependence

The value $u(x_0, t_0)$ depends only on the initial data at the single point $x_0 - ct_0$ (for the transport equation). More generally, the domain of dependence of $(x_0, t_0)$ is the set of initial points whose data influences $u(x_0, t_0)$. For the transport equation, this is exactly one characteristic passing through $(x_0, t_0)$.

Remark on the Method

The method of characteristics works for any first-order PDE where characteristics are well-defined. The key assumption is that the initial curve $\Gamma$ is non-characteristic: it must be transverse to the characteristic direction $(a,b)$ at every point, so that initial data can be propagated uniquely. If $\Gamma$ is characteristic, the problem is either over-determined or under-determined.
