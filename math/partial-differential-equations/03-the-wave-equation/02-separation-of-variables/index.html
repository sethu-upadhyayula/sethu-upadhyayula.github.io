3.2 Separation of Variables

For the wave equation on a bounded interval with Dirichlet boundary conditions, d'Alembert's formula no longer applies directly. The method of separation of variables reduces the PDE to eigenvalue problems, yielding a series solution.

Setup - Wave Equation on $[0,L]$

Consider the IBVP:

$$u_{tt} = c^2 u_{xx}, \quad 0 < x < L,\; t > 0$$

with Dirichlet BCs $u(0,t) = u(L,t) = 0$ and initial conditions $u(x,0) = \phi(x)$, $u_t(x,0) = \psi(x)$.

Assume $u(x,t) = X(x)T(t)$. Substituting: $X T'' = c^2 X'' T$, so $\frac{T''}{c^2 T} = \frac{X''}{X} = -\lambda$ (separation constant).

Theorem 3.2.1 - Eigenvalue Problem

The spatial ODE is $X'' + \lambda X = 0$ with $X(0) = X(L) = 0$. Non-trivial solutions exist only for:

$$\lambda_n = \left(\frac{n\pi}{L}\right)^2, \quad X_n(x) = \sin\!\left(\frac{n\pi x}{L}\right), \quad n = 1, 2, 3, \ldots$$

The temporal ODE becomes $T_n'' + c^2\lambda_n T_n = 0$, with general solution $T_n(t) = A_n\cos(\omega_n t) + B_n\sin(\omega_n t)$ where $\omega_n = cn\pi/L$.

Theorem 3.2.2 - Series Solution

By superposition, the general solution satisfying the BCs is:

$$u(x,t) = \sum_{n=1}^{\infty}\left[A_n\cos\!\left(\frac{cn\pi t}{L}\right) + B_n\sin\!\left(\frac{cn\pi t}{L}\right)\right]\sin\!\left(\frac{n\pi x}{L}\right)$$

Imposing initial conditions: $A_n = \frac{2}{L}\int_0^L \phi(x)\sin\!\left(\frac{n\pi x}{L}\right)dx$ and $B_n = \frac{2}{cn\pi}\int_0^L \psi(x)\sin\!\left(\frac{n\pi x}{L}\right)dx$.

Each term $u_n = \sin(n\pi x/L)\cos(\omega_n t)$ is a normal mode oscillating at frequency $\omega_n = cn\pi/L$.

Physical Interpretation - Standing Waves

The $n$-th mode oscillates at frequency $\omega_n = cn\pi/L$ with $n$ half-waves fitting in $[0,L]$. For a guitar string of length $L$: $\omega_1 = c\pi/L$ is the fundamental; $\omega_2 = 2c\pi/L$ is the first overtone (octave). The general solution is a superposition of all harmonics.
