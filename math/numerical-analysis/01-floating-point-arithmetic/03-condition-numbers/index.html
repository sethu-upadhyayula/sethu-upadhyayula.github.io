1.3 Condition Numbers

The condition number of a problem measures its intrinsic sensitivity to small perturbations in the input, independently of the algorithm used to solve it. A large condition number signals that the problem itself amplifies errors; even an exact algorithm would give a poor answer when fed slightly perturbed data.

Absolute and Relative Condition Numbers

Definition - Absolute Condition Number

Let $f : \mathbb{R}^m \to \mathbb{R}^n$ be differentiable at $x$. The absolute condition number of $f$ at $x$ is:

$$\hat{\kappa}(x) = \lim_{\delta \to 0} \max_{\|\delta x\| \le \delta} \frac{\|f(x + \delta x) - f(x)\|}{\|\delta x\|} = \|J_f(x)\|,$$

where $J_f(x)$ is the Jacobian of $f$ at $x$ and $\|\cdot\|$ denotes an appropriate operator norm.

Definition - Relative Condition Number

The relative condition number normalises by the magnitudes of input and output:

$$\kappa(x) = \frac{\|J_f(x)\| \cdot \|x\|}{\|f(x)\|}.$$

It measures the ratio of relative output perturbation to relative input perturbation. A problem with $\kappa \approx 1$ is well-conditioned; one with $\kappa \gg 1$ is ill-conditioned.

Remark - Interpretation

If the input $x$ has relative error $\varepsilon_{\text{data}}$ (e.g., from measurement noise or prior computation), then the output $f(x)$ has relative error at most approximately $\kappa(x)\,\varepsilon_{\text{data}}$. For a backward-stable algorithm with backward error of order $\varepsilon_{\text{mach}}$, the forward error in the computed output is therefore bounded by $\kappa(x)\,\varepsilon_{\text{mach}}$.

Matrix Condition Number

Definition - Matrix Condition Number

For a nonsingular matrix $A \in \mathbb{R}^{n\times n}$, the condition number with respect to a matrix norm $\|\cdot\|$ is:

$$\kappa(A) = \|A\| \cdot \|A^{-1}\|.$$

For the 2-norm, $\kappa_2(A) = \sigma_{\max}(A)/\sigma_{\min}(A)$, the ratio of the largest to smallest singular values. We always have $\kappa(A) \ge 1$.

Theorem - Perturbation of Linear Systems

Let $Ax = b$ with $A$ nonsingular. Suppose $b$ is perturbed to $b + \delta b$, giving $A(x + \delta x) = b + \delta b$. Then:

$$\frac{\|\delta x\|}{\|x\|} \le \kappa(A) \cdot \frac{\|\delta b\|}{\|b\|}.$$

Similarly, if $A$ is perturbed to $A + \delta A$ with $\|\delta A\| \cdot \|A^{-1}\| < 1$, then $\|\delta x\|/\|x\| \le \kappa(A)\,\|\delta A\|/\|A\| + O(\|\delta A\|^2)$. Thus a linear system can be solved to relative accuracy at most $\kappa(A)$ times the relative perturbation in the data.

Proposition - Condition Number and Singular Values

For the 2-norm, $\kappa_2(A) = \sigma_1/\sigma_n$ where $\sigma_1 \ge \sigma_2 \ge \cdots \ge \sigma_n > 0$ are the singular values. A matrix is singular if and only if $\sigma_n = 0$, equivalently $\kappa_2(A) = \infty$. Orthogonal matrices satisfy $\kappa_2(Q) = 1$, the minimum possible value, which is why QR factorisation is numerically preferred over methods that destroy orthogonality.

The Hilbert Matrix - A Notoriously Ill-Conditioned Example

Definition - Hilbert Matrix

The $n \times n$ Hilbert matrix $H_n$ has entries $(H_n)_{ij} = 1/(i+j-1)$. It is symmetric positive definite and arises naturally in polynomial least-squares fitting. Its condition number grows as:

$$\kappa_2(H_n) \approx e^{3.5n}.$$

Remark - Condition vs. Stability

Condition number is a property of the problem, not the algorithm. An ill-conditioned problem cannot be solved accurately by any algorithm regardless of how clever it is - the input data simply does not contain enough information. Stability, by contrast, is a property of the algorithm: a stable algorithm does not introduce errors beyond what the condition number requires. The interplay is captured by the maxim: accuracy = stability × (1/conditioning).
