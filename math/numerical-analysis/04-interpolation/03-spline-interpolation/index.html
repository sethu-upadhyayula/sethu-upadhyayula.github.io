4.3 Spline Interpolation

Splines replace a single high-degree polynomial with piecewise low-degree polynomials joined smoothly at the data points (knots). The cubic spline achieves $C^2$ smoothness — making it visually smooth and physically meaningful — while avoiding the Runge phenomenon entirely and attaining fourth-order convergence.

Cubic Spline Definition

Definition — Cubic Spline

Given nodes $a = x_0 < x_1 < \cdots < x_n = b$ and values $y_i = f(x_i)$, a cubic spline $S$ satisfies:

  1. $S(x_i) = y_i$ for all $i = 0, 1, \ldots, n$ (interpolation).
  2. $S|_{[x_{i-1},x_i]}$ is a polynomial of degree $\le 3$ for each $i = 1,\ldots,n$.
  3. $S$, $S'$, and $S''$ are continuous on $[a,b]$ (i.e., $S \in C^2[a,b]$).

This leaves 2 free parameters, requiring 2 additional boundary conditions to determine $S$ uniquely.

Natural and Clamped Boundary Conditions

Definition — Boundary Conditions
  • Natural (free) spline: $S''(x_0) = S''(x_n) = 0$. This minimises $\int_a^b [S''(x)]^2\,dx$ among all interpolants, giving the physically natural "relaxed ruler" shape.
  • Clamped spline: $S'(x_0) = f'(x_0)$, $S'(x_n) = f'(x_n)$. Requires knowing the derivative at the endpoints; gives better approximation when $f'$ is available.
  • Not-a-knot: forces $S'''$ to be continuous at $x_1$ and $x_{n-1}$, eliminating two interior knots. This is the default in many software packages.

Tridiagonal System for Second Derivatives

Theorem — Construction via Second Derivatives

Let $M_i = S''(x_i)$ and $h_i = x_i - x_{i-1}$. The $C^2$ continuity conditions lead to the tridiagonal system (for the natural spline with $M_0 = M_n = 0$):

$$h_i M_{i-1} + 2(h_i + h_{i+1})M_i + h_{i+1}M_{i+1} = 6\left(\frac{y_{i+1}-y_i}{h_{i+1}} - \frac{y_i - y_{i-1}}{h_i}\right), \quad i=1,\ldots,n-1.$$

This is a strictly diagonally dominant tridiagonal system, solvable in $O(n)$ by the Thomas algorithm. Once the $M_i$ are found, each cubic piece is determined on $[x_{i-1},x_i]$.

Approximation Error

Theorem — Fourth-Order Convergence

If $f \in C^4[a,b]$ and $S$ is the natural or clamped cubic spline with uniform mesh spacing $h = (b-a)/n$, then:

$$\|f - S\|_\infty \le C h^4 \|f^{(4)}\|_\infty,$$

for a constant $C$ depending on the boundary conditions. This optimal $O(h^4)$ rate matches that of fourth-degree polynomial interpolation, but is achieved robustly without Runge oscillations for any smooth $f$.

B-Splines

Definition — B-Spline Basis

The B-spline basis functions $B_{i,k}(x)$ of degree $k$ are defined recursively via the Cox–de Boor formula:

$$B_{i,0}(x) = \begin{cases}1 & x_{i} \le x < x_{i+1}\\ 0 & \text{otherwise}\end{cases},$$ $$B_{i,k}(x) = \frac{x - x_i}{x_{i+k}-x_i}B_{i,k-1}(x) + \frac{x_{i+k+1}-x}{x_{i+k+1}-x_{i+1}}B_{i+1,k-1}(x).$$

Each $B_{i,k}$ has compact support $[x_i, x_{i+k+1}]$ and $B_{i,k} \ge 0$. The basis is numerically stable (the condition number of the B-spline collocation matrix is $O(1)$ regardless of $n$), unlike the monomial or Lagrange bases. The de Boor algorithm evaluates $\sum_i c_i B_{i,k}(x)$ in $O(k^2)$ operations via a triangular recurrence, analogous to Horner's method.

Remark — NURBS

NURBS (Non-Uniform Rational B-Splines) extend B-splines by allowing non-uniform knot vectors and rational (weighted) control points: $C(x) = \sum_i w_i c_i B_{i,k}(x) / \sum_i w_i B_{i,k}(x)$. NURBS can represent conic sections and other algebraic curves exactly, making them the standard curve representation in CAD software (IGES, STEP) and computer graphics (OpenGL, DirectX surface patches).
