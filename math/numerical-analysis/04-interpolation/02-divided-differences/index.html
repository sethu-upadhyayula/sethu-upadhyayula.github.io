4.2 Divided Differences

Newton's form of the interpolating polynomial uses divided differences as coefficients. It is equivalent to the Lagrange form but has superior computational properties: adding a new data point requires only $O(n)$ work to update the coefficients, and evaluation via Horner's method is stable and efficient.

Divided Differences

Definition — Divided Differences

The divided differences of $f$ at nodes $x_i, \ldots, x_{i+k}$ are defined recursively:

$$f[x_i] = f(x_i),$$ $$f[x_i, x_{i+1}] = \frac{f[x_{i+1}] - f[x_i]}{x_{i+1} - x_i},$$ $$f[x_i, x_{i+1}, \ldots, x_{i+k}] = \frac{f[x_{i+1}, \ldots, x_{i+k}] - f[x_i, \ldots, x_{i+k-1}]}{x_{i+k} - x_i}.$$

Divided differences are symmetric: $f[x_i, \ldots, x_{i+k}]$ is independent of the ordering of its arguments.

Newton's Interpolating Polynomial

Theorem — Newton's Divided-Difference Form

The unique interpolating polynomial of degree $\le n$ through $(x_0, f(x_0)), \ldots, (x_n, f(x_n))$ is:

$$p_n(x) = f[x_0] + f[x_0, x_1](x-x_0) + f[x_0,x_1,x_2](x-x_0)(x-x_1) + \cdots$$ $$= \sum_{k=0}^n f[x_0, \ldots, x_k]\prod_{j=0}^{k-1}(x - x_j),$$

where the empty product ($k=0$) equals 1.

Connection to Derivatives

Theorem — Divided Differences and Derivatives

If $f \in C^n$, then for any $n+1$ distinct points $x_0, \ldots, x_n$ there exists $\xi$ in their convex hull such that: $$f[x_0, x_1, \ldots, x_n] = \frac{f^{(n)}(\xi)}{n!}.$$ In the limit as all nodes collapse to a single point $x$, $f[x,x,\ldots,x] = f^{(n)}(x)/n!$, which is used to define divided differences for repeated nodes (Hermite interpolation).

Horner Evaluation and Incremental Updates

Algorithm — Horner's Method for Newton Form

Evaluate $p_n(x) = \sum_{k=0}^n c_k \prod_{j=0}^{k-1}(x-x_j)$ where $c_k = f[x_0,\ldots,x_k]$ via nested multiplication:

b = c_n
for k = n-1, n-2, ..., 0:
    b = c_k + (x - x_k) * b
return b

Cost: $O(n)$ multiplications and additions. Adding a new data point $(x_{n+1}, f(x_{n+1}))$ requires computing only one new diagonal entry $f[x_0, \ldots, x_{n+1}]$ in $O(n)$ operations, making Newton's form ideal for on-line interpolation.
