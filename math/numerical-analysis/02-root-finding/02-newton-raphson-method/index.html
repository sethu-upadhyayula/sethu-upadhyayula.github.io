2.2 Newton-Raphson Method

The Newton-Raphson method is the workhorse of numerical root-finding when derivatives are available. Starting from an initial guess, it iterates by following the tangent line to the curve, achieving quadratic convergence near a simple root — a remarkable rate that roughly doubles the number of correct digits at each step.

The Iteration

Definition — Newton-Raphson Iteration

Given a differentiable function $f$ and an initial guess $x_0$, the Newton-Raphson iteration is:

$$x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}, \qquad n = 0, 1, 2, \ldots$$

Geometric interpretation: the new iterate $x_{n+1}$ is the $x$-intercept of the tangent line to $y = f(x)$ at the point $(x_n, f(x_n))$. The tangent line is $y - f(x_n) = f'(x_n)(x - x_n)$; setting $y = 0$ yields the formula above.

Quadratic Convergence

Theorem — Quadratic Convergence

Let $r$ be a simple root of $f$ (i.e., $f(r) = 0$, $f'(r) \ne 0$) and suppose $f$ is twice continuously differentiable near $r$. If $x_0$ is sufficiently close to $r$, then the Newton-Raphson iterates converge to $r$ with:

$$|e_{n+1}| \le \frac{M}{2m} |e_n|^2,$$

where $e_n = x_n - r$, $M = \max_{x \in I}|f''(x)|$, and $m = \min_{x \in I}|f'(x)|$ on some interval $I$ containing $r$ and all iterates. This is quadratic convergence: once $|e_n|$ is small, $|e_{n+1}|$ is proportional to $|e_n|^2$.

Proof Sketch

Expand $f$ in a Taylor series around $r$: $$f(x_n) = f(r) + f'(r)(x_n - r) + \tfrac{1}{2}f''(\xi_n)(x_n - r)^2 = f'(r)\,e_n + \tfrac{1}{2}f''(\xi_n)\,e_n^2,$$ for some $\xi_n$ between $r$ and $x_n$ (Lagrange remainder). Similarly, $f'(x_n) = f'(r) + O(e_n)$. Then: $$e_{n+1} = x_{n+1} - r = x_n - \frac{f(x_n)}{f'(x_n)} - r = e_n - \frac{f'(r)\,e_n + \tfrac{1}{2}f''(\xi_n)\,e_n^2}{f'(x_n)} = \frac{-\tfrac{1}{2}f''(\xi_n)\,e_n^2}{f'(x_n)}.$$ Taking absolute values and using $|f'(x_n)| \ge m > 0$ near $r$ gives $|e_{n+1}| \le \tfrac{M}{2m}|e_n|^2$. $\square$

Failure Modes

Remark — When Newton-Raphson Fails
  • Stationary point encountered: if $f'(x_n) = 0$ for some $n$, the method is undefined (division by zero).
  • Poor initial guess: far from the root, the tangent line may point away from the root or toward a different root.
  • Oscillation: for some functions and starting points the iterates cycle without converging.
  • Multiple root ($m \ge 2$): convergence degrades to linear with rate $(m-1)/m$. The modified iteration $x_{n+1} = x_n - m\,f(x_n)/f'(x_n)$ restores quadratic convergence.
Definition — Modified Newton for Multiple Roots

If $r$ is a root of multiplicity $m$, the modified Newton iteration:

$$x_{n+1} = x_n - m \cdot \frac{f(x_n)}{f'(x_n)}$$

recovers quadratic convergence. Alternatively, define $u(x) = f(x)/f'(x)$, which has a simple root at $r$ regardless of the multiplicity of $f$; then Newton's method applied to $u$ converges quadratically.

Example: Computing $\sqrt{2}$

Example — Square Root via Newton-Raphson

Find $r = \sqrt{2}$ as the root of $f(x) = x^2 - 2$, so $f'(x) = 2x$. The iteration is: $$x_{n+1} = x_n - \frac{x_n^2 - 2}{2x_n} = \frac{x_n}{2} + \frac{1}{x_n}.$$ Starting from $x_0 = 1$:

The error roughly squares at each step: $8.58\times10^{-2} \to 2.45\times10^{-3} \to 2.12\times10^{-6} \to 1.59\times10^{-12}$, illustrating quadratic convergence. Full double-precision accuracy is reached in 5 iterations from a poor initial guess.
