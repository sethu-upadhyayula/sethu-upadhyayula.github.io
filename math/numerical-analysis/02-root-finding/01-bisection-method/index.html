2.1 Bisection Method

The bisection method is the simplest and most reliable root-finding algorithm. It requires only that $f$ be continuous and that a bracket $[a,b]$ containing a sign change is known. Each iteration halves the bracket, guaranteeing linear convergence regardless of the shape of $f$.

The Intermediate Value Theorem

Theorem - Intermediate Value Theorem

Let $f : [a,b] \to \mathbb{R}$ be continuous. If $f(a) \cdot f(b) < 0$, then there exists at least one $r \in (a,b)$ with $f(r) = 0$.

Remark - Bracket Condition

The condition $f(a)f(b) < 0$ means $f$ has opposite signs at the endpoints. Finding an initial bracket is a separate task; for many practical problems it can be done by sign sampling or physical reasoning. The bisection method itself only requires function evaluations, not derivatives or any other smoothness.

The Algorithm

Algorithm - Bisection

Input: continuous $f$, bracket $[a_0, b_0]$ with $f(a_0)f(b_0) < 0$, tolerance $\text{tol} > 0$.

Iteration: for $n = 0, 1, 2, \ldots$:

  1. Set $c_n = (a_n + b_n)/2$ (midpoint).
  2. If $|b_n - a_n| < \text{tol}$ or $f(c_n) = 0$, return $c_n$.
  3. If $f(a_n) f(c_n) < 0$, set $a_{n+1} = a_n$, $b_{n+1} = c_n$.
  4. Else set $a_{n+1} = c_n$, $b_{n+1} = b_n$.

Each iteration requires exactly one new function evaluation (since $f(a_n)$ and $f(b_n)$ are known from the previous step).

Convergence Analysis

Theorem - Linear Convergence of Bisection

After $n$ steps, the error satisfies:

$$|c_n - r| \le \frac{b_0 - a_0}{2^{n+1}},$$

where $r$ is any root in $(a_0, b_0)$. This is linear convergence with rate $1/2$: each iteration reduces the error bound by a factor of 2. Equivalently, to achieve $|c_n - r| < \varepsilon$, one needs $n \ge \log_2\!\left((b_0-a_0)/\varepsilon\right)$ iterations. To gain one decimal digit of accuracy requires approximately $\log_2(10) \approx 3.32$ iterations.

Remark - Stopping Criteria

Two natural stopping criteria are: (i) bracket width $|b_n - a_n| < \text{tol}$, which controls the absolute error in $x$; (ii) residual $|f(c_n)| < \text{tol}_f$, which checks how close $c_n$ is to a root in function value. For ill-conditioned functions, criterion (i) is preferable; for well-conditioned ones, (ii) may stop sooner.

Advantages and Disadvantages

Remark - Strengths and Limitations
  • Guaranteed convergence: provided the initial bracket is valid and $f$ is continuous, bisection cannot fail.
  • No derivative required: only function evaluations are needed.
  • Simple implementation: trivial to code and diagnose.
  • Slow convergence: linear rate of $1/2$ is slower than Newton-Raphson (quadratic) or secant (superlinear).
  • Bracket required: finding an initial bracket containing a root may itself be non-trivial for some functions.
  • Scalar only: bisection does not generalise directly to systems of equations in higher dimensions.
