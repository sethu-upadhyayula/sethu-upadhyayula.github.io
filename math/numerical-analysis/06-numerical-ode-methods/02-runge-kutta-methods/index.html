6.2 Runge-Kutta Methods

Runge-Kutta (RK) methods achieve higher order accuracy by evaluating $f$ at several intermediate stages within each step, then combining the stage values to cancel low-order error terms. The classical fourth-order method (RK4) is one of the most widely used algorithms in all of computational science.

RK2: Heun's Method

Definition - Heun's Method (Explicit Trapezoid)

$$k_1 = h\,f(t_n, y_n), \qquad k_2 = h\,f(t_n+h,\; y_n+k_1),$$ $$y_{n+1} = y_n + \tfrac{1}{2}(k_1+k_2).$$ This is the explicit trapezoidal rule. It has LTE $O(h^2)$ and global error $O(h^2)$, hence is second order.

The Classical RK4 Method

Definition - Classical Fourth-Order Runge-Kutta

$$k_1 = h\,f(t_n,\,y_n),$$ $$k_2 = h\,f\!\left(t_n+\tfrac{h}{2},\;y_n+\tfrac{k_1}{2}\right),$$ $$k_3 = h\,f\!\left(t_n+\tfrac{h}{2},\;y_n+\tfrac{k_2}{2}\right),$$ $$k_4 = h\,f(t_n+h,\;y_n+k_3),$$ $$y_{n+1} = y_n + \frac{k_1 + 2k_2 + 2k_3 + k_4}{6}.$$

The method has LTE $O(h^4)$ and global error $O(h^4)$: fourth order. Four function evaluations per step buy a factor of $\sim 10^4$ reduction in error when $h$ is halved.

Butcher Tableau Notation

Definition - Butcher Tableau

An $s$-stage explicit RK method is defined by the tableau $$\begin{array}{c|c} \mathbf{c} & A \\ \hline & \mathbf{b}^T \end{array}$$ where $A = (a_{ij})$ is strictly lower triangular (explicit), $\mathbf{c}$ are the stage abscissae, and $\mathbf{b}$ are the output weights. The stage values are: $$k_i = h\,f\!\left(t_n + c_i h,\; y_n + \sum_{j=1}^{i-1} a_{ij} k_j\right), \quad y_{n+1} = y_n + \sum_{i=1}^s b_i k_i.$$

For RK4: $\mathbf{c}=(0,\tfrac{1}{2},\tfrac{1}{2},1)^T$, $\mathbf{b}=(\tfrac{1}{6},\tfrac{1}{3},\tfrac{1}{3},\tfrac{1}{6})^T$, and $a_{31}=0, a_{32}=\tfrac{1}{2}$, etc.

Order Conditions

Theorem - Butcher's Order Conditions

A RK method has order $p$ iff it satisfies a system of algebraic equations indexed by rooted trees of order $\le p$. There are 1, 1, 2, 4, 9, 20, 51, \ldots conditions at orders 1, 2, 3, 4, 5, 6, 7, \ldots. For $p \ge 5$, more stages are required than $p$ (the minimum number of stages for order 5 is 6). RK4 satisfies all $1+1+2+4 = 8$ conditions at orders 1-4 using exactly 4 stages.

Embedded Methods and Adaptive Step Control

Definition - Embedded Runge-Kutta Pair

An embedded pair provides two solutions of order $p$ and $\hat{p} = p+1$ (or $p-1$) using the same stage values. The difference $e_{n+1} = \hat{y}_{n+1} - y_{n+1}$ estimates the local error.

The Dormand-Prince RK45 pair (the basis of MATLAB's ode45) has 6 stages, order 5 primary and order 4 embedded. The step is controlled by: $$h_{\text{new}} = h \cdot \left(\frac{\text{tol}}{\|e_{n+1}\|}\right)^{1/5}$$ with safety factor $0.9$ and min/max scale factors.

Example - Dormand-Prince Butcher Tableau (abridged)

The 7-stage (FSAL) tableau has $c = (0, 1/5, 3/10, 4/5, 8/9, 1, 1)^T$. The 5th-order weights $b$ and 4th-order weights $\hat{b}$ share all stage evaluations; the last stage of one step is the first of the next (FSAL: first-same-as-last), reducing the effective cost to 5 evaluations per step after the first.
