6.1 Euler's Method

Euler's method is the simplest numerical scheme for the initial value problem. Though rarely used in practice due to its low accuracy, it introduces all the key ideas — local truncation error, global error accumulation, stability, consistency, and convergence — that carry over to higher-order methods.

The Initial Value Problem

Definition — Initial Value Problem

The initial value problem (IVP) is: find $y:[t_0,T]\to\mathbb{R}$ satisfying $$y'(t) = f(t,y(t)), \quad y(t_0) = y_0.$$ Under the Picard–Lindelöf theorem, a unique solution exists on $[t_0,T]$ whenever $f$ is continuous and Lipschitz in $y$: $|f(t,u)-f(t,v)|\le L|u-v|$.

Forward (Explicit) Euler

Definition — Forward Euler Method

Partition $[t_0,T]$ into $N$ equal steps of size $h = (T-t_0)/N$, so $t_n = t_0+nh$. The forward Euler scheme is: $$y_{n+1} = y_n + h\,f(t_n,y_n), \quad n = 0,1,\ldots,N-1.$$

This follows from truncating the Taylor expansion $y(t_{n+1}) = y(t_n) + h\,y'(t_n) + \tfrac{h^2}{2}y''(t_n) + \cdots$ after the first-order term and replacing $y'(t_n)$ by $f(t_n,y_n)$.

Local Truncation Error

Definition — Local Truncation Error

The local truncation error (LTE) is the residual when the exact solution is substituted into the scheme: $$\tau_n = \frac{y(t_{n+1})-y(t_n)}{h} - f(t_n,y(t_n)) = \frac{h}{2}y''(t_n) + O(h^2).$$ So $\tau_n = O(h)$, and the forward Euler method is first-order consistent.

Global Error

Theorem — Global Error of Forward Euler

Let $e_n = y(t_n) - y_n$ be the global error. Under the Lipschitz condition with constant $L$, $$|e_n| \le \frac{h\,M_2}{2L}\bigl(e^{L(t_n-t_0)}-1\bigr),$$ where $M_2 = \max_{[t_0,T]}|y''(t)|$. In particular the global error is $O(h)$: halving $h$ halves the error. Note that $n = (T-t_0)/h$ steps each contribute LTE $O(h^2)$, and $n \cdot O(h^2) = O(h)$.

Backward (Implicit) Euler

Definition — Backward Euler Method

The backward Euler scheme evaluates $f$ at the new time level: $$y_{n+1} = y_n + h\,f(t_{n+1},y_{n+1}).$$ This is an implicit equation in $y_{n+1}$ and must be solved at each step, typically by Newton's method. It has the same first-order accuracy as forward Euler ($\tau_n = O(h)$), but its stability properties are dramatically better.

Stability on the Test Equation

Definition — Dahlquist Test Equation

The test equation is $y' = \lambda y$ with $\lambda \in \mathbb{C}$, $\text{Re}\,\lambda < 0$ (so the exact solution $y(t) = y_0 e^{\lambda t}\to 0$). Applied to this equation, any one-step method produces $y_{n+1} = R(h\lambda)\,y_n$ for some stability function $R(z)$. The method is stable on this equation iff $|R(h\lambda)| \le 1$.

Proposition — Stability Regions

For forward Euler, $R(z) = 1+z$, so $|R(h\lambda)|\le 1$ iff $h\lambda$ lies in the unit disk centred at $-1$: the stability region is $\{z\in\mathbb{C}: |1+z|\le 1\}$.

For backward Euler, $R(z)=1/(1-z)$, so $|R(h\lambda)|\le 1$ for all $\text{Re}(h\lambda)\le 0$: backward Euler is unconditionally stable (A-stable) for the test equation. This is why implicit methods are essential for stiff problems.

Dahlquist's Convergence Theorem

Theorem — Dahlquist: Consistency + Zero-Stability $\Rightarrow$ Convergence

A linear multistep method is convergent (global error $\to 0$ as $h\to 0$) if and only if it is consistent (LTE $\to 0$) and zero-stable (all roots of the first characteristic polynomial lie in the unit disk, with those on the boundary being simple). Both forward and backward Euler satisfy this theorem and converge at first order.
