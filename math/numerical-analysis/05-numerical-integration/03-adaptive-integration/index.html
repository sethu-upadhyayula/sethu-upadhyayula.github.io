5.3 Adaptive Integration

Fixed step-size rules work poorly when $f$ is smooth in some regions and rapidly varying in others. Adaptive quadrature concentrates function evaluations where the integrand is hardest to approximate, automatically achieving a user-specified tolerance with the fewest possible evaluations.

Local Error Estimation by Subdivision

Definition - Local Error Estimate

Let $I_1$ be a quadrature approximation to $\int_a^b f$ on the whole interval, and let $I_2 = I_L + I_R$ be the sum of approximations on $[a,m]$ and $[m,b]$ (where $m = (a+b)/2$) using the same rule. If the rule has order $p$ (i.e., error $\sim C h^{p+1}$), then: $$\int_a^b f - I_2 \approx \frac{I_2 - I_1}{2^p - 1}.$$ This estimate improves as $h \to 0$.

The algorithm accepts the approximation on $[a,b]$ and uses $I_2$ as the result if $|I_2 - I_1| \le \varepsilon_{\text{tol}}$, and subdivides otherwise.

Recursive Adaptive Algorithm

Definition - Recursive Adaptive Quadrature

Given tolerance $\delta$ for the subinterval $[a,b]$:

  1. Compute $I_1 = Q(f, a, b)$ and $I_2 = Q(f, a, m) + Q(f, m, b)$.
  2. If $|I_2 - I_1| \le 15\,\delta$, return $I_2 + (I_2-I_1)/15$ (Richardson-corrected).
  3. Otherwise, return $\text{Adapt}(f, a, m, \delta/2) + \text{Adapt}(f, m, b, \delta/2)$.
Remark - Global vs Local Error

The recursive halving of tolerance aims to control the global error $|\int_a^b f - \text{result}| \le \delta_{\text{global}}$. The subdivision assigns $\delta/2$ to each half, so when all subintervals are accepted the total error is $\le \sum_k \delta_k$. This is an error budget allocation; in practice, the algorithm distributes work where $f$ is most irregular.

Gauss-Kronrod Rules

Definition - Gauss-Kronrod G7-K15

The Kronrod extension of an $n$-point Gauss rule adds $n+1$ extra nodes interleaved with the Gaussian nodes to produce a $(2n+1)$-point rule of higher order. The original $n$ evaluations are reused, making the error estimate virtually free.

The standard G7-K15 pair: the 7-point Gauss rule (exact for degree $\le 13$) is extended to a 15-point Kronrod rule (exact for degree $\le 23$). The difference $|K_{15} - G_7|$ is used as the error estimate. This is the workhorse of QUADPACK's DQAGSE routine, which underlies MATLAB's integral and many scientific libraries.

Clenshaw-Curtis Quadrature

Definition - Clenshaw-Curtis Rule

Expand $f$ in a Chebyshev series on $[-1,1]$: $f(x) = \sum_{k=0}^\infty c_k T_k(x)$. Integrating term by term gives exact weights for the nodes $x_j = \cos(j\pi/n)$, $j=0,\ldots,n$. The DCT (discrete cosine transform) computes the weights in $O(n\log n)$ time. Nested grids (nodes for level $\ell$ are a subset of those for level $\ell+1$) allow adaptive refinement without redundant evaluations.

Practical Considerations

Remark - Failure Modes

Adaptive methods can fail or become very slow for:

  • Integrable singularities (e.g., $f(x) = x^{-1/2}$ near 0): subdivision converges but slowly. Use a singularity-extracting transformation or Gauss-Jacobi weights.
  • Highly oscillatory integrands (e.g., $\int_0^{1000} \sin(1000\pi x)\,dx$): exponentially many subintervals needed. Use Levin collocation or stationary phase methods.
  • Infinite intervals: map to $[0,1]$ via $x = t/(1-t)$, or use Gauss-Laguerre/Hermite rules.
