5.2 Gaussian Quadrature

Newton-Cotes rules fix nodes at prescribed points. Gaussian quadrature instead chooses both nodes and weights optimally, achieving degree of exactness $2n-1$ with only $n$ function evaluations - twice the accuracy for the same cost.

Gauss-Legendre Quadrature

Definition - $n$-Point Gaussian Quadrature Rule

An $n$-point quadrature rule approximates $$\int_{-1}^{1} f(x)\,dx \approx \sum_{i=1}^{n} w_i\, f(x_i).$$ The Gauss-Legendre rule chooses $x_1,\ldots,x_n$ and $w_1,\ldots,w_n$ to maximise the degree of exactness.

Theorem - Optimality of Gauss-Legendre Nodes

The $n$-point Gauss-Legendre rule is exact for all polynomials of degree $\le 2n-1$. No $n$-point rule can be exact for all polynomials of degree $\ge 2n$.

The nodes $x_1 < \cdots < x_n$ are the $n$ zeros of the Legendre polynomial $P_n(x)$, and the weights are $$w_i = \frac{2}{(1-x_i^2)\,[P_n'(x_i)]^2}.$$

Remark - Why Legendre Zeros?

For any quadrature rule to be exact for all polynomials of degree $\le 2n-1$, the node polynomial $\omega(x) = \prod_{i=1}^n (x-x_i)$ must be orthogonal to every polynomial of degree $< n$ with respect to the weight function $w \equiv 1$ on $[-1,1]$. The unique monic polynomial with this property is $P_n(x)/P_n(1)$, i.e., the Legendre polynomial.

Transformation to a General Interval

Proposition - Change of Interval

To integrate over $[a,b]$, use the affine substitution $x = \tfrac{b-a}{2}t + \tfrac{a+b}{2}$: $$\int_a^b f(x)\,dx = \frac{b-a}{2}\int_{-1}^{1} f\!\left(\tfrac{b-a}{2}t + \tfrac{a+b}{2}\right)dt \approx \frac{b-a}{2}\sum_{i=1}^{n} w_i\, f\!\left(\tfrac{b-a}{2}x_i + \tfrac{a+b}{2}\right).$$

Error Formula

Theorem - Gauss-Legendre Error

If $f \in C^{2n}[-1,1]$, the error of the $n$-point Gauss-Legendre rule is $$E_n(f) = \frac{2^{2n+1}(n!)^4}{(2n+1)\,[(2n)!]^3}\,f^{(2n)}(\xi), \quad \xi \in (-1,1).$$ This decays super-algebraically in $n$ when $f$ is analytic.

Other Gaussian Rules

Definition - Gauss-Laguerre, Gauss-Hermite, Gauss-Chebyshev

Different weight functions $w(x)$ on different intervals yield other Gaussian rules. The key identity is $\int w(x)f(x)\,dx \approx \sum w_i f(x_i)$ where nodes and weights come from the relevant orthogonal polynomials.

  • Gauss-Laguerre: $\int_0^\infty e^{-x}f(x)\,dx$; nodes = zeros of Laguerre polynomial $L_n$.
  • Gauss-Hermite: $\int_{-\infty}^{\infty} e^{-x^2}f(x)\,dx$; nodes = zeros of Hermite polynomial $H_n$.
  • Gauss-Chebyshev: $\int_{-1}^{1}(1-x^2)^{-1/2}f(x)\,dx$; nodes are equally spaced in angle: $x_i = \cos\!\left(\tfrac{(2i-1)\pi}{2n}\right)$; weights $w_i = \pi/n$.

Comparison with Newton-Cotes

Remark - Efficiency

For $n$ function evaluations, the $n$-point Gauss-Legendre rule integrates polynomials of degree $2n-1$ exactly, while the $n$-point Newton-Cotes rule integrates polynomials of degree at most $n-1$ (or $n$ if $n$ is even). For smooth $f$, Gaussian quadrature thus requires roughly half the evaluations for the same accuracy. For non-smooth or highly oscillatory $f$, adaptive rules (Section 5.3) are often preferable.
