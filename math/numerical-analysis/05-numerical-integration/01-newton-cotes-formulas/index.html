5.1 Newton-Cotes Formulas

Newton-Cotes formulas approximate $\int_a^b f(x)\,dx$ by integrating the polynomial that interpolates $f$ at equally spaced nodes on $[a,b]$. The three classical rules - midpoint, trapezoidal, and Simpson's - form the foundation, and their composite versions with error analysis guide the choice of step size.

Basic Rules on a Single Interval

Let $m = \tfrac{a+b}{2}$ denote the midpoint of $[a,b]$.

Definition - Midpoint Rule

The midpoint rule approximates the integral by the rectangle of height $f(m)$: $$M(f) = (b-a)\,f(m).$$

Theorem - Midpoint Rule Error

If $f \in C^2[a,b]$, then $$\int_a^b f(x)\,dx - M(f) = \frac{(b-a)^3}{24}\,f''(\xi)$$ for some $\xi \in (a,b)$. The midpoint rule is exact for polynomials of degree $\le 1$.

Definition - Trapezoidal Rule

The trapezoidal rule uses the linear interpolant through $(a,f(a))$ and $(b,f(b))$: $$T(f) = \frac{b-a}{2}\bigl(f(a)+f(b)\bigr).$$

Theorem - Trapezoidal Rule Error

If $f \in C^2[a,b]$, then $$\int_a^b f(x)\,dx - T(f) = -\frac{(b-a)^3}{12}\,f''(\xi)$$ for some $\xi \in (a,b)$. The trapezoidal rule is also exact for degree $\le 1$.

Definition - Simpson's Rule

Simpson's rule integrates the quadratic interpolant through $(a,f(a))$, $(m,f(m))$, $(b,f(b))$: $$S(f) = \frac{b-a}{6}\bigl(f(a)+4f(m)+f(b)\bigr).$$

Theorem - Simpson's Rule Error

If $f \in C^4[a,b]$, then $$\int_a^b f(x)\,dx - S(f) = -\frac{(b-a)^5}{2880}\,f^{(4)}(\xi)$$ for some $\xi \in (a,b)$. Simpson's rule is exact for polynomials of degree $\le 3$ - one degree higher than expected from a 3-point rule - because the error term for degree 3 vanishes by symmetry.

Composite Rules

To integrate over $[a,b]$, partition it into $n$ equal subintervals of length $h = (b-a)/n$, with nodes $x_i = a + ih$.

Definition - Composite Trapezoidal Rule

$$T_n(f) = h\left[\frac{f(a)}{2} + f(x_1)+\cdots+f(x_{n-1}) + \frac{f(b)}{2}\right] = \frac{h}{2}\sum_{i=0}^{n-1}\bigl(f(x_i)+f(x_{i+1})\bigr).$$

Theorem - Composite Trapezoidal Error

If $f\in C^2[a,b]$, $$\int_a^b f(x)\,dx - T_n(f) = -\frac{(b-a)}{12}\,h^2\,f''(\xi)$$ for some $\xi\in(a,b)$. The global error is $O(h^2)$.

Definition - Composite Simpson's Rule

Requires $n$ even. With midpoints $m_i = x_{i-1}+h/2$ of each subinterval: $$S_n(f) = \frac{h}{6}\sum_{i=1}^{n}\bigl(f(x_{i-1})+4f(m_i)+f(x_i)\bigr) = \frac{h}{3}\left[f(x_0)+4f(x_1)+2f(x_2)+\cdots+4f(x_{n-1})+f(x_n)\right]$$ (with alternating coefficients 4, 2 on interior nodes when written over $2n$ subintervals of length $h/2$).

The global error is $O(h^4)$: if $f\in C^4[a,b]$, the error is $-\tfrac{(b-a)}{180}h^4 f^{(4)}(\xi)$.

Richardson Extrapolation and the Romberg Method

Theorem - Euler-Maclaurin Error Expansion

The composite trapezoidal rule satisfies the asymptotic expansion $$\int_a^b f\,dx = T_n(f) + c_2 h^2 + c_4 h^4 + c_6 h^6 + \cdots$$ where the constants $c_{2k}$ depend on $f$ and its derivatives at the endpoints but are independent of $h$. This is the Euler-Maclaurin formula.

Definition - Richardson Extrapolation

Given two trapezoidal approximations $T_n$ (step $h$) and $T_{2n}$ (step $h/2$), the leading $h^2$ error term can be eliminated: $$R = \frac{4\,T_{2n} - T_n}{3} = S_{2n}.$$ This is precisely the composite Simpson's rule on the combined grid, exact to $O(h^4)$.

Definition - Romberg Triangle

The Romberg method applies Richardson extrapolation recursively. Define $R(n,0) = T_{2^n}$. Then $$R(n,k) = \frac{4^k\,R(n,k-1) - R(n-1,k-1)}{4^k - 1}, \quad k \ge 1.$$ Each column of the Romberg triangle has error one order higher: $R(n,k) = O(h^{2k+2})$ where $h = (b-a)/2^n$.

Example: $\int_0^1 e^x\,dx$

Example - Integrating $e^x$ on $[0,1]$

The Romberg iterate $R(3,3)$ achieves error below $10^{-13}$ using only $2^3+1=9$ function evaluations.
