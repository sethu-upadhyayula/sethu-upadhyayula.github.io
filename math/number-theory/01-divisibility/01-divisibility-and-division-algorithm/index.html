1.1 Divisibility and the Division Algorithm

We begin with the most fundamental relation in number theory: divisibility. The simple idea that one integer "goes into" another evenly underlies the entire structure of the integers.

Definition 1.1.1 - Divisibility

Let $a, b \in \mathbb{Z}$ with $a \neq 0$. We say $a$ divides $b$, written $a \mid b$, if there exists an integer $k$ such that $b = ka$. We say $a$ is a divisor (or factor) of $b$, and $b$ is a multiple of $a$. If $a \nmid b$, we say $a$ does not divide $b$.

Basic Properties of Divisibility

Theorem 1.1.2 - Properties of Divisibility

Let $a, b, c \in \mathbb{Z}$ (with divisors nonzero). Then:

  • Transitivity: If $a \mid b$ and $b \mid c$, then $a \mid c$.
  • Linear combinations: If $a \mid b$ and $a \mid c$, then $a \mid mb + nc$ for all $m, n \in \mathbb{Z}$.
  • Bound: If $a \mid b$ and $b > 0$, then $a \le b$.
  • Units: $1 \mid b$ for all $b$, and $b \mid b$ for all $b \neq 0$.
Proof

Transitivity: If $b = ka$ and $c = lb$, then $c = l(ka) = (lk)a$, so $a \mid c$.

Linear combinations: If $b = k_1 a$ and $c = k_2 a$, then $mb + nc = m k_1 a + n k_2 a = (mk_1 + nk_2)a$, so $a \mid mb + nc$.

Bound: If $b = ka$ with $b > 0$, then $k \ge 1$ (since $b > 0$ and $a \ge 1$ forces $k \ge 1$), so $b = ka \ge a$. $\square$

Example 1.1.3

Since $6 \mid 18$ and $18 \mid 54$, transitivity gives $6 \mid 54$. Indeed $54 = 9 \cdot 6$.

Since $3 \mid 12$ and $3 \mid 15$, we get $3 \mid (5 \cdot 12 - 2 \cdot 15) = 60 - 30 = 30$. Indeed $30 = 10 \cdot 3$.

The Division Algorithm

The division algorithm formalises the familiar process of long division, guaranteeing a unique quotient and remainder.

Theorem 1.1.4 - Division Algorithm

For every $a \in \mathbb{Z}$ and $b \in \mathbb{Z}^+$, there exist unique integers $q$ and $r$ such that $$a = bq + r \quad \text{and} \quad 0 \le r < b.$$ The integer $q$ is the quotient and $r$ is the remainder when $a$ is divided by $b$. We also write $r = a \bmod b$.

Proof

Existence. Consider the set $S = \{a - bk : k \in \mathbb{Z},\; a - bk \ge 0\}$. Since $b > 0$, taking $k$ sufficiently negative makes $a - bk$ positive, so $S$ is nonempty. By the Well-Ordering Principle, $S$ has a least element $r = a - bq$ for some integer $q$.

We have $r \ge 0$ by definition of $S$. Suppose $r \ge b$. Then $r - b = a - b(q+1) \ge 0$, so $r - b \in S$. But $r - b < r$, contradicting minimality of $r$. Therefore $0 \le r < b$.

Uniqueness. Suppose $a = bq_1 + r_1 = bq_2 + r_2$ with $0 \le r_1, r_2 < b$. Then $b(q_1 - q_2) = r_2 - r_1$. Taking absolute values, $b|q_1 - q_2| = |r_2 - r_1| < b$. Thus $|q_1 - q_2| < 1$, so $q_1 = q_2$ and consequently $r_1 = r_2$. $\square$

Example 1.1.5 - Applying the Division Algorithm

Divide $a = 37$ by $b = 5$: $37 = 5 \cdot 7 + 2$, so $q = 7$ and $r = 2$.

Divide $a = -14$ by $b = 3$: we need $r \ge 0$, so $-14 = 3 \cdot (-5) + 1$, giving $q = -5$ and $r = 1$.

Note: the remainder is always non-negative, so $(-14) \bmod 3 = 1$, not $-2$.

Remark

The proof uses the Well-Ordering Principle: every nonempty set of non-negative integers has a least element. This is equivalent to the Principle of Mathematical Induction and is a fundamental property of $\mathbb{N}$. The division algorithm is the engine that drives the Euclidean algorithm in Section 1.3.
