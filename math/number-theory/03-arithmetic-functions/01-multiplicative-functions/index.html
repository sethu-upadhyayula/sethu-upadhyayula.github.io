3.1 Multiplicative Functions

An arithmetic function is any function $f : \mathbb{Z}^+ \to \mathbb{C}$. The multiplicative functions form a particularly well-behaved class — they are determined by their values on prime powers — and arise naturally throughout number theory.

Definition 3.1.1 — Multiplicative and Completely Multiplicative

An arithmetic function $f$ (not identically zero) is:

  • Multiplicative if $f(mn) = f(m)f(n)$ whenever $\gcd(m,n) = 1$.
  • Completely multiplicative if $f(mn) = f(m)f(n)$ for all $m, n \geq 1$.

For a multiplicative function with $n = p_1^{a_1} \cdots p_k^{a_k}$: $f(n) = f(p_1^{a_1}) \cdots f(p_k^{a_k})$.

Example 3.1.2 — Divisor Functions

For $k \geq 0$, define $\sigma_k(n) = \sum_{d \mid n} d^k$. Special cases:

  • $\sigma_0(n) = d(n) = \tau(n)$: number of divisors of $n$.
  • $\sigma_1(n) = \sigma(n)$: sum of divisors of $n$.

These are multiplicative. For prime powers: $\sigma_k(p^a) = 1 + p^k + p^{2k} + \cdots + p^{ak} = \frac{p^{(a+1)k}-1}{p^k - 1}$ (for $k \geq 1$).

Definition 3.1.3 — Dirichlet Convolution

The Dirichlet convolution of arithmetic functions $f$ and $g$ is: $$(f * g)(n) = \sum_{d \mid n} f(d)\,g(n/d).$$ The set of arithmetic functions forms a commutative ring under pointwise addition and Dirichlet convolution. The identity element is $\varepsilon$, where $\varepsilon(1) = 1$ and $\varepsilon(n) = 0$ for $n > 1$. The convolution of two multiplicative functions is multiplicative.

Theorem 3.1.4 — Dirichlet Inverse

Every arithmetic function $f$ with $f(1) \neq 0$ has a unique Dirichlet inverse $f^{-1}$ satisfying $f * f^{-1} = \varepsilon$. The inverse of the constant function $\mathbf{1}$ (where $\mathbf{1}(n) = 1$ for all $n$) is the Möbius function $\mu$: $\mathbf{1} * \mu = \varepsilon$.

Example 3.1.5 — Key Identities
  • $d = \mathbf{1} * \mathbf{1}$: the number of divisors is the Dirichlet self-convolution of the constant $1$.
  • $\sigma = \text{id} * \mathbf{1}$ where $\text{id}(n) = n$: sum of divisors.
  • $\phi = \text{id} * \mu$: Euler's totient (proved in the next section).
  • $n = \phi * \mathbf{1}$: $\sum_{d \mid n} \phi(d) = n$.
