4.1 Legendre Symbol

Throughout this section, $p$ denotes an odd prime and $a$ an integer with $\gcd(a, p) = 1$. We study which elements of $(\mathbb{Z}/p\mathbb{Z})^*$ are squares — a question that turns out to have deep arithmetic significance.

Quadratic Residues and Non-Residues

Definition 4.1.1 — Quadratic Residue

Let $p$ be an odd prime and $\gcd(a, p) = 1$. We say $a$ is a quadratic residue (QR) mod $p$ if the congruence $x^2 \equiv a \pmod{p}$ has a solution. Otherwise, $a$ is a quadratic non-residue (QNR) mod $p$.

When $p \mid a$, we handle this case separately in the Legendre symbol by setting the value to $0$.

Theorem 4.1.2 — Count of Quadratic Residues

Among the integers $\{1, 2, \ldots, p-1\}$, exactly $\frac{p-1}{2}$ are quadratic residues mod $p$ and $\frac{p-1}{2}$ are quadratic non-residues.

Proof

The map $x \mapsto x^2$ on $(\mathbb{Z}/p\mathbb{Z})^*$ is exactly 2-to-1: since $x^2 \equiv (-x)^2 \pmod{p}$ and $x \not\equiv -x \pmod{p}$ (as $p$ is odd and $p \nmid x$), each nonzero square has exactly two square roots $\pm x$. The $(p-1)$ elements pair up into $\frac{p-1}{2}$ pairs $\{x, -x\}$, each giving one distinct square. $\square$

Example — QRs mod 7

Compute $1^2=1$, $2^2=4$, $3^2=2$, $4^2=2$, $5^2=4$, $6^2=1$ (all mod 7). The distinct squares are $\{1, 2, 4\}$, so the QRs mod 7 are $1, 2, 4$ and the QNRs are $3, 5, 6$. Indeed $\frac{7-1}{2} = 3$ of each.

The Legendre Symbol

Definition 4.1.3 — Legendre Symbol

For an odd prime $p$ and any integer $a$, the Legendre symbol $\left(\frac{a}{p}\right)$ is defined by: $$\left(\frac{a}{p}\right) = \begin{cases} 0 & \text{if } p \mid a, \\ +1 & \text{if } a \text{ is a QR mod } p, \\ -1 & \text{if } a \text{ is a QNR mod } p. \end{cases}$$

The Legendre symbol depends only on $a \bmod p$.

Euler's Criterion

Theorem 4.1.4 — Euler's Criterion

For an odd prime $p$ and $\gcd(a, p) = 1$: $$\left(\frac{a}{p}\right) \equiv a^{(p-1)/2} \pmod{p}.$$ In particular, $a^{(p-1)/2} \equiv \pm 1 \pmod{p}$, with $+1$ iff $a$ is a QR.

Proof

By Fermat's Little Theorem, $a^{p-1} \equiv 1 \pmod{p}$, so $\left(a^{(p-1)/2}\right)^2 \equiv 1 \pmod{p}$. Since $p$ is prime, the only square roots of $1$ mod $p$ are $\pm 1$, so $a^{(p-1)/2} \equiv \pm 1 \pmod{p}$.

If $a \equiv x^2 \pmod{p}$ is a QR, then $a^{(p-1)/2} \equiv x^{p-1} \equiv 1 \pmod{p}$ by Fermat.

Conversely, since $(\mathbb{Z}/p\mathbb{Z})^*$ is cyclic of order $p-1$, the polynomial $t^{(p-1)/2} - 1 \equiv 0 \pmod{p}$ has at most $(p-1)/2$ solutions, which are precisely the QRs (the squares of a generator of the group). So $a^{(p-1)/2} \equiv 1$ iff $a$ is a QR. $\square$

Example

Is 3 a QR mod 11? Compute $3^{(11-1)/2} = 3^5 = 243 \equiv 243 - 22 \cdot 11 = 243 - 242 = 1 \pmod{11}$. So $\left(\frac{3}{11}\right) = 1$ and 3 is a QR mod 11. (Check: $5^2 = 25 \equiv 3 \pmod{11}$. ✓)

Multiplicativity

Theorem 4.1.5 — Multiplicativity of the Legendre Symbol

For any integers $a, b$ with $p \nmid ab$: $$\left(\frac{ab}{p}\right) = \left(\frac{a}{p}\right)\left(\frac{b}{p}\right).$$

Proof

By Euler's criterion: $\left(\frac{ab}{p}\right) \equiv (ab)^{(p-1)/2} = a^{(p-1)/2} \cdot b^{(p-1)/2} \equiv \left(\frac{a}{p}\right)\left(\frac{b}{p}\right) \pmod{p}$. Since all three quantities are in $\{-1, +1\}$ and $p > 2$, equality holds as integers. $\square$

The Two Supplements

Theorem 4.1.6 — First Supplement

$$\left(\frac{-1}{p}\right) = (-1)^{(p-1)/2} = \begin{cases} +1 & \text{if } p \equiv 1 \pmod{4}, \\ -1 & \text{if } p \equiv 3 \pmod{4}. \end{cases}$$ That is, $-1$ is a quadratic residue mod $p$ if and only if $p \equiv 1 \pmod{4}$.

Proof

Immediate from Euler's criterion: $\left(\frac{-1}{p}\right) \equiv (-1)^{(p-1)/2} \pmod{p}$. Since $(p-1)/2$ is even iff $p \equiv 1 \pmod{4}$, the result follows. $\square$

Theorem 4.1.7 — Second Supplement

$$\left(\frac{2}{p}\right) = (-1)^{(p^2-1)/8} = \begin{cases} +1 & \text{if } p \equiv \pm 1 \pmod{8}, \\ -1 & \text{if } p \equiv \pm 3 \pmod{8}. \end{cases}$$ That is, $2$ is a quadratic residue mod $p$ if and only if $p \equiv \pm 1 \pmod{8}$.

Remark

The second supplement is harder to prove than the first; one standard route uses Gauss's lemma, which counts the number of "negative representatives" in the set $\{a, 2a, \ldots, \frac{p-1}{2}a\}$ reduced mod $p$ to the interval $(-p/2, p/2)$. The exponent $(p^2-1)/8$ is always an integer since $p$ is odd.

Example — Combining Supplements and Multiplicativity

Evaluate $\left(\frac{-2}{7}\right)$. We have $\left(\frac{-2}{7}\right) = \left(\frac{-1}{7}\right)\left(\frac{2}{7}\right)$. Now $\frac{7-1}{2} = 3$ is odd, so $\left(\frac{-1}{7}\right) = -1$. Also $7 \equiv -1 \pmod 8$, so $\left(\frac{2}{7}\right) = +1$. Therefore $\left(\frac{-2}{7}\right) = (-1)(+1) = -1$: $-2$ is a QNR mod 7.

Check: the QRs mod 7 are $\{1, 2, 4\}$, and $-2 \equiv 5 \pmod 7$ is indeed not among them. ✓
