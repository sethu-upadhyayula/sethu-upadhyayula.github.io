5.3 Pell's Equation

Pell's equation $x^2 - Dy^2 = 1$ is a Diophantine equation with a rich history stretching back to ancient India (Brahmagupta, 628 AD) and medieval Europe. Despite its simple appearance, it always has infinitely many solutions when $D$ is a positive non-square integer, and the complete solution theory relies beautifully on continued fractions.

Statement and Existence

Definition 5.3.1 — Pell's Equation

Pell's equation is $x^2 - Dy^2 = 1$, where $D$ is a positive integer that is not a perfect square. We seek positive integer solutions $(x, y)$.

Theorem 5.3.2 — Existence of Solutions

For any positive non-square integer $D$, Pell's equation $x^2 - Dy^2 = 1$ has infinitely many positive integer solutions. In particular, a minimal (fundamental) solution $(x_1, y_1)$ with $x_1, y_1 > 0$ always exists.

Periodic Continued Fractions of $\sqrt{D}$

Theorem 5.3.3 — Periodicity of $\sqrt{D}$

The continued fraction expansion of $\sqrt{D}$ (for non-square $D > 0$) is eventually periodic of the form: $$\sqrt{D} = [a_0; \overline{a_1, a_2, \ldots, a_{k-1}, 2a_0}]$$ where $a_0 = \lfloor \sqrt{D} \rfloor$ and the period ends with $2a_0$. The period length $k$ depends on $D$.

This is a special case of Lagrange's theorem (Theorem 5.1.4): $\sqrt{D}$ is a quadratic irrational, so its CF is eventually periodic.

Fundamental Solution via Convergents

Theorem 5.3.4 — Fundamental Solution

Let $\sqrt{D} = [a_0; \overline{a_1, \ldots, a_k}]$ with period length $k$, and let $p_n/q_n$ denote the convergents.

  • If $k$ is even: the fundamental solution is $(x_1, y_1) = (p_{k-1}, q_{k-1})$.
  • If $k$ is odd: $p_{k-1}^2 - D q_{k-1}^2 = -1$ (a solution to the negative Pell equation), and the fundamental solution to $x^2 - Dy^2 = 1$ is $(x_1, y_1) = (p_{2k-1}, q_{2k-1})$.

In both cases, $x_1 + y_1 \sqrt{D} = (p_{k-1} + q_{k-1}\sqrt{D})^{1 \text{ or } 2}$.

All Solutions

Theorem 5.3.5 — All Solutions of Pell's Equation

If $(x_1, y_1)$ is the fundamental solution, then all positive integer solutions $(x_n, y_n)$ are given by: $$x_n + y_n \sqrt{D} = (x_1 + y_1 \sqrt{D})^n, \quad n = 1, 2, 3, \ldots$$ The sequences $x_n$ and $y_n$ satisfy: $$x_n = 2x_1 x_{n-1} - x_{n-2}, \qquad y_n = 2x_1 y_{n-1} - y_{n-2}.$$

The solutions grow exponentially: $x_n \approx \frac{(x_1 + y_1\sqrt{D})^n}{2}$ as $n \to \infty$.

Worked Example

Example — $x^2 - 2y^2 = 1$

Step 1: CF of $\sqrt{2}$. We found $\sqrt{2} = [1; \overline{2}]$, so $a_0 = 1$, period $= (2)$, $k = 1$ (odd).

Step 2: Convergents. $p_{-1}=1$, $q_{-1}=0$; $p_0=1$, $q_0=1$; $p_1 = 2\cdot 1 + 1 = 3$, $q_1 = 2\cdot 1 + 0 = 2$.

Step 3: Fundamental solution. Since $k=1$ is odd, check $p_0^2 - 2q_0^2 = 1 - 2 = -1$: this solves the negative Pell equation. The fundamental solution to $x^2 - 2y^2 = 1$ uses $p_{2k-1} = p_1 = 3$, $q_{2k-1} = q_1 = 2$: check $3^2 - 2 \cdot 2^2 = 9 - 8 = 1$. ✓ So $(x_1, y_1) = (3, 2)$.

Step 4: All solutions. $(x_n + y_n\sqrt{2}) = (3 + 2\sqrt{2})^n$:

  • $n=1$: $(3, 2)$, check $9 - 8 = 1$ ✓
  • $n=2$: $(3+2\sqrt{2})^2 = 17 + 12\sqrt{2}$, so $(17, 12)$: $289 - 2\cdot144 = 289-288=1$ ✓
  • $n=3$: $(3+2\sqrt{2})^3 = 99 + 70\sqrt{2}$, so $(99, 70)$: $9801 - 2\cdot4900 = 9801-9800=1$ ✓

Example — $x^2 - 3y^2 = 1$

$\sqrt{3} = [1; \overline{1, 2}]$, period $k = 2$ (even). Convergents: $p_0=1, q_0=1$; $p_1 = 1\cdot1+1=2, q_1=1\cdot1+0=1$; $p_2 = 2\cdot2+1=7/3$... Let me recompute: $a_0=1, a_1=1, a_2=2$. $p_1 = a_1 p_0 + p_{-1} = 1\cdot1+1=2$, $q_1 = 1\cdot1+0=1$. $p_2 = 2\cdot2+1=5$... Wait, $a_2 p_1 + p_0 = 2\cdot2+1=5$, $q_2 = 2\cdot1+1=3$. But $k-1 = 1$, so fundamental solution is $(p_1, q_1) = (2, 1)$: check $4 - 3 = 1$ ✓. So $(x_1,y_1)=(2,1)$.

Next solution: $(2+\sqrt{3})^2 = 7+4\sqrt{3}$, giving $(7,4)$: $49-48=1$ ✓.

Remark — Historical Misnaming

Pell's equation is named after John Pell (1611-1685), though Pell had little to do with it. Euler mistakenly attributed a solution method to Pell when it was actually due to Brouncker. The equation was studied extensively by Fermat (who posed it as a challenge) and solved completely by Lagrange (1768), who proved both the periodicity of the CF of $\sqrt{D}$ and the connection to Pell's equation. Indian mathematicians (Brahmagupta, 628 AD; Bhaskara II, 1150 AD) had solved special cases with the elegant "chakravala" (cyclic) method centuries earlier.
