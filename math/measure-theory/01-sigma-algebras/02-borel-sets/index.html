1.2 Borel Sets

The most important $\sigma$-algebra for analysis on $\mathbb{R}$ is the Borel $\sigma$-algebra, generated by the open sets. Every set one encounters in classical analysis - open, closed, $G_\delta$, $F_\sigma$, and their countable combinations - is a Borel set. However, the Borel hierarchy is infinite and strict: non-Borel sets exist.

The Borel Sigma-Algebra on $\mathbb{R}$

Definition 1.2.1 - Borel $\sigma$-Algebra

The Borel $\sigma$-algebra on $\mathbb{R}$, denoted $\mathcal{B}(\mathbb{R})$, is the $\sigma$-algebra generated by the collection of all open subsets of $\mathbb{R}$: $$\mathcal{B}(\mathbb{R}) = \sigma(\{\text{open sets in } \mathbb{R}\}).$$ Elements of $\mathcal{B}(\mathbb{R})$ are called Borel sets.

Theorem 1.2.2 - Equivalent Generators

The Borel $\sigma$-algebra on $\mathbb{R}$ is generated by any of the following classes:

  • All open intervals $(a,b)$, $a < b$.
  • All closed intervals $[a,b]$.
  • All half-open intervals $(-\infty, c)$ for $c \in \mathbb{R}$.
  • All half-open intervals $(c, \infty)$ for $c \in \mathbb{R}$.
Each generating class yields the same $\sigma$-algebra $\mathcal{B}(\mathbb{R})$.

The Borel Hierarchy

Definition 1.2.3 - $G_\delta$ and $F_\sigma$ Sets

A $G_\delta$ set is a countable intersection of open sets. A $F_\sigma$ set is a countable union of closed sets. Both classes are Borel sets:

  • Every open set is trivially $G_\delta$ (intersection of one open set).
  • Every closed set is trivially $F_\sigma$ (union of one closed set).
  • The irrationals $\mathbb{R} \setminus \mathbb{Q}$ form a $G_\delta$ set.
  • The rationals $\mathbb{Q}$ form an $F_\sigma$ set (countable union of singletons).
Theorem 1.2.4 - Open Sets in $\mathbb{R}$

Every open set $U \subseteq \mathbb{R}$ is a countable union of pairwise disjoint open intervals. (Each connected component of $U$ is an open interval - bounded or unbounded - and there are at most countably many since each contains a rational.)

Proof Sketch

Define an equivalence relation on $U$ by $x \sim y$ iff $[\min(x,y),\max(x,y)] \subseteq U$. Each equivalence class is a connected open subset of $\mathbb{R}$, hence an open interval. These intervals are pairwise disjoint and cover $U$. Each contains a distinct rational (by density of $\mathbb{Q}$), so there are countably many. $\square$

Borel Sets in $\mathbb{R}^n$

Definition 1.2.5 - Borel Sets in $\mathbb{R}^n$

The Borel $\sigma$-algebra $\mathcal{B}(\mathbb{R}^n)$ is the $\sigma$-algebra on $\mathbb{R}^n$ generated by all open sets in the standard (Euclidean) topology. Equivalently, $\mathcal{B}(\mathbb{R}^n) = \mathcal{B}(\mathbb{R}) \otimes \cdots \otimes \mathcal{B}(\mathbb{R})$ ($n$ times), the product $\sigma$-algebra, generated by rectangles $I_1 \times \cdots \times I_n$ with each $I_k$ an interval.

Limits of the Borel Hierarchy

Remark - Non-Borel Sets

The Borel hierarchy is indexed by countable ordinals and is strictly increasing: at each level new sets appear that are not in the previous levels. In particular, there exist analytic sets (continuous images of Borel sets) that are not Borel.

However, every Lebesgue measurable set differs from a Borel set by at most a null set. So for integration purposes, the distinction between Borel and Lebesgue measurable is minor.

Example 1.2.6 - Some Borel Sets
  • Every singleton $\{x\} = \bigcap_{n=1}^\infty (x - 1/n, x + 1/n)$ is a $G_\delta$ (and hence Borel) set.
  • The Cantor set $C$ is closed, hence Borel.
  • Any countable set is $F_\sigma$, hence Borel.
  • The set $\{(x, \sin(1/x)) : x > 0\}$ in $\mathbb{R}^2$ is a Borel set (it is closed in a suitable sense).
