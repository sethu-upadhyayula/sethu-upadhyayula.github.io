1.1 Sigma-Algebras and Generated Sigma-Algebras

A $\sigma$-algebra (sigma-algebra) on a set $X$ is a collection of subsets of $X$ that is closed under the Boolean operations and countable unions that arise naturally when working with limits of sets. The prefix $\sigma$ refers to countable (from German Summe).

Definition

Definition 1.1.1 — $\sigma$-Algebra

A collection $\mathcal{M} \subseteq 2^X$ of subsets of $X$ is a $\sigma$-algebra (or $\sigma$-field) on $X$ if:

  1. $X \in \mathcal{M}$.
  2. If $A \in \mathcal{M}$, then $A^c = X \setminus A \in \mathcal{M}$ (closed under complements).
  3. If $A_1, A_2, A_3, \ldots \in \mathcal{M}$, then $\bigcup_{n=1}^\infty A_n \in \mathcal{M}$ (closed under countable unions).

The pair $(X, \mathcal{M})$ is called a measurable space. Elements of $\mathcal{M}$ are called measurable sets.

Proposition 1.1.2 — Closure under Countable Intersections

Every $\sigma$-algebra $\mathcal{M}$ is also closed under countable intersections: if $A_1, A_2, \ldots \in \mathcal{M}$ then $\bigcap_{n=1}^\infty A_n \in \mathcal{M}$. Moreover $\emptyset \in \mathcal{M}$, and $\mathcal{M}$ is closed under set differences $A \setminus B = A \cap B^c$.

Proof

Since $X \in \mathcal{M}$ and $\mathcal{M}$ is closed under complements, $\emptyset = X^c \in \mathcal{M}$. For intersections: $\bigcap A_n = \left(\bigcup A_n^c\right)^c$. Each $A_n^c \in \mathcal{M}$ by (2), the union is in $\mathcal{M}$ by (3), and the complement is in $\mathcal{M}$ by (2). $\square$

Standard Examples

Example 1.1.3 — Trivial and Discrete $\sigma$-Algebras

On any set $X$:

  • The trivial $\sigma$-algebra $\{\emptyset, X\}$ is the smallest possible $\sigma$-algebra on $X$.
  • The discrete $\sigma$-algebra $2^X$ (the power set) is the largest possible $\sigma$-algebra on $X$.

For $X = \{a, b, c\}$, the collection $\{\emptyset, \{a\}, \{b,c\}, X\}$ is a $\sigma$-algebra, but $\{\emptyset, \{a\}, \{b\}, X\}$ is not (it is not closed under unions: $\{a\} \cup \{b\} = \{a,b\} \notin \mathcal{M}$).

Generated Sigma-Algebras

Definition 1.1.4 — Generated $\sigma$-Algebra

Let $\mathcal{C} \subseteq 2^X$ be any collection of subsets. The $\sigma$-algebra generated by $\mathcal{C}$, denoted $\sigma(\mathcal{C})$, is defined as $$\sigma(\mathcal{C}) = \bigcap \{\mathcal{M} : \mathcal{M} \text{ is a } \sigma\text{-algebra on } X \text{ with } \mathcal{C} \subseteq \mathcal{M}\}.$$

This intersection is taken over all $\sigma$-algebras containing $\mathcal{C}$; it is non-empty since $2^X$ always works. The resulting $\sigma(\mathcal{C})$ is itself a $\sigma$-algebra, and it is the smallest $\sigma$-algebra containing $\mathcal{C}$.

Theorem 1.1.5 — Existence and Minimality

For any $\mathcal{C} \subseteq 2^X$, the generated $\sigma$-algebra $\sigma(\mathcal{C})$ exists and satisfies: (i) $\mathcal{C} \subseteq \sigma(\mathcal{C})$; (ii) $\sigma(\mathcal{C})$ is a $\sigma$-algebra; (iii) if $\mathcal{M}$ is any $\sigma$-algebra with $\mathcal{C} \subseteq \mathcal{M}$, then $\sigma(\mathcal{C}) \subseteq \mathcal{M}$.

Remark — Monotonicity of Generation

If $\mathcal{C}_1 \subseteq \mathcal{C}_2$, then $\sigma(\mathcal{C}_1) \subseteq \sigma(\mathcal{C}_2)$. Also, if $\mathcal{C}$ is already a $\sigma$-algebra, then $\sigma(\mathcal{C}) = \mathcal{C}$. Generation is idempotent: $\sigma(\sigma(\mathcal{C})) = \sigma(\mathcal{C})$.

Why Not All Subsets? The Vitali Construction

Remark — Non-Measurable Sets

One might ask: why not take $\mathcal{M} = 2^{\mathbb{R}}$ and try to define a measure on all subsets of $\mathbb{R}$? The Vitali theorem (1905) shows this is impossible if we require translation invariance and $\lambda([0,1]) = 1$.

Using the Axiom of Choice, one constructs a set $V \subseteq [0,1]$ containing exactly one representative from each equivalence class of $\mathbb{R}/\mathbb{Q}$. Any translation-invariant, countably additive measure assigning $[0,1]$ length 1 must give $V$ a value that is simultaneously 0 and 1 — a contradiction. Hence $V$ is non-measurable, and we must restrict to a proper $\sigma$-algebra.
