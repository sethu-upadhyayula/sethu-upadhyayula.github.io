7.1 Absolutely Continuous Functions

A function is absolutely continuous if it cannot have "hidden" changes concentrated on small sets. This is the precise condition under which the Fundamental Theorem of Calculus holds for the Lebesgue integral.

Definition

Definition 7.1.1 — Absolute Continuity

A function $f : [a,b] \to \mathbb{R}$ is absolutely continuous if for every $\varepsilon > 0$ there exists $\delta > 0$ such that for every finite collection of pairwise disjoint subintervals $(a_1, b_1), \ldots, (a_n, b_n) \subset [a,b]$: $$\sum_{i=1}^n (b_i - a_i) < \delta \implies \sum_{i=1}^n |f(b_i) - f(a_i)| < \varepsilon.$$

We write $f \in AC([a,b])$. Clearly, absolute continuity implies uniform continuity (take $n = 1$).

Example 7.1.2 — The Cantor Function is Not AC

The Cantor function $\varphi : [0,1] \to [0,1]$ is continuous and non-decreasing, with $\varphi' = 0$ a.e. (its derivative vanishes on the complement of the Cantor set, which has measure 1). Yet $\varphi(0) = 0$ and $\varphi(1) = 1$. If $\varphi$ were absolutely continuous, the FTC would give $\varphi(1) - \varphi(0) = \int_0^1 \varphi' = 0$, a contradiction. So the Cantor function is not absolutely continuous, despite being uniformly continuous.

This shows that uniform continuity does not imply absolute continuity.

Characterisation and the FTC

Theorem 7.1.3 — Characterisation of Absolutely Continuous Functions

A function $f : [a,b] \to \mathbb{R}$ is absolutely continuous if and only if there exists $g \in L^1([a,b], \lambda)$ such that: $$f(x) = f(a) + \int_a^x g(t) \, dt \quad \text{for all } x \in [a,b].$$

In this case, $f$ is differentiable $\lambda$-a.e. and $f'(x) = g(x)$ a.e.

Theorem 7.1.4 — Fundamental Theorem of Calculus for Lebesgue Integrals

Let $g \in L^1([a,b])$ and define $F(x) = \int_a^x g(t)\,dt$. Then:

  1. $F$ is absolutely continuous on $[a,b]$.
  2. $F'(x) = g(x)$ for $\lambda$-a.e. $x \in [a,b]$.

Conversely, if $F$ is absolutely continuous on $[a,b]$, then $F' \in L^1([a,b])$ and $F(b) - F(a) = \int_a^b F'(t)\,dt$.

The condition "$F$ absolutely continuous" is precisely what's needed to avoid the failure seen with the Cantor function.

Remark 7.1.5 — Bounded Variation

Recall that $f$ has bounded variation on $[a,b]$ if $\sup_P \sum |f(x_i) - f(x_{i-1})| < \infty$ over all partitions $P$. Every absolutely continuous function has bounded variation: $\text{TV}(f) \leq \int_a^b |f'|\,dt < \infty$. The Cantor function has bounded variation (it is non-decreasing and bounded) but is not absolutely continuous. The hierarchy is: $$\text{Lipschitz} \subset AC \subset BV \cap C \subset BV.$$

Example 7.1.6 — Integral of $L^1$ Function

Let $g(x) = x^{-1/2}$ on $(0,1]$, extended by $g(0) = 0$. Then $g \in L^1([0,1])$ (since $\int_0^1 x^{-1/2}\,dx = 2$). Define $F(x) = \int_0^x g = 2\sqrt{x}$. Then $F \in AC([0,1])$ and $F'(x) = x^{-1/2} = g(x)$ for all $x > 0$. Note that $F'$ is unbounded, yet the FTC holds because $F$ is absolutely continuous.
