3.2 Simple Functions

Simple functions are finite linear combinations of indicator functions of measurable sets. They are the "step functions" of measure theory and serve as the building blocks from which the Lebesgue integral is defined. Their integral is elementary to define and behaves exactly as expected.

Definition

Definition 3.2.1 - Simple Function

A function $s : X \to [0, \infty)$ is a non-negative simple function if it takes only finitely many distinct values $c_1 < c_2 < \cdots < c_n$ (all non-negative). Writing $E_i = \{x : s(x) = c_i\}$, we have the canonical form: $$s = \sum_{i=1}^n c_i \mathbf{1}_{E_i},$$ where $E_1, \ldots, E_n$ are pairwise disjoint measurable sets with $X = \bigsqcup E_i$. Here $\mathbf{1}_{E_i}$ is the indicator function of $E_i$: $\mathbf{1}_{E_i}(x) = 1$ if $x \in E_i$, else $0$.

$s$ is a simple function if and only if it is measurable and takes finitely many values. Note that a simple function need not have the canonical form; any representation $s = \sum_j a_j \mathbf{1}_{F_j}$ with measurable $F_j$ (not necessarily disjoint or distinct $a_j$) defines the same function.

Integral of a Simple Function

Definition 3.2.2 - Integral of a Simple Function

For a non-negative simple function $s = \sum_{i=1}^n c_i \mathbf{1}_{E_i}$ in canonical form, define $$\int s \, d\mu = \sum_{i=1}^n c_i \, \mu(E_i) \in [0, \infty],$$ using the convention $0 \cdot \infty = 0$. For a measurable set $A \in \mathcal{M}$: $$\int_A s \, d\mu = \int s \mathbf{1}_A \, d\mu = \sum_{i=1}^n c_i \, \mu(E_i \cap A).$$

This definition is independent of the representation chosen for $s$.

Theorem 3.2.3 - Properties of the Simple Function Integral

For non-negative simple functions $s, t$ and $a, b \geq 0$:

  1. Linearity: $\int (as + bt) \, d\mu = a \int s \, d\mu + b \int t \, d\mu$.
  2. Monotonicity: if $s \leq t$ pointwise, then $\int s \, d\mu \leq \int t \, d\mu$.
  3. Null sets: if $\mu(E) = 0$ and $s = c \cdot \mathbf{1}_E$, then $\int s \, d\mu = 0$.
  4. Measure function: the set function $A \mapsto \int_A s \, d\mu$ is a measure on $(X, \mathcal{M})$.
Example 3.2.4

On $([0,1], \mathcal{L}, \lambda)$, consider $s = 2 \cdot \mathbf{1}_{[0,1/2)} + 5 \cdot \mathbf{1}_{[1/2,1]}$. Then: $$\int s \, d\lambda = 2 \cdot \lambda([0,1/2)) + 5 \cdot \lambda([1/2,1]) = 2 \cdot \tfrac{1}{2} + 5 \cdot \tfrac{1}{2} = \tfrac{7}{2}.$$

This equals the Riemann integral since $s$ is a step function, but simple functions can be non-zero on far more exotic measurable sets.

Remark - Generalization

A general simple function (not necessarily non-negative) can be written as $s = s^+ - s^-$ where $s^+ = \max(s, 0)$ and $s^- = \max(-s, 0)$ are both non-negative simple functions. Its integral is $\int s = \int s^+ - \int s^-$, provided this is not of the form $\infty - \infty$.
