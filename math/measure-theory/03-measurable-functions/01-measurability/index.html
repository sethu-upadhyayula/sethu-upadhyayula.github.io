3.1 Measurability

A measurable function is one that "respects" the $\sigma$-algebra structure: preimages of measurable sets are measurable. This is the measure-theoretic analogue of continuity in topology, and is the minimal requirement for a function to be integrable.

Definition and Equivalent Conditions

Definition 3.1.1 — Measurable Function

Let $(X, \mathcal{M})$ be a measurable space. A function $f : X \to \mathbb{R}$ (or $f : X \to [-\infty, +\infty]$) is measurable (or $\mathcal{M}$-measurable) if $$f^{-1}(B) = \{x \in X : f(x) \in B\} \in \mathcal{M}$$ for every Borel set $B \in \mathcal{B}(\mathbb{R})$. If $\mathcal{M} = \mathcal{B}(X)$ (the Borel $\sigma$-algebra on a topological space), $f$ is called Borel measurable.

Theorem 3.1.2 — Equivalent Conditions for Measurability

The following are equivalent for $f : X \to \mathbb{R}$:

  1. $f$ is measurable (preimages of Borel sets are measurable).
  2. $\{x : f(x) > c\} \in \mathcal{M}$ for all $c \in \mathbb{R}$.
  3. $\{x : f(x) \geq c\} \in \mathcal{M}$ for all $c \in \mathbb{R}$.
  4. $\{x : f(x) < c\} \in \mathcal{M}$ for all $c \in \mathbb{R}$.
  5. $\{x : f(x) \leq c\} \in \mathcal{M}$ for all $c \in \mathbb{R}$.
Proof Sketch

The sets $(c, \infty)$, $[c, \infty)$, $(-\infty, c)$, $(-\infty, c]$ each generate $\mathcal{B}(\mathbb{R})$. So condition (1) is equivalent to each of (2)–(5) by the minimality of $\sigma$-generation. $\square$

Stability under Operations

Theorem 3.1.3 — Algebraic Operations

If $f, g : X \to \mathbb{R}$ are measurable, then so are $f + g$, $fg$, $|f|$, $\max(f, g)$, $\min(f, g)$, and $f/g$ (where $g \neq 0$). Also $f^+ = \max(f, 0)$ and $f^- = \max(-f, 0)$ are measurable, and $f = f^+ - f^-$.

Theorem 3.1.4 — Pointwise Limits

If $f_n : X \to [-\infty, \infty]$ are measurable, then $\sup_n f_n$, $\inf_n f_n$, $\limsup_{n \to \infty} f_n$, and $\liminf_{n \to \infty} f_n$ are all measurable. In particular, if $f_n \to f$ pointwise (or pointwise a.e.), then $f$ is measurable.

Proof

$\{\sup_n f_n > c\} = \bigcup_n \{f_n > c\} \in \mathcal{M}$, a countable union of measurable sets. Similarly $\inf_n f_n = -\sup_n(-f_n)$, $\limsup f_n = \inf_n \sup_{k \geq n} f_k$, $\liminf f_n = \sup_n \inf_{k \geq n} f_k$. $\square$

Continuous and Borel Measurable Functions

Theorem 3.1.5 — Continuous Functions are Borel Measurable

Every continuous function $f : \mathbb{R} \to \mathbb{R}$ is Borel measurable. More generally, any function measurable with respect to a finer $\sigma$-algebra is also measurable with respect to any coarser one: if $\mathcal{M}_1 \subseteq \mathcal{M}_2$ and $f$ is $\mathcal{M}_1$-measurable, it is also $\mathcal{M}_2$-measurable.

Remark — Compositions

If $f : (X, \mathcal{M}) \to (\mathbb{R}, \mathcal{B})$ is measurable and $g : (\mathbb{R}, \mathcal{B}) \to (\mathbb{R}, \mathcal{B})$ is Borel measurable, then $g \circ f$ is $\mathcal{M}$-measurable. In particular, $e^f$, $\log f$, $\sin f$, $|f|^p$ are all measurable whenever $f$ is.
