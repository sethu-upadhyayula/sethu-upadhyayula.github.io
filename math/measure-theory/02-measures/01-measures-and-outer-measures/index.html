2.1 Measures and Outer Measures

A measure formalises the intuitive notion of "size" - length, area, volume, probability - in a framework that handles countable operations without paradox. Outer measures, defined on all subsets, provide a flexible intermediate step in the construction of measures.

Measures

Definition 2.1.1 - Measure

Let $(X, \mathcal{M})$ be a measurable space. A function $\mu : \mathcal{M} \to [0, \infty]$ is a measure if:

  1. $\mu(\emptyset) = 0$.
  2. Countable additivity: if $A_1, A_2, \ldots \in \mathcal{M}$ are pairwise disjoint, then $\mu\!\left(\bigsqcup_{n=1}^\infty A_n\right) = \sum_{n=1}^\infty \mu(A_n)$.

The triple $(X, \mathcal{M}, \mu)$ is called a measure space. If $\mu(X) = 1$, $\mu$ is a probability measure. If $\mu(X) < \infty$, $\mu$ is a finite measure. If $X = \bigcup_n X_n$ with $\mu(X_n) < \infty$, $\mu$ is $\sigma$-finite.

Theorem 2.1.2 - Basic Properties of Measures

Let $(X, \mathcal{M}, \mu)$ be a measure space. Then:

  1. Monotonicity: $A \subseteq B \Rightarrow \mu(A) \leq \mu(B)$.
  2. Countable subadditivity: $\mu\!\left(\bigcup_{n=1}^\infty A_n\right) \leq \sum_{n=1}^\infty \mu(A_n)$ for any $A_n \in \mathcal{M}$.
  3. Continuity from below: if $A_1 \subseteq A_2 \subseteq \cdots$ and $A = \bigcup A_n$, then $\mu(A_n) \nearrow \mu(A)$.
  4. Continuity from above: if $A_1 \supseteq A_2 \supseteq \cdots$ and $A = \bigcap A_n$ and $\mu(A_1) < \infty$, then $\mu(A_n) \searrow \mu(A)$.

(The finiteness condition in (4) is necessary: consider $\mu = \lambda$ on $\mathbb{R}$ with $A_n = [n, \infty)$.)

Proof of Continuity from Below

Write $A = A_1 \sqcup (A_2 \setminus A_1) \sqcup (A_3 \setminus A_2) \sqcup \cdots$ as a disjoint union. By countable additivity, $\mu(A) = \mu(A_1) + \sum_{k=1}^\infty \mu(A_{k+1} \setminus A_k)$. The partial sums equal $\mu(A_n)$, so $\mu(A_n) \to \mu(A)$. $\square$

Example 2.1.3 - Standard Measures
  • Counting measure: $\mu(A) = |A|$ (cardinality) on $2^X$ for any $X$.
  • Dirac measure: $\delta_x(A) = \mathbf{1}_A(x)$ for a fixed $x \in X$.
  • Lebesgue measure: $\lambda$ on $(\mathbb{R}, \mathcal{B}(\mathbb{R}))$ extending length.
  • Gaussian measure: $d\gamma = \frac{1}{\sqrt{2\pi}}e^{-x^2/2}\,dx$ on $\mathbb{R}$, a probability measure.

Outer Measures

Definition 2.1.4 - Outer Measure

A function $\mu^* : 2^X \to [0, \infty]$ is an outer measure if:

  1. $\mu^*(\emptyset) = 0$.
  2. Monotonicity: $A \subseteq B \Rightarrow \mu^*(A) \leq \mu^*(B)$.
  3. Countable subadditivity: $\mu^*\!\left(\bigcup_{n=1}^\infty A_n\right) \leq \sum_{n=1}^\infty \mu^*(A_n)$ for any $A_n \subseteq X$.

Note that outer measures are defined on all subsets of $X$, not just measurable ones. The lack of countable additivity (only subadditivity) is the key difference from a measure.

Remark - Constructing Outer Measures

A standard method: let $\mathcal{E} \subseteq 2^X$ be any collection with $\emptyset \in \mathcal{E}$, and let $\rho : \mathcal{E} \to [0,\infty]$ with $\rho(\emptyset) = 0$. Then $$\mu^*(A) = \inf\left\{\sum_{n=1}^\infty \rho(E_n) : A \subseteq \bigcup_{n=1}^\infty E_n,\; E_n \in \mathcal{E}\right\}$$ defines an outer measure. This "covering" construction is how Lebesgue measure is built from interval lengths.
