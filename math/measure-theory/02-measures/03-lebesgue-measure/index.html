2.3 Lebesgue Measure

Lebesgue measure $\lambda$ on $\mathbb{R}$ is the unique translation-invariant Borel measure assigning length 1 to the unit interval. It extends the intuitive notion of "length" from intervals to a vastly larger class of sets, while remaining countably additive.

Construction via Outer Measure

Definition 2.3.1 - Lebesgue Outer Measure

For any $E \subseteq \mathbb{R}$, the Lebesgue outer measure is $$\lambda^*(E) = \inf\left\{\sum_{k=1}^\infty |I_k| : E \subseteq \bigcup_{k=1}^\infty I_k,\; I_k \text{ open intervals}\right\},$$ where $|I_k| = b_k - a_k$ for $I_k = (a_k, b_k)$. The infimum is over all countable covers of $E$ by open intervals.

Definition 2.3.2 - Lebesgue Measurable Sets and Measure

A set $E \subseteq \mathbb{R}$ is Lebesgue measurable if it is Carathéodory measurable with respect to $\lambda^*$, i.e., for all $A \subseteq \mathbb{R}$: $\lambda^*(A) = \lambda^*(A \cap E) + \lambda^*(A \cap E^c)$. The Lebesgue measure $\lambda$ is the restriction of $\lambda^*$ to the $\sigma$-algebra $\mathcal{L}$ of Lebesgue measurable sets.

Theorem 2.3.3 - Properties of Lebesgue Measure
  • $\lambda([a,b]) = \lambda([a,b)) = \lambda((a,b]) = \lambda((a,b)) = b - a$ for $a \leq b$.
  • Translation invariance: $\lambda(E + t) = \lambda(E)$ for all $t \in \mathbb{R}$, where $E + t = \{x + t : x \in E\}$.
  • $\mathcal{B}(\mathbb{R}) \subseteq \mathcal{L}$: all Borel sets are Lebesgue measurable.
  • $(\mathbb{R}, \mathcal{L}, \lambda)$ is a complete measure space.
  • $\lambda$ is $\sigma$-finite: $\mathbb{R} = \bigcup_n [-n,n]$ with $\lambda([-n,n]) = 2n < \infty$.

Null Sets

Definition 2.3.4 - Null Sets

A set $N \subseteq \mathbb{R}$ is a null set (or set of measure zero) if $\lambda(N) = 0$. By completeness, every subset of a null set is also measurable (and null). We say a property holds almost everywhere (a.e.) if it holds outside a null set.

Example 2.3.5 - Null Sets
  • Every countable set has Lebesgue measure zero: $\lambda(\{x\}) = 0$, and countable additivity gives $\lambda(\mathbb{Q}) = 0$.
  • The Cantor set $C$ is constructed by removing the middle third of $[0,1]$ repeatedly. It is closed, uncountable, has empty interior, and yet $\lambda(C) = 0$.
  • Smooth curves in $\mathbb{R}^2$ have 2-dimensional Lebesgue measure zero.

The Cantor Set

Definition 2.3.6 - Cantor Set

Define $C_0 = [0,1]$, $C_1 = [0,1/3] \cup [2/3,1]$, and in general $C_{n+1}$ is obtained from $C_n$ by removing the open middle third of each interval. Set $C = \bigcap_{n=0}^\infty C_n$.

$C$ is closed (intersection of closed sets), has no interior, is uncountable (homeomorphic to $\{0,1\}^\mathbb{N}$), and $\lambda(C) = \lim_{n \to \infty} (2/3)^n = 0$.

Non-Measurable Sets

Theorem 2.3.7 - Vitali Non-Measurable Set

Assuming the Axiom of Choice, there exists $V \subseteq [0,1]$ that is not Lebesgue measurable. The construction: partition $[0,1]$ into equivalence classes modulo $\mathbb{Q}$; by AC, choose one representative from each class. Any translation-invariant extension of $\lambda$ to $V$ leads to a contradiction: the interval $[0,2]$ would simultaneously have measure 0 and measure $\geq 1$ after decomposing $[0,1]$ into countably many translates of $V$ (modulo $\mathbb{Z}$).
