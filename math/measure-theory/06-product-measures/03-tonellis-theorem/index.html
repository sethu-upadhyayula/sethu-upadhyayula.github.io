6.3 Tonelli's Theorem

Tonelli's theorem is the non-negative analogue of Fubini's theorem — it requires no integrability hypothesis but only that $f \geq 0$. It is primarily used as a tool to verify that a function is integrable, after which Fubini's theorem applies.

Statement

Theorem 6.3.1 — Tonelli's Theorem

Let $(X, \mathcal{A}, \mu)$ and $(Y, \mathcal{B}, \nu)$ be $\sigma$-finite measure spaces. Let $f : X \times Y \to [0, \infty]$ be $(\mathcal{A} \otimes \mathcal{B})$-measurable and non-negative. Then:

  1. For $\mu$-a.e. $x$: $y \mapsto f(x,y)$ is $\mathcal{B}$-measurable.
  2. For $\nu$-a.e. $y$: $x \mapsto f(x,y)$ is $\mathcal{A}$-measurable.
  3. The iterated integrals equal the double integral (all possibly $+\infty$): $$\int_{X\times Y} f \, d(\mu\otimes\nu) = \int_X \left(\int_Y f(x,y) \, d\nu(y)\right) d\mu(x) = \int_Y \left(\int_X f(x,y) \, d\mu(x)\right) d\nu(y).$$

In particular, if any one of these three expressions is finite, then so are the other two, and $f \in L^1(\mu \otimes \nu)$.

Remark 6.3.2 — Tonelli vs. Fubini

The key difference:

  • Tonelli: $f \geq 0$, no integrability required; all three expressions are always equal (in $[0,\infty]$).
  • Fubini: $f \in L^1$; iterated integrals are finite and equal the double integral.

To use Fubini: first apply Tonelli to $|f|$ to check $\int|f| < \infty$, then apply Fubini to $f$.

Applications

Example 6.3.3 — Verifying Fubini Applies

Consider $f(x,y) = e^{-xy}\sin x$ on $X = Y = (0,\infty)$ with Lebesgue measure. We want $\int_0^\infty\!\!\int_0^\infty f\,dy\,dx$. First check integrability via Tonelli on $|f| = e^{-xy}|\sin x|$: $$\int_0^\infty \left(\int_0^\infty e^{-xy}|\sin x|\,dy\right)dx = \int_0^\infty \frac{|\sin x|}{x}\,dx = +\infty.$$ So $f \notin L^1((0,\infty)^2)$ and Fubini does not apply in the usual sense. However, for each fixed $a > 0$, restricting to $(0,a)\times(0,\infty)$ the function is $L^1$, and Fubini applies there. This illustrates the importance of checking Tonelli first.

Example 6.3.4 — Computing with Tonelli

For non-negative $f(x,y) = e^{-(x+y)}$ on $(0,\infty)^2$: $$\iint e^{-(x+y)}\,d(\lambda\otimes\lambda) = \int_0^\infty\!\int_0^\infty e^{-x}e^{-y}\,dy\,dx = \int_0^\infty e^{-x}\left(\int_0^\infty e^{-y}\,dy\right)dx = \int_0^\infty e^{-x} \cdot 1\,dx = 1.$$ The double integral is 1, which equals $(\int_0^\infty e^{-x}\,dx)^2 = 1^2 = 1$ as expected. Tonelli guarantees all three expressions equal 1.

Example 6.3.5 — Gaussian Integral via Tonelli

To compute $I = \int_{-\infty}^\infty e^{-x^2}\,dx$, write $I^2 = \int\!\int e^{-(x^2+y^2)}\,dx\,dy$. By Tonelli (integrand non-negative), convert to polar coordinates: $$I^2 = \int_0^{2\pi}\!\int_0^\infty e^{-r^2} r\,dr\,d\theta = 2\pi \cdot \frac{1}{2} = \pi.$$ So $I = \sqrt{\pi}$. Tonelli justifies the interchange and the polar coordinate change.

Corollary 6.3.6 — Fubini-Tonelli Theorem

Combining both: let $f$ be measurable on $X\times Y$ ($\sigma$-finite). If $\int_X\left(\int_Y |f(x,y)|\,d\nu\right)d\mu < \infty$ (or with the order of integration reversed), then $f \in L^1(\mu\otimes\nu)$ and all three integrals are equal. This combined statement is often what is applied in practice.
