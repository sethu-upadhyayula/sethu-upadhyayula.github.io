5.2 Hölder and Minkowski Inequalities

Two fundamental inequalities establish that $L^p$ spaces are normed vector spaces. Hölder's inequality bounds the integral of a product, and Minkowski's inequality is the triangle inequality for the $p$-norm. Both are proved using the elementary Young's inequality.

Young's Inequality

Lemma 5.2.1 - Young's Inequality

Let $p, q > 1$ with $\frac{1}{p} + \frac{1}{q} = 1$ (conjugate exponents). For any $a, b \geq 0$: $$ab \leq \frac{a^p}{p} + \frac{b^q}{q}.$$

Equality holds iff $a^p = b^q$. Proof: The function $t \mapsto \ln t$ is concave. Apply concavity to $\ln(ab) = \frac{1}{p}\ln(a^p) + \frac{1}{q}\ln(b^q) \leq \ln\!\left(\frac{a^p}{p} + \frac{b^q}{q}\right)$. Exponentiate. $\square$

Hölder's Inequality

Theorem 5.2.2 - Hölder's Inequality

Let $1 \leq p \leq \infty$ and $q$ its conjugate ($1/p + 1/q = 1$, with $q = \infty$ when $p = 1$ and vice versa). If $f \in L^p(\mu)$ and $g \in L^q(\mu)$, then $fg \in L^1(\mu)$ and: $$\int |fg| \, d\mu \leq \|f\|_p \|g\|_q.$$

Proof for $1 < p < \infty$

If $\|f\|_p = 0$ or $\|g\|_q = 0$, both sides are 0. Otherwise, set $\tilde{f} = f/\|f\|_p$ and $\tilde{g} = g/\|g\|_q$ so $\|\tilde{f}\|_p = \|\tilde{g}\|_q = 1$. By Young's inequality pointwise: $$|\tilde{f}(x)\tilde{g}(x)| \leq \frac{|\tilde{f}(x)|^p}{p} + \frac{|\tilde{g}(x)|^q}{q}.$$ Integrating: $\int |\tilde{f}\tilde{g}| \leq \frac{1}{p} + \frac{1}{q} = 1$, so $\int |fg| \leq \|f\|_p \|g\|_q$. $\square$

For $p = 1$, $q = \infty$: $|f(x)g(x)| \leq |f(x)| \cdot \|g\|_\infty$ a.e., integrating gives $\int|fg| \leq \|f\|_1 \|g\|_\infty$.

Corollary 5.2.3 - Cauchy-Schwarz Inequality

For $p = q = 2$: $\int |fg| \leq \|f\|_2 \|g\|_2$, or equivalently $|\langle f, g\rangle| \leq \|f\|_2 \|g\|_2$ where $\langle f,g\rangle = \int f g \, d\mu$. This is the Cauchy-Schwarz inequality for $L^2$.

Corollary 5.2.4 - Generalised Hölder

If $f_1 \in L^{p_1}, \ldots, f_n \in L^{p_n}$ with $\sum 1/p_i = 1$, then $\int |f_1 \cdots f_n| \leq \prod \|f_i\|_{p_i}$. Applied repeatedly.

Minkowski's Inequality

Theorem 5.2.5 - Minkowski's Inequality

For $1 \leq p \leq \infty$ and $f, g \in L^p(\mu)$: $$\|f + g\|_p \leq \|f\|_p + \|g\|_p.$$

This is the triangle inequality for the $p$-norm, establishing that $L^p$ is a normed vector space.

Proof for $1 < p < \infty$

Write $|f+g|^p = |f+g| \cdot |f+g|^{p-1} \leq (|f| + |g|)|f+g|^{p-1}$. Integrate: $$\|f+g\|_p^p \leq \int |f| |f+g|^{p-1} + \int |g| |f+g|^{p-1}.$$ Apply Hölder with exponents $p$ and $q = p/(p-1)$ to each term. Since $\|{|f+g|^{p-1}}\|_q = \|f+g\|_p^{p-1}$: $$\|f+g\|_p^p \leq (\|f\|_p + \|g\|_p) \|f+g\|_p^{p-1}.$$ Divide by $\|f+g\|_p^{p-1}$ (if nonzero) to get Minkowski's inequality. $\square$

Remark 5.2.6 - $p < 1$ and the Triangle Inequality

For $0 < p < 1$, $\|\cdot\|_p$ fails the triangle inequality. Instead, $d(f,g) = \int |f-g|^p \, d\mu$ is a metric (without the $p$-th root), making $L^p$ a complete metric space but not a normed space.

$L^p$ as a Normed Space

Theorem 5.2.7 - $L^p$ is a Normed Vector Space

For $1 \leq p \leq \infty$, $L^p(\mu)$ with the $p$-norm is a normed vector space. The norm axioms are:

  1. $\|f\|_p \geq 0$ with $\|f\|_p = 0 \Leftrightarrow f = 0$ a.e. (i.e., $f = 0$ in $L^p$).
  2. $\|\alpha f\|_p = |\alpha| \|f\|_p$ (homogeneity).
  3. $\|f+g\|_p \leq \|f\|_p + \|g\|_p$ (triangle inequality, Minkowski).

That $L^p$ is also complete (i.e., a Banach space) is the content of the Riesz-Fischer theorem in Section 5.3.
