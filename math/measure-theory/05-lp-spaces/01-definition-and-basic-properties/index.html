5.1 Definition and Basic Properties

We define the $L^p$ spaces rigorously as quotient spaces of measurable functions modulo equality almost everywhere, equipped with the $p$-norm.

The Space $L^p(\mu)$ for $1 \leq p < \infty$

Definition 5.1.1 — $L^p$ Space

Let $(X, \mathcal{M}, \mu)$ be a measure space and $1 \leq p < \infty$. Define: $$\mathcal{L}^p(\mu) = \left\{ f : X \to \mathbb{R} \text{ measurable} : \int_X |f|^p \, d\mu < \infty \right\}.$$

The $p$-norm of $f \in \mathcal{L}^p$ is $\|f\|_p = \left(\int |f|^p \, d\mu\right)^{1/p}$.

Since $\|f\|_p = 0$ iff $f = 0$ $\mu$-a.e. (not necessarily everywhere), $\|\cdot\|_p$ is only a seminorm on $\mathcal{L}^p$. We define the true $L^p$ space as the quotient: $$L^p(\mu) = \mathcal{L}^p(\mu) / \sim \quad \text{where } f \sim g \iff f = g \text{ $\mu$-a.e.}$$

Elements of $L^p(\mu)$ are equivalence classes of functions equal a.e. We typically write $f \in L^p$ for a representative, with the understanding that statements hold a.e.

Definition 5.1.2 — $L^\infty$ Space

The essential supremum of a measurable function $f$ is: $$\|f\|_\infty = \inf\{M \geq 0 : \mu(\{|f| > M\}) = 0\} = \text{ess}\sup |f|.$$

The space $L^\infty(\mu)$ consists of equivalence classes (mod a.e. equality) of measurable functions $f$ with $\|f\|_\infty < \infty$ (essentially bounded functions).

Basic Properties

Proposition 5.1.3 — $L^p$ is a Vector Space

For $1 \leq p \leq \infty$, $L^p(\mu)$ is a vector space over $\mathbb{R}$. In particular, if $f, g \in L^p$ and $\alpha, \beta \in \mathbb{R}$, then $\alpha f + \beta g \in L^p$.

Proof: For $1 \leq p < \infty$, use the inequality $(a+b)^p \leq 2^{p-1}(a^p + b^p)$ for $a,b \geq 0$ (convexity of $t^p$). Then $\|f + g\|_p^p \leq 2^{p-1}(\|f\|_p^p + \|g\|_p^p) < \infty$. For $L^\infty$: $\|f+g\|_\infty \leq \|f\|_\infty + \|g\|_\infty$. $\square$

Proposition 5.1.4 — Density of Simple Functions

For $1 \leq p < \infty$, the simple functions in $L^p(\mu)$ are dense: for every $f \in L^p$ and $\varepsilon > 0$, there exists a simple function $s$ with $\|f - s\|_p < \varepsilon$.

Proof: By the approximation theorem, there exist simple functions $s_n \to f$ pointwise with $|s_n| \leq |f|$. Then $|f - s_n|^p \leq (2|f|)^p \in L^1$, so DCT gives $\|f - s_n\|_p^p \to 0$. $\square$

Proposition 5.1.5 — Inclusions on Finite Measure Spaces

If $\mu(X) < \infty$ and $1 \leq p \leq q \leq \infty$, then $L^q(\mu) \subseteq L^p(\mu)$ and $\|f\|_p \leq \mu(X)^{1/p - 1/q} \|f\|_q$.

Proof: Apply Hölder's inequality to $|f|^p \cdot 1$ with exponents $q/p$ and $(q/p)'$. $\square$

Remark 5.1.6 — Inclusions on $\mathbb{N}$

For the counting measure on $\mathbb{N}$ (sequences): $\ell^p \subseteq \ell^q$ for $p \leq q$, i.e., the inclusions reverse compared to finite measure spaces. Here $\ell^p = \{(a_n) : \sum |a_n|^p < \infty\}$ with $\|(a_n)\|_p = (\sum |a_n|^p)^{1/p}$.

Examples

Example 5.1.7 — Power Functions on $(0,1]$

Let $f(x) = x^{-\alpha}$ on $(0,1]$ with Lebesgue measure. Then $\int_0^1 |f|^p \, dx = \int_0^1 x^{-\alpha p}\,dx$ converges iff $\alpha p < 1$, i.e., $\alpha < 1/p$. So $f \in L^p(0,1)$ iff $\alpha < 1/p$. In particular, $f \in L^1$ iff $\alpha < 1$, and $f \in L^2$ iff $\alpha < 1/2$. As $p \to \infty$, the condition becomes $\alpha \leq 0$, consistent with $f \in L^\infty$ iff $f$ is essentially bounded (which requires $\alpha \leq 0$ here).

Example 5.1.8 — Power Functions on $[1,\infty)$

Let $f(x) = x^{-\beta}$ on $[1,\infty)$. Then $\int_1^\infty x^{-\beta p}\,dx$ converges iff $\beta p > 1$, i.e., $\beta > 1/p$. So $f \in L^p(1,\infty)$ iff $\beta > 1/p$. Note: on $(0,1]$, higher $p$ is more restrictive; on $[1,\infty)$, lower $p$ is more restrictive. These "dual" behaviors reflect the different sources of failure of integrability (singularity at 0 vs. decay at $\infty$).
