4.4 Comparison with Riemann Integration

A natural question is: how does the Lebesgue integral relate to the Riemann integral one learns in calculus? The answer is that Lebesgue integration is a strict generalisation - every Riemann-integrable function is Lebesgue-integrable with the same value, but not vice versa.

Review of the Riemann Integral

Definition 4.4.1 - Riemann Integral

Let $f : [a,b] \to \mathbb{R}$ be bounded. For a partition $P = \{a = x_0 < x_1 < \cdots < x_n = b\}$, define: $$U(f, P) = \sum_{i=1}^n \sup_{x \in [x_{i-1}, x_i]} f(x) \cdot (x_i - x_{i-1}), \quad L(f, P) = \sum_{i=1}^n \inf_{x \in [x_{i-1}, x_i]} f(x) \cdot (x_i - x_{i-1}).$$

The upper and lower Riemann integrals are $\overline{\int} f = \inf_P U(f,P)$ and $\underline{\int} f = \sup_P L(f,P)$. $f$ is Riemann integrable if $\overline{\int} f = \underline{\int} f$, and this common value is $\int_a^b f(x)\,dx$.

Lebesgue's Criterion

Lebesgue gave a complete characterisation of Riemann-integrable functions in terms of continuity almost everywhere.

Theorem 4.4.2 - Lebesgue's Criterion for Riemann Integrability

A bounded function $f : [a,b] \to \mathbb{R}$ is Riemann integrable if and only if the set of its discontinuities has Lebesgue measure zero: $$\lambda(\{x \in [a,b] : f \text{ is discontinuous at } x\}) = 0.$$

Equivalently, $f$ is Riemann integrable iff $f$ is bounded and continuous $\lambda$-a.e. on $[a,b]$.

Proof Sketch

Define the oscillation of $f$ at $x$: $\omega_f(x) = \lim_{\delta\to 0} \sup_{|y-x|<\delta} |f(y) - f(x)|$. Then $f$ is continuous at $x$ iff $\omega_f(x) = 0$. The key fact is that $U(f,P) - L(f,P) = \sum_i \omega_f([x_{i-1},x_i]) \cdot \Delta x_i$, and $f$ is Riemann integrable iff this can be made arbitrarily small, which happens exactly when the set $\{x : \omega_f(x) \geq \varepsilon\}$ has measure zero for every $\varepsilon > 0$, i.e., the discontinuity set has measure zero. $\square$

Riemann Implies Lebesgue

Theorem 4.4.3 - Inclusion of Riemann in Lebesgue

If $f : [a,b] \to \mathbb{R}$ is Riemann integrable, then $f$ is Lebesgue measurable, $f \in L^1([a,b], \lambda)$, and: $$\int_a^b f(x) \, dx \text{ (Riemann)} = \int_{[a,b]} f \, d\lambda \text{ (Lebesgue)}.$$

This follows because by Lebesgue's criterion, $f$ is bounded and continuous a.e., hence measurable, and the Riemann sums converge to both integrals.

Lebesgue is Strictly More General

Example 4.4.4 - The Dirichlet Function

Define $f : [0,1] \to \mathbb{R}$ by $f(x) = \mathbf{1}_\mathbb{Q}(x)$ (1 on rationals, 0 on irrationals). This is the Dirichlet function.

Riemann integral: For any partition $P$, every subinterval contains both rationals and irrationals, so $\sup f = 1$ and $\inf f = 0$ on every subinterval. Thus $U(f,P) = 1$ and $L(f,P) = 0$ for all $P$. So $\overline{\int} f = 1 \neq 0 = \underline{\int} f$ and $f$ is not Riemann integrable.

Lebesgue integral: The set of discontinuities of $f$ is all of $[0,1]$, which has measure 1. So Lebesgue's criterion confirms it fails Riemann integrability. But $f = \mathbf{1}_{\mathbb{Q} \cap [0,1]}$ is measurable (countable set), and: $$\int_{[0,1]} f \, d\lambda = \lambda(\mathbb{Q} \cap [0,1]) = 0,$$ since countable sets have Lebesgue measure zero. Lebesgue handles this effortlessly.

Example 4.4.5 - Limit of Riemann-Integrable Functions

Enumerate the rationals in $[0,1]$ as $q_1, q_2, \ldots$ and define $f_n = \mathbf{1}_{\{q_1,\ldots,q_n\}}$. Each $f_n$ is Riemann integrable (finitely many discontinuities, all with measure zero) with $\int f_n = 0$. But $f_n \nearrow f = \mathbf{1}_\mathbb{Q}$, which is not Riemann integrable.

Lebesgue integration handles this via MCT: $\int f \, d\lambda = \lim \int f_n \, d\lambda = 0$. The Lebesgue integral is closed under monotone limits; the Riemann integral is not.

Improper Riemann Integrals

Remark 4.4.6 - Improper Integrals

The relationship between improper Riemann integrals and Lebesgue integrals requires care. If $f \geq 0$ and the improper Riemann integral $\int_a^\infty f\,dx$ converges, then $f$ is Lebesgue integrable and the values agree.

However, conditional convergence can fail. Consider $f(x) = \frac{\sin x}{x}$ on $(0,\infty)$. The improper Riemann integral $\int_0^\infty \frac{\sin x}{x}\,dx = \frac{\pi}{2}$ converges conditionally. But $\int_0^\infty \left|\frac{\sin x}{x}\right| dx = +\infty$, so $f \notin L^1(0,\infty)$ in the Lebesgue sense. Lebesgue integration requires absolute integrability; it does not handle conditionally convergent integrals directly.

Remark 4.4.7 - Summary

In short: Lebesgue $\supset$ Riemann (for bounded functions on bounded intervals, with agreement of values). Lebesgue allows: unbounded functions (if $\int|f| < \infty$), integrals over arbitrary measurable sets, interchange of limits (MCT, DCT), and a complete normed space ($L^1$) of integrable functions. These features make Lebesgue integration the foundation of modern analysis.
