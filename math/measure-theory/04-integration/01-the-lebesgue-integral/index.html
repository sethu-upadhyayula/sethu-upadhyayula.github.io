4.1 The Lebesgue Integral

The Lebesgue integral is defined in two steps: first for non-negative functions via a supremum over simple functions, then for general (signed) functions by decomposing $f = f^+ - f^-$. The approach ensures that the integral is well-defined for a far broader class of functions than the Riemann integral.

Integral of Non-Negative Functions

Definition 4.1.1 — Lebesgue Integral for $f \geq 0$

Let $(X, \mathcal{M}, \mu)$ be a measure space and $f : X \to [0, \infty]$ a non-negative measurable function. The Lebesgue integral of $f$ with respect to $\mu$ is: $$\int f \, d\mu = \int_X f \, d\mu = \sup\left\{\int s \, d\mu : s \text{ simple}, 0 \leq s \leq f\right\} \in [0, \infty].$$

For a measurable set $A \in \mathcal{M}$, $\int_A f \, d\mu = \int f \cdot \mathbf{1}_A \, d\mu$.

Remark — Consistency with Simple Functions

If $f = s$ is itself a simple function, both definitions agree. Moreover, by the approximation theorem, the supremum is achieved as a limit: if $s_n \nearrow f$, then $\int s_n \nearrow \int f$ (this will be the Monotone Convergence Theorem).

General Integral

Definition 4.1.2 — Lebesgue Integral for General $f$

For a measurable function $f : X \to [-\infty, \infty]$, decompose $f = f^+ - f^-$ where $f^+(x) = \max(f(x), 0) \geq 0$ and $f^-(x) = \max(-f(x), 0) \geq 0$. Both $f^+$ and $f^-$ are non-negative measurable.

Define $\int f \, d\mu = \int f^+ \, d\mu - \int f^- \, d\mu$, provided this is not of the form $\infty - \infty$.

$f$ is integrable (or $f \in L^1(X, \mu)$) if $\int |f| \, d\mu = \int f^+ \, d\mu + \int f^- \, d\mu < \infty$, equivalently both $\int f^+$ and $\int f^-$ are finite.

Properties

Theorem 4.1.3 — Basic Properties

For integrable $f, g$ and $a, b \in \mathbb{R}$:

  1. Linearity: $\int (af + bg) \, d\mu = a \int f \, d\mu + b \int g \, d\mu$.
  2. Monotonicity: if $f \leq g$ a.e., then $\int f \, d\mu \leq \int g \, d\mu$.
  3. Triangle inequality: $\left|\int f \, d\mu\right| \leq \int |f| \, d\mu$.
  4. Null sets: if $\mu(A) = 0$ and $f \geq 0$, then $\int_A f \, d\mu = 0$. Conversely, if $\int f \, d\mu = 0$ and $f \geq 0$, then $f = 0$ a.e. (assuming $\mu$ is complete or $f$ is measurable).
  5. Almost everywhere equality: if $f = g$ a.e., then $\int f \, d\mu = \int g \, d\mu$.
Example 4.1.4 — Dirichlet Function

The Dirichlet function $f = \mathbf{1}_\mathbb{Q}$ (indicator of the rationals) is not Riemann integrable on $[0,1]$ (upper Riemann sum = 1, lower Riemann sum = 0). However, it is Lebesgue integrable: $\int_0^1 \mathbf{1}_\mathbb{Q} \, d\lambda = \lambda(\mathbb{Q} \cap [0,1]) = 0$, since $\mathbb{Q}$ is countable.

Remark — Notation

On $(\mathbb{R}, \mathcal{L}, \lambda)$, the Lebesgue integral is often written $\int f(x)\,dx$ or $\int_a^b f(x)\,dx$ for the integral over $[a,b]$. These coincide with the Riemann integral when both are defined.
