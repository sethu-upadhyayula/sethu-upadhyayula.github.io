5.2 Multiple Regression

Multiple regression extends simple linear regression to $p$ predictors, allowing the modelling of complex relationships while controlling for confounding variables. The matrix formulation unifies estimation and inference elegantly.

Definition 5.2.1 — Multiple Regression Model

With $n$ observations and $p$ predictors (plus intercept), the model is $$\mathbf{Y} = \mathbf{X}\boldsymbol{\beta} + \boldsymbol{\varepsilon}, \quad \boldsymbol{\varepsilon} \sim N(\mathbf{0}, \sigma^2 \mathbf{I}_n),$$ where $\mathbf{Y} \in \mathbb{R}^n$, $\mathbf{X} \in \mathbb{R}^{n\times(p+1)}$ is the design matrix (first column all 1s), $\boldsymbol{\beta} \in \mathbb{R}^{p+1}$, and $\mathbf{I}_n$ is the $n\times n$ identity matrix. We assume $\mathbf{X}$ has full column rank $p+1$.

Theorem 5.2.2 — OLS Estimator

Minimising $\|\mathbf{Y} - \mathbf{X}\boldsymbol{\beta}\|^2$ yields the unique OLS estimator $$\hat{\boldsymbol{\beta}} = (\mathbf{X}^\top \mathbf{X})^{-1}\mathbf{X}^\top \mathbf{Y}.$$

This is the BLUE (Gauss-Markov). Under normality, $\hat{\boldsymbol{\beta}} \sim N(\boldsymbol{\beta},\, \sigma^2(\mathbf{X}^\top\mathbf{X})^{-1})$. The fitted values are $\hat{\mathbf{Y}} = \mathbf{H}\mathbf{Y}$ where the hat matrix $\mathbf{H} = \mathbf{X}(\mathbf{X}^\top\mathbf{X})^{-1}\mathbf{X}^\top$ is the orthogonal projection onto the column space of $\mathbf{X}$.

Proposition 5.2.3 — Properties of the Hat Matrix

$\mathbf{H}$ is symmetric ($\mathbf{H}^\top = \mathbf{H}$) and idempotent ($\mathbf{H}^2 = \mathbf{H}$) with $\mathrm{rank}(\mathbf{H}) = p+1$. The residuals are $\hat{\boldsymbol{\varepsilon}} = (\mathbf{I}-\mathbf{H})\mathbf{Y}$, and $\mathrm{SSE} = \hat{\boldsymbol{\varepsilon}}^\top\hat{\boldsymbol{\varepsilon}}$ with $E[\mathrm{SSE}] = (n-p-1)\sigma^2$, giving unbiased estimator $\hat{\sigma}^2 = \mathrm{SSE}/(n-p-1)$.

$F$-Test for Overall Significance

Theorem 5.2.4 — Global $F$-Test

Test $H_0:\beta_1=\cdots=\beta_p=0$ (intercept-only model). Under $H_0$: $$F = \frac{\mathrm{SSR}/p}{\mathrm{SSE}/(n-p-1)} \sim F_{p,\,n-p-1}.$$

The adjusted $R^2$ corrects for the number of predictors: $$R^2_{\mathrm{adj}} = 1 - \frac{\mathrm{SSE}/(n-p-1)}{\mathrm{SST}/(n-1)}.$$ Unlike $R^2$, $R^2_{\mathrm{adj}}$ can decrease when an uninformative predictor is added.

Multicollinearity and VIF

Definition 5.2.5 — Variance Inflation Factor

Multicollinearity refers to near-linear dependencies among predictors. It inflates $\mathrm{Var}(\hat{\beta}_j)$ without biasing $\hat{\boldsymbol{\beta}}$. The Variance Inflation Factor for predictor $j$ is $\mathrm{VIF}_j = 1/(1-R_j^2)$, where $R_j^2$ is the $R^2$ from regressing $x_j$ on all other predictors. $\mathrm{VIF}_j > 10$ is often taken as indicating problematic collinearity.

Model Selection Criteria

Definition 5.2.6 — AIC and BIC

For a model with $p+1$ parameters fitted by maximum likelihood: $$\mathrm{AIC} = -2\ell(\hat{\boldsymbol{\theta}}) + 2(p+1), \qquad \mathrm{BIC} = -2\ell(\hat{\boldsymbol{\theta}}) + (p+1)\log n.$$ Lower values indicate a better bias-variance trade-off. BIC penalises complexity more heavily and is consistent (selects the true model as $n\to\infty$ if it is among the candidates), while AIC targets predictive performance.

Ridge Regression

Definition 5.2.7 — Ridge Estimator

When $\mathbf{X}^\top\mathbf{X}$ is ill-conditioned, the ridge estimator adds an $L_2$ penalty: $$\hat{\boldsymbol{\beta}}_{\mathrm{ridge}} = \underset{\boldsymbol{\beta}}{\arg\min}\left\{\|\mathbf{Y}-\mathbf{X}\boldsymbol{\beta}\|^2 + \lambda\|\boldsymbol{\beta}\|^2\right\} = (\mathbf{X}^\top\mathbf{X}+\lambda\mathbf{I})^{-1}\mathbf{X}^\top\mathbf{Y}, \quad \lambda > 0.$$

The ridge estimator is biased but can substantially reduce variance, improving mean squared error when predictors are highly correlated. The tuning parameter $\lambda$ is typically chosen by cross-validation. Ridge is equivalent to placing a $N(0,\sigma^2/\lambda)$ prior on each coefficient in a Bayesian framework.
