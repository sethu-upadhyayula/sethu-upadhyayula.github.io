4. Hypothesis Testing

Hypothesis testing provides a formal framework for making decisions from data. Given competing claims about an unknown parameter, we collect evidence and assess whether the data are consistent with a baseline (null) hypothesis or whether they compel us to favour an alternative.

This chapter develops the foundational concepts of the Neyman-Pearson framework: formulating null and alternative hypotheses, quantifying error rates, and applying canonical tests to the most common inferential problems in statistics.

Chapter Contents

  • 4.1 Null and Alternative Hypotheses — $H_0$, $H_1$, test statistics, $p$-values, and the Neyman-Pearson lemma.
  • 4.2 Type I and Type II Errors — Error rates, power functions, sample size determination, and multiple testing corrections.
  • 4.3 Common Tests — $t$-tests, $\chi^2$ tests, $F$-tests, and one-way ANOVA.
