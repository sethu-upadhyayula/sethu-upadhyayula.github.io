4.2 Type I and Type II Errors

Any decision rule based on finite data is susceptible to two fundamental types of mistakes. Understanding and controlling these error probabilities is central to the design and interpretation of statistical tests.

Definition 4.2.1 — Error Types

For a test of $H_0$ vs.\ $H_1$:

  • A Type I error (false positive) occurs when $H_0$ is rejected even though $H_0$ is true. Its probability is $\alpha = P(\text{reject } H_0 \mid H_0 \text{ true})$.
  • A Type II error (false negative) occurs when $H_0$ is not rejected even though $H_1$ is true. Its probability is $\beta = P(\text{fail to reject } H_0 \mid H_1 \text{ true})$.
  • The power of the test is $1 - \beta = P(\text{reject } H_0 \mid H_1 \text{ true})$.

The Error-Power Trade-off

Proposition 4.2.2 — Trade-off Between $\alpha$ and $\beta$

For a fixed sample size $n$ and a fixed alternative $\theta_1$, reducing $\alpha$ (tightening the rejection region) necessarily increases $\beta$ (reduces power). To simultaneously decrease both $\alpha$ and $\beta$, one must increase $n$. The power function $\beta(\theta) = P_\theta(\text{reject } H_0)$ increases monotonically toward 1 as $|\theta - \theta_0| \to \infty$ for well-designed tests.

Sample Size Calculation

Theorem 4.2.3 — Sample Size for $Z$-Test

To detect a true mean $\mu_1 = \mu_0 + \delta$ ($\delta > 0$) in a one-sided $Z$-test at level $\alpha$ with power $1 - \beta$, we need $$n = \left(\frac{(z_\alpha + z_\beta)\,\sigma}{\delta}\right)^2,$$

where $z_\alpha, z_\beta$ are standard normal upper quantiles. For a two-sided test at level $\alpha$ detecting $|\delta|$, replace $z_\alpha$ with $z_{\alpha/2}$. The effect size $\delta/\sigma$ is the key driver: doubling the required effect size reduces $n$ by a factor of 4.

Multiple Testing

When $m$ tests are conducted simultaneously, the probability of at least one false rejection inflates dramatically under per-test significance level $\alpha$.

Definition 4.2.4 — Family-Wise Error Rate (FWER)

The family-wise error rate is $\mathrm{FWER} = P(\text{at least one Type I error among all } m \text{ tests})$. Under independence, $\mathrm{FWER} = 1 - (1-\alpha)^m \to 1$ as $m \to \infty$. The Bonferroni correction controls FWER $\le \alpha$ by testing each hypothesis at level $\alpha/m$: reject $H_{0j}$ if $p_j \le \alpha/m$. This is conservative (FWER can be well below $\alpha$) and has low power when $m$ is large.

Definition 4.2.5 — False Discovery Rate (FDR)

The FDR is the expected proportion of rejected nulls that are true nulls: $$\mathrm{FDR} = E\!\left[\frac{V}{R \vee 1}\right],$$ where $V$ = number of false rejections and $R$ = total rejections.

The Benjamini-Hochberg procedure controls FDR $\le \alpha$: order the $p$-values $p_{(1)} \le p_{(2)} \le \cdots \le p_{(m)}$, let $k = \max\{j : p_{(j)} \le j\alpha/m\}$, and reject $H_{0(1)},\ldots,H_{0(k)}$. FDR control is less stringent than FWER control and hence more powerful for large $m$, making it the standard in genomics and large-scale inference.

Remark 4.2.6 — ROC Curves

A receiver operating characteristic (ROC) curve plots power $= 1 - \beta$ (true positive rate) against $\alpha$ (false positive rate) as the threshold varies. The area under the ROC curve (AUC) summarises overall discriminatory ability: $\mathrm{AUC} = 0.5$ corresponds to random guessing, $\mathrm{AUC} = 1$ to perfect separation. The Neyman-Pearson lemma guarantees that the likelihood ratio test traces out the optimal ROC curve — no other test achieves higher power for any given level.
