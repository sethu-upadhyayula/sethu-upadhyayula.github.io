1.1 Separable Equations

A separable ODE is one in which the variables $x$ and $y$ can be algebraically separated to opposite sides of the equation. This is the most elementary solution technique and applies to a surprisingly wide class of problems in science and engineering.

Definition 1.1.1 — Separable ODE

An ODE is called separable if it can be written in the form $$\frac{dy}{dx} = f(x)\,g(y)$$ for some functions $f$ of $x$ alone and $g$ of $y$ alone.

Method of Separation

Assuming $g(y) \neq 0$, divide both sides by $g(y)$ and multiply by $dx$:

$$\frac{dy}{g(y)} = f(x)\,dx.$$

Integrate both sides:

$$\int \frac{dy}{g(y)} = \int f(x)\,dx + C.$$

The constant of integration $C$ is determined by an initial condition $y(x_0) = y_0$. Note that the points where $g(y) = 0$ give constant (equilibrium) solutions $y = $ const., which must be checked separately.

Theorem 1.1.2 — Solution of Separable ODE

If $f$ is continuous on an interval $I$ and $g$ is continuous and nonzero on an interval $J$, then for any $(x_0, y_0) \in I \times J$, the IVP $y' = f(x)g(y)$, $y(x_0)=y_0$ has a unique solution given implicitly by $G(y) = F(x) + C$, where $G' = 1/g$ and $F' = f$, and $C = G(y_0) - F(x_0)$.

Example 1.1.3 — Exponential Growth

Solve $\frac{dy}{dx} = ky$ with $y(0) = y_0$, where $k$ is a constant.

Separate: $\frac{dy}{y} = k\,dx$. Integrate: $\ln|y| = kx + C_1$, so $|y| = e^{C_1}e^{kx}$. Writing $A = \pm e^{C_1}$, the general solution is $y = Ae^{kx}$.

Applying $y(0) = y_0$ gives $A = y_0$, so $y(x) = y_0 e^{kx}$. For $k > 0$ this models exponential growth; for $k < 0$ it models radioactive decay.

Example 1.1.4 — The Logistic Equation

The logistic model for population growth is $$\frac{dP}{dt} = rP\!\left(1 - \frac{P}{K}\right),$$ where $r > 0$ is the intrinsic growth rate and $K > 0$ is the carrying capacity.

Separate and use partial fractions: $$\frac{dP}{P(1-P/K)} = r\,dt \implies \int\left(\frac{1}{P} + \frac{1/K}{1-P/K}\right)dP = rt + C.$$ Integrating: $\ln|P| - \ln|1-P/K| = rt + C$, which gives $$\ln\!\left|\frac{P}{K-P}\right| = rt + C.$$

Solving for $P$ with $P(0) = P_0$: $$P(t) = \frac{K P_0}{P_0 + (K - P_0)e^{-rt}}.$$ As $t\to\infty$, $P(t)\to K$ regardless of the initial condition $0 < P_0 < K$: the population always approaches the carrying capacity.

Remark — Equilibrium Solutions

In the logistic equation, $g(P) = rP(1-P/K) = 0$ at $P = 0$ and $P = K$. These are the equilibrium solutions, valid for all $t$. Note that $P = K$ is a stable equilibrium (nearby solutions tend toward it) while $P = 0$ is unstable (nearby solutions move away from it).

Example 1.1.5 — Bernoulli Substitution

The ODE $\frac{dy}{dx} = x\sqrt{1-y^2}$ is separable. Separating: $$\frac{dy}{\sqrt{1-y^2}} = x\,dx \implies \arcsin y = \frac{x^2}{2} + C.$$

So $y = \sin\!\left(\frac{x^2}{2} + C\right)$. The constant solutions $y = \pm 1$ also satisfy the ODE (check: LHS $= 0 =$ RHS when $y = \pm 1$).
