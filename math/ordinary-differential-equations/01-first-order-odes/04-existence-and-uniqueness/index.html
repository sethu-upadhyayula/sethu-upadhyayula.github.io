1.4 Existence and Uniqueness

Before solving an IVP we should ask: does a solution exist, and is it unique? The Picard-Lindelöf theorem gives definitive affirmative answers under mild regularity assumptions. Its proof constructs the solution explicitly via successive approximations, known as Picard iteration.

Definition 1.4.1 - Lipschitz Condition

A function $f(x,y)$ satisfies a Lipschitz condition in $y$ on a set $D$ if there is a constant $L > 0$ such that $$|f(x,y_1) - f(x,y_2)| \leq L|y_1 - y_2|$$ for all $(x,y_1), (x,y_2) \in D$. The constant $L$ is the Lipschitz constant. If $\partial f/\partial y$ exists and is bounded on $D$, then $f$ is Lipschitz with $L = \sup_D |\partial f/\partial y|$.

Theorem 1.4.2 - Picard-Lindelöf

Let $f(x,y)$ be continuous on a rectangle $R = \{|x-x_0| \leq a,\, |y-y_0| \leq b\}$ and Lipschitz in $y$ on $R$ with constant $L$. Let $M = \max_R |f|$ and $h = \min(a, b/M)$.

Then the IVP $y' = f(x,y)$, $y(x_0) = y_0$ has a unique solution $y(x)$ on $|x - x_0| \leq h$.

Picard Iteration

The IVP $y' = f(x,y)$, $y(x_0)=y_0$ is equivalent to the integral equation $$y(x) = y_0 + \int_{x_0}^x f(t, y(t))\,dt.$$ Define the Picard iterates $\phi_0(x) = y_0$ and $$\phi_{n+1}(x) = y_0 + \int_{x_0}^x f(t, \phi_n(t))\,dt, \quad n = 0, 1, 2, \ldots$$

Theorem 1.4.3 - Convergence of Picard Iterates

Under the hypotheses of Theorem 1.4.2, the sequence $\{\phi_n\}$ converges uniformly on $|x-x_0| \leq h$ to the unique solution $y(x)$. Moreover, the error satisfies $$|\phi_n(x) - y(x)| \leq \frac{ML^n h^{n+1}}{(n+1)!} \to 0.$$

Example 1.4.4 - Picard Iteration for $y' = y$

Solve $y' = y$, $y(0) = 1$ by Picard iteration. Set $\phi_0 = 1$.

$\phi_1(x) = 1 + \int_0^x 1\,dt = 1 + x.$

$\phi_2(x) = 1 + \int_0^x (1+t)\,dt = 1 + x + x^2/2.$

$\phi_n(x) = \sum_{k=0}^n x^k/k!$, which converges to $e^x$ as $n\to\infty$.

The Picard iterates are the partial sums of the Taylor series of $e^x$.

Non-uniqueness Without Lipschitz

Example 1.4.5 - Non-uniqueness: $y' = y^{1/3}$

Consider $y' = y^{1/3}$, $y(0) = 0$. Here $f(x,y) = y^{1/3}$ is continuous but $\partial f/\partial y = \frac{1}{3}y^{-2/3}$ is unbounded near $y = 0$, so $f$ is not Lipschitz there.

One solution is $y \equiv 0$. Separating variables: $y^{-1/3}\,dy = dx$, giving $\frac{3}{2}y^{2/3} = x + C$, so $y = \left(\frac{2(x-c)}{3}\right)^{3/2}$ for any $c \geq 0$.

By patching: $y(x) = 0$ for $x \leq c$ and $y(x) = \left(\frac{2(x-c)}{3}\right)^{3/2}$ for $x > c$ is a valid solution for every $c \geq 0$. There are infinitely many solutions through $(0,0)$.

Remark - Peano's Theorem

Even without the Lipschitz condition, Peano's existence theorem guarantees at least one local solution whenever $f$ is merely continuous. Uniqueness is the property that requires Lipschitz continuity. Thus continuity alone gives existence but not uniqueness.
