1.2 Linear First-Order Equations

A first-order ODE is linear if it is linear in both $y$ and $y'$. Linear equations admit a complete, explicit solution via the method of integrating factors, making them one of the most tractable classes of ODEs.

Definition 1.2.1 — Linear First-Order ODE

A first-order linear ODE has the standard form $$y' + p(x)\,y = q(x),$$ where $p$ and $q$ are given functions of $x$. The equation is homogeneous if $q \equiv 0$ and nonhomogeneous otherwise.

The Integrating Factor Method

The key idea is to multiply both sides by a function $\mu(x)$ chosen so that the left-hand side becomes an exact derivative $\frac{d}{dx}[\mu(x) y]$.

Expanding $\frac{d}{dx}[\mu y] = \mu y' + \mu' y$. For this to equal $\mu(y' + py)$ we need $\mu' = \mu p$, i.e., $\mu'/\mu = p$. This separable equation gives:

$$\mu(x) = e^{\int p(x)\,dx}.$$
Theorem 1.2.2 — General Solution

Let $p, q$ be continuous on an interval $I$. Let $\mu(x) = e^{\int p(x)\,dx}$. Then the general solution of $y' + p(x)y = q(x)$ on $I$ is $$y(x) = \frac{1}{\mu(x)}\left[\int \mu(x)\,q(x)\,dx + C\right].$$

Every IVP $y(x_0) = y_0$ with $x_0 \in I$ has a unique solution on all of $I$.

Derivation

Multiply $y' + py = q$ by $\mu$: $\mu y' + \mu p y = \mu q$. The left side is $\frac{d}{dx}(\mu y)$. Integrating both sides:

$\mu y = \int \mu q\,dx + C$, so $y = \mu^{-1}\!\left[\int \mu q\,dx + C\right]$. $\square$

Example 1.2.3 — First-Order Linear Equation

Solve $y' - 2xy = x$ with $y(0) = 1$.

Here $p(x) = -2x$, so $\mu = e^{\int -2x\,dx} = e^{-x^2}$. Multiply through: $$\frac{d}{dx}\!\left[e^{-x^2} y\right] = x e^{-x^2}.$$ Integrate: $e^{-x^2} y = \int x e^{-x^2}\,dx = -\tfrac{1}{2}e^{-x^2} + C$.

So $y = -\tfrac{1}{2} + Ce^{x^2}$. Applying $y(0) = 1$: $1 = -\tfrac{1}{2} + C$, giving $C = \tfrac{3}{2}$. The solution is $y = \tfrac{3}{2}e^{x^2} - \tfrac{1}{2}$.

Application: RC Circuit

Example 1.2.4 — RC Circuit

In a series RC circuit driven by voltage $V(t)$, Kirchhoff's voltage law gives $$R\,\frac{dq}{dt} + \frac{q}{C} = V(t),$$ where $q(t)$ is the charge on the capacitor, $R$ the resistance, and $C$ the capacitance. Dividing by $R$: $$q' + \frac{1}{RC}\,q = \frac{V(t)}{R}.$$ This is linear with $p = 1/(RC)$ and $\mu = e^{t/(RC)}$. For constant $V$: $$q(t) = CV + (q_0 - CV)e^{-t/(RC)},$$

where $q_0 = q(0)$. The steady state is $q_\infty = CV$ (full charge), approached exponentially with time constant $\tau = RC$.

Remark — Superposition

The general solution $y = y_h + y_p$ is the sum of the homogeneous solution $y_h = Ce^{-\int p\,dx}$ and any particular solution $y_p$. This superposition principle is a hallmark of linear equations and extends to all orders.
