7.1 Two-Point Boundary Problems

A two-point boundary value problem (BVP) specifies conditions on a solution at two distinct points rather than prescribing all data at a single point. This seemingly small change has profound consequences for existence and uniqueness.

Problem Formulation

Definition 7.1.1 — Two-Point BVP

A two-point boundary value problem consists of the ODE $$y'' = f(x, y, y'), \quad a \leq x \leq b,$$ together with boundary conditions $y(a) = \alpha$ and $y(b) = \beta$ (Dirichlet conditions). More generally, boundary conditions may involve $y'$, e.g. $y'(a) = \alpha$ (Neumann) or $y(a) + \gamma y'(a) = \alpha$ (Robin). Unlike IVPs, the BVP may have zero, one, or infinitely many solutions.

Example 7.1.2 — Non-uniqueness and Non-existence

Consider $y'' + y = 0$, $y(0) = 0$, $y(\pi) = 0$. The general solution is $y = A\sin x + B\cos x$. Applying $y(0)=0$: $B=0$, so $y = A\sin x$. Then $y(\pi) = 0$ for every $A$ — infinitely many solutions. Changing the right endpoint: $y'' + y = 0$, $y(0)=0$, $y(\pi) = 1$ has no solution (since $y(\pi) = 0$ for all solutions satisfying $y(0)=0$).

Contrast with the IVP $y''+y=0$, $y(0)=0$, $y'(0)=1$, which has the unique solution $y=\sin x$.

Shooting Method

Definition 7.1.3 — Shooting Method

Convert the BVP $y'' = f(x,y,y')$, $y(a)=\alpha$, $y(b)=\beta$ into a root-finding problem. For a parameter $s$ (the initial slope), let $y(x;s)$ solve the IVP $$y'' = f(x,y,y'), \quad y(a)=\alpha,\quad y'(a) = s.$$ Define the shooting function $\phi(s) = y(b;s) - \beta$. We seek $s^*$ with $\phi(s^*)=0$.

Secant method update: Given iterates $s_k, s_{k-1}$: $$s_{k+1} = s_k - \phi(s_k)\,\frac{s_k - s_{k-1}}{\phi(s_k) - \phi(s_{k-1})}.$$

For linear BVPs, $\phi(s)$ is affine in $s$, so one step suffices: compute $y(x;s_1)$ and $y(x;s_2)$, then $s^* = s_1 - \phi(s_1)(s_2-s_1)/(\phi(s_2)-\phi(s_1))$.

Finite Difference Discretisation

Definition 7.1.4 — Finite Difference Method

Discretise $[a,b]$ with a uniform grid $x_i = a + ih$, $h = (b-a)/N$, $i = 0, 1, \ldots, N$. Replace $y''$ by the second-order central difference: $$y''(x_i) \approx \frac{y_{i-1} - 2y_i + y_{i+1}}{h^2}.$$ For the linear BVP $y'' + p(x)y' + q(x)y = r(x)$, $y_0=\alpha$, $y_N=\beta$, this gives a tridiagonal linear system for the unknowns $y_1, \ldots, y_{N-1}$: $$\frac{y_{i-1} - 2y_i + y_{i+1}}{h^2} + p_i \frac{y_{i+1}-y_{i-1}}{2h} + q_i y_i = r_i, \quad i=1,\ldots,N-1.$$ In matrix form $T\mathbf{y} = \mathbf{r}$ where $T$ is tridiagonal (Thomas algorithm solves this in $O(N)$ operations). The method has global truncation error $O(h^2)$.

Green's Function

Definition 7.1.5 — Green's Function

For the linear BVP $\mathcal{L}[y] \equiv y'' + p(x)y' + q(x)y = f(x)$, $y(a)=0$, $y(b)=0$, the Green's function $G(x,\xi)$ is defined by $\mathcal{L}_x G(x,\xi) = \delta(x-\xi)$ with homogeneous BCs. The solution is then $$y(x) = \int_a^b G(x,\xi)\,f(\xi)\,d\xi.$$

Explicitly, if $y_1, y_2$ are solutions of the homogeneous equation $\mathcal{L}[y]=0$ satisfying $y_1(a)=0$ and $y_2(b)=0$ respectively, and $W = y_1 y_2' - y_1' y_2$ is their (constant, up to $p$-factor) Wronskian, then: $$G(x,\xi) = \begin{cases} \dfrac{y_1(x)\,y_2(\xi)}{W(\xi)}, & a \leq x \leq \xi,\\[6pt] \dfrac{y_1(\xi)\,y_2(x)}{W(\xi)}, & \xi \leq x \leq b.\end{cases}$$

$G$ is symmetric: $G(x,\xi)=G(\xi,x)$ (for self-adjoint $\mathcal{L}$). Green's functions exist precisely when $\mathcal{L}[y]=0$ has only the trivial solution satisfying the BCs, i.e., when the associated homogeneous BVP has no non-trivial solution.

Example 7.1.6 — Green's Function for $-y'' = f$

Solve $-y'' = f(x)$, $y(0)=0$, $y(1)=0$. Homogeneous solutions: $y_1 = x$ (satisfies $y_1(0)=0$) and $y_2 = 1-x$ (satisfies $y_2(1)=0$). Wronskian: $W = y_1 y_2' - y_1' y_2 = x(-1) - (1)(1-x) = -x - 1 + x = -1$. So $W = -1$. $$G(x,\xi) = \begin{cases} \dfrac{x(1-\xi)}{1}, & 0 \leq x \leq \xi,\\[6pt] \dfrac{\xi(1-x)}{1}, & \xi \leq x \leq 1.\end{cases}$$ (Here the sign from $-y''=f$ is absorbed.) The solution is $y(x) = \int_0^1 G(x,\xi)f(\xi)\,d\xi.$

Verification: $-G_{xx} = \delta(x-\xi)$. For $x\neq\xi$, $G_{xx}=0$; the jump condition $[G_x]_{x=\xi} = G_x(\xi^+,\xi) - G_x(\xi^-,\xi) = -(1-\xi) - \xi = -1$ gives $-[G_x]=-1\cdot(-1)=1=\delta$-coefficient ✓.
