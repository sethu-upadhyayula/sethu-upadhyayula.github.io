2.1 Constant Coefficient Equations

The equation $ay'' + by' + cy = g(x)$ with constant $a,b,c$ is the workhorse of classical physics. The homogeneous version is solved completely by the characteristic equation, and non-homogeneous versions by undetermined coefficients or variation of parameters.

Definition 2.1.1 — Characteristic Equation

For $ay'' + by' + cy = 0$, we try $y = e^{rx}$. Substituting gives $e^{rx}(ar^2 + br + c) = 0$. Since $e^{rx} \neq 0$, the characteristic equation is $ar^2 + br + c = 0$ with roots $r = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$.

Three Cases

Case 1 — Two Distinct Real Roots

If $b^2 - 4ac > 0$, the roots $r_1 \neq r_2$ are real. The general solution is $y = c_1 e^{r_1 x} + c_2 e^{r_2 x}$. For the damped oscillator $my'' + cy' + ky = 0$, large damping $c^2 > 4mk$ gives two real roots and overdamped behavior.

Case 2 — Repeated Real Root

If $b^2 - 4ac = 0$, there is one root $r = -b/(2a)$. The general solution is $y = (c_1 + c_2 x)e^{rx}$. The second solution $xe^{rx}$ is found by reduction of order. This corresponds to critical damping.

Case 3 — Complex Conjugate Roots

If $b^2 - 4ac < 0$, the roots are $r = \alpha \pm \beta i$ where $\alpha = -b/(2a)$ and $\beta = \sqrt{4ac-b^2}/(2a)$. By Euler's formula $e^{(\alpha+\beta i)x} = e^{\alpha x}(\cos\beta x + i\sin\beta x)$, the real general solution is $y = e^{\alpha x}(c_1 \cos\beta x + c_2 \sin\beta x)$.

Example 2.1.2 — Harmonic Oscillator

Solve $y'' + 4y = 0$ with $y(0)=1$, $y'(0)=2$. Characteristic equation: $r^2 + 4 = 0$, roots $r = \pm 2i$. General solution: $y = c_1\cos 2x + c_2\sin 2x$.

$y(0) = c_1 = 1$; $y'(0) = 2c_2 = 2$ so $c_2 = 1$. Solution: $y = \cos 2x + \sin 2x = \sqrt{2}\sin(2x + \pi/4)$.

Undetermined Coefficients

For $ay'' + by' + cy = g(x)$ where $g$ is a polynomial, exponential, sine, cosine, or product thereof, we guess the form of $y_p$ based on the form of $g$, then determine coefficients by substitution.

Method of Undetermined Coefficients

If $g(x) = P_n(x)e^{\alpha x}$ (polynomial of degree $n$ times exponential), the trial solution is:

  • $y_p = x^s Q_n(x) e^{\alpha x}$ where $s = 0$ if $\alpha$ is not a characteristic root, $s = 1$ if $\alpha$ is a simple root, $s = 2$ if $\alpha$ is a repeated root.
  • For $g = P_n(x)e^{\alpha x}\cos\beta x$ or $g = P_n(x)e^{\alpha x}\sin\beta x$: use $y_p = x^s e^{\alpha x}[Q_n(x)\cos\beta x + R_n(x)\sin\beta x]$.
Example 2.1.3 — Non-homogeneous

Solve $y'' - 3y' + 2y = e^x$. Characteristic roots: $r=1,2$. Since $\alpha=1$ is a simple root, try $y_p = Axe^x$.

$y_p' = A(1+x)e^x$, $y_p'' = A(2+x)e^x$. Substituting: $A(2+x)e^x - 3A(1+x)e^x + 2Axe^x = Ae^x(2+x-3-3x+2x) = -Ae^x = e^x$.

So $A = -1$ and $y_p = -xe^x$. General solution: $y = c_1 e^x + c_2 e^{2x} - xe^x$.
