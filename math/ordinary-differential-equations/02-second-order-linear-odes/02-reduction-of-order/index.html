2.2 Reduction of Order

When one solution $y_1$ of a second-order linear ODE is known, a second linearly independent solution $y_2$ can always be found by reducing the problem to a first-order ODE. This technique, called reduction of order, is both theoretically important and practically useful for equations with non-constant coefficients.

The Method

Consider $y'' + p(x)y' + q(x)y = 0$ and suppose $y_1$ is a known nonzero solution. Let $y = v(x)y_1(x)$. Then:

$$y' = v'y_1 + vy_1', \quad y'' = v''y_1 + 2v'y_1' + vy_1''.$$

Substituting and using $y_1'' + py_1' + qy_1 = 0$:

$$y_1 v'' + (2y_1' + py_1)v' = 0.$$

Setting $w = v'$ gives the first-order linear equation $y_1 w' + (2y_1' + py_1)w = 0$.

Theorem 2.2.1 - Reduction of Order

If $y_1 \neq 0$ is a solution of $y'' + py' + qy = 0$, then $y_2 = y_1 \int \frac{e^{-\int p\,dx}}{y_1^2}\,dx$ is a second linearly independent solution, and $W[y_1,y_2] \neq 0$.

Example 2.2.2

Given $y_1 = x$ satisfies $x^2 y'' - xy' + y = 0$ (i.e., $y'' - (1/x)y' + (1/x^2)y = 0$), find $y_2$.

Here $p(x) = -1/x$. The formula gives: $$y_2 = x \int \frac{e^{-\int -1/x\,dx}}{x^2}\,dx = x\int\frac{e^{\ln x}}{x^2}\,dx = x\int\frac{x}{x^2}\,dx = x\int\frac{1}{x}\,dx = x\ln x.$$

Check: $y_2 = x\ln x$, $y_2' = 1 + \ln x$, $y_2'' = 1/x$. Substituting: $x^2(1/x) - x(1+\ln x) + x\ln x = x - x - x\ln x + x\ln x = 0$. Confirmed.

Abel's Formula and the Wronskian

Theorem 2.2.3 - Abel's Formula

If $y_1, y_2$ are any two solutions of $y'' + p(x)y' + q(x)y = 0$ on an interval $I$, then their Wronskian satisfies $$W[y_1,y_2](x) = W_0 \exp\!\left(-\int_{x_0}^x p(t)\,dt\right)$$ for some constant $W_0$. Consequently, $W$ is either identically zero (if $y_1, y_2$ are linearly dependent) or never zero on $I$ (if they are linearly independent).

Proof

Differentiate $W = y_1 y_2' - y_1' y_2$: $W' = y_1 y_2'' - y_1'' y_2$. Substituting $y_i'' = -py_i' - qy_i$: $W' = y_1(-py_2' - qy_2) - (-py_1' - qy_1)y_2 = -p(y_1 y_2' - y_1' y_2) = -pW.$

This gives $W'/W = -p$, so $W(x) = W_0 e^{-\int_{x_0}^x p}$. $\square$

Remark - Fundamental Solution Sets

Any pair $\{y_1, y_2\}$ with $W[y_1,y_2] \neq 0$ is called a fundamental solution set. Every solution of the homogeneous equation can be written as $y = c_1 y_1 + c_2 y_2$ for unique constants $c_1, c_2$. This two-dimensional structure is a consequence of the linearity and second-order nature of the equation.
