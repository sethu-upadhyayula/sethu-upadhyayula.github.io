2.3 Variation of Parameters

Variation of parameters is a general method for finding a particular solution to the nonhomogeneous equation $y'' + p(x)y' + q(x)y = g(x)$. Unlike undetermined coefficients, it works for any continuous $g$ and requires only that we know a fundamental solution set $\{y_1, y_2\}$ of the homogeneous equation.

Derivation

We seek $y_p = u_1(x)y_1(x) + u_2(x)y_2(x)$ where $u_1, u_2$ are functions to be determined. We impose the auxiliary condition $u_1'y_1 + u_2'y_2 = 0$ to simplify. Then:

$$y_p' = u_1 y_1' + u_2 y_2', \quad y_p'' = u_1 y_1'' + u_2 y_2'' + u_1' y_1' + u_2' y_2'.$$

Substituting into the ODE and using $y_i'' + py_i' + qy_i = 0$:

$$u_1' y_1' + u_2' y_2' = g.$$

Combined with the auxiliary condition, we have the linear system: $$\begin{pmatrix} y_1 & y_2 \\ y_1' & y_2' \end{pmatrix}\begin{pmatrix} u_1' \\ u_2' \end{pmatrix} = \begin{pmatrix} 0 \\ g \end{pmatrix}.$$ The determinant of the coefficient matrix is $W = W[y_1,y_2] \neq 0$, so by Cramer's rule:

Theorem 2.3.1 — Variation of Parameters Formulas

Let $\{y_1, y_2\}$ be a fundamental solution set of $y'' + py' + qy = 0$ with Wronskian $W$. A particular solution of $y'' + py' + qy = g$ is $y_p = u_1 y_1 + u_2 y_2$ where $$u_1'(x) = -\frac{y_2(x)\,g(x)}{W(x)}, \qquad u_2'(x) = \frac{y_1(x)\,g(x)}{W(x)}.$$

Integrating: $u_1 = -\int \frac{y_2 g}{W}\,dx$ and $u_2 = \int \frac{y_1 g}{W}\,dx$. The general solution is $y = c_1 y_1 + c_2 y_2 + y_p$.

Example 2.3.2 — $y'' + y = \sec x$

The homogeneous equation $y'' + y = 0$ has solutions $y_1 = \cos x$, $y_2 = \sin x$, and $W = 1$.

$u_1' = -\sin x \cdot \sec x = -\tan x$, so $u_1 = \ln|\cos x|$.

$u_2' = \cos x \cdot \sec x = 1$, so $u_2 = x$.

Therefore $y_p = \cos x \ln|\cos x| + x\sin x$, and the general solution is $y = c_1\cos x + c_2\sin x + \cos x\ln|\cos x| + x\sin x$. This result cannot be obtained by undetermined coefficients since $\sec x$ is not of the appropriate form.

Example 2.3.3 — $y'' + y = \tan x$

With $y_1 = \cos x$, $y_2 = \sin x$, $W = 1$, $g = \tan x$:

$u_1' = -\sin x\tan x = -\sin^2 x/\cos x$, integrated to $u_1 = \sin x - \ln|\sec x + \tan x|$.

$u_2' = \cos x\tan x = \sin x$, giving $u_2 = -\cos x$.

So $y_p = \cos x(\sin x - \ln|\sec x+\tan x|) - \cos x\sin x = -\cos x\ln|\sec x+\tan x|$. The general solution is $y = c_1\cos x + c_2\sin x - \cos x\ln|\sec x+\tan x|$.

Remark — Green's Function

The variation of parameters formula can be written as a convolution: $$y_p(x) = \int_{x_0}^x G(x,t)\,g(t)\,dt, \quad G(x,t) = \frac{y_1(t)y_2(x) - y_1(x)y_2(t)}{W(t)}.$$ The function $G(x,t)$ is the Green's function for the operator $L[y] = y'' + py' + qy$. It encodes the response to a unit impulse at $t$ and is fundamental to the study of boundary value problems.
