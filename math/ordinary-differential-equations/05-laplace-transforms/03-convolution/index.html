5.3 Convolution

Convolution is the operation that corresponds, under the Laplace transform, to multiplication of transforms. It provides a unified framework for writing the particular solution of any linear constant-coefficient ODE as an integral against the forcing function — the integral kernel being the impulse response of the system.

Convolution Integral

Definition 5.3.1 — Convolution

For functions $f, g: [0,\infty) \to \mathbb{R}$, the convolution $f * g$ is defined by $$(f * g)(t) = \int_0^t f(\tau)\,g(t - \tau)\,d\tau.$$ Convolution is commutative: $f*g = g*f$ (substitute $u = t-\tau$). It is also associative and distributes over addition, making the space of piecewise-continuous functions of exponential order a commutative algebra under convolution.

Convolution Theorem

Theorem 5.3.2 — Convolution Theorem

If $\mathcal{L}\{f\} = F$ and $\mathcal{L}\{g\} = G$ (both of exponential order $\alpha$), then for $\text{Re}(s) > \alpha$: $$\mathcal{L}\{f * g\}(s) = F(s)\,G(s).$$ Equivalently, $\mathcal{L}^{-1}\{F(s)G(s)\} = (f*g)(t)$.

Proof. Compute directly: $$\mathcal{L}\{f*g\}(s) = \int_0^\infty e^{-st}\!\int_0^t f(\tau)g(t-\tau)\,d\tau\,dt.$$ Switch the order of integration (Fubini, justified by absolute convergence for $\text{Re}(s) > \alpha$): integrate over the region $0 \leq \tau \leq t < \infty$, equivalent to $0 \leq \tau < \infty$ and $\tau \leq t < \infty$: $$= \int_0^\infty f(\tau)\!\int_\tau^\infty e^{-st}g(t-\tau)\,dt\,d\tau.$$ Substitute $u = t - \tau$: $$= \int_0^\infty f(\tau)e^{-s\tau}\!\int_0^\infty e^{-su}g(u)\,du\,d\tau = F(s)\,G(s). \qquad \square$$

The theorem is extraordinarily useful: products in $s$-space are difficult to invert directly but easy to recognise as convolutions.

Impulse Response and Transfer Function

Definition 5.3.3 — Transfer Function and Impulse Response

Consider a linear constant-coefficient ODE (with zero initial conditions) $$a_n y^{(n)} + \cdots + a_1 y' + a_0 y = f(t).$$ Taking the Laplace transform gives $P(s)Y(s) = F(s)$ where $P(s) = a_n s^n + \cdots + a_0$. The transfer function is $$H(s) = \frac{Y(s)}{F(s)} = \frac{1}{P(s)}.$$ The impulse response $h(t) = \mathcal{L}^{-1}\{H(s)\}$ is the output when $f(t) = \delta(t)$. By the convolution theorem, $$y(t) = (h * f)(t) = \int_0^t h(t-\tau)f(\tau)\,d\tau.$$

This expresses $y$ as a superposition of delayed copies of $h$, weighted by $f$. The impulse response fully characterises the zero-state response of a linear system to any input.

Volterra Integral Equations

Definition 5.3.4 — Volterra Integral Equation of the Second Kind

A Volterra integral equation of the second kind has the form $$y(t) = f(t) + \int_0^t K(t-\tau)\,y(\tau)\,d\tau,$$ i.e., $y = f + K*y$. Taking the Laplace transform: $Y = F + KY$ (where $K = \mathcal{L}\{K\}$), so $$Y(s) = \frac{F(s)}{1 - \mathcal{L}\{K\}(s)}, \quad \text{provided } \mathcal{L}\{K\}(s) \neq 1.$$ Inverting gives $y(t)$. The Laplace transform thus reduces a Volterra integral equation to an algebraic equation.

Worked Example

Example 5.3.5 — Using Convolution to Invert $1/(s(s^2+1))$

Find $\mathcal{L}^{-1}\!\left\{\dfrac{1}{s(s^2+1)}\right\}$ via the convolution theorem.

Write $\dfrac{1}{s(s^2+1)} = \dfrac{1}{s}\cdot\dfrac{1}{s^2+1}$. We have $f_1(t) = \mathcal{L}^{-1}\{1/s\} = 1$ and $f_2(t) = \mathcal{L}^{-1}\{1/(s^2+1)\} = \sin t$. Hence $$\mathcal{L}^{-1}\!\left\{\frac{1}{s(s^2+1)}\right\} = (1 * \sin)(t) = \int_0^t \sin\tau\,d\tau = [-\cos\tau]_0^t = 1 - \cos t.$$

This matches the partial-fraction result $\mathcal{L}^{-1}\!\left\{1/s - s/(s^2+1)\right\} = 1 - \cos t$, confirming consistency.

Example 5.3.6 — Volterra Integral Equation

Solve $y(t) = 1 + \int_0^t \sin(t-\tau)\,y(\tau)\,d\tau$.

This is $y = 1 + \sin * y$. Taking transforms: $Y = 1/s + Y/(s^2+1)$, so $$Y\!\left(1 - \frac{1}{s^2+1}\right) = \frac{1}{s}, \quad Y\cdot\frac{s^2}{s^2+1} = \frac{1}{s},$$ $$Y(s) = \frac{s^2+1}{s^3} = \frac{1}{s} + \frac{1}{s^3}.$$

Inverting: $y(t) = 1 + t^2/2$.
