5.1 Definition and Properties

The Laplace transform is an integral transform that maps a function of time $t \geq 0$ to a function of a complex frequency variable $s$. By trading differentiation in $t$ for multiplication by $s$, it converts linear constant-coefficient ODEs into algebraic equations that are often easy to solve and invert.

Definition and Region of Convergence

Definition 5.1.1 — Laplace Transform

Let $f:[0,\infty)\to\mathbb{R}$ be locally integrable. The Laplace transform of $f$ is $$\mathcal{L}\{f\}(s) = F(s) = \int_0^\infty e^{-st}f(t)\,dt,$$ defined for all $s \in \mathbb{C}$ for which the integral converges absolutely. The set of such $s$ is called the region of convergence (ROC).

Theorem 5.1.2 — Existence and ROC

If $f$ is piecewise continuous on $[0,\infty)$ and of exponential order $\alpha$, meaning $|f(t)| \leq Me^{\alpha t}$ for some $M > 0$ and all $t \geq T$, then $F(s)$ is defined and analytic for $\text{Re}(s) > \alpha$. The ROC is always a right half-plane $\{\text{Re}(s) > \alpha_0\}$ for some $\alpha_0 \in [-\infty, \infty)$, called the abscissa of convergence.

Linearity

Proposition 5.1.3 — Linearity

For constants $\alpha, \beta$ and functions $f, g$ of exponential order, $$\mathcal{L}\{\alpha f + \beta g\}(s) = \alpha F(s) + \beta G(s),$$ valid in the intersection of the ROCs of $f$ and $g$. Linearity follows directly from the linearity of the integral.

Transform of Derivatives

Theorem 5.1.4 — Derivative Rule

If $f, f', \ldots, f^{(n-1)}$ are continuous and $f^{(n)}$ is piecewise continuous, all of exponential order $\alpha$, then for $\text{Re}(s) > \alpha$: $$\mathcal{L}\{f^{(n)}\}(s) = s^n F(s) - s^{n-1}f(0) - s^{n-2}f'(0) - \cdots - f^{(n-1)}(0).$$ In particular, $$\mathcal{L}\{f'\}(s) = sF(s) - f(0), \qquad \mathcal{L}\{f''\}(s) = s^2 F(s) - sf(0) - f'(0).$$

Proof sketch. Integrate by parts: $\int_0^\infty e^{-st}f'(t)\,dt = [e^{-st}f(t)]_0^\infty + s\int_0^\infty e^{-st}f(t)\,dt = -f(0) + sF(s)$, using $e^{-st}f(t)\to 0$ as $t\to\infty$ when $\text{Re}(s) > \alpha$. Iterate for higher derivatives.

Remark — Initial Conditions

The derivative rule automatically incorporates the initial conditions $f(0), f'(0), \ldots$ into the algebraic equation for $F(s)$. This is the key advantage over classical undetermined-coefficient methods: there is no need to add a particular solution and then match initial data separately.

First Shift Theorem

Theorem 5.1.5 — First Shift Theorem (s-shifting)

If $\mathcal{L}\{f\}(s) = F(s)$ for $\text{Re}(s) > \alpha$, then $$\mathcal{L}\{e^{at}f(t)\}(s) = F(s - a), \quad \text{Re}(s) > \alpha + \text{Re}(a).$$ Proof. $\int_0^\infty e^{-st}e^{at}f(t)\,dt = \int_0^\infty e^{-(s-a)t}f(t)\,dt = F(s-a)$. Application: $\mathcal{L}\{e^{at}\cos(\omega t)\} = \dfrac{s-a}{(s-a)^2+\omega^2}$; $\mathcal{L}\{e^{at}\sin(\omega t)\} = \dfrac{\omega}{(s-a)^2+\omega^2}$.

Second Shift Theorem

Theorem 5.1.6 — Second Shift Theorem (t-shifting)

Let $H(t)$ denote the Heaviside unit step function, $H(t) = 0$ for $t < 0$ and $H(t) = 1$ for $t \geq 0$. If $\mathcal{L}\{f\}(s) = F(s)$, then for $a > 0$: $$\mathcal{L}\{H(t-a)\,f(t-a)\}(s) = e^{-as}F(s).$$

Proof. Substitute $u = t - a$: $\int_a^\infty e^{-st}f(t-a)\,dt = e^{-as}\int_0^\infty e^{-su}f(u)\,du = e^{-as}F(s)$. The second shift theorem handles forcing functions that switch on at time $t = a$.

Additional Properties

Proposition 5.1.8 — Multiplication by $t^n$

$$\mathcal{L}\{t^n f(t)\}(s) = (-1)^n \frac{d^n}{ds^n}F(s).$$ Proof. Differentiate $F(s) = \int_0^\infty e^{-st}f(t)\,dt$ under the integral sign: $F'(s) = -\int_0^\infty t e^{-st}f(t)\,dt = -\mathcal{L}\{tf\}(s)$. Iterate.

Proposition 5.1.9 — Division by $t$ (Integration in $s$)

If $\lim_{t\to 0^+} f(t)/t$ exists, then $$\mathcal{L}\!\left\{\frac{f(t)}{t}\right\}(s) = \int_s^\infty F(\sigma)\,d\sigma.$$ Example: $\mathcal{L}\!\left\{\dfrac{\sin t}{t}\right\}(s) = \int_s^\infty \dfrac{d\sigma}{\sigma^2+1} = \dfrac{\pi}{2} - \arctan s$.

Proposition 5.1.10 — Integration (Transform of $\int_0^t f$)

$$\mathcal{L}\!\left\{\int_0^t f(\tau)\,d\tau\right\}(s) = \frac{F(s)}{s}.$$ This follows from the derivative rule applied to $g(t) = \int_0^t f(\tau)\,d\tau$: $\mathcal{L}\{g'\} = sG(s)-g(0) = sG(s)$, so $G(s) = F(s)/s$.

Theorem 5.1.11 — Initial and Final Value Theorems

Provided the relevant limits exist:

$\bullet$ Initial value: $\lim_{t\to 0^+} f(t) = \lim_{s\to\infty} sF(s).$

$\bullet$ Final value: $\lim_{t\to\infty} f(t) = \lim_{s\to 0^+} sF(s),$ provided all poles of $sF(s)$ lie in $\text{Re}(s) < 0$. These theorems allow direct reading of limiting behaviour from $F(s)$ without inverting the transform.
