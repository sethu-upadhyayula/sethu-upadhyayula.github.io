6.2 Lyapunov Functions

Lyapunov's direct (or second) method proves stability without solving the ODE, by constructing a scalar "energy-like" function $V$ whose behaviour along trajectories certifies stability. The idea generalises the observation that if a physical system's energy is strictly decreasing, it must converge to a minimum.

Lyapunov Functions and Stability Theorems

Definition 6.2.1 - Positive Definite Function

A function $V: \Omega \to \mathbb{R}$ defined on an open neighbourhood $\Omega$ of $\mathbf{0}$ is positive definite if $V(\mathbf{0}) = 0$ and $V(\mathbf{x}) > 0$ for all $\mathbf{x} \in \Omega \setminus\{\mathbf{0}\}$. It is positive semi-definite if $V(\mathbf{x}) \geq 0$ everywhere. The orbital derivative (Lie derivative along $\mathbf{f}$) is $$\dot{V}(\mathbf{x}) = \nabla V(\mathbf{x}) \cdot \mathbf{f}(\mathbf{x}) = \sum_{i=1}^n \frac{\partial V}{\partial x_i}\,f_i(\mathbf{x}).$$ If $\mathbf{x}(t)$ is a solution, then $\frac{d}{dt}V(\mathbf{x}(t)) = \dot{V}(\mathbf{x}(t))$ by the chain rule.

Theorem 6.2.2 - Lyapunov Stability Theorem

Let $V: \Omega \to \mathbb{R}$ be $C^1$, positive definite on $\Omega \ni \mathbf{0}$.

  • If $\dot{V}(\mathbf{x}) \leq 0$ for all $\mathbf{x} \in \Omega$, then $\mathbf{0}$ is Lyapunov stable.
  • If $\dot{V}(\mathbf{x}) < 0$ for all $\mathbf{x} \in \Omega \setminus\{\mathbf{0}\}$, then $\mathbf{0}$ is asymptotically stable.

Such a $V$ is called a Lyapunov function. The sublevel sets $\{V \leq c\}$ are positively invariant when $\dot{V} \leq 0$, providing the $\delta$ for the $\varepsilon$-$\delta$ definition.

Theorem 6.2.3 - Global Asymptotic Stability

If $V: \mathbb{R}^n \to \mathbb{R}$ is $C^1$, positive definite, radially unbounded ($V(\mathbf{x}) \to \infty$ as $\|\mathbf{x}\| \to \infty$), and $\dot{V}(\mathbf{x}) < 0$ for all $\mathbf{x} \neq \mathbf{0}$, then $\mathbf{0}$ is globally asymptotically stable.

Chetaev Instability Theorem

Theorem 6.2.4 - Chetaev Instability

Suppose there exist a $C^1$ function $V$ and a region $U$ touching $\mathbf{0}$ on its boundary such that $V(\mathbf{0}) = 0$, $V(\mathbf{x}) > 0$ on $U$, and $\dot{V}(\mathbf{x}) > 0$ on $U$ (with $V = 0$ on $\partial U \cap \Omega$). Then $\mathbf{0}$ is unstable. Intuitively, trajectories starting in $U$ near $\mathbf{0}$ are pushed by the increasing $V$ toward $\partial U$ and hence away from $\mathbf{0}$.

Quadratic Lyapunov Functions and the Lyapunov Matrix Equation

Proposition 6.2.5 - Lyapunov Matrix Equation

For the linear system $\mathbf{x}' = A\mathbf{x}$, take $V(\mathbf{x}) = \mathbf{x}^\top P \mathbf{x}$ where $P$ is symmetric positive definite. Then $$\dot{V}(\mathbf{x}) = \mathbf{x}^\top(PA + A^\top P)\mathbf{x}.$$ We need $PA + A^\top P$ to be negative definite. Given any symmetric positive definite $Q$, the Lyapunov matrix equation $$PA + A^\top P = -Q$$ has a unique symmetric positive definite solution $P$ if and only if all eigenvalues of $A$ have strictly negative real part. Thus $V(\mathbf{x}) = \mathbf{x}^\top P \mathbf{x}$ is a quadratic Lyapunov function for $\mathbf{x}' = A\mathbf{x}$ whenever $A$ is Hurwitz (all eigenvalues in $\text{Re} < 0$).

The solution is given by the integral formula $P = \int_0^\infty e^{A^\top t}Q e^{At}\,dt$ (converges by the Hurwitz condition).

LaSalle's Invariance Principle

Theorem 6.2.6 - LaSalle's Invariance Principle

Let $\Omega_c = \{\mathbf{x}: V(\mathbf{x}) \leq c\}$ be a compact positively-invariant set for $\mathbf{x}' = \mathbf{f}(\mathbf{x})$. Suppose $\dot{V}(\mathbf{x}) \leq 0$ on $\Omega_c$. Define $$E = \{\mathbf{x} \in \Omega_c : \dot{V}(\mathbf{x}) = 0\}$$ and let $M$ be the largest positively invariant set contained in $E$. Then every solution starting in $\Omega_c$ converges to $M$ as $t \to \infty$.

LaSalle's principle extends asymptotic stability analysis to cases where $\dot{V} = 0$ on a set larger than $\{\mathbf{0}\}$ (so $\dot{V} < 0$ everywhere cannot hold), as long as no entire trajectory stays in the set $\{\dot{V}=0\}$ except $\mathbf{0}$.

Example: Damped Harmonic Oscillator

Example 6.2.7 - Damped Harmonic Oscillator

Consider $x'' + bx' + kx = 0$ with $b, k > 0$. In state form ($x_1 = x$, $x_2 = x'$): $$x_1' = x_2, \qquad x_2' = -kx_1 - bx_2.$$ Try the energy-like Lyapunov function $V(x_1, x_2) = \tfrac{1}{2}kx_1^2 + \tfrac{1}{2}x_2^2$ (potential + kinetic energy).

Compute: $$\dot{V} = kx_1 x_1' + x_2 x_2' = kx_1 x_2 + x_2(-kx_1 - bx_2) = -bx_2^2.$$ We have $\dot{V} \leq 0$ everywhere, but $\dot{V} = 0$ when $x_2 = 0$ (not just at the origin). By LaSalle: $E = \{x_2 = 0\}$. On $E$, $x_2 = 0$ implies $x_2' = 0$, so $-kx_1 - b \cdot 0 = 0 \Rightarrow x_1 = 0$. Thus the largest invariant set in $E$ is $M = \{(0,0)\}$, and all bounded solutions converge to the origin.

Since $V$ is radially unbounded and $\Omega_c$ is compact for every $c > 0$, the origin is globally asymptotically stable for all $b, k > 0$. This matches the eigenvalue criterion: the eigenvalues of $A = \bigl(\begin{smallmatrix}0&1\\-k&-b\end{smallmatrix}\bigr)$ are $\frac{-b \pm \sqrt{b^2-4k}}{2}$, both with negative real part.
