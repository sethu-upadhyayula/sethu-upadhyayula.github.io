3.2 Phase Plane Analysis

For a two-dimensional autonomous system $x' = f(x,y)$, $y' = g(x,y)$, the phase plane is the $(x,y)$-plane where each point represents the state of the system. A trajectory (orbit) is a curve traced by the solution $(x(t), y(t))$ as $t$ varies. Phase plane analysis reveals global behaviour without solving explicitly.

Definition 3.2.1 — Phase Portrait

The phase portrait of the system $(x', y') = (f(x,y), g(x,y))$ is the collection of all orbits in the phase plane, together with arrows indicating the direction of increasing $t$. Orbits are the level curves of $dy/dx = g(x,y)/f(x,y)$ (when $f \neq 0$).

Nullclines and Direction Fields

Definition 3.2.2 — Nullclines

The $x$-nullcline is the set $\{f(x,y) = 0\}$ where $x' = 0$ (trajectories cross vertically). The $y$-nullcline is $\{g(x,y) = 0\}$ where $y' = 0$ (trajectories cross horizontally). Their intersections are the equilibrium points.

The direction field assigns the vector $(f(x,y), g(x,y))$ at each point. Trajectories are tangent to the direction field everywhere. Between nullclines, $f$ and $g$ maintain constant sign, giving a consistent direction of motion in each region.

Classification of Equilibria for Linear Systems

For the linear system $\mathbf{x}' = A\mathbf{x}$ near $\mathbf{0}$, the type of equilibrium is determined entirely by the eigenvalues $\lambda_1, \lambda_2$ of $A$. Let $\tau = \text{tr}(A) = \lambda_1 + \lambda_2$ and $\delta = \det(A) = \lambda_1 \lambda_2$.

Theorem 3.2.3 — Classification by Eigenvalues

The equilibrium $\mathbf{0}$ of $\mathbf{x}' = A\mathbf{x}$ is classified as follows:

  • Saddle: $\delta < 0$, i.e., $\lambda_1 < 0 < \lambda_2$ (real, opposite signs). Unstable.
  • Unstable node: $\delta > 0$, $\tau > 0$, $\tau^2 - 4\delta \geq 0$ (both real, positive). Trajectories diverge.
  • Stable node: $\delta > 0$, $\tau < 0$, $\tau^2 - 4\delta \geq 0$ (both real, negative). Trajectories converge.
  • Unstable spiral: $\tau^2 - 4\delta < 0$, $\tau > 0$ (complex with positive real part). Spirals outward.
  • Stable spiral: $\tau^2 - 4\delta < 0$, $\tau < 0$ (complex with negative real part). Spirals inward.
  • Centre: $\tau = 0$, $\delta > 0$ (purely imaginary eigenvalues). Closed orbits (periodic solutions).

Saddle Points

For a saddle, $A$ has real eigenvalues $\lambda_1 < 0 < \lambda_2$ with eigenvectors $\mathbf{v}_1, \mathbf{v}_2$. The stable manifold (solutions approaching $\mathbf{0}$ as $t \to +\infty$) is the line through $\mathbf{v}_1$; the unstable manifold is the line through $\mathbf{v}_2$. All other trajectories approach the unstable manifold as $t \to +\infty$ and the stable manifold as $t \to -\infty$.

Nodes

For a stable node, both eigenvalues are negative. If $\lambda_1 < \lambda_2 < 0$, trajectories approach $\mathbf{0}$ tangent to the $\mathbf{v}_2$ direction (the slow eigendirection). For an improper (star) node where $\lambda_1 = \lambda_2$, trajectories approach radially if $A = \lambda I$, or tangent to the single eigenvector if $A$ has a non-trivial Jordan block.

Spirals and Centres

With complex eigenvalues $\lambda = \alpha \pm i\beta$, trajectories rotate. The angular speed is $\beta/r$ (approximately, near the origin) and the radius grows like $e^{\alpha t}$: inward if $\alpha < 0$, outward if $\alpha > 0$, and constant (closed ellipses) if $\alpha = 0$. The sense of rotation (clockwise or counterclockwise) is determined by the sign of the off-diagonal entries of $A$.

Example 3.2.4 — Simple Pendulum (Linearised)

The nonlinear pendulum equation $\theta'' + \sin\theta = 0$ becomes the system $x' = y$, $y' = -\sin x$. Nullclines: $y' = 0$ on $x = n\pi$, $x' = 0$ on $y = 0$.

Near $x = 0$: $\sin x \approx x$, giving the linear system with $A = \begin{pmatrix}0&1\\-1&0\end{pmatrix}$. Eigenvalues $\pm i$: this is a centre — the linearised pendulum has periodic orbits. Near $x = \pi$ (unstable equilibrium): $\sin x \approx -(x-\pi)$, giving $A = \begin{pmatrix}0&1\\1&0\end{pmatrix}$ with eigenvalues $\pm 1$ — a saddle.

Theorem 3.2.5 — Bendixson's Criterion

If $\partial f/\partial x + \partial g/\partial y$ is not identically zero and does not change sign in a simply connected region $D \subset \mathbb{R}^2$, then the system $x' = f$, $y' = g$ has no closed orbits in $D$.

Theorem 3.2.6 — Poincaré–Bendixson Theorem

If a trajectory of a planar autonomous system remains in a bounded closed region $D$ for all $t \geq t_0$, and $D$ contains only finitely many equilibria, then the $\omega$-limit set of the trajectory is either an equilibrium point, a closed orbit, or a heteroclinic/homoclinic orbit connecting equilibria.
