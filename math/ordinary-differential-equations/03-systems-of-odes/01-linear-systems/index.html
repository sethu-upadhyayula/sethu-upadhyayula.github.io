3.1 Linear Systems

A first-order linear system of ODEs has the form $\mathbf{x}' = A(t)\mathbf{x} + \mathbf{b}(t)$, where $\mathbf{x}(t) \in \mathbb{R}^n$ and $A(t)$ is an $n \times n$ matrix. When $A$ is constant and $\mathbf{b} = \mathbf{0}$, the system is autonomous homogeneous and admits a complete explicit solution via eigenvalue analysis.

Definition 3.1.1 — Linear Autonomous System

A linear autonomous system is $$\mathbf{x}' = A\mathbf{x}, \qquad \mathbf{x}(0) = \mathbf{x}_0,$$ where $A \in M_n(\mathbb{R})$ is a constant matrix. The solution is $\mathbf{x}(t) = e^{At}\mathbf{x}_0$.

Eigenvalue Method

We seek solutions of the form $\mathbf{x}(t) = e^{\lambda t}\mathbf{v}$ for some scalar $\lambda$ and vector $\mathbf{v} \neq \mathbf{0}$. Substituting gives $\lambda e^{\lambda t}\mathbf{v} = Ae^{\lambda t}\mathbf{v}$, so $A\mathbf{v} = \lambda\mathbf{v}$: $\lambda$ must be an eigenvalue and $\mathbf{v}$ an eigenvector of $A$.

Theorem 3.1.2 — General Solution (Distinct Eigenvalues)

If $A$ has $n$ distinct eigenvalues $\lambda_1, \ldots, \lambda_n$ with corresponding eigenvectors $\mathbf{v}_1, \ldots, \mathbf{v}_n$, then the general solution of $\mathbf{x}' = A\mathbf{x}$ is $$\mathbf{x}(t) = c_1 e^{\lambda_1 t}\mathbf{v}_1 + c_2 e^{\lambda_2 t}\mathbf{v}_2 + \cdots + c_n e^{\lambda_n t}\mathbf{v}_n,$$ where $c_1, \ldots, c_n$ are arbitrary constants determined by initial conditions.

Fundamental Matrix and Matrix Exponential

Definition 3.1.3 — Fundamental Matrix

A fundamental matrix $\Phi(t)$ for $\mathbf{x}' = A\mathbf{x}$ is an $n \times n$ matrix whose columns are $n$ linearly independent solutions. The principal fundamental matrix satisfies $\Phi(0) = I$ and equals the matrix exponential $e^{At}$.

Definition 3.1.4 — Matrix Exponential

For $A \in M_n(\mathbb{R})$, the matrix exponential is defined by the convergent power series $$e^{At} = \sum_{k=0}^{\infty} \frac{(At)^k}{k!} = I + At + \frac{A^2 t^2}{2!} + \frac{A^3 t^3}{3!} + \cdots$$ This series converges absolutely for all $t$ and all matrices $A$.

Theorem 3.1.5 — Properties of $e^{At}$

The matrix exponential satisfies:

  • $\frac{d}{dt}e^{At} = Ae^{At} = e^{At}A$
  • $e^{A \cdot 0} = I$
  • $(e^{At})^{-1} = e^{-At}$
  • If $AB = BA$, then $e^{(A+B)t} = e^{At}e^{Bt}$
  • If $A = PDP^{-1}$ (diagonalisable), then $e^{At} = Pe^{Dt}P^{-1}$

Complex Eigenvalues

If $A$ is real and $\lambda = \alpha + i\beta$ (with $\beta \neq 0$) is a complex eigenvalue with eigenvector $\mathbf{v} = \mathbf{a} + i\mathbf{b}$, then $\bar{\lambda} = \alpha - i\beta$ is also an eigenvalue with eigenvector $\bar{\mathbf{v}}$. The corresponding real solutions are:

$$\mathbf{x}_1(t) = e^{\alpha t}(\mathbf{a}\cos\beta t - \mathbf{b}\sin\beta t), \qquad \mathbf{x}_2(t) = e^{\alpha t}(\mathbf{a}\sin\beta t + \mathbf{b}\cos\beta t).$$

Repeated Eigenvalues and Jordan Form

When an eigenvalue $\lambda$ has algebraic multiplicity $m$ but geometric multiplicity $k < m$, there are not enough eigenvectors. We need generalised eigenvectors satisfying $(A - \lambda I)^j \mathbf{v} = \mathbf{0}$ for $j \leq m$.

Definition 3.1.6 — Generalised Eigenvector

A vector $\mathbf{v}_k$ is a generalised eigenvector of rank $k$ for eigenvalue $\lambda$ if $(A-\lambda I)^k \mathbf{v}_k = \mathbf{0}$ but $(A - \lambda I)^{k-1}\mathbf{v}_k \neq \mathbf{0}$. The chain $\mathbf{v}_1 = (A-\lambda I)\mathbf{v}_2, \ldots$ forms a Jordan chain.

Theorem 3.1.7 — Solution via Jordan Block

For a Jordan block $J = \lambda I + N$ (where $N$ is the nilpotent shift matrix), the matrix exponential is $$e^{Jt} = e^{\lambda t}\begin{pmatrix}1 & t & \frac{t^2}{2!} & \cdots \\ 0 & 1 & t & \cdots \\ \vdots & & \ddots & \\ 0 & \cdots & 0 & 1\end{pmatrix}.$$ This produces solutions of the form $e^{\lambda t}(c_0 + c_1 t + \cdots + c_{m-1}t^{m-1})$ multiplied by eigenvectors and generalised eigenvectors.

Example 3.1.8 — $2\times 2$ System with Repeated Eigenvalue

Consider $A = \begin{pmatrix}3 & 1 \\ 0 & 3\end{pmatrix}$. The only eigenvalue is $\lambda = 3$ with eigenvector $\mathbf{v}_1 = (1,0)^T$. To find the generalised eigenvector, solve $(A - 3I)\mathbf{v}_2 = \mathbf{v}_1$: $$\begin{pmatrix}0 & 1 \\ 0 & 0\end{pmatrix}\mathbf{v}_2 = \begin{pmatrix}1 \\ 0\end{pmatrix} \implies \mathbf{v}_2 = \begin{pmatrix}0 \\ 1\end{pmatrix}.$$

The general solution is $$\mathbf{x}(t) = e^{3t}\left[c_1\begin{pmatrix}1\\0\end{pmatrix} + c_2\left(t\begin{pmatrix}1\\0\end{pmatrix}+\begin{pmatrix}0\\1\end{pmatrix}\right)\right] = e^{3t}\begin{pmatrix}c_1 + c_2 t \\ c_2\end{pmatrix}.$$

Remark — Variation of Parameters for Non-Homogeneous Systems

For $\mathbf{x}' = A\mathbf{x} + \mathbf{b}(t)$, the solution with $\mathbf{x}(0) = \mathbf{x}_0$ is given by the variation of parameters formula: $$\mathbf{x}(t) = e^{At}\mathbf{x}_0 + \int_0^t e^{A(t-s)}\mathbf{b}(s)\,ds.$$ This generalises the scalar integrating factor formula to systems.
