9.1 Fixed Point Iteration

A fixed point of a function $g$ is a value $x^*$ satisfying $g(x^*) = x^*$. Many root-finding problems can be reformulated as fixed point problems: solving $f(x) = 0$ is equivalent to finding a fixed point of $g(x) = x - f(x)$ or any other equivalent rearrangement $g(x) = x$.

Definition - Fixed Point Iteration

Given a function $g$, the fixed point iteration generates a sequence by choosing an initial guess $x_0$ and repeatedly applying: $$x_{n+1} = g(x_n), \quad n = 0, 1, 2, \ldots$$ If the sequence $(x_n)$ converges to some $x^*$, and $g$ is continuous, then $x^* = g(x^*)$ is a fixed point of $g$.

Banach Contraction Mapping Theorem

The central theoretical result guaranteeing convergence of fixed point iteration is the Banach (or Picard) Contraction Mapping Theorem. It provides conditions under which the iteration converges to a unique fixed point regardless of the starting point.

Theorem - Banach Contraction Mapping

Let $g:[a,b]\to[a,b]$ be continuous, and suppose there exists a constant $k\in[0,1)$ such that $$|g'(x)| \le k \quad \text{for all } x \in (a,b).$$ Then:

  1. Existence and uniqueness: There is a unique fixed point $x^* \in [a,b]$.
  2. Global convergence: For any initial guess $x_0 \in [a,b]$, the iteration $x_{n+1} = g(x_n)$ converges to $x^*$.
  3. Error bound: $\displaystyle|x_n - x^*| \le \frac{k^n}{1-k}|x_1 - x_0|$ for all $n \ge 1$.
Proof Sketch

Step 1 - $(x_n)$ is Cauchy. By the Mean Value Theorem applied to $g$: $$|x_{n+1} - x_n| = |g(x_n) - g(x_{n-1})| \le k|x_n - x_{n-1}|.$$ By induction, $|x_{n+1} - x_n| \le k^n|x_1 - x_0|$. For $m > n$: $$|x_m - x_n| \le \sum_{j=n}^{m-1}|x_{j+1}-x_j| \le |x_1-x_0|\sum_{j=n}^{m-1}k^j \le \frac{k^n}{1-k}|x_1-x_0|.$$ Since $k < 1$, this tends to $0$ as $n \to \infty$, so $(x_n)$ is Cauchy, hence convergent in $[a,b]$.

Step 2 - The limit is a fixed point. Let $x^* = \lim x_n$. By continuity of $g$: $$g(x^*) = g(\lim_{n\to\infty} x_n) = \lim_{n\to\infty} g(x_n) = \lim_{n\to\infty} x_{n+1} = x^*.$$

Step 3 - Uniqueness. If $x^*$ and $y^*$ are both fixed points in $[a,b]$: $$|x^* - y^*| = |g(x^*) - g(y^*)| \le k|x^* - y^*|.$$ Since $k < 1$, we must have $|x^* - y^*| = 0$, i.e., $x^* = y^*$. $\square$

Remark - Contraction Constant

The condition $|g'(x)| \le k < 1$ is called the contraction condition. The smaller $k$ is, the faster the convergence: the error bound decays geometrically as $k^n$. A function satisfying this is called a contraction mapping on $[a,b]$.

Example 1: Solving $x = \cos x$ (Dottie Number)

Example - $g(x) = \cos x$

We wish to find $x^*$ such that $x^* = \cos x^*$. Take $g(x) = \cos x$. Then $g'(x) = -\sin x$, so $|g'(x)| = |\sin x|$.

On the interval $[0.7, 0.8]$, we have $|\sin x| \le \sin(0.8) \approx 0.717 < 1$, so the contraction condition holds. Starting from $x_0 = 1$:

  • $x_1 = \cos(1) \approx 0.5403$
  • $x_2 = \cos(0.5403) \approx 0.8576$
  • $x_3 = \cos(0.8576) \approx 0.6543$
  • $x_4 = \cos(0.6543) \approx 0.7935$
  • $\vdots$
  • $x_n \to x^* \approx 0.7391$ (the Dottie number)

The Dottie number is the unique fixed point of cosine; it satisfies $\cos(x^*) = x^*$.

Example 2: Solving $x^3 - x - 1 = 0$

Example - $g(x) = \sqrt[3]{x+1}$

Rearrange $x^3 = x + 1$ as $x = (x+1)^{1/3}$. Set $g(x) = (x+1)^{1/3}$. Then: $$g'(x) = \frac{1}{3}(x+1)^{-2/3}.$$ Near the root (approximately $x \approx 1.3247$): $g'(1.3) \approx \frac{1}{3(2.3)^{2/3}} \approx \frac{1}{3 \cdot 1.69} \approx 0.197 < \frac{1}{3}$.

The contraction condition holds on any interval around the root. Starting from $x_0 = 1.5$:

  • $x_1 = (2.5)^{1/3} \approx 1.3572$
  • $x_2 = (2.3572)^{1/3} \approx 1.3309$
  • $x_3 \approx 1.3253$
  • $x_4 \approx 1.3248$
  • $x_n \to x^* \approx 1.3247$

Since $k < 1/3$, convergence is rapid. The other rearrangement $g(x) = x^3 - 1$ gives $g'(x) = 3x^2 > 1$ near the root, and diverges - highlighting that the rearrangement matters.

Remark - Choice of $g$

Many different rearrangements of $f(x) = 0$ as $x = g(x)$ are possible. Only those for which $|g'|$ is strictly less than $1$ near the root will converge. In practice one tries rearrangements and checks the derivative condition. Newton's method (Section 9.2) is a systematic way to construct a $g$ with $g'(x^*) = 0$, giving quadratic convergence.
