5.3 Intermediate Value Property

The Intermediate Value Theorem (IVT) captures the intuition that a continuous function cannot "jump" from one value to another without passing through every value in between. It is one of the most practically useful theorems in analysis, providing existence proofs for solutions of equations.

Theorem 1 - Intermediate Value Theorem

Let $f:[a,b]\to\mathbb{R}$ be continuous. If $k$ is any value strictly between $f(a)$ and $f(b)$ (i.e., $f(a)k>f(b)$), then there exists $c\in(a,b)$ such that $f(c)=k$.

Corollary (sign-change form): if $f(a)$ and $f(b)$ have opposite signs, then $f$ has a zero in $(a,b)$.

An Application

Example - $\cos x = x$ has a solution in $(0, \pi/2)$

Let $g(x)=\cos x - x$. This is continuous on $\mathbb{R}$.

  • $g(0)=\cos 0 - 0 = 1 > 0$.
  • $g(\pi/2)=\cos(\pi/2)-\pi/2 = 0-\pi/2 < 0$.

By the IVT (sign-change corollary), there exists $c\in(0,\pi/2)$ with $g(c)=0$, i.e., $\cos c=c$.

Numerically, $c\approx 0.7391$ (the "Dottie number"). We revisit this in Section 5.4.

IVP Does Not Imply Continuity

Remark - Darboux Functions

The Intermediate Value Property (IVP) - that $f$ takes every intermediate value - is a consequence of continuity but not equivalent to it. Functions satisfying the IVP but not continuous are called Darboux functions.

The classic example is $f(x)=\sin(1/x)$ for $x\ne 0$ and $f(0)=0$: this satisfies IVP on $[-1,1]$ (for any $k\in[-1,1]$, $f$ takes the value $k$ near $0$) but is discontinuous at $0$. A more exotic example is the derivative of a differentiable function that is not itself continuous: by Darboux's theorem, all derivatives satisfy IVP.
