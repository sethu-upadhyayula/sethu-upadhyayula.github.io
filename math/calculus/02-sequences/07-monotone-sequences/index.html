2.7 Monotone Sequences

The Monotone Convergence Theorem is one of the most useful tools in analysis: it provides convergence without requiring us to know the limit in advance. It relies fundamentally on the completeness of $\mathbb{R}$.

Definition 2.7.1 - Monotone Sequences

A sequence $(a_n)$ is:

  • Increasing (non-decreasing) if $a_{n+1} \ge a_n$ for all $n$.
  • Strictly increasing if $a_{n+1} > a_n$ for all $n$.
  • Decreasing (non-increasing) if $a_{n+1} \le a_n$ for all $n$.
  • Monotone if it is either increasing or decreasing.
Theorem 2.7.2 - Monotone Convergence Theorem

Every bounded monotone sequence converges. Specifically:

  • If $(a_n)$ is increasing and bounded above, then $a_n \to \sup\{a_n : n \in \mathbb{N}\}$.
  • If $(a_n)$ is decreasing and bounded below, then $a_n \to \inf\{a_n : n \in \mathbb{N}\}$.
Proof (increasing case)

Let $S = \{a_n : n \in \mathbb{N}\}$. Since $(a_n)$ is bounded above, $S$ has a supremum $L = \sup S$ by the Completeness Axiom of $\mathbb{R}$.

Let $\varepsilon > 0$. Since $L - \varepsilon$ is not an upper bound of $S$, there exists $N$ with $a_N > L - \varepsilon$. Since $(a_n)$ is increasing, $a_n \ge a_N > L - \varepsilon$ for all $n \ge N$. Also $a_n \le L$ for all $n$ (as $L$ is an upper bound). Therefore: $$L - \varepsilon < a_n \le L \quad \Rightarrow \quad |a_n - L| < \varepsilon \quad \text{for all } n \ge N.$$

Since $\varepsilon > 0$ was arbitrary, $a_n \to L$. $\square$

Example: Newton's Method for $\sqrt{2}$

Example - Recursive Sequence Converging to $\sqrt{2}$

Define $a_1 = 1$ and $a_{n+1} = \frac{1}{2}\!\left(a_n + \frac{2}{a_n}\right)$. This is Newton's method applied to $f(x) = x^2 - 2$.

Bounded below by $\sqrt{2}$: By AM-GM, $a_{n+1} = \frac{1}{2}(a_n + 2/a_n) \ge \sqrt{a_n \cdot 2/a_n} = \sqrt{2}$ for all $n \ge 1$.

Decreasing (for $n \ge 2$): Since $a_n \ge \sqrt{2}$ for $n \ge 2$: $$a_{n+1} - a_n = \frac{2 - a_n^2}{2a_n} \le 0 \quad (\text{as } a_n^2 \ge 2).$$

By MCT, $(a_n)$ converges to some $L \ge \sqrt{2}$. Taking limits: $L = \frac{1}{2}(L + 2/L)$, so $2L = L + 2/L$, giving $L^2 = 2$, hence $L = \sqrt{2}$. The convergence is extremely fast: $a_1 = 1$, $a_2 = 1.5$, $a_3 \approx 1.4167$, $a_4 \approx 1.41422$.

The Number $e$

Theorem 2.7.3 - Definition of $e$

The sequence $e_n = \left(1 + \frac{1}{n}\right)^n$ is strictly increasing and bounded above by $3$. Therefore it converges; its limit is Euler's number $e = \lim_{n\to\infty}(1 + 1/n)^n \approx 2.71828\ldots$

Proof Sketch

Increasing: By the AM-GM inequality applied to the $n$ numbers $(1 + 1/n)$ and one copy of $1$: $$\left(\frac{n \cdot (1+1/n) + 1}{n+1}\right)^{n+1} \ge \left(1+\frac{1}{n}\right)^n,$$ which simplifies (after algebra) to $e_{n+1} \ge e_n$.

Bounded above by $3$: By the binomial theorem: $$e_n = \sum_{k=0}^{n}\binom{n}{k}\frac{1}{n^k} = 1 + 1 + \sum_{k=2}^{n}\frac{1}{k!}\cdot\frac{n(n-1)\cdots(n-k+1)}{n^k} < \sum_{k=0}^{n}\frac{1}{k!} < \sum_{k=0}^{\infty}\frac{1}{2^{k-1}} = 3.$$

By MCT, $e_n$ converges to a limit we call $e$. $\square$

Remark

The Monotone Convergence Theorem is equivalent to the Completeness Axiom: each implies the other. This is why it fails in $\mathbb{Q}$: the sequence of rational approximations to $\sqrt{2}$ via Newton's method is increasing, bounded above by $3/2$, yet has no limit in $\mathbb{Q}$ since $\sqrt{2} \notin \mathbb{Q}$.
