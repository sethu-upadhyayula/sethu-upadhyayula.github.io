2.1 Definition

A sequence is one of the most fundamental objects in analysis. Informally, a sequence is an ordered list of real numbers indexed by the natural numbers. Formally, we treat it as a function.

Definition 2.1.1 - Sequence

A sequence of real numbers is a function $a : \mathbb{N} \to \mathbb{R}$. We write the sequence as $(a_n)_{n=1}^\infty$ or simply $(a_n)$, where $a_n = a(n)$ is the $n$th term.

Although a sequence is technically a function, we think of it as an ordered list $a_1, a_2, a_3, \ldots$ The distinction from a set is that order matters and repetition is allowed.

Example - Basic Sequences

The following are standard examples:

  • $a_n = \dfrac{1}{n}$: the sequence $1, \tfrac{1}{2}, \tfrac{1}{3}, \tfrac{1}{4}, \ldots$ - terms shrink toward $0$.
  • $a_n = (-1)^n$: the sequence $-1, 1, -1, 1, \ldots$ - terms alternate between $\pm 1$.
  • $a_n = n^2$: the sequence $1, 4, 9, 16, \ldots$ - terms grow without bound.
  • $a_n = \left(1 + \dfrac{1}{n}\right)^n$: the sequence $2, \tfrac{9}{4}, \tfrac{64}{27}, \ldots$ - converges to Euler's number $e \approx 2.718$.

Bounded Sequences

Definition 2.1.2 - Bounded Sequence

A sequence $(a_n)$ is bounded above if there exists $M \in \mathbb{R}$ such that $a_n \le M$ for all $n \in \mathbb{N}$.

It is bounded below if there exists $m \in \mathbb{R}$ such that $a_n \ge m$ for all $n$.

It is bounded if there exists $M > 0$ such that $|a_n| \le M$ for all $n \in \mathbb{N}$, i.e., it is bounded both above and below.

Example - Bounded vs Unbounded

$a_n = (-1)^n$ is bounded: $|(-1)^n| = 1 \le 1$ for all $n$, so $M = 1$ works.

$a_n = n^2$ is not bounded above: for any $M > 0$, choosing $n > \sqrt{M}$ gives $a_n = n^2 > M$.

Tails and Eventual Properties

Definition 2.1.3 - Tail of a Sequence

The tail of $(a_n)$ starting at $N$ is the sequence $(a_n)_{n=N}^\infty = a_N, a_{N+1}, a_{N+2}, \ldots$ A property holds eventually if it holds for all $n \ge N$ for some $N \in \mathbb{N}$.

For example, $a_n = (-1)^n / n$ is eventually positive for all even $n$ beyond some point, but the alternation means it is not eventually positive overall. On the other hand, $a_n = -1/n + 2$ is eventually positive since $a_n > 0$ for all $n \ge 1$.

Arithmetic and Geometric Progressions

Definition 2.1.4 - Arithmetic and Geometric Progressions

An arithmetic progression is a sequence of the form $$a_n = a_1 + (n-1)d$$ where $a_1$ is the first term and $d$ is the common difference. Consecutive terms differ by $d$.

A geometric progression is a sequence of the form $$a_n = a_1 \cdot r^{n-1}$$ where $a_1 \ne 0$ is the first term and $r$ is the common ratio. Consecutive terms have ratio $r$.

Example

The sequence $2, 5, 8, 11, \ldots$ is an arithmetic progression with $a_1 = 2$, $d = 3$, so $a_n = 2 + 3(n-1) = 3n - 1$.

The sequence $3, 6, 12, 24, \ldots$ is a geometric progression with $a_1 = 3$, $r = 2$, so $a_n = 3 \cdot 2^{n-1}$.

Recursively Defined Sequences

Many important sequences are defined recursively: the first term (or first few terms) are specified, and each subsequent term is given by a formula involving previous terms.

Example - A Recursive Sequence

Define $a_1 = 1$ and $a_{n+1} = \sqrt{2 + a_n}$ for all $n \ge 1$. The first several terms are: $$a_1 = 1, \quad a_2 = \sqrt{3} \approx 1.732, \quad a_3 = \sqrt{2 + \sqrt{3}} \approx 1.932, \quad a_4 \approx 1.983, \ldots$$

The sequence appears to be increasing and bounded above. If it converges to a limit $L$, then taking limits on both sides of $a_{n+1} = \sqrt{2 + a_n}$ gives $L = \sqrt{2 + L}$, so $L^2 = 2 + L$, giving $L^2 - L - 2 = 0$ and thus $L = 2$ (taking the positive root). We will prove convergence rigorously in Section 2.7 using the Monotone Convergence Theorem.

Remark

The Fibonacci sequence $1, 1, 2, 3, 5, 8, 13, \ldots$, defined by $F_1 = F_2 = 1$ and $F_{n+2} = F_{n+1} + F_n$, is another fundamental recursive sequence. The ratio $F_{n+1}/F_n$ converges to the golden ratio $\varphi = (1 + \sqrt{5})/2 \approx 1.618$.
