2.3 Limit Theorems

The $\varepsilon$-$N$ definition is powerful but cumbersome to apply directly every time. The limit theorems below allow us to compute limits of complicated sequences by combining simpler ones. Throughout, assume $a_n \to a$ and $b_n \to b$ as $n \to \infty$.

Theorem 2.3.1 — Algebra of Limits

Let $a_n \to a$ and $b_n \to b$. Then:

  1. $a_n + b_n \to a + b$
  2. $c \cdot a_n \to c \cdot a$ for any constant $c \in \mathbb{R}$
  3. $a_n b_n \to ab$
  4. If $b \ne 0$ and $b_n \ne 0$ for all large $n$: $\dfrac{a_n}{b_n} \to \dfrac{a}{b}$
Proof of (i): Sum Rule

Let $\varepsilon > 0$. Since $a_n \to a$, $\exists N_1$ such that $|a_n - a| < \varepsilon/2$ for $n \ge N_1$. Since $b_n \to b$, $\exists N_2$ such that $|b_n - b| < \varepsilon/2$ for $n \ge N_2$. Set $N = \max(N_1, N_2)$. For $n \ge N$: $$|(a_n + b_n) - (a + b)| \le |a_n - a| + |b_n - b| < \frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon. \quad \square$$

Proof of (iii): Product Rule

We use the identity: $$|a_n b_n - ab| = |a_n b_n - a_n b + a_n b - ab| \le |a_n||b_n - b| + |b||a_n - a|.$$

Since $(a_n)$ converges, it is bounded (Theorem 2.6.1): $\exists M > 0$ with $|a_n| \le M$ for all $n$. Given $\varepsilon > 0$, choose $N_1$ so that $|b_n - b| < \varepsilon/(2M)$ for $n \ge N_1$, and choose $N_2$ so that $|a_n - a| < \varepsilon/(2(|b|+1))$ for $n \ge N_2$. For $n \ge N = \max(N_1, N_2)$: $$|a_n b_n - ab| \le M \cdot \frac{\varepsilon}{2M} + |b| \cdot \frac{\varepsilon}{2(|b|+1)} < \frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon. \quad \square$$

Corollary 2.3.2 — Absolute Value

If $a_n \to a$ then $|a_n| \to |a|$. This follows from the reverse triangle inequality: $\bigl||a_n| - |a|\bigr| \le |a_n - a|$.

Examples with Rational Functions

Example — Polynomial and Rational Limits

Since $1/n \to 0$, we have by the algebra of limits: $$\lim_{n\to\infty} \frac{3n^2 + 2n + 1}{n^2 - 5} = \lim_{n\to\infty} \frac{3 + 2/n + 1/n^2}{1 - 5/n^2} = \frac{3 + 0 + 0}{1 - 0} = 3.$$

The technique is to divide numerator and denominator by the highest power of $n$ appearing. More generally, for any polynomial $p$ of degree $k$ and $q$ of degree $\ell$: $$\lim_{n\to\infty} \frac{p(n)}{q(n)} = \begin{cases} \text{ratio of leading coefficients} & \text{if } k = \ell \\ 0 & \text{if } k < \ell \\ \pm\infty & \text{if } k > \ell. \end{cases}$$

For example, $\lim (n^3 + 1)/(2n^3 - n) = 1/2$ and $\lim (n+1)/n^2 = 0$.

Remark — Order of Arguments

The proof of the product rule uses the fact that convergent sequences are bounded. This is proved in Section 2.6 using the $\varepsilon$-$N$ definition directly. The logical order is: convergence $\Rightarrow$ bounded (§2.6), and then we use this in proving the product rule (§2.3). In practice, both results are used together without circularity because the boundedness proof does not rely on the product rule.
