1.2 Order and Bounds

$\mathbb{R}$ is not merely a field — it is an ordered field. The order structure allows us to compare magnitudes, define intervals, and ultimately talk about limits and convergence. In this section we lay out the axioms of order, define what it means for a set to be bounded, and prove the indispensable triangle inequality.

Ordered Field Axioms

Definition 1.2.1 — Ordered Field

An ordered field is a field $F$ together with a relation $<$ satisfying:

  1. Trichotomy: For any $x, y \in F$, exactly one of $x < y$, $x = y$, $y < x$ holds.
  2. Transitivity: $x < y$ and $y < z$ imply $x < z$.
  3. Addition compatibility: $x < y$ implies $x + z < y + z$ for all $z$.
  4. Multiplication compatibility: $x < y$ and $0 < z$ imply $xz < yz$.

Both $\mathbb{Q}$ and $\mathbb{R}$ are ordered fields under the usual ordering.

Bounds

Definition 1.2.2 — Upper and Lower Bounds

Let $S \subseteq \mathbb{R}$ be a nonempty set.

  • A number $M \in \mathbb{R}$ is an upper bound for $S$ if $x \leq M$ for all $x \in S$. We say $S$ is bounded above if it has an upper bound.
  • A number $m \in \mathbb{R}$ is a lower bound for $S$ if $m \leq x$ for all $x \in S$. We say $S$ is bounded below if it has a lower bound.
  • $S$ is bounded if it is both bounded above and bounded below.

If $M \in S$ and $M$ is an upper bound for $S$, then $M$ is the maximum of $S$, written $\max S$. The minimum is defined analogously.

Example

The set $S = \{1, 1/2, 1/3, 1/4, \ldots\} = \{1/n : n \in \mathbb{N}\}$ is bounded: $0$ is a lower bound and $1$ is an upper bound. Furthermore, $\max S = 1 \in S$, but $\min S$ does not exist (the infimum is $0 \notin S$).

The set $\mathbb{N} = \{1, 2, 3, \ldots\}$ is bounded below (by $1$) but not bounded above.

Absolute Value and the Triangle Inequality

Definition 1.2.3 — Absolute Value

For $x \in \mathbb{R}$, the absolute value is defined by $$|x| = \begin{cases} x & \text{if } x \geq 0 \\ -x & \text{if } x < 0 \end{cases}$$ Geometrically, $|x|$ is the distance from $x$ to $0$, and $|x - y|$ is the distance from $x$ to $y$.

Theorem 1.2.4 — Triangle Inequality

For all $x, y \in \mathbb{R}$: $$|x + y| \leq |x| + |y|$$

Proof

We use the fact that $-|x| \leq x \leq |x|$ and $-|y| \leq y \leq |y|$ for all real $x, y$. Adding these inequalities:

$$-(|x| + |y|) \leq x + y \leq |x| + |y|$$

This is equivalent to $|x + y| \leq |x| + |y|$. $\square$

Theorem 1.2.5 — Reverse Triangle Inequality

For all $x, y \in \mathbb{R}$: $$\bigl||x| - |y|\bigr| \leq |x - y|$$

Proof

By the triangle inequality, $|x| = |(x - y) + y| \leq |x - y| + |y|$, so $|x| - |y| \leq |x - y|$. By symmetry, $|y| - |x| \leq |y - x| = |x - y|$. Combining: $\bigl||x| - |y|\bigr| \leq |x - y|$. $\square$

Remark

The triangle inequality generalises to any finite sum: $|x_1 + x_2 + \cdots + x_n| \leq |x_1| + |x_2| + \cdots + |x_n|$. It is ubiquitous in analysis — nearly every limit proof uses it to break apart error terms.
