Chapter 10: Integration

Integration formalises the intuitive idea of area under a curve. The Riemann integral provides a rigorous definition via approximation by sums, and the Fundamental Theorem of Calculus connects integration with differentiation, making systematic computation possible.

Partitions and Darboux Sums

Definition - Partition and Darboux Sums

A partition of $[a,b]$ is a finite set $P = \{x_0, x_1, \ldots, x_n\}$ with $a = x_0 < x_1 < \cdots < x_n = b$. Let $\Delta x_i = x_i - x_{i-1}$ and set: $$M_i = \sup_{x \in [x_{i-1},x_i]} f(x), \qquad m_i = \inf_{x \in [x_{i-1},x_i]} f(x).$$ The upper Darboux sum and lower Darboux sum are: $$U(f,P) = \sum_{i=1}^n M_i \Delta x_i, \qquad L(f,P) = \sum_{i=1}^n m_i \Delta x_i.$$ The upper integral is $\overline{\int_a^b} f = \inf_P U(f,P)$, and the lower integral is $\underline{\int_a^b} f = \sup_P L(f,P)$.

Definition - Riemann Integrability

A bounded function $f:[a,b]\to\mathbb{R}$ is Riemann integrable if $\overline{\int_a^b} f = \underline{\int_a^b} f$, and the common value is the Riemann integral $\int_a^b f(x)\,dx$. Equivalently (Riemann's criterion): $f$ is integrable if and only if for every $\varepsilon > 0$ there exists a partition $P$ such that $U(f,P) - L(f,P) < \varepsilon$.

Theorem - Continuous Functions Are Integrable

Every continuous function $f:[a,b]\to\mathbb{R}$ is Riemann integrable. More generally, every bounded function with only finitely many discontinuities is integrable.

Properties of the Integral

Theorem - Basic Properties

Let $f,g$ be integrable on $[a,b]$ and $a < c < b$.

  • Linearity: $\displaystyle\int_a^b (\alpha f + \beta g)\,dx = \alpha\int_a^b f\,dx + \beta\int_a^b g\,dx$.
  • Monotonicity: If $f(x) \le g(x)$ for all $x \in [a,b]$, then $\displaystyle\int_a^b f \le \int_a^b g$.
  • Additivity: $\displaystyle\int_a^b f\,dx = \int_a^c f\,dx + \int_c^b f\,dx$.
  • Triangle inequality: $\displaystyle\left|\int_a^b f\,dx\right| \le \int_a^b |f|\,dx$.

Fundamental Theorem of Calculus

Theorem - FTC Part 1

Let $f$ be continuous on $[a,b]$ and define $F(x) = \int_a^x f(t)\,dt$ for $x \in [a,b]$. Then $F$ is differentiable on $(a,b)$ and $F'(x) = f(x)$. Thus $F$ is an antiderivative of $f$.

Theorem - FTC Part 2

Let $f$ be continuous on $[a,b]$ and $F$ any antiderivative of $f$ (i.e., $F' = f$). Then: $$\int_a^b f(x)\,dx = F(b) - F(a).$$

Example - Using FTC

$\displaystyle\int_0^{\pi} \sin x\,dx = [-\cos x]_0^{\pi} = -\cos\pi + \cos 0 = 1 + 1 = 2.$

Integration Techniques

Integration by Parts

If $u$ and $v$ are differentiable with $u'v$ integrable: $$\int u\,dv = uv - \int v\,du.$$ Example: $\int x e^x\,dx = xe^x - e^x + C$.

Substitution Rule

Let $g:[a,b]\to\mathbb{R}$ be differentiable with $g'$ continuous, and $f$ continuous on the range of $g$. Then: $$\int_a^b f(g(x))\,g'(x)\,dx = \int_{g(a)}^{g(b)} f(u)\,du.$$ Example: $\int_0^1 2x\,e^{x^2}\,dx \xrightarrow{u=x^2} \int_0^1 e^u\,du = e - 1$.

Partial Fractions

To integrate rational functions $P(x)/Q(x)$, decompose into partial fractions. Example: $$\int \frac{1}{x^2-1}\,dx = \int\!\left(\frac{1/2}{x-1} - \frac{1/2}{x+1}\right)dx = \frac{1}{2}\ln|x-1| - \frac{1}{2}\ln|x+1| + C.$$

Improper Integrals

Definition - Improper Integrals

Type I (infinite interval): $$\int_a^\infty f(x)\,dx = \lim_{R\to\infty}\int_a^R f(x)\,dx.$$ Type II (integrand unbounded near $b$): $$\int_a^b f(x)\,dx = \lim_{\varepsilon\to 0^+}\int_a^{b-\varepsilon} f(x)\,dx.$$ Both converge if the limit exists and is finite.

Theorem - $p$-Test

$$\int_1^\infty \frac{1}{x^p}\,dx \text{ converges} \iff p > 1.$$ For $p > 1$: $\displaystyle\int_1^\infty x^{-p}\,dx = \frac{1}{p-1}$. For $p = 1$: $\int_1^\infty x^{-1}\,dx$ diverges (equals $\ln R \to \infty$).

Example - Gaussian Integral

The celebrated result $\displaystyle\int_{-\infty}^\infty e^{-x^2}\,dx = \sqrt{\pi}$ is proved by squaring and converting to polar coordinates. It is fundamental in probability (the normal distribution).
