Chapter 13: Multivariable Calculus

Multivariable calculus extends differentiation and integration to functions of several variables $f:\mathbb{R}^n\to\mathbb{R}$. New phenomena arise: limits must hold along all paths, the derivative becomes a linear map (the Jacobian), and optimisation requires the gradient and Hessian.

Limits in $\mathbb{R}^n$

Definition — Multivariable Limit

$\lim_{\mathbf{x}\to\mathbf{a}} f(\mathbf{x}) = L$ means: for every $\varepsilon > 0$ there exists $\delta > 0$ such that $0 < \|\mathbf{x} - \mathbf{a}\| < \delta$ implies $|f(\mathbf{x}) - L| < \varepsilon$. The limit must be the same along all paths approaching $\mathbf{a}$.

Example — Path-Dependence

Consider $f(x,y) = xy/(x^2+y^2)$ at $(0,0)$. Along $y = 0$: $f = 0$. Along $y = x$: $f = x^2/(2x^2) = 1/2$. Since different paths give different limits, $\lim_{(x,y)\to(0,0)} f$ does not exist.

Partial Derivatives and the Gradient

Definition — Partial Derivative

The partial derivative of $f$ with respect to $x_i$ at $\mathbf{x}$ is: $$\frac{\partial f}{\partial x_i}(\mathbf{x}) = \lim_{h\to 0} \frac{f(\mathbf{x} + h\mathbf{e}_i) - f(\mathbf{x})}{h},$$ where $\mathbf{e}_i$ is the $i$-th standard basis vector. The gradient is: $$\nabla f(\mathbf{x}) = \left(\frac{\partial f}{\partial x_1}, \ldots, \frac{\partial f}{\partial x_n}\right).$$

Theorem — Directional Derivative

For a unit vector $\mathbf{u}$, the directional derivative $D_\mathbf{u} f(\mathbf{x}) = \nabla f(\mathbf{x}) \cdot \mathbf{u}$. The gradient $\nabla f(\mathbf{x})$ points in the direction of steepest ascent; $|\nabla f(\mathbf{x})|$ is the steepest rate of change. Level sets of $f$ are perpendicular to $\nabla f$.

Total Derivative and Jacobian

Definition — Total Derivative

$f:\mathbb{R}^n\to\mathbb{R}$ is differentiable at $\mathbf{x}$ if there exists a linear map $L:\mathbb{R}^n\to\mathbb{R}$ such that: $$\lim_{\mathbf{h}\to\mathbf{0}} \frac{|f(\mathbf{x}+\mathbf{h}) - f(\mathbf{x}) - L(\mathbf{h})|}{\|\mathbf{h}\|} = 0.$$ Then $L(\mathbf{h}) = \nabla f(\mathbf{x}) \cdot \mathbf{h}$, and the Jacobian of $f:\mathbb{R}^n\to\mathbb{R}^m$ is the $m\times n$ matrix $Df$ of all partial derivatives $(Df)_{ij} = \partial f_i/\partial x_j$.

Theorem — Chain Rule

If $\mathbf{x}:[a,b]\to\mathbb{R}^n$ is differentiable and $f:\mathbb{R}^n\to\mathbb{R}$ is differentiable, then: $$\frac{d}{dt}f(\mathbf{x}(t)) = \nabla f(\mathbf{x}(t)) \cdot \mathbf{x}'(t).$$ More generally, $(f \circ g)'(\mathbf{x}) = Df(g(\mathbf{x})) \cdot Dg(\mathbf{x})$ (product of Jacobians).

Hessian and Second-Order Analysis

Definition — Hessian Matrix

The Hessian of $f$ at $\mathbf{x}$ is the $n\times n$ matrix $H_{ij} = \partial^2 f/\partial x_i \partial x_j$. By Clairaut's theorem, if $f$ has continuous second partial derivatives, then $H$ is symmetric: $\partial^2 f/\partial x_i\partial x_j = \partial^2 f/\partial x_j\partial x_i$.

Second Derivative Test (2D)

At a critical point $(a,b)$ where $\nabla f = \mathbf{0}$, let $D = f_{xx}f_{yy} - (f_{xy})^2$.

  • If $D > 0$ and $f_{xx} > 0$: local minimum.
  • If $D > 0$ and $f_{xx} < 0$: local maximum.
  • If $D < 0$: saddle point.
  • If $D = 0$: test inconclusive.

Lagrange Multipliers

Theorem — Lagrange Multipliers

To optimise $f(\mathbf{x})$ subject to a constraint $g(\mathbf{x}) = 0$ (where $\nabla g \ne \mathbf{0}$ on the constraint), any extremum satisfies: $$\nabla f(\mathbf{x}) = \lambda\,\nabla g(\mathbf{x})$$ for some scalar $\lambda$ (the Lagrange multiplier). Solve the system $\nabla f = \lambda\nabla g$, $g(\mathbf{x}) = 0$ simultaneously.

Example — Maximum on a Circle

Maximise $f(x,y) = x+y$ on the circle $g(x,y) = x^2+y^2-1=0$. $\nabla f = (1,1)$, $\nabla g = (2x,2y)$. Setting $\nabla f = \lambda\nabla g$: $1 = 2\lambda x$, $1 = 2\lambda y$, so $x = y$. From the constraint: $2x^2 = 1$, giving $x = y = 1/\sqrt{2}$ (max) or $x = y = -1/\sqrt{2}$ (min). Max value $= \sqrt{2}$.

Double Integrals

Fubini's Theorem

For a continuous $f$ on a region $D = \{(x,y): a\le x\le b,\, c(x)\le y\le d(x)\}$: $$\iint_D f(x,y)\,dA = \int_a^b\int_{c(x)}^{d(x)} f(x,y)\,dy\,dx.$$ The order of integration may be switched when $D$ admits both descriptions.

Change of Variables

If $x = x(u,v)$, $y = y(u,v)$ defines a bijection with Jacobian $J = \partial(x,y)/\partial(u,v)$: $$\iint_D f(x,y)\,dx\,dy = \iint_{D'} f(x(u,v),y(u,v))\,|J|\,du\,dv.$$ Polar coordinates: $x = r\cos\theta$, $y = r\sin\theta$; Jacobian $J = r$, so $dx\,dy = r\,dr\,d\theta$.

Example — Area of Disk by Polar Coordinates

$\iint_{x^2+y^2\le R^2} dA = \int_0^{2\pi}\int_0^R r\,dr\,d\theta = 2\pi \cdot \frac{R^2}{2} = \pi R^2$.
