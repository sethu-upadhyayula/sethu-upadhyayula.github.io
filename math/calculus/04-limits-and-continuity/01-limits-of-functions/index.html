4.1 Limits of Functions

We extend the concept of limit from sequences to functions. The key idea is the same - we want to describe the behaviour of $f(x)$ as $x$ gets arbitrarily close to some value $c$ - but now $x$ varies continuously rather than through a discrete index.

Definition 1 - Limit of a Function ($\varepsilon$-$\delta$)

Let $f$ be defined on a punctured neighbourhood of $c$ (i.e., on $(c-r, c+r)\setminus\{c\}$ for some $r>0$). We say $\lim_{x\to c}f(x)=L$ if: $$\forall\,\varepsilon>0,\;\exists\,\delta>0:\quad 0<|x-c|<\delta\;\Rightarrow\;|f(x)-L|<\varepsilon.$$

Key point: the condition $0 < |x-c|$ deliberately excludes $x=c$. The value $f(c)$ (or whether it is even defined) is irrelevant to the limit.

One-Sided Limits

Definition 2 - One-Sided Limits

The right-hand limit $\lim_{x\to c^+}f(x)=L$ means: $\forall\,\varepsilon>0,\,\exists\,\delta>0: 0

The left-hand limit $\lim_{x\to c^-}f(x)=L$ means: $\forall\,\varepsilon>0,\,\exists\,\delta>0: 0

Theorem 1 - Existence Criterion

$\lim_{x\to c}f(x)=L$ exists if and only if both one-sided limits exist and are equal: $$\lim_{x\to c^-}f(x)=\lim_{x\to c^+}f(x)=L.$$

An Epsilon-Delta Proof

Example - Proving $\lim_{x\to 2}(3x-1)=5$

We need: given $\varepsilon>0$, find $\delta>0$ such that $0<|x-2|<\delta\Rightarrow|(3x-1)-5|<\varepsilon$.

Simplify the conclusion: $|(3x-1)-5|=|3x-6|=3|x-2|$. So we need $3|x-2|<\varepsilon$, i.e., $|x-2|<\varepsilon/3$.

Choose $\delta=\varepsilon/3$. Then $0<|x-2|<\delta$ gives $|(3x-1)-5|=3|x-2|<3\cdot(\varepsilon/3)=\varepsilon$. $\square$

Limit Laws

Theorem 2 - Limit Laws

If $\lim_{x\to c}f(x)=L$ and $\lim_{x\to c}g(x)=M$, then:

  • Sum: $\lim_{x\to c}(f+g)(x)=L+M$.
  • Product: $\lim_{x\to c}(fg)(x)=LM$.
  • Quotient: $\lim_{x\to c}\frac{f(x)}{g(x)}=\frac{L}{M}$, provided $M\ne 0$.
  • Scalar: $\lim_{x\to c}(\alpha f)(x)=\alpha L$ for any constant $\alpha$.

These mirror the sequence limit laws and are proved analogously by choosing $\delta$ as the minimum of the $\delta$'s for $f$ and $g$.

A Limit That Does Not Exist

Example - $\lim_{x\to 0}\sin\!\left(\tfrac{1}{x}\right)$ does not exist

As $x\to 0$, the argument $1/x\to\pm\infty$ and $\sin(1/x)$ oscillates between $-1$ and $1$ infinitely often, never settling near any fixed value. More precisely, the sequences $x_n=\frac{1}{2n\pi}$ and $x_n'=\frac{1}{\pi/2+2n\pi}$ both tend to $0$, yet $\sin(1/x_n)=0$ and $\sin(1/x_n')=1$. Since these sequences give different "limiting values", no single $L$ can satisfy the definition.

This non-existence will be revisited in the sequential characterization of continuity.
