5.3 Classification of Singularities

The Laurent series gives a precise language for describing isolated singularities. The number of nonzero terms in the principal part completely determines whether a singularity is removable, a pole, or essential - with dramatically different behavior in each case.

Definition 5.3.1 - Isolated Singularity

A point $z_0$ is an isolated singularity of $f$ if $f$ is analytic in some punctured disk $0 < |z-z_0| < r$ but not at $z_0$ itself. In this case $f$ has a Laurent expansion $\sum_{n=-\infty}^\infty a_n(z-z_0)^n$ on $0 < |z-z_0| < r$.

The Three Types

Definition 5.3.2 - Removable Singularity

$z_0$ is a removable singularity if the principal part vanishes: $a_n = 0$ for all $n < 0$. Then $f(z) = \sum_{n=0}^\infty a_n(z-z_0)^n$ and one simply defines $f(z_0) = a_0$ to make $f$ analytic at $z_0$.

Characterization: $z_0$ is removable if and only if $|f(z)|$ is bounded in some punctured neighborhood of $z_0$.

Theorem 5.3.3 - Riemann's Removable Singularity Theorem

If $f$ is analytic for $0 < |z-z_0| < r$ and $\lim_{z\to z_0}(z-z_0)f(z) = 0$, then $z_0$ is a removable singularity. In particular, if $f$ is bounded near $z_0$, then $z_0$ is removable.

Definition 5.3.4 - Pole of Order $m$

$z_0$ is a pole of order $m$ ($m \ge 1$) if the principal part has exactly $m$ nonzero terms: $a_{-m} \ne 0$ and $a_n = 0$ for $n < -m$. That is, $$f(z) = \frac{a_{-m}}{(z-z_0)^m} + \cdots + \frac{a_{-1}}{z-z_0} + a_0 + a_1(z-z_0) + \cdots$$

Equivalently, $f(z) = g(z)/(z-z_0)^m$ where $g$ is analytic at $z_0$ with $g(z_0) = a_{-m} \ne 0$. A pole of order $1$ is called a simple pole. At a pole, $|f(z)| \to \infty$ as $z \to z_0$.

Definition 5.3.5 - Essential Singularity

$z_0$ is an essential singularity if the principal part has infinitely many nonzero terms: $a_n \ne 0$ for infinitely many $n < 0$. The behavior of $f$ near an essential singularity is drastically more complicated than near a pole.

Behavior Near Each Type

Theorem 5.3.6 - Casorati-Weierstrass

If $z_0$ is an essential singularity of $f$, then for every $\varepsilon > 0$ the image $f(\{0 < |z-z_0| < \varepsilon\})$ is dense in $\mathbb{C}$. That is, $f$ comes arbitrarily close to every complex number in any punctured neighborhood of $z_0$.

Theorem 5.3.7 - Picard's Great Theorem

In any punctured neighborhood of an essential singularity, $f$ takes every complex value in $\mathbb{C}$ infinitely often, with at most one exceptional value. For example, $e^{1/z}$ near $z=0$ omits only $0$.

Examples

Example 5.3.8 - Identifying Singularity Types
  • $f(z) = \dfrac{\sin z}{z}$: Laurent expansion gives $f(z) = 1 - z^2/6 + \cdots$, no negative powers. So $z=0$ is removable, and setting $f(0)=1$ makes $f$ entire.
  • $f(z) = \dfrac{1}{(z-1)^3}$: pole of order $3$ at $z=1$.
  • $f(z) = \dfrac{e^z - 1}{z^2}$: near $z=0$, $e^z - 1 = z + z^2/2 + \cdots$, so $f(z) = 1/z + 1/2 + z/6 + \cdots$. Simple pole at $z=0$.
  • $f(z) = e^{1/z}$: essential singularity at $z=0$ (infinitely many negative powers in Laurent expansion).
  • $f(z) = \sin(1/z)$: essential singularity at $z=0$ since $\sin w = w - w^3/6 + \cdots$ gives $\sin(1/z) = 1/z - 1/(6z^3) + \cdots$
Remark - Meromorphic Functions

A function $f$ is meromorphic on a domain $D$ if it is analytic on $D$ except at isolated poles. Rational functions are meromorphic on $\mathbb{C}$; $\tan z$, $\cot z$, and $\sec z$ are meromorphic on $\mathbb{C}$. The residue theorem of Chapter 6 applies to meromorphic functions.
