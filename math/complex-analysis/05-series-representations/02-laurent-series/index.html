5.2 Laurent Series

When $f$ has an isolated singularity at $z_0$, it is analytic in a punctured disk $0 < |z-z_0| < R$ but not at $z_0$ itself. A Taylor series cannot capture this behavior; instead, we need a two-sided series — the Laurent series — which allows negative powers of $(z-z_0)$.

Theorem 5.2.1 — Laurent's Theorem

Let $f$ be analytic in the annulus $A = \{z : r < |z-z_0| < R\}$ (where $0 \le r < R \le \infty$). Then $f$ has a unique representation $$f(z) = \sum_{n=-\infty}^\infty a_n (z-z_0)^n$$ convergent absolutely and uniformly on every compact sub-annulus $r < \rho_1 \le |z-z_0| \le \rho_2 < R$. The coefficients are $$a_n = \frac{1}{2\pi i} \oint_{|z-z_0|=\rho} \frac{f(z)}{(z-z_0)^{n+1}}\,dz$$ for any $r < \rho < R$. By Cauchy's theorem, the value of $a_n$ is independent of the choice of $\rho$.

Principal Part and Analytic Part

Definition 5.2.2 — Principal Part

Write the Laurent series as $f(z) = P(z) + A(z)$ where: $$P(z) = \sum_{n=1}^\infty \frac{a_{-n}}{(z-z_0)^n} \quad\text{(principal part)}$$ $$A(z) = \sum_{n=0}^\infty a_n (z-z_0)^n \quad\text{(analytic part)}.$$

The analytic part converges for $|z-z_0| < R$ (it is a Taylor series); the principal part converges for $|z-z_0| > r$. The nature of the principal part — zero, finitely many terms, or infinitely many terms — classifies the singularity at $z_0$.

Remark — Uniqueness

The Laurent expansion on a given annulus is unique: if $f(z) = \sum b_n(z-z_0)^n$ also converges on $A$ and equals $f$, then $b_n = a_n$ for all $n$. This follows from the integral formula for the coefficients.

Examples

Example 5.2.3 — $e^{1/z}$

Substitute $w = 1/z$ into the exponential series $e^w = \sum_{n=0}^\infty w^n/n!$: $$e^{1/z} = \sum_{n=0}^\infty \frac{1}{n!\, z^n} = 1 + \frac{1}{z} + \frac{1}{2!\,z^2} + \frac{1}{3!\,z^3} + \cdots$$ valid for $0 < |z| < \infty$. The principal part has infinitely many nonzero terms, so $z=0$ is an essential singularity.

Example 5.2.4 — Partial Fractions in Two Annuli

Consider $f(z) = \dfrac{1}{z(z-1)}$. Write via partial fractions: $f(z) = \dfrac{1}{z-1} - \dfrac{1}{z}$.

Annulus $0 < |z| < 1$: Expand $\dfrac{1}{z-1} = -\dfrac{1}{1-z} = -\sum_{n=0}^\infty z^n$. So $$f(z) = -\sum_{n=0}^\infty z^n - \frac{1}{z} = -\frac{1}{z} - 1 - z - z^2 - \cdots$$ The principal part is $-1/z$; hence $z=0$ is a simple pole.

Annulus $|z| > 1$: Expand $\dfrac{1}{z-1} = \dfrac{1}{z}\cdot\dfrac{1}{1-1/z} = \sum_{n=0}^\infty z^{-(n+1)}$. So $$f(z) = \sum_{n=0}^\infty \frac{1}{z^{n+1}} - \frac{1}{z} = \frac{1}{z^2} + \frac{1}{z^3} + \cdots$$ No principal part at $\infty$; both singularities ($z=0,1$) lie inside $|z|=1$.

Example 5.2.5 — Reading off Coefficients

For $f(z) = \dfrac{\sin z}{z^3}$, use $\sin z = z - z^3/6 + z^5/120 - \cdots$: $$\frac{\sin z}{z^3} = \frac{1}{z^2} - \frac{1}{6} + \frac{z^2}{120} - \cdots$$

So $a_{-2} = 1$, $a_{-1} = 0$, $a_0 = -1/6$, etc. Since $a_{-1} = 0$, the residue at $z=0$ is zero.

Remark — Laurent vs Taylor

When $f$ is analytic at $z_0$ (no singularity), the Laurent series on $0 < |z-z_0| < R$ automatically has $a_n = 0$ for all $n < 0$, and coincides with the Taylor series on $|z-z_0| < R$. Laurent series strictly generalize Taylor series by allowing the inner radius $r > 0$.
