2.3 Harmonic Functions

The real and imaginary parts of every analytic function satisfy Laplace's equation. This deep connection between complex analysis and potential theory makes harmonic functions central to both mathematics and physics.

Harmonic Functions and the Laplacian

Definition 2.3.1 - Harmonic Function

A real-valued function $u : D \to \mathbb{R}$ on an open set $D \subseteq \mathbb{R}^2$ is harmonic if it is twice continuously differentiable and satisfies Laplace's equation: $$\Delta u = \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 0 \quad \text{on } D.$$

Theorem 2.3.2 - Analytic Functions Have Harmonic Parts

If $f = u + iv$ is analytic on an open set $D$, then both $u$ and $v$ are harmonic on $D$.

Proof. From the Cauchy-Riemann equations: $u_x = v_y$ and $u_y = -v_x$. Differentiating: $u_{xx} = v_{yx}$ and $u_{yy} = -v_{xy}$. Since $v$ is smooth (analyticity implies $C^\infty$, proved later), $v_{xy} = v_{yx}$, giving $u_{xx} + u_{yy} = 0$. Symmetrically, $\Delta v = 0$. $\square$

Harmonic Conjugates

Definition 2.3.3 - Harmonic Conjugate

Given a harmonic function $u$ on a domain $D$, a harmonic function $v$ on $D$ is called a harmonic conjugate of $u$ if $f = u + iv$ is analytic on $D$. Equivalently, $u$ and $v$ satisfy the Cauchy-Riemann equations on $D$.

Theorem 2.3.4 - Existence of Harmonic Conjugates on Simply Connected Domains

If $u$ is harmonic on a simply connected domain $D$, then $u$ has a harmonic conjugate $v$ on $D$, unique up to an additive constant. The conjugate can be recovered by integrating the CR relations: $$v(x,y) = \int_{(x_0,y_0)}^{(x,y)} \left(-\frac{\partial u}{\partial y}\,dx + \frac{\partial u}{\partial x}\,dy\right) + C.$$ Simple connectivity ensures path independence of this line integral.

Example 2.3.5 - Conjugate of $u = x^2 - y^2$

Check: $u_{xx} = 2$, $u_{yy} = -2$, so $\Delta u = 0$. The CR equations require $v_y = u_x = 2x$ and $v_x = -u_y = 2y$. Integrating the first: $v = 2xy + g(x)$. Substituting into the second: $v_x = 2y + g'(x) = 2y$, so $g'(x) = 0$, giving $g(x) = C$.

Hence $v = 2xy + C$, and $f(z) = (x^2-y^2) + i(2xy) = z^2$, confirming this is the real part of $z^2$.

Example 2.3.6 - $u = \log|z|$ is Harmonic on $\mathbb{C} \setminus \{0\}$

For $z = x + iy \neq 0$, $u = \tfrac{1}{2}\log(x^2 + y^2)$. Computing: $$u_x = \frac{x}{x^2+y^2}, \quad u_{xx} = \frac{y^2-x^2}{(x^2+y^2)^2}; \quad u_y = \frac{y}{x^2+y^2}, \quad u_{yy} = \frac{x^2-y^2}{(x^2+y^2)^2}.$$ Hence $\Delta u = u_{xx} + u_{yy} = 0$. The harmonic conjugate (on $\mathbb{C} \setminus (-\infty,0]$) is $v = \arg(z) = \arctan(y/x)$, and $f(z) = \log|z| + i\arg(z) = \mathrm{Log}\,z$.

The Mean Value Property and the Dirichlet Problem

Theorem 2.3.7 - Mean Value Property

If $u$ is harmonic on a domain containing the closed disk $\overline{B(z_0,r)}$, then $$u(z_0) = \frac{1}{2\pi} \int_0^{2\pi} u(z_0 + re^{i\theta})\,d\theta.$$ That is, the value of $u$ at any point equals the average of $u$ over any circle centred at that point. This characterises harmonic functions and immediately implies the maximum principle: a non-constant harmonic function on a domain attains neither its maximum nor minimum in the interior.

Definition 2.3.8 - Dirichlet Problem

Given a domain $D$ with boundary $\partial D$ and a continuous function $\phi : \partial D \to \mathbb{R}$, the Dirichlet problem asks for a harmonic function $u$ on $D$ satisfying $u = \phi$ on $\partial D$. For the unit disk, the explicit solution is given by the Poisson integral formula: $$u(re^{i\theta}) = \frac{1}{2\pi}\int_0^{2\pi} \frac{1 - r^2}{1 - 2r\cos(\theta-t) + r^2}\,\phi(e^{it})\,dt, \quad r < 1.$$
