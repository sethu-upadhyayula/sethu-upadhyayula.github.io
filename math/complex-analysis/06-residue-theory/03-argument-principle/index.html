6.3 Argument Principle

The argument principle gives a topological interpretation of the contour integral of $f'/f$: it counts the net number of zeros minus poles of $f$ inside a contour. This is one of the first results in complex analysis that is genuinely topological in nature.

Theorem 6.3.1 — The Argument Principle

Let $f$ be meromorphic inside and on a simple closed positively-oriented contour $\gamma$, with no zeros or poles on $\gamma$. Let $N$ denote the number of zeros of $f$ inside $\gamma$ (counting multiplicity) and $P$ the number of poles (counting order). Then: $$\frac{1}{2\pi i} \oint_\gamma \frac{f'(z)}{f(z)}\,dz = N - P.$$

Proof

The integrand $f'/f$ is meromorphic. Near a zero of order $n$ at $z_j$: $f(z) = (z-z_j)^n g(z)$ with $g(z_j)\ne 0$, so $f'/f = n/(z-z_j) + g'/g$. The residue of $f'/f$ at $z_j$ is $n$.

Near a pole of order $m$ at $p_k$: $f(z) = h(z)/(z-p_k)^m$ with $h(p_k)\ne 0$, so $f'/f = -m/(z-p_k) + h'/h$. The residue of $f'/f$ at $p_k$ is $-m$.

By the residue theorem: $\frac{1}{2\pi i}\oint_\gamma \frac{f'}{f}\,dz = \sum_j n_j - \sum_k m_k = N - P$. $\square$

Theorem 6.3.2 — Geometric Interpretation

The quantity $N-P$ equals the winding number of the image curve $f \circ \gamma$ around the origin in the $w$-plane. As $z$ traverses $\gamma$ once, $f(z)$ winds $(N-P)$ times around $0$. This is why the theorem is named after the change in argument of $f$ along $\gamma$: $N - P = \frac{1}{2\pi}\Delta_\gamma \arg f(z)$.

Rouché's Theorem

Theorem 6.3.3 — Rouché's Theorem

Let $f$ and $g$ be analytic inside and on a simple closed contour $\gamma$. If $|f(z)| > |g(z)|$ for all $z \in \gamma$, then $f$ and $f + g$ have the same number of zeros inside $\gamma$ (counting multiplicity).

Proof idea: Consider $h_t(z) = f(z) + tg(z)$ for $t \in [0,1]$. On $\gamma$, $|h_t(z)| \ge |f(z)| - t|g(z)| > 0$, so $h_t$ has no zeros on $\gamma$. The number of zeros $N(t)$ is continuous (integer-valued) in $t$, hence constant. $N(0) = N(f) = N(f+g) = N(1)$. $\square$

Example 6.3.4 — Fundamental Theorem of Algebra

Let $p(z) = z^n + a_{n-1}z^{n-1} + \cdots + a_0$ be a monic polynomial of degree $n \ge 1$. We show $p$ has exactly $n$ zeros in $\mathbb{C}$.

Set $f(z) = z^n$ and $g(z) = a_{n-1}z^{n-1} + \cdots + a_0$. On the circle $|z| = R$ for sufficiently large $R$: $$|g(z)| \le |a_{n-1}|R^{n-1} + \cdots + |a_0| < R^n = |f(z)|.$$ By Rouché's theorem, $f(z) + g(z) = p(z)$ has the same number of zeros inside $|z| = R$ as $f(z) = z^n$, which has exactly $n$ zeros (all at the origin). Hence $p$ has exactly $n$ zeros. $\square$

Example 6.3.5 — Locating Zeros

How many zeros does $p(z) = z^4 + z + 1$ have inside the unit disk $|z| < 1$?

Let $f(z) = 1$ and $g(z) = z^4 + z$. On $|z| = 1$: $|g(z)| \le |z|^4 + |z| = 1 + 1 = 2 > 1 = |f(z)|$. That fails. Instead, let $f(z) = z + 1$ and $g(z) = z^4$. On $|z|=1$: $|g(z)| = 1$ and we need $|f(z)| > 1$; this is not uniform. A more careful argument using $f(z) = 1$ fails; instead one checks that $p(z)$ has exactly $1$ zero inside $|z| < 1$ by careful estimation — the zero near $z \approx -1$ lies on the boundary region.

A correct application: on $|z|=1$, take $f = z^4 + 1$ and $g = z$. Then $|g| = 1 < \sqrt{2} \le |f|$ requires further analysis; in practice one uses the argument principle numerically or via explicit root location.

Remark — Open Mapping Theorem

A corollary of the argument principle is the Open Mapping Theorem: a non-constant analytic function maps open sets to open sets. This in turn implies the Maximum Modulus Principle: $|f|$ cannot attain a local maximum in the interior of a domain.
