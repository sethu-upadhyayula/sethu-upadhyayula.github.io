3.3 Branch Cuts

Multi-valued functions such as $\log z$ and $z^{1/2}$ are not functions in the strict sense. A branch cut is a curve removed from the domain that prevents the function from returning to a different value when $z$ traverses a loop. Removing this ambiguity locally defines an analytic branch; the global structure is captured by Riemann surfaces.

Branch Points and Monodromy

Definition 3.3.1 - Branch Point

A point $z_0 \in \mathbb{C} \cup \{\infty\}$ is a branch point of a multi-valued function $f$ if, when $z$ traverses a small loop around $z_0$, the value of $f(z)$ undergoes a non-trivial permutation among its branches. If $z_0$ is a branch point and $f(z)$ returns to its original value after $n$ circuits, $z_0$ is a branch point of order $n-1$ (or an algebraic branch point). If it never returns, $z_0$ is a logarithmic branch point.

Example 3.3.2 - Branch Points of $\log z$

Starting with $\log z = \ln r + i\theta$ at $z = re^{i\theta}$, and traversing a loop $\theta \mapsto \theta + 2\pi$ around the origin, the argument increases by $2\pi$, shifting the value of $\log z$ by $2\pi i$. Thus $z = 0$ is a logarithmic branch point of $\log z$.

Similarly, $z = \infty$ is a branch point (the argument changes by $-2\pi$ as $z$ loops around $\infty$). The function $\log z$ has exactly these two branch points: $0$ and $\infty$.

Definition 3.3.3 - Branch Cut

A branch cut for a multi-valued function $f$ is a curve (or union of curves) connecting branch points, whose removal from $\mathbb{C}$ makes $f$ single-valued and analytic on the remaining domain. The branch cut for the principal logarithm $\operatorname{Log} z$ is the non-positive real axis $(-\infty, 0]$, connecting the branch points $0$ and $\infty$.

Remark - Freedom in Choosing Branch Cuts

The branch cut need not be the negative real axis. Any curve from $0$ to $\infty$ in $\mathbb{C}$ that does not self-intersect works. For example, the positive imaginary axis $[0, +i\infty)$ defines a different branch of $\log z$, analytic on $\mathbb{C}$ minus that ray, with argument range $(-\tfrac{\pi}{2}, \tfrac{3\pi}{2})$. The choice of branch cut is a matter of convention and convenience for a given problem.

Analytic Continuation and Monodromy

Example 3.3.4 - Continuation Around the Branch Point Changes the Sheet

Start with $\operatorname{Log}(1) = 0$ (principal value). Analytically continue $\log z$ along a path encircling the origin counterclockwise: $z(t) = e^{2\pi i t}$ for $t \in [0,1]$. The argument increases continuously from $0$ to $2\pi$. At $t = 1$ we return to $z = 1$, but $\log(1) = 2\pi i$ on this new branch - not $0$.

Continuing once more around the origin gives $\log(1) = 4\pi i$, and so on. The monodromy transformation is $\log z \mapsto \log z + 2\pi i$, which generates an infinite cyclic group - consistent with $\log z$ having infinitely many branches.

Riemann Surfaces

Definition 3.3.5 - Riemann Surface (Informal)

A Riemann surface is a one-dimensional complex manifold on which a multi-valued function becomes genuinely single-valued. For $\log z$, the Riemann surface is constructed by taking countably many copies of $\mathbb{C} \setminus \{0\}$ (the sheets) and gluing them along branch cuts so that traversing a loop around $0$ continuously passes from one sheet to the next. The result is a single-sheeted surface homeomorphic to $\mathbb{C}$, on which $\log z$ is single-valued and analytic. For $z^{1/n}$, the surface has $n$ sheets and the monodromy is cyclic of order $n$.

Example 3.3.6 - Branch Cuts for $\sqrt{z^2-1}$

The function $\sqrt{z^2-1} = \sqrt{(z-1)(z+1)}$ has branch points at $z = \pm 1$ (and possibly $\infty$). A standard choice of branch cut connects $-1$ to $1$ along the real interval $[-1,1]$, on which the function is single-valued and analytic off the cut.

Alternatively, two separate branch cuts can run from $-1$ to $-\infty$ and from $1$ to $+\infty$, yielding a different analytic branch on $\mathbb{C} \setminus ((-\infty,-1] \cup [1,\infty))$. The correct choice depends on the application.
