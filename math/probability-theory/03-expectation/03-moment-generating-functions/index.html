3.3 Moment Generating Functions

The moment generating function encodes all the moments of a random variable in a single analytic object. When it exists in a neighbourhood of zero, it uniquely characterises the distribution.

Definition 3.3.1 - Moment Generating Function

The moment generating function (MGF) of $X$ is $$M_X(t) = E\!\left[e^{tX}\right], \quad t \in \mathbb{R},$$ whenever this expectation is finite. The MGF may not exist for all $t$; we say it exists if it is finite on an open interval containing $0$.

For a discrete $X$: $M_X(t) = \sum_x e^{tx} p(x)$. For a continuous $X$: $M_X(t) = \int e^{tx} f(x)\, dx$.

Theorem 3.3.2 - Moments via Differentiation

If $M_X(t)$ is finite on an open interval containing $0$, then all moments exist and $$E[X^n] = M_X^{(n)}(0) = \left.\frac{d^n}{dt^n} M_X(t)\right|_{t=0}.$$ The Taylor expansion $M_X(t) = \sum_{n=0}^\infty \frac{E[X^n]}{n!} t^n$ holds in a neighbourhood of $0$.

Theorem 3.3.3 - Uniqueness

If $M_X(t)$ and $M_Y(t)$ are both finite on an open interval containing $0$ and $M_X(t) = M_Y(t)$ for all $t$ in that interval, then $X$ and $Y$ have the same distribution. The MGF (when it exists) uniquely determines the distribution.

Theorem 3.3.4 - MGF of a Sum of Independent Variables

If $X$ and $Y$ are independent, then $$M_{X+Y}(t) = M_X(t) \cdot M_Y(t).$$ This product rule is the key tool for deriving the distribution of sums of independent random variables.

MGFs of Common Distributions

Example 3.3.5 - Normal MGF

If $X \sim N(\mu, \sigma^2)$, then $$M_X(t) = \exp\!\left(\mu t + \frac{\sigma^2 t^2}{2}\right), \quad t \in \mathbb{R}.$$

Differentiating: $M_X'(0) = \mu = E[X]$, $M_X''(0) = \mu^2 + \sigma^2 = E[X^2]$, giving $\text{Var}(X) = \sigma^2$. The MGF also shows that the sum of independent normals is normal.

Example 3.3.6 - Poisson and Exponential MGFs

Poisson($\lambda$): $M_X(t) = e^{\lambda(e^t - 1)}$, $t \in \mathbb{R}$.

Exponential($\lambda$): $M_X(t) = \lambda/(\lambda - t)$ for $t < \lambda$.

Characteristic Functions

Definition 3.3.7 - Characteristic Function

The characteristic function of $X$ is $$\phi_X(t) = E\!\left[e^{itX}\right] = E[\cos(tX)] + i\, E[\sin(tX)], \quad t \in \mathbb{R}.$$

Unlike the MGF, the characteristic function always exists (since $|e^{itX}| = 1$). It also uniquely determines the distribution and is used in proofs of the Central Limit Theorem via convergence of characteristic functions (Lévy's continuity theorem). For $X \sim N(\mu, \sigma^2)$: $\phi_X(t) = e^{i\mu t - \sigma^2 t^2/2}$.
