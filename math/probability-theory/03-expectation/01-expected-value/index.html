3.1 Expected Value

The expected value (or expectation, or mean) of a random variable is its probability-weighted average. It is the fundamental summary statistic of a distribution.

Definition 3.1.1 - Expected Value

The expected value of a random variable $X$ is:

  • Discrete: $\displaystyle E[X] = \sum_{x} x\, p(x)$, provided $\sum_x |x|\, p(x) < \infty$.
  • Continuous: $\displaystyle E[X] = \int_{-\infty}^{\infty} x\, f(x)\, dx$, provided $\int |x|\, f(x)\, dx < \infty$.

When the sum/integral diverges absolutely, we say $E[X]$ does not exist (e.g., Cauchy distribution).

Theorem 3.1.2 - Law of the Unconscious Statistician (LOTUS)

For any measurable function $g$: $$E[g(X)] = \sum_x g(x)\, p(x) \quad \text{(discrete)}, \qquad E[g(X)] = \int_{-\infty}^\infty g(x)\, f(x)\, dx \quad \text{(continuous)}.$$ This allows computing $E[g(X)]$ directly from the distribution of $X$ without finding the distribution of $g(X)$.

Theorem 3.1.3 - Linearity of Expectation

For constants $a, b$ and random variables $X, Y$ (with finite expectations): $$E[aX + b] = aE[X] + b, \qquad E[X + Y] = E[X] + E[Y].$$ Linearity holds regardless of whether $X$ and $Y$ are independent.

Theorem 3.1.4 - Tail Formula

For a non-negative random variable $X \ge 0$: $$E[X] = \int_0^\infty P(X > t)\, dt.$$

More generally, for integer-valued $X \ge 0$: $E[X] = \sum_{k=1}^\infty P(X \ge k)$. This formula is often simpler than the direct definition and is useful in computing expectations via survival functions.

Proof (continuous case)

$\displaystyle\int_0^\infty P(X > t)\, dt = \int_0^\infty \int_t^\infty f(x)\, dx\, dt = \int_0^\infty \int_0^x dt\, f(x)\, dx = \int_0^\infty x f(x)\, dx = E[X]$ by Fubini's theorem. $\square$

Jensen's Inequality

Theorem 3.1.5 - Jensen's Inequality

If $g$ is a convex function (i.e., $g''(x) \ge 0$ for all $x$), then $$E[g(X)] \ge g(E[X]).$$ If $g$ is concave, the inequality reverses.

Examples: $g(x) = x^2$ (convex) gives $E[X^2] \ge (E[X])^2$; $g(x) = \log x$ (concave) gives $E[\log X] \le \log E[X]$ (AM-GM in disguise).

Example 3.1.6 - Expectations of Standard Distributions

Using LOTUS and linearity:

  • $X \sim \text{Binomial}(n,p)$: write $X = \sum_{i=1}^n \mathbf{1}_{A_i}$; by linearity $E[X] = np$.
  • $X \sim \text{Poisson}(\lambda)$: $E[X] = \sum_{k=0}^\infty k \frac{e^{-\lambda}\lambda^k}{k!} = \lambda$.
  • $X \sim \text{Exp}(\lambda)$: $E[X] = \int_0^\infty \lambda x e^{-\lambda x}\, dx = 1/\lambda$.
  • $X \sim N(\mu,\sigma^2)$: $E[X] = \mu$ (by symmetry of the density about $\mu$).

Tail formula check for $X \sim \text{Exp}(\lambda)$: $E[X] = \int_0^\infty P(X > t)\, dt = \int_0^\infty e^{-\lambda t}\, dt = 1/\lambda$. Agrees.
