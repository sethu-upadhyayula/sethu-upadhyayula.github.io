7.1 Definition and Examples

A martingale formalises the concept of a fair game: given all information to date, the expected future value equals the present value. The key inputs are a filtration (representing the flow of information) and an adapted process.

Definition 7.1.1 — Filtration and Adapted Process

A filtration is an increasing family $(\mathcal{F}_n)_{n \ge 0}$ of sub-$\sigma$-algebras of $\mathcal{F}$: $\mathcal{F}_0 \subseteq \mathcal{F}_1 \subseteq \cdots \subseteq \mathcal{F}$. Intuitively, $\mathcal{F}_n$ is the information available at time $n$.

A stochastic process $(M_n)_{n \ge 0}$ is adapted to $(\mathcal{F}_n)$ if $M_n$ is $\mathcal{F}_n$-measurable for each $n$. The natural filtration of $(M_n)$ is $\mathcal{F}_n^M = \sigma(M_0, M_1, \ldots, M_n)$.

Definition 7.1.2 — Martingale, Submartingale, Supermartingale

An adapted integrable process $(M_n, \mathcal{F}_n)$ is a:

  • Martingale if $E[M_{n+1} \mid \mathcal{F}_n] = M_n$ a.s. for all $n \ge 0$;
  • Submartingale if $E[M_{n+1} \mid \mathcal{F}_n] \ge M_n$ a.s. (tends to increase);
  • Supermartingale if $E[M_{n+1} \mid \mathcal{F}_n] \le M_n$ a.s. (tends to decrease).

Note: a martingale is both a sub- and supermartingale. If $(M_n)$ is a martingale then $E[M_n] = E[M_0]$ for all $n$ (constant mean).

Standard Examples

Example 7.1.3 — Simple Random Walk

Let $\xi_1, \xi_2, \ldots$ be i.i.d. with $P(\xi_i = +1) = P(\xi_i = -1) = 1/2$ and $S_n = \xi_1 + \cdots + \xi_n$, $S_0 = 0$. With $\mathcal{F}_n = \sigma(\xi_1,\ldots,\xi_n)$:

  • $S_n$ is a martingale: $E[S_{n+1}\mid\mathcal{F}_n] = S_n + E[\xi_{n+1}] = S_n$.
  • $M_n = S_n^2 - n$ is a martingale: $E[S_{n+1}^2 - (n+1)\mid\mathcal{F}_n] = S_n^2 + 1 - (n+1) = S_n^2 - n$.
  • For $\theta \ne 0$, $Z_n = e^{\theta S_n}/(\cosh\theta)^n$ is a martingale (the exponential martingale).

Example 7.1.4 — Product Martingale

Let $Y_1, Y_2, \ldots$ be i.i.d. with $E[Y_i] = 1$ and $Y_i \ge 0$. Define $M_0 = 1$ and $M_n = Y_1 Y_2 \cdots Y_n$. Then $$E[M_{n+1}\mid\mathcal{F}_n] = M_n \cdot E[Y_{n+1}] = M_n,$$ so $(M_n)$ is a non-negative martingale. This is the Radon-Nikodym derivative of a tilted measure with respect to the original measure, and arises in likelihood ratio tests and sequential analysis.

Example 7.1.5 — Doob Martingale

Let $X \in L^1(\mathcal{F})$ and $(\mathcal{F}_n)$ a filtration with $\mathcal{F}_n \nearrow \mathcal{F}$. Define $$M_n = E[X \mid \mathcal{F}_n].$$ By the tower property, $E[M_{n+1}\mid\mathcal{F}_n] = E[E[X\mid\mathcal{F}_{n+1}]\mid\mathcal{F}_n] = E[X\mid\mathcal{F}_n] = M_n$, so $(M_n)$ is a martingale.

The Doob martingale is the fundamental construction linking martingales to conditional expectation. Lévy's upward theorem says $M_n \to X$ a.s. and in $L^1$.

Stopped Processes

Proposition 7.1.6 — Optional Sampling

Let $\tau$ be a stopping time. The stopped process $M^\tau_n = M_{\min(n,\tau)}$ is also a martingale (resp. sub/supermartingale) whenever $(M_n)$ is. This follows because $$M^\tau_{n+1} - M^\tau_n = (M_{n+1} - M_n)\mathbf{1}_{\tau > n},$$ and $\mathbf{1}_{\tau > n}$ is $\mathcal{F}_n$-measurable (since $\tau$ is a stopping time).
