7.3 Convergence Theorems

The martingale convergence theorems are among the deepest results in probability. They give sufficient conditions for a martingale to converge, and include as special cases both the Strong Law of Large Numbers and many results in ergodic theory.

Doob's Upcrossing Inequality

Definition — Upcrossings

Given a sequence $(M_n)$ and $a < b$, an upcrossing of $[a,b]$ by time $N$ is a pair of times $s < t \le N$ with $M_s \le a$ and $M_t \ge b$. Let $U_N[a,b]$ denote the number of complete upcrossings of $[a,b]$ by time $N$.

Theorem 7.3.1 — Doob's Upcrossing Inequality

If $(M_n)$ is a supermartingale, then for all $a < b$ and $N \ge 1$: $$(b-a)\,E[U_N[a,b]] \le E[(M_N - a)^-] \le E[|M_N|] + |a|.$$

This is the key lemma: it bounds the expected number of oscillations of the process across any interval $[a,b]$ in terms of $E[|M_N|]$.

Almost Sure Convergence

Theorem 7.3.2 — Martingale Convergence Theorem (a.s.)

Let $(M_n)$ be a supermartingale with $\sup_n E[M_n^-] < \infty$ (equivalently, $\sup_n E[|M_n|] < \infty$ for $L^1$-bounded supermartingales). Then there exists an integrable random variable $M_\infty$ such that $$M_n \xrightarrow{a.s.} M_\infty \quad \text{as } n \to \infty.$$

Proof sketch: By the upcrossing inequality, $E[U_\infty[a,b]] < \infty$ for every $a < b$. On the event $\{\liminf M_n < a < b < \limsup M_n\}$, the process upcrosses $[a,b]$ infinitely often, which has probability zero. Taking a countable union over rationals $a < b$ gives $P(\lim M_n \text{ does not exist}) = 0$. $\square$

Remark — Without $L^1$ Convergence

Almost sure convergence does not imply $L^1$ convergence. Example: the simple random walk martingale $S_n$ converges a.s. on the event $\{S_n \to -\infty\}$ (which has probability 1 if $S_n \to -\infty$), but $E[S_n] = 0$ while $S_n$ does not converge in $L^1$ to any finite limit. A non-negative martingale (like $M_n = e^{\theta S_n}/(\cosh\theta)^n$) converges a.s. but may converge to $0$ with probability 1.

$L^p$ Convergence

Theorem 7.3.3 — $L^p$ Martingale Convergence ($p > 1$)

Let $(M_n)$ be a martingale with $\sup_n E[|M_n|^p] < \infty$ for some $p > 1$. Then $M_n \to M_\infty$ both a.s. and in $L^p$. Moreover, $E[|M_\infty|^p] < \infty$ and $M_n = E[M_\infty \mid \mathcal{F}_n]$.

Proof uses Doob's maximal inequality: $E[\sup_{n \le N}|M_n|^p] \le (p/(p-1))^p E[|M_N|^p]$, which gives uniform integrability of $(|M_n|^p)$.

Uniform Integrability and $L^1$ Convergence

Theorem 7.3.4 — UI Martingale Convergence

For a martingale $(M_n)$, the following are equivalent:

  1. $(M_n)$ is uniformly integrable;
  2. $M_n \to M_\infty$ in $L^1$;
  3. There exists $X \in L^1$ with $M_n = E[X \mid \mathcal{F}_n]$ for all $n$.
When these hold, $M_\infty = E[X \mid \mathcal{F}_\infty]$ where $\mathcal{F}_\infty = \sigma(\bigcup_n \mathcal{F}_n)$.

Backward Martingales and the SLLN

Theorem 7.3.5 — Lévy's Downward Theorem

Let $(\mathcal{F}_n)_{n \le 0}$ be a decreasing filtration (i.e. $\mathcal{F}_{-1} \supseteq \mathcal{F}_{-2} \supseteq \cdots$) and $X \in L^1$. Then $$E[X \mid \mathcal{F}_{-n}] \xrightarrow{a.s. \text{ and } L^1} E[X \mid \mathcal{F}_{-\infty}], \quad \mathcal{F}_{-\infty} = \bigcap_n \mathcal{F}_{-n}.$$

Backward martingales are automatically uniformly integrable (since $E[|M_n|] \le E[|X|] < \infty$ for all $n$).

Corollary 7.3.6 — SLLN via Backward Martingales

Let $X_1, X_2, \ldots$ be i.i.d. with $E[|X_1|] < \infty$. Set $\mathcal{G}_n = \sigma(X_n, X_{n+1}, \ldots)$ (the tail $\sigma$-algebra of the future). Define the backward martingale $M_n = E[X_1 \mid \mathcal{G}_n] = \frac{X_1 + \cdots + X_n}{n}$ (by exchangeability). By Lévy's downward theorem, $\bar{X}_n \to E[X_1 \mid \mathcal{G}_\infty]$ a.s. By the Hewitt-Savage 0-1 law, $\mathcal{G}_\infty$ is trivial, so $\bar{X}_n \to E[X_1] = \mu$ a.s. $\square$

Theorem 7.3.7 — Lévy's Upward Theorem

If $X \in L^1$ and $\mathcal{F}_n \nearrow \mathcal{F}_\infty$, then $E[X \mid \mathcal{F}_n] \to E[X \mid \mathcal{F}_\infty]$ both a.s. and in $L^1$. In particular, if $\mathcal{F}_\infty = \mathcal{F}$, then $E[X \mid \mathcal{F}_n] \to X$ a.s. and in $L^1$. This justifies the Doob martingale as a genuine approximation to $X$.
