7.2 Optional Stopping Theorem

The Optional Stopping Theorem (OST) says that the martingale property is preserved when we stop at a stopping time: you cannot beat a fair game by choosing a clever stopping rule. This powerful result has applications ranging from the gambler's ruin problem to sequential hypothesis testing.

Definition 7.2.1 — Stopping Time

A random variable $\tau: \Omega \to \{0, 1, 2, \ldots\} \cup \{\infty\}$ is a stopping time with respect to $(\mathcal{F}_n)$ if $\{\tau \le n\} \in \mathcal{F}_n$ for all $n \ge 0$. Equivalently, the decision to stop at time $n$ depends only on the information $\mathcal{F}_n$ available at time $n$, not on the future.

Theorem 7.2.2 — Optional Stopping Theorem

Let $(M_n, \mathcal{F}_n)$ be a martingale and $\tau$ a stopping time. Then $E[M_\tau] = E[M_0]$ holds under any one of the following sufficient conditions:

  1. $\tau \le N$ a.s. for some constant $N < \infty$ (bounded stopping time);
  2. $\tau < \infty$ a.s., $|M_n| \le C$ a.s. for all $n$ (bounded increments), and $E[\tau] < \infty$;
  3. $(M_{\min(n,\tau)})$ is uniformly integrable.

Proof sketch for (1): $E[M_\tau] = E[M_{\tau \wedge N}] = E[E[M_N \mid \mathcal{F}_{\tau \wedge N}]] = E[M_N] = E[M_0]$ by the martingale property and the tower property. $\square$

Wald's Identity

Theorem 7.2.3 — Wald's Identity

Let $X_1, X_2, \ldots$ be i.i.d. with $E[X_i] = \mu$ and $S_n = X_1 + \cdots + X_n$. Let $\tau$ be a stopping time with $E[\tau] < \infty$. Then $$E[S_\tau] = \mu \cdot E[\tau].$$ Similarly, if $E[X_i^2] < \infty$ with variance $\sigma^2$, then $E[S_\tau^2] = \sigma^2 E[\tau] + \mu^2 (E[\tau])^2$.

Proof: $M_n = S_n - n\mu$ is a martingale. OST gives $E[S_\tau - \tau\mu] = E[M_0] = 0$, hence $E[S_\tau] = \mu E[\tau]$. $\square$

Gambler's Ruin

Example 7.2.4 — Gambler's Ruin

A gambler starts with $\$k$ and plays a fair game ($p = 1/2$) until reaching $\$N$ (win) or $\$0$ (ruin). Let $S_n$ be the gambler's fortune (a simple random walk starting at $k$) and $\tau = \min\{n : S_n = 0 \text{ or } S_n = N\}$.

Since $\tau \le N^2$ a.s. (the walk must eventually reach a boundary), OST applied to the martingale $S_n$ gives $$k = E[S_0] = E[S_\tau] = N \cdot P(\text{win}) + 0 \cdot P(\text{ruin}),$$ so $P(\text{win}) = k/N$ and $P(\text{ruin}) = 1 - k/N = (N-k)/N$.

For the general case starting at $0$ with absorbing barriers at $-b$ and $a$ (i.e. $k = b$, $N = a+b$): $$P(\text{hit } a \text{ before } {-b} \mid S_0 = 0) = \frac{b}{a+b}.$$

Applying OST to $M_n = S_n^2 - n$ gives $E[\tau] = k(N-k) = b \cdot a$, the expected duration of the game.

Example 7.2.5 — Expected Hitting Time via OST

Let $S_n$ be simple symmetric random walk, $S_0 = 0$, and $\tau_a = \min\{n : S_n = a\}$ for $a > 0$. Apply OST to $M_n = S_n^2 - n$ on $[0, \tau_a \wedge N]$: $$0 = E[S_{\tau_a \wedge N}^2] - E[\tau_a \wedge N].$$ Letting $N \to \infty$: by monotone convergence $E[\tau_a] = E[S_{\tau_a}^2] = a^2$. So the simple random walk reaches level $a$ in expected time $a^2$.
