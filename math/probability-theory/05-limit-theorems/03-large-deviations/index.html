5.3 Large Deviations

The Law of Large Numbers says $\bar{X}_n \to \mu$ almost surely, and the CLT quantifies fluctuations of order $1/\sqrt{n}$. Large deviation theory addresses a complementary question: how fast does $P(\bar{X}_n \ge x)$ decay to zero for $x > \mu$? The answer is exponential in $n$, with a precise rate given by the rate function.

The Cramér Rate Function

Definition — Log-Moment Generating Function and Rate Function

For a random variable $X$ with moment generating function $M(\theta) = E[e^{\theta X}]$, define the log-moment generating function (cumulant generating function): $$\Lambda(\theta) = \log M(\theta) = \log E[e^{\theta X}].$$ The Cramér rate function (or Legendre-Fenchel transform of $\Lambda$) is $$I(x) = \sup_{\theta \in \mathbb{R}} \{\theta x - \Lambda(\theta)\}.$$ $I$ is convex, lower semicontinuous, non-negative, and $I(\mu) = 0$. It measures the "cost" of the deviation $\bar{X}_n \approx x$.

Theorem 5.3.1 — Cramér's Theorem

Let $X_1, X_2, \ldots$ be i.i.d. with $E[e^{\theta X_1}] < \infty$ for all $\theta$ in a neighbourhood of $0$. Then for any $x > \mu = E[X_1]$: $$\lim_{n\to\infty} \frac{1}{n} \log P(\bar{X}_n \ge x) = -I(x).$$ More precisely, for any closed set $F$ and open set $G$ in $\mathbb{R}$: $$\limsup_{n\to\infty} \frac{1}{n}\log P(\bar{X}_n \in F) \le -\inf_{x \in F} I(x), \quad \liminf_{n\to\infty} \frac{1}{n}\log P(\bar{X}_n \in G) \ge -\inf_{x \in G} I(x).$$

Example — Gaussian Rate Function

If $X \sim N(\mu, \sigma^2)$, then $\Lambda(\theta) = \mu\theta + \sigma^2\theta^2/2$. The rate function is $$I(x) = \sup_\theta\left\{\theta x - \mu\theta - \frac{\sigma^2\theta^2}{2}\right\} = \frac{(x-\mu)^2}{2\sigma^2}.$$ This is just the squared Mahalanobis distance. Cramér's theorem gives $P(\bar{X}_n \ge x) \approx e^{-n(x-\mu)^2/(2\sigma^2)}$, consistent with the known Gaussian tail bound.

Example — Bernoulli Rate Function

If $X \sim \text{Bernoulli}(p)$, then $M(\theta) = 1 - p + pe^\theta$ and $\Lambda(\theta) = \log(1-p+pe^\theta)$. The rate function is the relative entropy $$I(x) = x\log\frac{x}{p} + (1-x)\log\frac{1-x}{1-p} = D_{\text{KL}}(\text{Ber}(x) \,\|\, \text{Ber}(p))$$ for $x \in [0,1]$, and $I(x) = \infty$ otherwise. This is Sanov's theorem in the binary case.

Gärtner-Ellis Theorem

Theorem 5.3.2 — Gärtner-Ellis Theorem

Let $\Lambda_n(\theta) = \frac{1}{n}\log E[e^{\theta S_n}]$ where $S_n = X_1 + \cdots + X_n$. Suppose $\Lambda(\theta) = \lim_{n\to\infty} \Lambda_n(\theta)$ exists in $[-\infty, +\infty]$ for all $\theta$, and that $\Lambda$ is differentiable on $\{\Lambda < \infty\}$. Then the sequence $(\bar{X}_n)$ satisfies a large deviation principle with rate function $$I(x) = \sup_{\theta}\{\theta x - \Lambda(\theta)\}.$$

This extends Cramér's theorem to non-i.i.d. sequences (e.g. Markov chains) and provides a systematic framework for computing large deviation rates.

Sanov's Theorem

Theorem 5.3.3 — Sanov's Theorem

Let $X_1, \ldots, X_n$ be i.i.d. with law $P$. The empirical measure $L_n = \frac{1}{n}\sum_{i=1}^n \delta_{X_i}$ satisfies a large deviation principle on the space of probability measures with rate function $$I(Q) = D_{\text{KL}}(Q \,\|\, P) = E_Q\!\left[\log\frac{dQ}{dP}\right]$$ (the relative entropy of $Q$ with respect to $P$). Explicitly: $$P(L_n \approx Q) \approx e^{-n D_{\text{KL}}(Q\|P)}.$$

Sanov's theorem is fundamental in information theory, statistics, and statistical mechanics. It shows that the probability of seeing an empirical distribution $Q$ when the true distribution is $P$ decays exponentially with rate equal to the KL divergence.

Remark — Connection to Statistical Mechanics

In statistical mechanics, large deviation theory underlies the derivation of thermodynamic quantities. The rate function $I$ plays the role of entropy: equilibrium states are those minimising $I$. The Gibbs variational principle $F = -\frac{1}{\beta}\log Z$ is a large deviation statement about empirical averages of energy.
