6.1 Conditional Expectation

The elementary definition $E[X \mid A] = E[X \mathbf{1}_A]/P(A)$ works for a single event but fails when conditioning on a continuous random variable (since $P(Y=y)=0$). The measure-theoretic framework resolves this by defining conditional expectation as a random variable, not a number.

Definition 6.1.1 - Conditional Expectation

Let $(\Omega, \mathcal{F}, P)$ be a probability space and $\mathcal{G} \subseteq \mathcal{F}$ a sub-$\sigma$-algebra. For $X \in L^1(\Omega, \mathcal{F}, P)$, the conditional expectation $E[X \mid \mathcal{G}]$ is any random variable $Y$ satisfying:

  1. $Y$ is $\mathcal{G}$-measurable;
  2. $\displaystyle\int_A Y\,dP = \int_A X\,dP$ for all $A \in \mathcal{G}$.
Such $Y$ exists and is unique $P$-a.s. (by the Radon-Nikodym theorem applied to the signed measure $A \mapsto \int_A X\,dP$ dominated by $P|_\mathcal{G}$).

We write $E[X \mid Y] := E[X \mid \sigma(Y)]$ when conditioning on a random variable $Y$.

Existence via Radon-Nikodym

Proof of Existence

Define the signed measure $\nu(A) = \int_A X\,dP$ for $A \in \mathcal{G}$. Then $\nu \ll P|_\mathcal{G}$. By the Radon-Nikodym theorem, there exists a $\mathcal{G}$-measurable $Y$ with $\nu(A) = \int_A Y\,dP$ for all $A \in \mathcal{G}$. This is $E[X\mid\mathcal{G}]$.

Uniqueness: if $Y$ and $Y'$ both satisfy the defining conditions, then $\int_A (Y-Y')\,dP = 0$ for all $A \in \mathcal{G}$. Taking $A = \{Y - Y' > \varepsilon\}$ gives $P(Y \ne Y') = 0$.

Properties

Theorem 6.1.2 - Properties of Conditional Expectation

Let $X, Y \in L^1$ and $\mathcal{H} \subseteq \mathcal{G} \subseteq \mathcal{F}$. Then:

  1. Linearity: $E[aX + bY \mid \mathcal{G}] = aE[X\mid\mathcal{G}] + bE[Y\mid\mathcal{G}]$ a.s.
  2. Tower property: $E[E[X\mid\mathcal{G}]\mid\mathcal{H}] = E[X\mid\mathcal{H}]$ a.s.
  3. Taking out what is known: If $Z$ is $\mathcal{G}$-measurable and $ZX \in L^1$, then $E[ZX\mid\mathcal{G}] = Z\,E[X\mid\mathcal{G}]$ a.s.
  4. Independence: If $X$ is independent of $\mathcal{G}$, then $E[X\mid\mathcal{G}] = E[X]$ a.s.
  5. Monotonicity: $X \le Y$ a.s. $\Rightarrow$ $E[X\mid\mathcal{G}] \le E[Y\mid\mathcal{G}]$ a.s.
  6. Jensen's inequality: For convex $\varphi$, $\varphi(E[X\mid\mathcal{G}]) \le E[\varphi(X)\mid\mathcal{G}]$ a.s.
  7. $L^2$ contraction: $\|E[X\mid\mathcal{G}]\|_2 \le \|X\|_2$.

Geometric Interpretation in $L^2$

Remark - Orthogonal Projection

The space $L^2(\mathcal{G}) = \{Z \in L^2 : Z \text{ is } \mathcal{G}\text{-measurable}\}$ is a closed subspace of $L^2(\mathcal{F})$. The map $X \mapsto E[X\mid\mathcal{G}]$ is the orthogonal projection $\Pi_\mathcal{G}: L^2(\mathcal{F}) \to L^2(\mathcal{G})$, i.e. $$E[X\mid\mathcal{G}] = \arg\min_{Z \in L^2(\mathcal{G})} E[(X-Z)^2].$$ The projection property says $E[(X - E[X\mid\mathcal{G}])Z] = 0$ for all $\mathcal{G}$-measurable $Z \in L^2$.

This geometric view makes many identities immediate: the tower property is transitivity of projections onto nested subspaces; the $L^2$ contraction is the Pythagorean theorem.

Example - Conditioning on a Discrete Random Variable

Let $Y$ take values $y_1, y_2, \ldots$ with positive probability. Then $\mathcal{G} = \sigma(Y)$ is generated by the partition $\{Y = y_k\}$, and $$E[X \mid Y] = \sum_k E[X \mid Y = y_k]\,\mathbf{1}_{\{Y = y_k\}}, \quad E[X \mid Y=y_k] = \frac{E[X\,\mathbf{1}_{\{Y=y_k\}}]}{P(Y=y_k)}.$$

The result is a function of $Y$: $E[X\mid Y] = g(Y)$ where $g(y_k) = E[X\mid Y=y_k]$.
