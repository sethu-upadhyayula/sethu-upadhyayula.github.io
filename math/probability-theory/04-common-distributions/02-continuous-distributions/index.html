4.2 Continuous Distributions

This section provides a systematic reference for the most important continuous probability distributions, with emphasis on their properties, special cases, and relationships.

Uniform$(a, b)$

PDF: $f(x) = 1/(b-a)$ on $[a,b]$, zero elsewhere.

Mean: $(a+b)/2$. Variance: $(b-a)^2/12$. MGF: $(e^{tb}-e^{ta})/(t(b-a))$ for $t \ne 0$. The uniform distribution is the continuous analog of "no information" about the value within the interval.

Exponential$(\lambda)$

PDF: $f(x) = \lambda e^{-\lambda x}$ on $[0,\infty)$.

Mean: $1/\lambda$. Variance: $1/\lambda^2$. MGF: $\lambda/(\lambda-t)$ for $t < \lambda$.

Memoryless: $P(X > s+t \mid X > s) = P(X > t)$; unique among continuous distributions. Models inter-arrival times in a Poisson process.

Normal$(\mu, \sigma^2)$

PDF: $f(x) = \frac{1}{\sigma\sqrt{2\pi}}\exp\!\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)$.

Mean: $\mu$. Variance: $\sigma^2$. MGF: $e^{\mu t + \sigma^2 t^2/2}$.

68-95-99.7 rule: $P(\mu - k\sigma \le X \le \mu + k\sigma) \approx 68\%$, $95\%$, $99.7\%$ for $k = 1,2,3$.

Standardisation: $Z = (X-\mu)/\sigma \sim N(0,1)$. Sum of independent normals is normal. The normal distribution is the attractor in the CLT.

Gamma$(\alpha, \beta)$

PDF: $f(x) = \frac{\beta^\alpha}{\Gamma(\alpha)} x^{\alpha-1} e^{-\beta x}$ on $(0,\infty)$, where $\Gamma(\alpha) = \int_0^\infty t^{\alpha-1}e^{-t}\,dt$.

Mean: $\alpha/\beta$. Variance: $\alpha/\beta^2$. MGF: $(\beta/(\beta-t))^\alpha$ for $t < \beta$.

Key facts: $\Gamma(n) = (n-1)!$ for $n \in \mathbb{N}$; $\Gamma(1/2) = \sqrt{\pi}$. Sum of independent Gamma$(\alpha_i, \beta)$ is Gamma$(\sum \alpha_i, \beta)$.

Special cases: Gamma$(1,\lambda)$ = Exp$(\lambda)$. Chi-squared: $\chi^2_k = \text{Gamma}(k/2, 1/2)$; if $Z_1,\ldots,Z_k \overset{\text{iid}}{\sim} N(0,1)$ then $\sum Z_i^2 \sim \chi^2_k$.

Beta$(\alpha, \beta)$

PDF: $f(x) = x^{\alpha-1}(1-x)^{\beta-1}/B(\alpha,\beta)$ on $(0,1)$, where $B(\alpha,\beta) = \Gamma(\alpha)\Gamma(\beta)/\Gamma(\alpha+\beta)$.

Mean: $\alpha/(\alpha+\beta)$. Variance: $\alpha\beta/((\alpha+\beta)^2(\alpha+\beta+1))$.

Special cases: Beta$(1,1)$ = Uniform$(0,1)$. Conjugacy: if $\theta \sim \text{Beta}(\alpha,\beta)$ (prior) and $X|\theta \sim \text{Bin}(n,\theta)$, then $\theta | X=k \sim \text{Beta}(\alpha+k, \beta+n-k)$ (posterior) — the Beta is the conjugate prior for the Binomial likelihood.

Log-Normal$(\mu, \sigma^2)$

$X$ is log-normal if $\log X \sim N(\mu, \sigma^2)$, i.e., $X = e^Z$ where $Z \sim N(\mu,\sigma^2)$.

PDF: $f(x) = \frac{1}{x\sigma\sqrt{2\pi}}\exp\!\left(-\frac{(\log x - \mu)^2}{2\sigma^2}\right)$ on $(0,\infty)$.

Mean: $e^{\mu+\sigma^2/2}$. Variance: $(e^{\sigma^2}-1)e^{2\mu+\sigma^2}$. Used to model strictly positive, right-skewed quantities such as stock prices and incomes.
