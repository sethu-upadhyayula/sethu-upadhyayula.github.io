2.2 Continuous Random Variables

A continuous random variable is characterised by a probability density function rather than a probability mass function. Probabilities are computed by integrating the density over a set.

Definition 2.2.1 - Probability Density Function

A random variable $X$ is continuous if there exists a function $f_X : \mathbb{R} \to [0, \infty)$ called the probability density function (PDF) such that for every Borel set $A \subseteq \mathbb{R}$, $$P(X \in A) = \int_A f_X(x)\, dx.$$

A valid PDF satisfies $f_X(x) \ge 0$ for all $x$ and $\displaystyle\int_{-\infty}^{\infty} f_X(x)\, dx = 1$. Note that $P(X = x) = 0$ for any single point $x$.

Standard Continuous Distributions

Definition 2.2.2 - Uniform Distribution

$X \sim \text{Uniform}(a, b)$ with $a < b$ has PDF $$f(x) = \frac{1}{b-a}, \quad a \le x \le b.$$ $E[X] = (a+b)/2$, $\text{Var}(X) = (b-a)^2/12$.

The uniform distribution models complete ignorance about the value of $X$ within $[a,b]$.

Definition 2.2.3 - Normal Distribution

$X \sim N(\mu, \sigma^2)$ with $\mu \in \mathbb{R}$, $\sigma > 0$ has PDF $$f(x) = \frac{1}{\sigma\sqrt{2\pi}} \exp\!\left(-\frac{(x-\mu)^2}{2\sigma^2}\right), \quad x \in \mathbb{R}.$$ $E[X] = \mu$, $\text{Var}(X) = \sigma^2$.

The standard normal $Z \sim N(0,1)$ has CDF $\Phi(z) = P(Z \le z)$. Any normal can be standardised: if $X \sim N(\mu, \sigma^2)$ then $Z = (X - \mu)/\sigma \sim N(0,1)$.

Definition 2.2.4 - Exponential Distribution

$X \sim \text{Exp}(\lambda)$ with $\lambda > 0$ has PDF $$f(x) = \lambda e^{-\lambda x}, \quad x \ge 0.$$ $E[X] = 1/\lambda$, $\text{Var}(X) = 1/\lambda^2$.

The Exponential distribution is the unique continuous distribution with the memoryless property: $P(X > s + t \mid X > s) = P(X > t)$ for all $s, t \ge 0$. It models waiting times in a Poisson process.

Definition 2.2.5 - Gamma Distribution

$X \sim \text{Gamma}(\alpha, \beta)$ with $\alpha, \beta > 0$ has PDF $$f(x) = \frac{\beta^\alpha}{\Gamma(\alpha)} x^{\alpha-1} e^{-\beta x}, \quad x > 0,$$ where $\Gamma(\alpha) = \int_0^\infty t^{\alpha-1} e^{-t}\, dt$ is the Gamma function. $E[X] = \alpha/\beta$, $\text{Var}(X) = \alpha/\beta^2$.

Special cases: $\text{Gamma}(1, \lambda) = \text{Exp}(\lambda)$; $\text{Gamma}(n, \lambda)$ is the distribution of the sum of $n$ independent $\text{Exp}(\lambda)$ random variables; $\text{Gamma}(k/2, 1/2)$ is the $\chi^2_k$ (chi-squared) distribution.

Definition 2.2.6 - Beta Distribution

$X \sim \text{Beta}(\alpha, \beta)$ with $\alpha, \beta > 0$ has PDF $$f(x) = \frac{x^{\alpha-1}(1-x)^{\beta-1}}{B(\alpha,\beta)}, \quad 0 < x < 1,$$ where $B(\alpha,\beta) = \Gamma(\alpha)\Gamma(\beta)/\Gamma(\alpha+\beta)$. $E[X] = \alpha/(\alpha+\beta)$, $\text{Var}(X) = \alpha\beta/((\alpha+\beta)^2(\alpha+\beta+1))$.

The Beta distribution is supported on $(0,1)$ and is the conjugate prior for the Binomial likelihood in Bayesian inference.

Mixed Distributions

Remark - Mixed Distributions

Some random variables are neither discrete nor continuous. A mixed distribution has both a discrete component (atoms with positive probability mass) and a continuous component (a density on some interval). For example, $X$ might equal $0$ with probability $1/2$ and be $\text{Exp}(1)$-distributed with probability $1/2$.

Such distributions arise naturally in models with excess zeros (e.g., insurance claims: no claim with positive probability, or a positive claim following a continuous distribution).
