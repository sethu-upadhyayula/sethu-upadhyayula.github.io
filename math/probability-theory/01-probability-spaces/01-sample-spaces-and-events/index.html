1.1 Sample Spaces and Events

Before we can define probability, we must specify the universe of possible outcomes and the collection of subsets to which we will assign probabilities. The fundamental objects are the sample space, events, and sigma-algebras.

Definition 1.1.1 — Sample Space

The sample space $\Omega$ is the set of all possible outcomes of a random experiment. Elements $\omega \in \Omega$ are called outcomes or sample points.

Example 1.1.2 — Sample Spaces

Some typical sample spaces:

  • Tossing a fair coin: $\Omega = \{H, T\}$
  • Rolling a die: $\Omega = \{1,2,3,4,5,6\}$
  • Lifetime of a light bulb: $\Omega = [0, \infty)$
  • Infinite coin tosses: $\Omega = \{H,T\}^{\mathbb{N}}$, an uncountable set

The structure of $\Omega$ (finite, countable, or uncountable) determines the appropriate probability framework.

Definition 1.1.3 — Event

An event is a subset $A \subseteq \Omega$ to which a probability can be assigned. We say the event $A$ occurs if the observed outcome $\omega$ belongs to $A$.

Sigma-Algebras

For finite or countably infinite sample spaces, we can assign probabilities to all subsets of $\Omega$. But for uncountable spaces such as $\Omega = \mathbb{R}$, not every subset can be assigned a measure in a consistent way (the Vitali set provides a counterexample). We therefore restrict attention to a special collection of "measurable" subsets.

Definition 1.1.4 — Sigma-Algebra

A collection $\mathcal{F}$ of subsets of $\Omega$ is called a $\sigma$-algebra (or $\sigma$-field) if:

  1. $\Omega \in \mathcal{F}$
  2. If $A \in \mathcal{F}$, then $A^c = \Omega \setminus A \in \mathcal{F}$ (closed under complementation)
  3. If $A_1, A_2, \ldots \in \mathcal{F}$, then $\bigcup_{n=1}^\infty A_n \in \mathcal{F}$ (closed under countable unions)

The pair $(\Omega, \mathcal{F})$ is called a measurable space.

Remark — Derived Closure Properties

From the three axioms one deduces:

  • $\emptyset \in \mathcal{F}$ (since $\emptyset = \Omega^c$)
  • Closed under countable intersections: $\bigcap_n A_n = \left(\bigcup_n A_n^c\right)^c \in \mathcal{F}$
  • Closed under set differences: $A \setminus B = A \cap B^c \in \mathcal{F}$

So $\sigma$-algebras are closed under all countable set operations.

Example 1.1.5 — Trivial and Discrete Sigma-Algebras

For any $\Omega$:

  • The trivial $\sigma$-algebra $\{\emptyset, \Omega\}$ is the smallest possible $\sigma$-algebra.
  • The power set $2^\Omega$ (all subsets) is the largest possible $\sigma$-algebra; used when $\Omega$ is countable.

For $\Omega = \{1,2,3,4\}$ and event $\{1,2\}$, the smallest $\sigma$-algebra containing $\{1,2\}$ is $\{\emptyset, \{1,2\}, \{3,4\}, \{1,2,3,4\}\}$.

The Borel Sigma-Algebra

The most important $\sigma$-algebra for probability on $\mathbb{R}$ is the Borel $\sigma$-algebra.

Definition 1.1.6 — Borel Sigma-Algebra

The Borel $\sigma$-algebra $\mathcal{B}(\mathbb{R})$ is the smallest $\sigma$-algebra on $\mathbb{R}$ containing all open sets (equivalently, all open intervals $(a,b)$).

Sets in $\mathcal{B}(\mathbb{R})$ are called Borel sets. They include all open sets, all closed sets, all countable intersections of open sets ($G_\delta$ sets), all countable unions of closed sets ($F_\sigma$ sets), and so on.

Theorem 1.1.7 — Generators of the Borel Sigma-Algebra

$\mathcal{B}(\mathbb{R})$ is generated by any of the following collections: all open intervals $(a,b)$; all closed intervals $[a,b]$; all half-lines $(-\infty, x]$ for $x \in \mathbb{R}$; all half-lines $(x, \infty)$ for $x \in \mathbb{R}$. In particular, every singleton $\{x\}$ and every interval is a Borel set.

Why Not All Subsets?

Remark — The Vitali Set

Using the Axiom of Choice, one can construct a Vitali set $V \subset [0,1]$ that is not a Borel set (or even Lebesgue measurable). Any attempt to assign a length (probability) to $V$ consistently with translation invariance leads to a contradiction. This is why we must restrict the domain of probability measures to $\mathcal{F}$, not all of $2^\Omega$.

The message: sigma-algebras are the mathematically correct domain for probability measures on uncountable spaces.
