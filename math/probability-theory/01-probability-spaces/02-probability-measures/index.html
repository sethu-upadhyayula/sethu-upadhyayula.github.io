1.2 Probability Measures

A probability measure is a function that assigns a number in $[0,1]$ to each event, satisfying natural consistency conditions. Kolmogorov's axioms capture precisely what we mean by "probability" and allow us to derive all classical probability rules.

Definition 1.2.1 — Probability Measure

A probability measure on the measurable space $(\Omega, \mathcal{F})$ is a function $P: \mathcal{F} \to [0,1]$ satisfying:

  1. Non-negativity: $P(A) \ge 0$ for all $A \in \mathcal{F}$
  2. Normalisation: $P(\Omega) = 1$
  3. Countable additivity ($\sigma$-additivity): For pairwise disjoint $A_1, A_2, \ldots \in \mathcal{F}$, $$P\!\left(\bigcup_{n=1}^\infty A_n\right) = \sum_{n=1}^\infty P(A_n)$$

The triple $(\Omega, \mathcal{F}, P)$ is called a probability space.

Basic Properties

Theorem 1.2.2 — Properties of Probability Measures

Let $(\Omega,\mathcal{F},P)$ be a probability space. Then:

  1. $P(\emptyset) = 0$
  2. Complementation: $P(A^c) = 1 - P(A)$
  3. Monotonicity: If $A \subseteq B$ then $P(A) \le P(B)$
  4. Inclusion-exclusion: $P(A \cup B) = P(A) + P(B) - P(A \cap B)$
  5. Sub-additivity: $P\!\left(\bigcup_n A_n\right) \le \sum_n P(A_n)$
Proof (selected)

For (1): $\Omega = \Omega \cup \emptyset \cup \emptyset \cup \cdots$, so $P(\Omega) = P(\Omega) + P(\emptyset) + P(\emptyset) + \cdots$, giving $P(\emptyset) = 0$.

For (2): $\Omega = A \cup A^c$ (disjoint), so $1 = P(\Omega) = P(A) + P(A^c)$.

For (3): $B = A \cup (B \setminus A)$ (disjoint), so $P(B) = P(A) + P(B \setminus A) \ge P(A)$ since $P(B\setminus A) \ge 0$. $\square$

Theorem 1.2.3 — Continuity of Probability

If $A_n \nearrow A$ (i.e., $A_1 \subseteq A_2 \subseteq \cdots$ and $\bigcup_n A_n = A$), then $P(A_n) \to P(A)$.

Similarly, if $A_n \searrow A$ (decreasing sets), then $P(A_n) \to P(A)$.

Discrete and Continuous Probability Spaces

Definition 1.2.4 — Discrete Probability Space

When $\Omega$ is countable, we take $\mathcal{F} = 2^\Omega$. A probability measure is determined by a probability mass function (PMF) $p: \Omega \to [0,1]$ with $\sum_{\omega \in \Omega} p(\omega) = 1$, via: $$P(A) = \sum_{\omega \in A} p(\omega)$$

For a fair die, $p(\omega) = 1/6$ for each $\omega \in \{1,\ldots,6\}$.

Definition 1.2.5 — Continuous Probability Space

When $\Omega = \mathbb{R}$ and $\mathcal{F} = \mathcal{B}(\mathbb{R})$, a probability measure is determined by a probability density function (PDF) $f: \mathbb{R} \to [0,\infty)$ with $\int_{-\infty}^\infty f(x)\,dx = 1$, via: $$P(A) = \int_A f(x)\,dx$$

Note $P(\{x\}) = 0$ for every single point $x$ in the continuous case.

Example 1.2.6 — Inclusion-Exclusion

For three events $A$, $B$, $C$: $$P(A \cup B \cup C) = P(A)+P(B)+P(C) - P(A\cap B) - P(A\cap C) - P(B\cap C) + P(A\cap B\cap C)$$

In general for $n$ events: $P\!\left(\bigcup_{i=1}^n A_i\right) = \sum_k (-1)^{k-1} \sum_{|S|=k} P\!\left(\bigcap_{i\in S} A_i\right)$.
