4.2 Linear SDEs

A linear SDE is one in which the coefficients are linear (affine) functions of the state. Such equations can always be solved explicitly using an integrating factor, a technique made rigorous by Itô's formula.

Definition 4.2.1 - General Linear SDE

A general linear SDE has the form

$$dX_t = \bigl(a(t)X_t + b(t)\bigr)dt + \bigl(c(t)X_t + d(t)\bigr)dW_t, \quad X_0 = x_0,$$ where $a, b, c, d: [0,T] \to \mathbb{R}$ are deterministic, continuous functions.

Solving the Homogeneous Linear SDE

We first consider the homogeneous case $b \equiv 0$, $d \equiv 0$:

$$dX_t = a(t)X_t\,dt + c(t)X_t\,dW_t.$$
Theorem 4.2.2 - Explicit Solution of the Homogeneous Linear SDE

The solution of $dX = a(t)X\,dt + c(t)X\,dW$ with $X_0 = x_0$ is

$$X_t = x_0 \exp\!\left(\int_0^t \!\bigl(a(s) - \tfrac{1}{2}c(s)^2\bigr)ds + \int_0^t c(s)\,dW_s\right).$$
Proof via Itô's Formula

Let $Z_t = \int_0^t (a(s) - \frac{1}{2}c(s)^2)ds + \int_0^t c(s)\,dW_s$, so $dZ_t = (a - \frac{1}{2}c^2)dt + c\,dW_t$. Apply Itô's formula to $f(z) = x_0 e^z$:

$$d(x_0 e^{Z_t}) = x_0 e^{Z_t} dZ_t + \frac{1}{2}x_0 e^{Z_t}(dZ_t)^2.$$ Since $(dZ_t)^2 = c(t)^2\,dt$: $$= x_0 e^{Z_t}\!\left[\bigl(a - \tfrac{1}{2}c^2\bigr)dt + c\,dW_t + \tfrac{1}{2}c^2\,dt\right] = x_0 e^{Z_t}\bigl(a\,dt + c\,dW_t\bigr) = a(t)X_t\,dt + c(t)X_t\,dW_t.$$ Hence $X_t = x_0 e^{Z_t}$ satisfies the SDE. $\square$

The Ornstein-Uhlenbeck Process

Definition 4.2.3 - Ornstein-Uhlenbeck Process

The Ornstein-Uhlenbeck (OU) process satisfies the SDE

$$dX_t = \theta(\mu - X_t)\,dt + \sigma\,dW_t, \quad X_0 = x_0,$$

where $\theta > 0$ is the mean-reversion speed, $\mu \in \mathbb{R}$ the long-run mean, and $\sigma > 0$ the diffusion coefficient. This is the canonical linear SDE with additive noise, widely used in interest rate modelling (Vasicek model) and physics (velocity in a fluid).

Theorem 4.2.4 - Explicit Solution of the OU Process

The OU process has the explicit solution

$$X_t = \mu + (x_0 - \mu)e^{-\theta t} + \sigma \int_0^t e^{-\theta(t-s)}\,dW_s.$$

Consequently, $X_t$ is Gaussian with

$$\mathbb{E}[X_t] = \mu + (x_0 - \mu)e^{-\theta t}, \qquad \mathrm{Var}(X_t) = \frac{\sigma^2}{2\theta}\bigl(1 - e^{-2\theta t}\bigr).$$
Proof via Integrating Factor

Multiply both sides of the SDE by the integrating factor $e^{\theta t}$ and apply Itô's product rule to $Y_t = e^{\theta t} X_t$:

$$dY_t = \theta e^{\theta t} X_t\,dt + e^{\theta t}\,dX_t = e^{\theta t}\bigl[\theta X_t\,dt + \theta(\mu - X_t)\,dt + \sigma\,dW_t\bigr] = \theta\mu e^{\theta t}\,dt + \sigma e^{\theta t}\,dW_t.$$ Integrating from $0$ to $t$: $$Y_t - x_0 = \mu(e^{\theta t} - 1) + \sigma\int_0^t e^{\theta s}\,dW_s.$$ Dividing by $e^{\theta t}$ gives the result. $\square$
Remark 4.2.5 - Stationary Distribution

As $t \to \infty$, $\mathbb{E}[X_t] \to \mu$ and $\mathrm{Var}(X_t) \to \frac{\sigma^2}{2\theta}$. The process converges in distribution to $\mathcal{N}\!\left(\mu, \frac{\sigma^2}{2\theta}\right)$, which is the unique stationary distribution of the OU process. This makes it the prototypical mean-reverting diffusion.

Example 4.2.6 - General Inhomogeneous Linear SDE

For the full equation $dX = (aX + b)\,dt + (cX + d)\,dW$, define the fundamental solution $\Phi_t$ by $d\Phi = a\Phi\,dt + c\Phi\,dW$, $\Phi_0 = 1$, i.e.,

$$\Phi_t = \exp\!\left(\bigl(a - \tfrac{1}{2}c^2\bigr)t + cW_t\right).$$ Then by the variation-of-constants formula: $$X_t = \Phi_t\!\left[x_0 + \int_0^t \Phi_s^{-1}(b - cd)\,ds + \int_0^t \Phi_s^{-1} d\,dW_s\right].$$ This reduces to the OU formula when $c = 0$ and generalises it to the multiplicative noise case.
