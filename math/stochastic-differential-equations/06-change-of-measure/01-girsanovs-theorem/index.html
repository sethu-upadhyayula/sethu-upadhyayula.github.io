6.1 Girsanov's Theorem

Girsanov's theorem specifies precisely how a Brownian motion with drift can be transformed into a standard Brownian motion by passing to an equivalent probability measure. It is the stochastic-calculus analogue of the classical change-of-variables formula for probability densities, and it is the mathematical foundation for risk-neutral pricing in mathematical finance.

Radon-Nikodym Derivative and Equivalent Measures

Definition 6.1.1 — Equivalent Probability Measures

Two probability measures $P$ and $Q$ on $(\Omega, \mathcal{F})$ are equivalent, written $P \sim Q$, if they have the same null sets: $P(A) = 0 \iff Q(A) = 0$. By the Radon-Nikodym theorem, $Q \ll P$ (meaning $Q$ is absolutely continuous with respect to $P$) if and only if there exists a non-negative $\mathcal{F}$-measurable random variable $Z$ — the Radon-Nikodym derivative — such that $Q(A) = E^P[Z\,\mathbf{1}_A]$ for all $A \in \mathcal{F}$. We write $dQ/dP = Z$. Equivalence $P \sim Q$ holds iff $Z > 0$ $P$-a.s.

The conditional Radon-Nikodym derivative is $Z_t = E^P[Z \mid \mathcal{F}_t]$, and $(Z_t)_{t \geq 0}$ is a non-negative $P$-martingale.

The Girsanov Exponential

Definition 6.1.2 — Doléans-Dade Exponential / Girsanov Density

Let $(\theta_t)_{0 \leq t \leq T}$ be a progressively measurable process. Define the Girsanov exponential (Doléans-Dade exponential of $-\int \theta\,dW$):

$$Z_t = \exp\!\left(-\int_0^t \theta_s\,dW_s - \frac{1}{2}\int_0^t \theta_s^2\,ds\right), \quad 0 \leq t \leq T.$$

By Itô's formula, $Z$ satisfies $dZ_t = -\theta_t Z_t\,dW_t$, so $Z$ is a non-negative local martingale with $Z_0 = 1$.

Novikov's condition: If $E^P\!\left[\exp\!\left(\frac{1}{2}\int_0^T \theta_s^2\,ds\right)\right] < \infty$, then $Z$ is a true $P$-martingale and in particular $E^P[Z_T] = 1$, so $dQ/dP = Z_T$ defines a genuine probability measure $Q \sim P$.

Girsanov's Theorem

Theorem 6.1.3 — Girsanov's Theorem

Let $W$ be a $P$-Brownian motion and let $\theta$ be progressively measurable satisfying Novikov's condition. Define $Q$ by $dQ/dP = Z_T$ as above. Then the process

$$\widetilde{W}_t = W_t + \int_0^t \theta_s\,ds, \quad 0 \leq t \leq T,$$

is a standard Brownian motion under $Q$.

Proof Sketch via Lévy's Characterisation

By Lévy's characterisation, it suffices to show that $\widetilde{W}$ is a continuous $Q$-martingale with quadratic variation $[\widetilde{W}]_t = t$. The quadratic variation $[\widetilde{W}]_t = [W]_t = t$ follows since adding the bounded variation term $\int \theta\,ds$ does not change the quadratic variation.

For the martingale property: any $P$-local martingale $M$ satisfies $M_t - \int_0^t \frac{d[M, Z]_s}{Z_s}$ is a $Q$-local martingale (Girsanov's formula). Apply this to $M = \widetilde{W}$: since $d[\widetilde{W}, Z]_t = d[W, Z]_t = -\theta_t Z_t\,dt$, the correction term is $-\int_0^t \theta_s\,ds$. Thus $\widetilde{W}_t - \int_0^t \theta_s\,ds + \int_0^t \theta_s\,ds = \widetilde{W}_t$ is a $Q$-local martingale. Being continuous and with finite variation term absent, it is a $Q$-martingale. $\square$

Corollary 6.1.4 — Change of Drift

Suppose under $P$ the process $X$ satisfies $dX_t = b(t,X_t)\,dt + \sigma(t,X_t)\,dW_t$. With $\theta_t$ such that $\sigma\theta = b - \tilde{b}$ for some target drift $\tilde{b}$, under $Q$ (with $dQ/dP = Z_T$) the process $X$ satisfies

$$dX_t = \tilde{b}(t,X_t)\,dt + \sigma(t,X_t)\,d\widetilde{W}_t.$$

In particular, choosing $\tilde{b} = 0$ removes the drift entirely, and choosing $\tilde{b} = r\sigma/\sigma = r$ (in one dimension) converts physical drift $\mu$ to risk-free rate $r$.

Proposition 6.1.5 — Abstract Bayes Formula

Let $Y$ be an integrable random variable and $0 \leq t \leq T$. Then

$$E^Q[Y \mid \mathcal{F}_t] = \frac{E^P[Z_T Y \mid \mathcal{F}_t]}{E^P[Z_T \mid \mathcal{F}_t]} = \frac{E^P[Z_T Y \mid \mathcal{F}_t]}{Z_t}.$$

This formula is essential for computing conditional expectations under $Q$ using $P$-based calculations.
