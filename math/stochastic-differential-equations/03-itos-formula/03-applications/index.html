3.3 Applications of Itô's Formula

We present three canonical applications of Itô's formula: the log transformation that solves GBM, the stochastic exponential (Doléans-Dade exponential) which is fundamental to Girsanov's theorem, and a preview of how the Black-Scholes PDE emerges from a hedging argument.

Application 1: Geometric Brownian Motion

Example 3.3.1 — Log Dynamics of GBM

Let $S_t$ satisfy $dS = \mu S\,dt + \sigma S\,dW$. Apply Itô's formula to $f(S) = \ln S$ (so $f_x = 1/S$, $f_{xx} = -1/S^2$, $f_t = 0$):

$$d\ln S_t = \frac{1}{S_t}\,dS_t - \frac{1}{2S_t^2}\sigma^2 S_t^2\,dt = \mu\,dt + \sigma\,dW_t - \frac{\sigma^2}{2}\,dt = \left(\mu - \frac{\sigma^2}{2}\right)dt + \sigma\,dW_t.$$

Since the right-hand side has constant coefficients, we can integrate directly:

$$\ln S_T - \ln S_0 = \left(\mu - \frac{\sigma^2}{2}\right)T + \sigma W_T,$$ $$S_T = S_0 \exp\!\left[\left(\mu - \frac{\sigma^2}{2}\right)T + \sigma W_T\right].$$

The $-\sigma^2/2$ correction is the Itô term. Without it, one would obtain the wrong drift: $e^{\mu T}$ versus $E[S_T] = S_0 e^{\mu T}$ (which is consistent: $E[e^{\sigma W_T}] = e^{\sigma^2 T/2}$ absorbs the correction).

Application 2: The Stochastic Exponential

Definition 3.3.2 — Doléans-Dade Exponential

For an adapted process $\theta$, the stochastic exponential (or Doléans-Dade exponential) is:

$$\mathcal{E}(\theta \cdot W)_t = \exp\!\left(\int_0^t \theta_s\,dW_s - \frac{1}{2}\int_0^t \theta_s^2\,ds\right).$$

It is the unique solution to $dZ_t = Z_t \theta_t\,dW_t$, $Z_0 = 1$.

Theorem 3.3.3 — Stochastic Exponential is a Local Martingale

$Z_t = \mathcal{E}(\theta \cdot W)_t$ is always a local martingale (since $dZ_t = Z_t\theta_t\,dW_t$ has no drift). Under Novikov's condition $E[e^{\frac{1}{2}\int_0^T\theta_s^2\,ds}] < \infty$, $Z_t$ is a true martingale with $E[Z_T] = 1$.

Proof — Via Itô's Formula

Let $M_t = \int_0^t\theta_s\,dW_s$ and $A_t = \frac{1}{2}\int_0^t\theta_s^2\,ds$. Then $Z_t = e^{M_t - A_t}$. Apply Itô's formula to $f(m,a) = e^{m-a}$:

$$dZ_t = e^{M_t-A_t}(dM_t - dA_t) + \frac{1}{2}e^{M_t-A_t}(dM_t)^2 = Z_t(\theta_t\,dW_t - \tfrac{1}{2}\theta_t^2\,dt + \tfrac{1}{2}\theta_t^2\,dt) = Z_t\theta_t\,dW_t.$$

So $Z_t$ satisfies $dZ_t = Z_t\theta_t\,dW_t$ — a pure martingale SDE with no drift.

Application 3: Black-Scholes PDE Preview

Example 3.3.4 — Discounted Price Process

Let $V_t = V(t, S_t)$ be an option price and $B_t = e^{rt}$ a bond. Apply Itô to $f(t,S) = e^{-rt}V(t,S)$:

$$d(e^{-rt}V) = e^{-rt}\left[-rV + V_t + \mu S V_S + \frac{1}{2}\sigma^2 S^2 V_{SS}\right]dt + e^{-rt}\sigma S V_S\,dW.$$

For $e^{-rt}V_t$ to be a martingale (no drift), we need the coefficient of $dt$ to vanish:

$$V_t + \mu S V_S + \frac{1}{2}\sigma^2 S^2 V_{SS} = rV.$$

Replacing $\mu$ by $r$ (risk-neutral pricing) gives the Black-Scholes PDE: $V_t + rSV_S + \frac{1}{2}\sigma^2 S^2 V_{SS} - rV = 0$. The full derivation via the delta-hedging argument appears in the Financial Mathematics notes.
