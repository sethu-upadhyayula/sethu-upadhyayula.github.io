4.1 Definition and Examples

A ring equips a set with two operations — addition and multiplication — where addition forms an abelian group and multiplication distributes over addition. Rings generalise the integers and arise throughout algebra, number theory, and geometry.

Definition 4.1.1 — Ring

A ring is a triple $(R, +, \cdot)$ where $R$ is a set with two binary operations satisfying:

  1. $(R, +)$ is an abelian group (with identity $0_R$).
  2. $\cdot$ is associative: $(ab)c = a(bc)$ for all $a,b,c \in R$.
  3. Distributivity: $a(b+c) = ab + ac$ and $(a+b)c = ac + bc$ for all $a,b,c \in R$.

Additional properties define special classes:

  • Commutative ring: $ab = ba$ for all $a,b \in R$.
  • Ring with unity: there exists $1_R \in R$ with $1_R \cdot a = a \cdot 1_R = a$ for all $a$.
  • Zero divisor: $a \ne 0$ such that $ab = 0$ for some $b \ne 0$.
  • Integral domain: commutative ring with unity and no zero divisors.
  • Field: commutative ring with unity where every nonzero element has a multiplicative inverse.

Examples

Example 4.1.2 — The Integers

$(\mathbb{Z}, +, \cdot)$ is a commutative ring with unity. It is an integral domain: if $ab = 0$ in $\mathbb{Z}$ then $a = 0$ or $b = 0$. It is not a field: $2$ has no multiplicative inverse in $\mathbb{Z}$.

Example 4.1.3 — Fields $\mathbb{Q}$, $\mathbb{R}$, $\mathbb{C}$

The rationals $\mathbb{Q}$, reals $\mathbb{R}$, and complex numbers $\mathbb{C}$ are all fields under standard addition and multiplication. Every field is an integral domain.

Example 4.1.4 — $\mathbb{Z}/n\mathbb{Z}$

$\mathbb{Z}/n\mathbb{Z}$ is a commutative ring with unity. It is an integral domain if and only if $n$ is prime (since $[a][b] = [0]$ iff $n \mid ab$, iff $n \mid a$ or $n \mid b$ when $n$ is prime). It is a field if and only if $n$ is prime.

Example 4.1.5 — Matrix Ring $M_n(\mathbb{R})$

The set of $n \times n$ real matrices under matrix addition and multiplication is a ring. For $n \ge 2$ it is non-commutative and has zero divisors: e.g., the matrix with $1$ in position $(1,2)$ and $0$ elsewhere multiplied by itself gives $0$.

Example 4.1.6 — Gaussian Integers $\mathbb{Z}[i]$

$\mathbb{Z}[i] = \{a + bi : a,b \in \mathbb{Z}\}$ with $(a+bi) + (c+di) = (a+c) + (b+d)i$ and $(a+bi)(c+di) = (ac-bd) + (ad+bc)i$ is a commutative ring with unity $1$. It is an integral domain (since $\mathbb{Z}[i] \subset \mathbb{C}$ which is a field). It is not a field: $2$ has no inverse in $\mathbb{Z}[i]$.

Example 4.1.7 — $\mathbb{Z}[\sqrt{-5}]$

$\mathbb{Z}[\sqrt{-5}] = \{a + b\sqrt{-5} : a,b \in \mathbb{Z}\}$ is a commutative integral domain. Unlike $\mathbb{Z}[i]$, it is not a unique factorisation domain: $6 = 2 \cdot 3 = (1+\sqrt{-5})(1-\sqrt{-5})$ gives two distinct factorisations into irreducibles.

Basic Properties

Theorem 4.1.8 — Elementary Ring Properties

In any ring $R$:

  • $0 \cdot a = a \cdot 0 = 0$ for all $a \in R$.
  • $(-a)b = a(-b) = -(ab)$ for all $a,b \in R$.
  • If $R$ has unity: $(-1)a = -a$ for all $a \in R$.
  • In an integral domain, cancellation holds: $ac = bc$ and $c \ne 0$ implies $a = b$.
