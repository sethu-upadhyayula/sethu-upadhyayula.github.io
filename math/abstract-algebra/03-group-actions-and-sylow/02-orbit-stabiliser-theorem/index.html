3.2 Orbit–Stabiliser Theorem

The orbit-stabiliser theorem is the key counting tool for group actions. It relates the size of an orbit to the index of the stabiliser, generalising Lagrange's theorem and leading to the class equation and Burnside's lemma.

Theorem 3.2.1 — Orbit–Stabiliser Theorem

Let $G$ be a finite group acting on a set $X$, and let $x \in X$. Then: $$|G \cdot x| \cdot |G_x| = |G|$$ equivalently, $|G \cdot x| = [G : G_x] = \dfrac{|G|}{|G_x|}$.

Proof

Define $\varphi : G/G_x \to G \cdot x$ by $\varphi(gG_x) = g \cdot x$.

  • Well-defined: If $gG_x = hG_x$ then $h^{-1}g \in G_x$, so $h^{-1}g \cdot x = x$, giving $g \cdot x = h \cdot x$.
  • Injective: If $g \cdot x = h \cdot x$ then $h^{-1}g \cdot x = x$, so $h^{-1}g \in G_x$, giving $gG_x = hG_x$.
  • Surjective: Every element of $G \cdot x$ has the form $g \cdot x = \varphi(gG_x)$.

So $|G \cdot x| = |G/G_x| = [G:G_x] = |G|/|G_x|$. $\square$

The Class Equation

Theorem 3.2.2 — Class Equation

Let $G$ act on itself by conjugation. The orbits are the conjugacy classes. Let $x_1, \ldots, x_k$ be representatives of conjugacy classes of size $> 1$. Then: $$|G| = |Z(G)| + \sum_{i=1}^{k} [G : C_G(x_i)]$$ where $C_G(x) = \{g \in G : gx = xg\}$ is the centraliser of $x$ (equal to the stabiliser under conjugation).

Example 3.2.3 — Groups of Order $p^2$

Let $|G| = p^2$ for a prime $p$. The class equation gives $p^2 = |Z(G)| + \sum [G : C_G(x_i)]$. Each term $[G:C_G(x_i)]$ divides $p^2$ and is $> 1$, so equals $p$ or $p^2$. But $[G:C_G(x_i)] = p^2$ forces $C_G(x_i) = \{e\}$, impossible since $e \in C_G(x_i)$. Hence each non-central class has size $p$, and $p \mid |Z(G)|$. Since $|Z(G)| \ge 1$ and $p \mid |Z(G)|$, we get $|Z(G)| \ge p$. One shows $Z(G) \ne G$ leads to a contradiction via cyclic quotients. Therefore $Z(G) = G$, i.e., every group of order $p^2$ is abelian.

Burnside's Lemma

Theorem 3.2.4 — Burnside's Lemma

Let $G$ be a finite group acting on a finite set $X$. The number of distinct orbits is: $$|\text{Orbits}| = \frac{1}{|G|} \sum_{g \in G} |X^g|$$ where $X^g = \{x \in X : g \cdot x = x\}$ is the set of points fixed by $g$.

Example 3.2.5 — Colouring a Necklace

Count distinct necklaces with 4 beads, each coloured red or blue (rotational equivalence only). The rotation group is $\mathbb{Z}_4 = \{r^0, r^1, r^2, r^3\}$ acting on $2^4 = 16$ colourings.

  • $r^0$ (identity): fixes all 16 colourings.
  • $r^1$ (rotate by 90°): fixes colourings where all beads are the same colour — 2 fixed.
  • $r^2$ (rotate by 180°): fixes colourings where bead 1 = bead 3 and bead 2 = bead 4 — $2^2 = 4$ fixed.
  • $r^3$ (rotate by 270°): same as $r^1$ — 2 fixed.

Distinct necklaces $= \tfrac{1}{4}(16 + 2 + 4 + 2) = \tfrac{24}{4} = 6$.
