2.1 Homomorphisms and Isomorphisms

A group homomorphism is a map between groups that respects the group structure. Homomorphisms and isomorphisms are the principal tools for comparing groups and transferring properties between them.

Definition 2.1.1 — Group Homomorphism

A map $\phi : G \to H$ between groups is a homomorphism if for all $a, b \in G$: $$\phi(a \cdot b) = \phi(a) \cdot \phi(b).$$ (Here $\cdot$ denotes the group operations in $G$ and $H$ respectively.)

Theorem 2.1.2 — Basic Properties of Homomorphisms

Let $\phi : G \to H$ be a group homomorphism. Then:

  1. $\phi(e_G) = e_H$.
  2. $\phi(g^{-1}) = \phi(g)^{-1}$ for all $g \in G$.
  3. $\phi(g^n) = \phi(g)^n$ for all $g \in G$, $n \in \mathbb{Z}$.
  4. If $|g|$ is finite, then $|\phi(g)|$ divides $|g|$.
Proof

For (1): $\phi(e_G) = \phi(e_G \cdot e_G) = \phi(e_G) \cdot \phi(e_G)$. Cancelling $\phi(e_G)$ gives $e_H = \phi(e_G)$.

For (2): $\phi(g)\phi(g^{-1}) = \phi(gg^{-1}) = \phi(e_G) = e_H$, so $\phi(g^{-1}) = \phi(g)^{-1}$. $\square$

Definition 2.1.3 — Types of Homomorphisms
  • Isomorphism: a bijective homomorphism. Write $G \cong H$.
  • Endomorphism: a homomorphism $G \to G$.
  • Automorphism: a bijective endomorphism.
  • Monomorphism: an injective homomorphism.
  • Epimorphism: a surjective homomorphism.

Kernel and Image

Definition 2.1.4 — Kernel and Image

Let $\phi : G \to H$ be a homomorphism.

  • The kernel is $\ker\phi = \{g \in G : \phi(g) = e_H\} = \phi^{-1}(e_H)$.
  • The image is $\text{im}\,\phi = \phi(G) = \{\phi(g) : g \in G\}$.
Theorem 2.1.5

Let $\phi : G \to H$ be a homomorphism. Then:

  1. $\ker\phi \trianglelefteq G$ (the kernel is a normal subgroup of $G$).
  2. $\text{im}\,\phi \le H$ (the image is a subgroup of $H$).
  3. $\phi$ is injective if and only if $\ker\phi = \{e_G\}$.
Proof of (3)

($\Rightarrow$) If $\phi$ is injective and $\phi(g) = e_H = \phi(e_G)$, then $g = e_G$.

($\Leftarrow$) If $\ker\phi = \{e_G\}$ and $\phi(g_1) = \phi(g_2)$, then $\phi(g_1 g_2^{-1}) = \phi(g_1)\phi(g_2)^{-1} = e_H$, so $g_1 g_2^{-1} \in \ker\phi = \{e_G\}$, giving $g_1 = g_2$. $\square$

Example 2.1.6
  • $\text{sgn} : S_n \to \{+1,-1\}$ is a surjective homomorphism with $\ker(\text{sgn}) = A_n$.
  • $\det : GL_n(\mathbb{R}) \to \mathbb{R}^*$ is a surjective homomorphism with $\ker(\det) = SL_n(\mathbb{R})$.
  • $\phi : \mathbb{Z} \to \mathbb{Z}/n\mathbb{Z}$, $\phi(k) = [k]$, is a surjective homomorphism with $\ker\phi = n\mathbb{Z}$.
  • $\exp : (\mathbb{R}, +) \to (\mathbb{R}^*, \cdot)$ given by $\exp(x) = e^x$ is an isomorphism (it is bijective and $e^{x+y} = e^x e^y$).
