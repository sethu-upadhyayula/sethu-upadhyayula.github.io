3.1 Limits of Functions

The limit of a function formalises the idea that $f(x)$ can be made arbitrarily close to $L$ by restricting $x$ close enough to $c$, without requiring anything about the value of $f$ at $c$ itself.

Definition 3.1.1 - Limit of a Function ($\varepsilon$-$\delta$)

Let $f$ be defined on a punctured neighbourhood of $c$. We write $\lim_{x\to c} f(x) = L$ if: $\forall\varepsilon > 0,\, \exists\delta > 0$ such that $0 < |x - c| < \delta \Rightarrow |f(x) - L| < \varepsilon$. Note $x = c$ is excluded; $f$ need not even be defined at $c$.

Theorem 3.1.2 - Heine's Theorem (Sequential Characterisation)

$\lim_{x\to c} f(x) = L$ if and only if $f(x_n) \to L$ for every sequence $(x_n)$ with $x_n \to c$ and $x_n \ne c$. This equivalence is extremely useful: to prove a limit, show every such sequence gives $L$; to disprove a limit, exhibit two sequences approaching $c$ that give different values.

Example 3.1.3 - Non-existence via Sequential Criterion

Let $f(x) = \sin(1/x)$ for $x \ne 0$. Along $x_n = 1/(2\pi n)$ we get $f(x_n) = 0$; along $y_n = 1/(\pi/2 + 2\pi n)$ we get $f(y_n) = 1$. Since two sequences tending to 0 give different values, $\lim_{x\to 0} f(x)$ does not exist.

Extensions of the Limit Concept

Definition 3.1.4 - One-Sided Limits

The right-hand limit $\lim_{x\to c^+} f(x) = L$ requires $0 < x - c < \delta \Rightarrow |f(x) - L| < \varepsilon$. The left-hand limit $\lim_{x\to c^-}$ is defined analogously.

$\lim_{x\to c} f(x)$ exists iff both one-sided limits exist and are equal.

Definition 3.1.5 - Infinite Limits and Limits at Infinity

$\lim_{x\to c} f(x) = +\infty$: $\forall M > 0,\, \exists\delta > 0,\, 0 < |x-c| < \delta \Rightarrow f(x) > M$.

$\lim_{x\to\infty} f(x) = L$: $\forall\varepsilon > 0,\, \exists R > 0,\, x > R \Rightarrow |f(x) - L| < \varepsilon$.

Limit Algebra and Squeeze Theorem

Theorem 3.1.6 - Limit Operations

If $\lim_{x\to c} f(x) = L$ and $\lim_{x\to c} g(x) = M$, then:

  • $\lim_{x\to c}(f \pm g)(x) = L \pm M$
  • $\lim_{x\to c}(fg)(x) = LM$
  • $\lim_{x\to c}(f/g)(x) = L/M$ provided $M \ne 0$
Theorem 3.1.7 - Squeeze Theorem for Functions

If $g(x) \le f(x) \le h(x)$ near $c$ (excluding $c$) and $\lim_{x\to c} g(x) = \lim_{x\to c} h(x) = L$, then $\lim_{x\to c} f(x) = L$. Standard application: $\lim_{x\to 0} x\sin(1/x) = 0$ since $|x\sin(1/x)| \le |x| \to 0$.
