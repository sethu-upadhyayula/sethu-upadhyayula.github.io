5.3 Improper Integrals

Improper integrals extend the Riemann integral to unbounded intervals or functions with singularities, defined as limits of proper integrals.

Definition 5.3.1 — Type I: Infinite Limits

$\displaystyle\int_a^\infty f(x)\,dx = \lim_{R\to\infty} \int_a^R f(x)\,dx$, provided the limit exists and is finite. Similarly for $\int_{-\infty}^b$ and $\int_{-\infty}^\infty f = \int_{-\infty}^c f + \int_c^\infty f$ (must both converge).

Definition 5.3.2 — Type II: Singularity at Endpoint

If $f$ is unbounded near $a$: $\displaystyle\int_a^b f(x)\,dx = \lim_{\varepsilon \to 0^+} \int_{a+\varepsilon}^b f(x)\,dx$. Singularities in the interior are split into two Type II integrals.

Theorem 5.3.3 — Comparison Test

If $0 \le f(x) \le g(x)$ for all large $x$ and $\int_a^\infty g < \infty$, then $\int_a^\infty f < \infty$. If $\int_a^\infty f = \infty$, then $\int_a^\infty g = \infty$. Absolute convergence ($\int|f| < \infty$) implies convergence.

Example 5.3.4 — $p$-Test for Integrals

$$\int_1^\infty x^{-p}\,dx \text{ converges} \iff p > 1. \qquad \int_0^1 x^{-p}\,dx \text{ converges} \iff p < 1.$$

For $p \ne 1$: $\int_1^R x^{-p}\,dx = \dfrac{R^{1-p} - 1}{1-p}$, which tends to $\dfrac{1}{p-1}$ as $R \to \infty$ iff $p > 1$. At $p = 1$: $\int_1^R x^{-1}\,dx = \ln R \to \infty$.

Definition 5.3.5 — The Gamma Function

For $s > 0$: $\displaystyle\Gamma(s) = \int_0^\infty t^{s-1} e^{-t}\,dt$. The integral converges (Type II near 0 when $s < 1$, Type I at $\infty$).

Integration by parts gives $\Gamma(s+1) = s\Gamma(s)$. Since $\Gamma(1) = 1$, we get $\Gamma(n) = (n-1)!$ for positive integers.

The value $\Gamma(1/2) = \sqrt{\pi}$ follows from the Gaussian integral computation below.

Example 5.3.6 — Gaussian Integral

$$\int_{-\infty}^\infty e^{-x^2}\,dx = \sqrt{\pi}.$$

Proof: let $I = \int_{-\infty}^\infty e^{-x^2}\,dx$. Then $I^2 = \int_{-\infty}^\infty\int_{-\infty}^\infty e^{-(x^2+y^2)}\,dx\,dy$. Convert to polar coordinates: $I^2 = \int_0^{2\pi}\int_0^\infty e^{-r^2} r\,dr\,d\theta = 2\pi \cdot \frac{1}{2} = \pi$. Hence $I = \sqrt{\pi}$.
