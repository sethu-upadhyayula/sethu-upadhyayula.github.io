2.3 Series and Convergence Tests

A series $\sum_{n=1}^\infty a_n$ is defined by its sequence of partial sums $S_N = \sum_{n=1}^N a_n$. The series converges to $S$ if $S_N \to S$, and diverges otherwise. A rich toolkit of convergence tests allows us to decide convergence without computing the limit explicitly.

Definition 2.3.1 - Series and Partial Sums

Given a sequence $(a_n)$, the series $\sum_{n=1}^\infty a_n$ is said to converge to $S\in\mathbb{R}$ if the partial sums $S_N = \sum_{n=1}^N a_n$ satisfy $S_N \to S$. Otherwise the series diverges.

Theorem 2.3.2 - Divergence Test

If $\sum a_n$ converges, then $a_n \to 0$. Equivalently: if $a_n \not\to 0$, then $\sum a_n$ diverges. Proof: $a_n = S_n - S_{n-1} \to S - S = 0$. The converse fails: $\sum 1/n$ diverges despite $1/n \to 0$.

Comparison Tests

Theorem 2.3.3 - Direct Comparison Test

Suppose $0 \le a_n \le b_n$ for all large $n$.

  • If $\sum b_n$ converges, then $\sum a_n$ converges.
  • If $\sum a_n$ diverges, then $\sum b_n$ diverges.
Theorem 2.3.4 - Limit Comparison Test

If $a_n, b_n > 0$ and $\lim_{n\to\infty} a_n/b_n = L$ with $0 < L < \infty$, then $\sum a_n$ and $\sum b_n$ either both converge or both diverge. If $L = 0$ and $\sum b_n < \infty$, then $\sum a_n < \infty$.

Ratio and Root Tests

Theorem 2.3.5 - Ratio Test

Let $r = \limsup_{n\to\infty} |a_{n+1}/a_n|$.

  • $r < 1$: $\sum a_n$ converges absolutely.
  • $r > 1$: $\sum a_n$ diverges.
  • $r = 1$: inconclusive (both $\sum 1/n$ and $\sum 1/n^2$ give $r=1$).
Theorem 2.3.6 - Root Test (Cauchy)

Let $r = \limsup_{n\to\infty} |a_n|^{1/n}$.

  • $r < 1$: $\sum a_n$ converges absolutely.
  • $r > 1$: $\sum a_n$ diverges.
  • $r = 1$: inconclusive.

The root test is at least as powerful as the ratio test: if the ratio test gives $r$, then the root test gives the same $r$. Strictly more powerful in general.

Alternating Series and Absolute Convergence

Theorem 2.3.7 - Alternating Series Test (Leibniz)

If $(a_n)$ is a decreasing sequence with $a_n \searrow 0$, then the alternating series $\sum_{n=1}^\infty (-1)^{n+1} a_n$ converges. Moreover, the error in truncating after $N$ terms satisfies $|S - S_N| \le a_{N+1}$.

Example: $\sum(-1)^{n+1}/n = \ln 2$, with error at most $1/(N+1)$ after $N$ terms.

Definition 2.3.8 - Absolute vs Conditional Convergence

$\sum a_n$ converges absolutely if $\sum |a_n| < \infty$. It converges conditionally if $\sum a_n$ converges but $\sum |a_n| = \infty$.

Absolute convergence implies convergence. The converse is false: $\sum(-1)^{n+1}/n$ converges conditionally.

Dirichlet's Test and Rearrangements

Theorem 2.3.9 - Dirichlet's Test

If the partial sums of $\sum b_n$ are bounded and $(a_n)$ is monotone decreasing to 0, then $\sum a_n b_n$ converges. This generalises the alternating series test (take $b_n = (-1)^{n+1}$).

Theorem 2.3.10 - Riemann Rearrangement Theorem

If $\sum a_n$ converges conditionally, then for any $L \in \mathbb{R} \cup \{-\infty, +\infty\}$, there exists a rearrangement of the series that converges to $L$. In contrast, any rearrangement of an absolutely convergent series converges to the same sum.

Example 2.3.11 - $p$-Series

The series $\sum_{n=1}^\infty n^{-p}$ converges if and only if $p > 1$. Proof via Cauchy condensation: $\sum n^{-p}$ and $\sum 2^n (2^n)^{-p} = \sum 2^{n(1-p)}$ have the same convergence behaviour, and the latter is a geometric series converging iff $2^{1-p} < 1$, i.e.\ $p > 1$.
