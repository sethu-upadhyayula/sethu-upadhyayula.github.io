7.1 Metric Spaces and Topology

A metric space generalises the notion of distance from $\mathbb{R}^n$ to an abstract set, providing the minimal structure needed to discuss limits, continuity, open and closed sets, and convergence.

Metric Spaces

Definition 7.1.1 - Metric Space

A metric on a set $X$ is a function $d: X \times X \to [0,\infty)$ such that for all $x,y,z \in X$:

  • Positive-definiteness: $d(x,y) \ge 0$, with $d(x,y)=0$ if and only if $x=y$.
  • Symmetry: $d(x,y) = d(y,x)$.
  • Triangle inequality: $d(x,z) \le d(x,y) + d(y,z)$.

The pair $(X,d)$ is called a metric space.

Example 7.1.2 - $\mathbb{R}^n$ with Various Metrics

On $\mathbb{R}^n$, the following are all metrics:

  • Euclidean metric: $d_2(x,y) = \left(\sum_{i=1}^n (x_i-y_i)^2\right)^{1/2}$.
  • Taxicab (Manhattan) metric: $d_1(x,y) = \sum_{i=1}^n |x_i-y_i|$.
  • Sup metric: $d_\infty(x,y) = \max_{1\le i\le n} |x_i-y_i|$.

More generally, $d_p(x,y) = \left(\sum_{i=1}^n |x_i-y_i|^p\right)^{1/p}$ for $1 \le p < \infty$ is a metric (the triangle inequality is the Minkowski inequality).

Example 7.1.3 - Discrete Metric

On any nonempty set $X$, define $d(x,y) = 0$ if $x=y$ and $d(x,y)=1$ if $x \ne y$. This is the discrete metric; every subset of $X$ is both open and closed in the resulting topology.

Example 7.1.4 - $C[a,b]$ with the Sup Metric

Let $X = C[a,b]$, the space of continuous real-valued functions on $[a,b]$. Define $d(f,g) = \|f-g\|_\infty = \sup_{x\in[a,b]} |f(x)-g(x)|$. The triangle inequality follows from $|f(x)-h(x)| \le |f(x)-g(x)| + |g(x)-h(x)|$ for each $x$, taking the supremum. Convergence in this metric is exactly uniform convergence.

Open and Closed Sets

Definition 7.1.5 - Open and Closed Balls

For $x_0 \in X$ and $r>0$, the open ball of radius $r$ about $x_0$ is $$B(x_0,r) = \{x \in X : d(x,x_0) < r\},$$ and the closed ball is $$\overline{B}(x_0,r) = \{x \in X : d(x,x_0) \le r\}.$$

In the discrete metric, $B(x_0,r) = \{x_0\}$ for $r \le 1$ and $B(x_0,r)=X$ for $r>1$.

Definition 7.1.6 - Open Sets

A set $U \subseteq X$ is open if for every $x \in U$ there exists $r>0$ such that $B(x,r) \subseteq U$. A set $F \subseteq X$ is closed if its complement $X \setminus F$ is open.

A point $x \in X$ is a limit point of $A \subseteq X$ if every open ball $B(x,r)$ contains a point of $A$ other than $x$. The closure $\overline{A}$ is $A$ together with all its limit points; $A$ is closed iff $A = \overline{A}$.

Theorem 7.1.7 - Open Balls Are Open

For any $x_0 \in X$ and $r>0$, the open ball $B(x_0,r)$ is an open set.

Proof

Let $x \in B(x_0,r)$, so $d(x,x_0) < r$. Set $\rho = r - d(x,x_0) > 0$. We claim $B(x,\rho) \subseteq B(x_0,r)$: if $y \in B(x,\rho)$, then by the triangle inequality $$d(y,x_0) \le d(y,x) + d(x,x_0) < \rho + d(x,x_0) = r,$$ so $y \in B(x_0,r)$.

Hence every point of $B(x_0,r)$ has a ball around it contained in $B(x_0,r)$, so $B(x_0,r)$ is open. $\square$

Theorem 7.1.8 - The Topology Induced by a Metric

Let $\mathcal{T}_d$ be the collection of all open subsets of $(X,d)$ as in Definition 7.1.6. Then $\mathcal{T}_d$ satisfies:

  • $\emptyset$ and $X$ are open.
  • Arbitrary unions of open sets are open.
  • Finite intersections of open sets are open.

Hence $\mathcal{T}_d$ is a topology on $X$, called the topology induced by $d$. This is the framework that makes precise the notions of convergence and continuity in $X$.

Continuity in Metric Spaces

Definition 7.1.9 - Continuity

Let $(X,d_X)$ and $(Y,d_Y)$ be metric spaces. A function $f: X \to Y$ is continuous at $x_0 \in X$ if for every $\varepsilon>0$ there exists $\delta>0$ such that $$d_X(x,x_0)<\delta \implies d_Y(f(x),f(x_0))<\varepsilon.$$ We say $f$ is continuous if it is continuous at every point of $X$.

Theorem 7.1.10 - Topological Characterisation of Continuity

A function $f: X \to Y$ is continuous if and only if $f^{-1}(V)$ is open in $X$ for every open set $V \subseteq Y$.

Proof

($\Rightarrow$) Suppose $f$ is continuous and $V \subseteq Y$ is open. Let $x_0 \in f^{-1}(V)$, so $f(x_0) \in V$. Since $V$ is open, there is $\varepsilon>0$ with $B(f(x_0),\varepsilon) \subseteq V$. By continuity, choose $\delta>0$ such that $d_X(x,x_0)<\delta \implies d_Y(f(x),f(x_0))<\varepsilon$, i.e.\ $f(B(x_0,\delta)) \subseteq B(f(x_0),\varepsilon) \subseteq V$. So $B(x_0,\delta)\subseteq f^{-1}(V)$, showing $f^{-1}(V)$ is open.

($\Leftarrow$) Suppose preimages of open sets are open. Fix $x_0 \in X$ and $\varepsilon>0$. The ball $B(f(x_0),\varepsilon)$ is open in $Y$, so $f^{-1}(B(f(x_0),\varepsilon))$ is open in $X$ and contains $x_0$. Hence there is $\delta>0$ with $B(x_0,\delta) \subseteq f^{-1}(B(f(x_0),\varepsilon))$, which is exactly the $\varepsilon$-$\delta$ statement of continuity at $x_0$. $\square$

Equivalent Metrics

Definition 7.1.11 - Equivalent Metrics

Two metrics $d_1, d_2$ on $X$ are equivalent if there exist constants $0 < c \le C$ such that $$c\,d_1(x,y) \le d_2(x,y) \le C\,d_1(x,y) \quad \text{for all } x,y \in X.$$ Equivalent metrics induce the same open sets, hence the same topology, and therefore the same notions of convergence and continuity.

Example 7.1.12 - Equivalence of $d_1, d_2, d_\infty$ on $\mathbb{R}^n$

For all $x \in \mathbb{R}^n$, $$d_\infty(x,0) \le d_2(x,0) \le d_1(x,0) \le n\, d_\infty(x,0).$$ Hence $d_1, d_2, d_\infty$ are all equivalent on $\mathbb{R}^n$, and a sequence converges in one if and only if it converges in the others - coordinatewise convergence. (In infinite dimensions, e.g.\ on $C[a,b]$, analogous norms are not equivalent.)
