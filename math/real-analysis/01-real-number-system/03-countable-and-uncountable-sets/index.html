1.3 Countable and Uncountable Sets

Cantor's theory of infinite sets reveals a hierarchy of infinities. The key distinction is between sets that can be listed in a sequence (countable) and those that cannot (uncountable). The real line $\mathbb{R}$ is strictly larger than $\mathbb{Q}$ in this sense.

Cardinality

Definition 1.3.1 - Equipotent Sets and Cardinality

Two sets $A$ and $B$ are equipotent (or have the same cardinality), written $|A|=|B|$, if there exists a bijection $f:A\to B$.

$|A|\leq|B|$ if there exists an injection $f:A\to B$. We say $|A|<|B|$ if $|A|\leq|B|$ but $|A|\ne|B|$.

A set $A$ is countably infinite if $|A|=|\mathbb{N}|$, i.e.\ if there is a bijection $A\to\mathbb{N}$. It is countable if it is finite or countably infinite. Otherwise it is uncountable.

Countable Sets

Theorem 1.3.2 - $\mathbb{Z}$ is countable

The bijection $f:\mathbb{N}\to\mathbb{Z}$ defined by $f(n)=n/2$ if $n$ even, $f(n)=-(n+1)/2$ if $n$ odd gives $0,1,-1,2,-2,3,-3,\ldots$

Theorem 1.3.3 - $\mathbb{N}\times\mathbb{N}$ is countable

List pairs $(m,n)$ along diagonals: $(1,1),(1,2),(2,1),(1,3),(2,2),(3,1),\ldots$ This enumerates all of $\mathbb{N}\times\mathbb{N}$. More precisely, the function $(m,n)\mapsto\frac{(m+n-2)(m+n-1)}{2}+m$ is a bijection $\mathbb{N}\times\mathbb{N}\to\mathbb{N}$.

Theorem 1.3.4 - $\mathbb{Q}$ is countable

Every rational $p/q$ (in lowest terms, $q>0$) is a pair $(p,q)\in\mathbb{Z}\times\mathbb{N}$. Since $\mathbb{Z}$ and $\mathbb{N}$ are countable, $\mathbb{Z}\times\mathbb{N}$ is countable (a countable union of countable sets is countable). Since $\mathbb{Q}$ injects into $\mathbb{Z}\times\mathbb{N}$, $\mathbb{Q}$ is countable.

More concretely: arrange all fractions in a grid and traverse the diagonals, skipping reducible fractions. This gives an explicit enumeration of $\mathbb{Q}$.

Uncountability of $\mathbb{R}$

Theorem 1.3.5 - $\mathbb{R}$ is uncountable (Cantor's Diagonal Argument)

The interval $(0,1)$ is uncountable. Hence $\mathbb{R}$ is uncountable.

Proof

Suppose for contradiction that $(0,1)$ is countable, say $(0,1)=\{x_1,x_2,x_3,\ldots\}$. Write each $x_n$ in decimal: $x_n=0.d_{n1}d_{n2}d_{n3}\cdots$

Define $y=0.y_1y_2y_3\cdots$ where $y_n=5$ if $d_{nn}\ne5$, and $y_n=6$ if $d_{nn}=5$. Then $y\in(0,1)$ but $y\ne x_n$ for all $n$ (they differ in the $n$-th decimal place). This contradicts the assumption that $\{x_n\}$ lists all of $(0,1)$. $\square$

Theorem 1.3.6 - Cantor-Bernstein Theorem

If $|A|\leq|B|$ and $|B|\leq|A|$, then $|A|=|B|$. That is: if there are injections $f:A\to B$ and $g:B\to A$, then there is a bijection $h:A\to B$. This lets us compare cardinalities without constructing an explicit bijection.

Remark - Cardinalities of $\mathbb{R}$

One can show: $|\mathbb{R}|=|(0,1)|=|2^\mathbb{N}|=|\mathbb{R}^n|$ for any $n\geq1$. (The last equality uses Cantor-Bernstein and interleaving of decimal expansions.) The continuum hypothesis - whether there is a set of cardinality strictly between $|\mathbb{N}|$ and $|\mathbb{R}|$ - is independent of ZFC set theory.

The cardinality $|\mathbb{R}|$ is denoted $\mathfrak{c}$ (the continuum) or $2^{\aleph_0}$.
