1.2 The Completeness Axiom

The ordered field axioms are satisfied by $\mathbb{Q}$. What distinguishes $\mathbb{R}$ is the Completeness Axiom — the assertion that $\mathbb{R}$ has no "gaps". This single axiom is the source of all the deep theorems of analysis.

Axiom (Least Upper Bound Property / Dedekind Completeness)

Every nonempty subset $S\subseteq\mathbb{R}$ that is bounded above has a supremum (least upper bound) in $\mathbb{R}$: there exists $s\in\mathbb{R}$ such that (i) $s$ is an upper bound of $S$, and (ii) if $t$ is any upper bound of $S$ then $s\leq t$. We write $s=\sup S$.

Example — $\mathbb{Q}$ is not complete

Let $S=\{x\in\mathbb{Q}: x^2<2\}$. This set is nonempty ($1\in S$) and bounded above (e.g.\ by $2$). We claim $S$ has no supremum in $\mathbb{Q}$.

Any upper bound $q\in\mathbb{Q}$ satisfies $q^2\geq 2$. But $\sqrt{2}\notin\mathbb{Q}$, so there is no least rational upper bound. In $\mathbb{R}$, however, $\sup S=\sqrt{2}$.

This gap in $\mathbb{Q}$ — at $\sqrt{2}$ — illustrates precisely what completeness fills in.

Constructions of $\mathbb{R}$

Remark — Dedekind Cuts

One rigorous construction: a Dedekind cut is a pair $(A,B)$ with $A\cup B=\mathbb{Q}$, $A\cap B=\emptyset$, $A\ne\emptyset$, $B\ne\emptyset$, every element of $A$ is less than every element of $B$, and $A$ has no greatest element. Each real number corresponds to a unique cut. For example, $\sqrt{2}$ corresponds to $A=\{q\in\mathbb{Q}:q\leq0\text{ or }q^2<2\}$.

One verifies that the set of all Dedekind cuts, with the natural ordering and arithmetic operations, is a complete ordered field.

Remark — Cauchy Completion

An alternative construction: identify $\mathbb{R}$ with equivalence classes of Cauchy sequences in $\mathbb{Q}$, where $(a_n)\sim(b_n)$ iff $a_n-b_n\to0$. This construction makes the completeness of $\mathbb{R}$ transparent and generalises to arbitrary metric spaces.

Theorem 1.2.3 — Existence of $n$-th Roots

For every $x>0$ and $n\in\mathbb{N}$, there exists a unique $y>0$ with $y^n=x$. Proof: let $S=\{t\geq0:t^n\leq x\}$; $S$ is bounded above, so $y=\sup S$ exists; one checks $y^n=x$ by ruling out $y^nx$ using the Archimedean property.

Theorem 1.2.4 — Nested Interval Property

If $[a_1,b_1]\supseteq[a_2,b_2]\supseteq\cdots$ is a nested sequence of closed bounded intervals, then $\bigcap_{n=1}^\infty[a_n,b_n]\ne\emptyset$. If additionally $b_n-a_n\to0$, the intersection is a single point.
