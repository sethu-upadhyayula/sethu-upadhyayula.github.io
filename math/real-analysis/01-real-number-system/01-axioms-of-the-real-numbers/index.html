1.1 Axioms of the Real Numbers

The real numbers $\mathbb{R}$ are characterised axiomatically. Rather than constructing $\mathbb{R}$ from scratch, we list the properties it must satisfy and prove that any two systems satisfying these properties are isomorphic — so $\mathbb{R}$ is unique up to isomorphism.

Field Axioms

Definition 1.1.1 — Field Axioms

$\mathbb{R}$ is equipped with operations $+$ (addition) and $\cdot$ (multiplication) satisfying, for all $a,b,c\in\mathbb{R}$:

  • Commutativity: $a+b=b+a$ and $ab=ba$.
  • Associativity: $(a+b)+c=a+(b+c)$ and $(ab)c=a(bc)$.
  • Distributivity: $a(b+c)=ab+ac$.
  • Additive identity: there exists $0\in\mathbb{R}$ with $a+0=a$.
  • Multiplicative identity: there exists $1\in\mathbb{R}$, $1\ne0$, with $a\cdot1=a$.
  • Additive inverses: for each $a$ there exists $-a$ with $a+(-a)=0$.
  • Multiplicative inverses: for each $a\ne0$ there exists $a^{-1}$ with $a\cdot a^{-1}=1$.

Any set with two operations satisfying these axioms is called a field.

Order Axioms

Definition 1.1.2 — Order Axioms

  • Compatibility with multiplication: if $a

A field with a compatible order is an ordered field. Both $\mathbb{Q}$ and $\mathbb{R}$ are ordered fields.

Remark — Consequences of Order

From the field and order axioms alone one can derive: $0 < 1$; $a^2 \geq 0$ for all $a$ (with equality iff $a=0$); $a>0$ iff $-a<0$; if $abc$ (multiplication by a negative reverses order).

These are not additional axioms — they follow from the ones listed above.

Uniqueness of $\mathbb{R}$

Theorem 1.1.3 — Uniqueness of the Complete Ordered Field

Up to order-preserving field isomorphism, there is exactly one complete ordered field. We call it $\mathbb{R}$. In particular, $\mathbb{Q}$ is not complete (the set $\{x\in\mathbb{Q}:x^2<2\}$ has no rational supremum), confirming that completeness is a genuinely new axiom.

Archimedean Property

Theorem 1.1.4 — Archimedean Property

For every $x\in\mathbb{R}$ there exists $n\in\mathbb{N}$ with $n>x$. Equivalently, for every $\varepsilon>0$ there exists $n\in\mathbb{N}$ with $1/n<\varepsilon$. This is a theorem, not an axiom — it follows from completeness.

Proof

Suppose for contradiction that $x\geq n$ for all $n\in\mathbb{N}$. Then $\mathbb{N}$ is a nonempty subset of $\mathbb{R}$ bounded above by $x$. By completeness, $s=\sup\mathbb{N}$ exists in $\mathbb{R}$.

Example — Key Consequences

From the field and order axioms we can prove: $a\cdot 0=0$ for all $a$ (since $a\cdot 0=a(0+0)=a\cdot 0+a\cdot 0$, then subtract $a\cdot 0$ from both sides); $(-a)(-b)=ab$; if $ab=0$ then $a=0$ or $b=0$ ($\mathbb{R}$ is an integral domain).

These elementary consequences are often taken for granted but follow rigorously from the axioms.
