2.2 Orthonormal Bases

An orthonormal basis (Hilbert basis) is an orthonormal system whose linear span is dense. It provides a coordinate system for the whole Hilbert space — every element can be expanded uniquely in terms of basis vectors, generalising Fourier series.

Orthonormal Systems

Definition 2.2.1 — Orthonormal System and Basis

A sequence $(e_n)$ in a Hilbert space $H$ is an orthonormal system (ONS) if $\langle e_i,e_j\rangle=\delta_{ij}$ for all $i,j$. It is an orthonormal basis (ONB) or Hilbert basis if additionally the closed linear span $\overline{\text{span}}\{e_n\}=H$.

The numbers $\hat x_n=\langle x,e_n\rangle$ are the Fourier coefficients of $x$ with respect to $(e_n)$.

Bessel's Inequality

Theorem 2.2.2 — Bessel's Inequality

For any ONS $(e_n)$ and any $x\in H$: $$\sum_{n=1}^\infty |\langle x,e_n\rangle|^2 \le \|x\|^2.$$

In particular the series $\sum|\langle x,e_n\rangle|^2$ always converges, and $\langle x,e_n\rangle\to0$ as $n\to\infty$.

Proof

For any finite $N$, expand $\left\|x-\sum_{n=1}^N\langle x,e_n\rangle e_n\right\|^2\ge0$. Using the ONS property this equals $\|x\|^2-\sum_{n=1}^N|\langle x,e_n\rangle|^2\ge0$.

Hence $\sum_{n=1}^N|\langle x,e_n\rangle|^2\le\|x\|^2$ for all $N$, and the result follows by letting $N\to\infty$. $\square$

Parseval's Identity and ONBs

Theorem 2.2.3 — Characterisations of an ONB

For an ONS $(e_n)$ in a Hilbert space $H$, the following are equivalent:

  1. $(e_n)$ is an ONB.
  2. Parseval's identity: $\|x\|^2=\sum_{n=1}^\infty|\langle x,e_n\rangle|^2$ for all $x\in H$.
  3. Expansion: $x=\sum_{n=1}^\infty\langle x,e_n\rangle e_n$ (convergence in $H$-norm) for all $x$.
  4. $\langle x,e_n\rangle=0$ for all $n$ implies $x=0$.
Theorem 2.2.4 — Separable Hilbert Spaces and $\ell^2$

Every separable infinite-dimensional Hilbert space has a countable ONB and is isometrically isomorphic to $\ell^2$: the map $x\mapsto(\langle x,e_n\rangle)_{n\ge1}$ is a surjective isometry $H\to\ell^2$.

Key Example: Fourier Basis

Example 2.2.5 — Fourier ONB for $L^2[-\pi,\pi]$

Define $e_n(x)=e^{inx}/\sqrt{2\pi}$ for $n\in\mathbb{Z}$. These form an ONB for $L^2[-\pi,\pi]$: $$\langle e_m,e_n\rangle = \frac{1}{2\pi}\int_{-\pi}^\pi e^{i(n-m)x}\,dx = \delta_{mn}.$$

The expansion $f=\sum_{n\in\mathbb{Z}}\langle f,e_n\rangle e_n$ is the Fourier series of $f$, converging in $L^2$-norm (not necessarily pointwise). Parseval's identity becomes $$\|f\|_{L^2}^2 = \sum_{n\in\mathbb{Z}}|\hat f(n)|^2$$ where $\hat f(n)=(1/\sqrt{2\pi})\int_{-\pi}^\pi f(x)e^{-inx}\,dx$ are the Fourier coefficients.

This is the rigorous foundation for Fourier analysis on $L^2$.
