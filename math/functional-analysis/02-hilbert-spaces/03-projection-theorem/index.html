2.3 Projection Theorem

The projection theorem is one of the most useful results in Hilbert space theory. It says that every closed convex set has a unique nearest point to any given point, and in the case of a closed subspace the projection is a bounded linear operator with a clean algebraic description.

Nearest Point in a Convex Set

Theorem 2.3.1 — Projection onto a Closed Convex Set

Let $H$ be a Hilbert space and $C\subseteq H$ a nonempty closed convex subset. For each $x\in H$ there exists a unique $Px\in C$ such that $$\|x-Px\| = d(x,C) = \inf_{y\in C}\|x-y\|.$$

The point $Px$ is characterised by: $Px\in C$ and $\operatorname{Re}\langle x-Px,y-Px\rangle\le0$ for all $y\in C$.

Proof Sketch

Let $d=d(x,C)$ and pick $(y_n)\subseteq C$ with $\|x-y_n\|\to d$. Using the parallelogram law applied to $x-y_n$ and $x-y_m$: $$\|y_n-y_m\|^2 = 2\|x-y_n\|^2+2\|x-y_m\|^2-4\left\|x-\frac{y_n+y_m}{2}\right\|^2.$$ Since $(y_n+y_m)/2\in C$ (convexity), the last term is $\ge4d^2$, so $\|y_n-y_m\|^2\le2\|x-y_n\|^2+2\|x-y_m\|^2-4d^2\to0$. Thus $(y_n)$ is Cauchy; let $Px=\lim y_n\in C$ (closed).

Uniqueness: if $Px$ and $Px'$ both achieve the minimum, the parallelogram law gives $\|Px-Px'\|=0$. $\square$

Projection onto a Closed Subspace

Theorem 2.3.2 — Orthogonal Projection Theorem

Let $M$ be a closed subspace of a Hilbert space $H$. For each $x\in H$, the unique nearest point $Px\in M$ is characterised by $$x - Px \perp M, \quad \text{i.e.} \quad \langle x-Px, m\rangle = 0 \text{ for all } m\in M.$$

The map $P:H\to M$ (orthogonal projection onto $M$) satisfies:

  • $P$ is linear and bounded with $\|P\|\le1$ ($\|P\|=1$ if $M\ne\{0\}$).
  • $P^2=P$ (idempotent).
  • $P^*=P$ (self-adjoint).
  • $H = M \oplus M^\perp$ (orthogonal direct sum), where $M^\perp=\{x\in H:\langle x,m\rangle=0\;\forall m\in M\}$.
Example 2.3.3 — Least Squares

In $\mathbb{R}^n$ with an overdetermined system $Ax\approx b$ (more equations than unknowns, $A\in\mathbb{R}^{m\times n}$, $m>n$), the least-squares solution minimises $\|Ax-b\|^2$.

The column space $\text{col}(A)$ is a closed subspace of $\mathbb{R}^m$. The best approximation is $P_{\text{col}(A)}b = A\hat x$ where $\hat x$ solves the normal equations: $$A^TA\hat x = A^Tb, \qquad \hat x = (A^TA)^{-1}A^Tb \text{ (when }A\text{ has full column rank)}.$$

The orthogonality condition $b-A\hat x\perp\text{col}(A)$ is exactly $A^T(b-A\hat x)=0$, which are the normal equations.

Remark — Characterisation of Orthogonal Projections

Conversely, any bounded linear operator $P:H\to H$ satisfying $P^2=P$ and $P^*=P$ is the orthogonal projection onto $\text{ran}(P)$. This algebraic characterisation is crucial in quantum mechanics, where projections onto eigenspaces represent measurement outcomes.
