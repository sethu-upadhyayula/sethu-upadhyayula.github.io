1.3 Bounded Linear Operators

Linear operators between normed spaces generalise matrices. In infinite dimensions, not all linear operators are continuous — the crucial condition is boundedness. Bounded linear operators are the morphisms of the category of normed spaces, and the space of all such operators is itself a normed space (and a Banach space when the codomain is Banach).

Bounded Operators and Continuity

Definition 1.3.1 — Bounded Linear Operator

A linear operator $T:X\to Y$ between normed spaces is bounded if there exists $M\ge0$ such that $$\|Tx\| \le M\|x\| \quad \text{for all } x\in X.$$

The infimum of all such $M$ is the operator norm $\|T\|=\sup_{\|x\|=1}\|Tx\|=\sup_{x\ne0}\frac{\|Tx\|}{\|x\|}$.

Theorem 1.3.2 — Boundedness Equals Continuity

For a linear operator $T:X\to Y$, the following are equivalent:

  1. $T$ is bounded.
  2. $T$ is continuous (everywhere).
  3. $T$ is continuous at $0$.
  4. $T$ maps bounded sets to bounded sets.
Proof

(1)$\Rightarrow$(2): If $x_n\to x$ then $\|Tx_n-Tx\|=\|T(x_n-x)\|\le M\|x_n-x\|\to0$.

(2)$\Rightarrow$(3): Trivial.

(3)$\Rightarrow$(1): Continuity at 0 gives $\delta>0$ such that $\|x\|\le\delta\Rightarrow\|Tx\|\le1$. For $x\ne0$, set $z=\delta x/\|x\|$; then $\|Tz\|\le1$, so $\|Tx\|=\|x\|/\delta\cdot\|Tz\|\le\|x\|/\delta$.

(1)$\Leftrightarrow$(4): Clear from definitions. $\square$

The Space $\mathcal{B}(X,Y)$

Theorem 1.3.3 — $\mathcal{B}(X,Y)$ Is a Banach Space

The set $\mathcal{B}(X,Y)$ of all bounded linear operators from $X$ to $Y$, with the operator norm, is a normed space. If $Y$ is a Banach space, then $\mathcal{B}(X,Y)$ is also a Banach space.

Moreover, if $T\in\mathcal{B}(X,Y)$ and $S\in\mathcal{B}(Y,Z)$ then $ST\in\mathcal{B}(X,Z)$ and $\|ST\|\le\|S\|\|T\|$ (submultiplicativity).

Examples

Example 1.3.4 — Differentiation Operator

Define $D:C^1[0,1]\to C[0,1]$ by $Df=f'$, where $C^1[0,1]$ carries the norm $\|f\|_{C^1}=\|f\|_\infty+\|f'\|_\infty$. Then $\|Df\|_\infty=\|f'\|_\infty\le\|f\|_{C^1}$, so $D$ is bounded with $\|D\|\le1$.

However, if $C^1[0,1]$ carries only the sup norm $\|f\|_\infty$, then $D$ is unbounded: take $f_n(t)=\sin(nt)/n$, then $\|f_n\|_\infty=1/n\to0$ but $\|Df_n\|_\infty=1$ for all $n$.

Example 1.3.5 — Multiplication Operator on $L^2$

Let $m\in L^\infty[0,1]$ and define $M_m:L^2[0,1]\to L^2[0,1]$ by $(M_m f)(t)=m(t)f(t)$. Then $$\|M_m f\|_2^2 = \int_0^1|m(t)|^2|f(t)|^2\,dt \le \|m\|_\infty^2\|f\|_2^2,$$ so $M_m$ is bounded with $\|M_m\|\le\|m\|_\infty$. In fact $\|M_m\|=\|m\|_\infty$.

Multiplication operators are fundamental in spectral theory: the spectral theorem represents self-adjoint operators as multiplication operators on an $L^2$ space.

Remark — Unbounded Operators

Unbounded linear operators (defined only on a dense domain) play a crucial role in quantum mechanics — the position and momentum operators are unbounded. Their analysis requires the theory of closed and closable operators, which builds on the closed graph theorem studied in Chapter 3.
