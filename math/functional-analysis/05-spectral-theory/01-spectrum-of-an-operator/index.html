5.1 Spectrum of an Operator

The spectrum of a bounded linear operator on a Banach space generalizes the notion of eigenvalues from finite-dimensional linear algebra. In infinite dimensions, the spectrum is richer — it decomposes into three parts with quite different characters — and is always a non-empty compact subset of $\mathbb{C}$.

Resolvent and Spectrum

Definition 5.1.1 — Resolvent Set and Spectrum

Let $X$ be a complex Banach space and $T \in B(X)$. The resolvent set of $T$ is

$$\rho(T) = \{\lambda \in \mathbb{C} : (\lambda I - T) \text{ is bijective, i.e., has a bounded inverse}\}.$$

For $\lambda \in \rho(T)$, the resolvent operator is $R(\lambda, T) = (\lambda I - T)^{-1} \in B(X)$.

The spectrum of $T$ is the complement $\sigma(T) = \mathbb{C} \setminus \rho(T)$.

Remark — Automatic Boundedness

By the bounded inverse theorem (Corollary 3.2.2), if $\lambda I - T$ is bijective then its inverse is automatically bounded. So $\lambda \in \rho(T)$ iff $\lambda I - T$ is a bijection, equivalently a bijective bounded linear operator with bounded inverse.

Decomposition of the Spectrum

Definition 5.1.2 — Three Parts of the Spectrum

The spectrum $\sigma(T)$ decomposes into three disjoint parts:

  • Point spectrum $\sigma_p(T)$: values $\lambda$ for which $\lambda I - T$ is not injective. These are the eigenvalues: $Tx = \lambda x$ for some $x \ne 0$.
  • Continuous spectrum $\sigma_c(T)$: values $\lambda$ for which $\lambda I - T$ is injective with dense but non-closed range.
  • Residual spectrum $\sigma_r(T)$: values $\lambda$ for which $\lambda I - T$ is injective but whose range is not dense.

Thus $\sigma(T) = \sigma_p(T) \sqcup \sigma_c(T) \sqcup \sigma_r(T)$. For self-adjoint operators on Hilbert spaces, the residual spectrum is empty.

Basic Properties

Theorem 5.1.3 — Properties of the Spectrum
  1. Boundedness: $\sigma(T) \subseteq \overline{B}(0, \|T\|)$. If $|\lambda| > \|T\|$ then $\lambda I - T$ is invertible by Neumann series: $(\lambda I - T)^{-1} = \frac{1}{\lambda}\sum_{n=0}^\infty \left(\frac{T}{\lambda}\right)^n$.
  2. Closedness: $\sigma(T)$ is closed (equivalently, $\rho(T)$ is open). The resolvent map $\lambda \mapsto R(\lambda, T)$ is analytic on $\rho(T)$.
  3. Non-emptiness: For a complex Banach space, $\sigma(T) \ne \emptyset$. (If $\sigma(T) = \emptyset$, the entire-function $\lambda \mapsto R(\lambda,T)$ would be bounded and hence constant by Liouville, contradicting $\|R(\lambda,T)\| \to 0$ as $|\lambda| \to \infty$.)
Example 5.1.4 — Spectrum of the Shift Operator

On $\ell^2$, let $S$ be the right shift: $S(x_1, x_2, \ldots) = (0, x_1, x_2, \ldots)$.

  • Point spectrum: $\sigma_p(S) = \emptyset$. If $Sx = \lambda x$, then $0 = \lambda x_1$, $x_1 = \lambda x_2$, $x_2 = \lambda x_3$, etc. For $|\lambda| < 1$, the only solution is $x = 0$; for $|\lambda| \ge 1$, $x \notin \ell^2$.
  • Spectrum: $\sigma(S) = \overline{\mathbb{D}} = \{\lambda : |\lambda| \le 1\}$ (the closed unit disk).
  • For $|\lambda| < 1$: $\lambda \in \sigma_c(S)$. The range of $\lambda I - S$ is dense but not all of $\ell^2$.

Spectral Radius

Theorem 5.1.5 — Spectral Radius Formula

The spectral radius $r(T) = \sup\{|\lambda| : \lambda \in \sigma(T)\}$ satisfies

$$r(T) = \lim_{n \to \infty} \|T^n\|^{1/n} = \inf_{n \ge 1} \|T^n\|^{1/n}.$$

Note that $r(T) \le \|T\|$ always, and equality can fail. For normal operators on Hilbert spaces, $r(T) = \|T\|$.

Proof Sketch

The Neumann series shows $\sigma(T) \subseteq \overline{B}(0, \|T^n\|^{1/n})$ for each $n$, giving $r(T) \le \inf_n \|T^n\|^{1/n}$. The Laurent expansion of $R(\lambda, T)$ about $\infty$ shows that $R(\lambda,T)$ has a pole or essential singularity at every point $\lambda$ with $|\lambda| = r(T)$, so $r(T)$ is the actual radius of convergence of the Neumann series, giving $r(T) = \lim_{n} \|T^n\|^{1/n}$. $\square$
