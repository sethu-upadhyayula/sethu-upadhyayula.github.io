5.2 Spectral Theorem

The spectral theorem is a profound generalization of the diagonalization theorem from linear algebra to infinite-dimensional Hilbert spaces. It asserts that self-adjoint (and more generally normal) operators can be "diagonalized" in a suitable sense, revealing their structure through eigenvalues or a spectral measure.

Compact Self-Adjoint Operators

Theorem 5.2.1 — Spectral Theorem for Compact Self-Adjoint Operators

Let $H$ be a Hilbert space and $T: H \to H$ a compact self-adjoint operator ($T^* = T$ and $T$ compact). Then:

  1. All eigenvalues of $T$ are real.
  2. Eigenvectors corresponding to distinct eigenvalues are orthogonal.
  3. The spectrum $\sigma(T) = \{0\} \cup \{\lambda_n\}$ where $(\lambda_n)$ is a sequence (finite or countably infinite) of eigenvalues with $\lambda_n \to 0$.
  4. There exists an orthonormal system $(e_n)$ of eigenvectors of $T$ such that for all $x \in H$:
$$Tx = \sum_n \lambda_n \langle x, e_n \rangle e_n.$$

If $H$ is separable and $\ker T = \{0\}$, then $(e_n)$ is an orthonormal basis of $H$.

Proof Outline

Eigenvalues are real: If $Tv = \lambda v$ with $v \ne 0$: $\lambda\|v\|^2 = \langle Tv, v\rangle = \langle v, Tv\rangle = \bar\lambda\|v\|^2$, so $\lambda = \bar\lambda$.

Existence of a first eigenvalue: $\|T\| = r(T) = \sup\{|\langle Tx,x\rangle| : \|x\|=1\}$ for self-adjoint $T$. This supremum is attained (by compactness, extract a convergent subsequence from a maximizing sequence), and the maximizer is an eigenvector with eigenvalue $\pm\|T\|$.

Induction: The orthogonal complement of each eigenspace is $T$-invariant (by self-adjointness). Restrict $T$ and repeat. Each eigenspace is finite-dimensional (since $T$ is compact). The eigenvalues must accumulate only at $0$.

Completeness: Any $x$ orthogonal to all eigenvectors lies in $\ker T$ (by the above), giving the decomposition. $\square$

Spectral Theorem for Bounded Self-Adjoint Operators

Theorem 5.2.2 — Spectral Theorem (Spectral Measure Form)

Let $H$ be a Hilbert space and $T \in B(H)$ self-adjoint. Then there exists a unique projection-valued measure (or spectral measure) $E$ on the Borel $\sigma$-algebra of $\mathbb{R}$, supported on $\sigma(T) \subseteq [-\|T\|, \|T\|]$, such that

$$T = \int_{\sigma(T)} \lambda\, dE(\lambda).$$

Here $E(B)$ is an orthogonal projection for each Borel set $B$, and the integral is interpreted as a strong operator limit of approximating sums. Moreover $E(\sigma(T)) = I$ and $E(B_1)E(B_2) = E(B_1 \cap B_2)$.

Remark — Point vs. Continuous Spectrum

In the spectral measure form, isolated points $\{\lambda\}$ in $\sigma(T)$ with $E(\{\lambda\}) \ne 0$ correspond to eigenvalues. If $\sigma(T)$ is purely continuous (no isolated points with nonzero measure), there are no eigenvectors — a feature impossible in finite dimensions but generic in infinite dimensions (e.g., the position operator $Xf(x) = xf(x)$ on $L^2(\mathbb{R})$ has only continuous spectrum).

Functional Calculus

Theorem 5.2.3 — Continuous Functional Calculus

Let $T \in B(H)$ be self-adjoint with spectral measure $E$. For any bounded Borel function $f: \sigma(T) \to \mathbb{C}$, define

$$f(T) = \int_{\sigma(T)} f(\lambda)\, dE(\lambda).$$

This gives a $C^*$-algebra homomorphism from bounded Borel functions on $\sigma(T)$ to $B(H)$ satisfying:

  • $(\text{id})(T) = T$ (where $\mathrm{id}(\lambda) = \lambda$),
  • $(fg)(T) = f(T)g(T)$,
  • $\overline{f}(T) = f(T)^*$,
  • $\|f(T)\| \le \|f\|_\infty$.
Example 5.2.4 — Square Root and Absolute Value

For a positive self-adjoint operator $T \ge 0$ (i.e., $\langle Tx,x\rangle \ge 0$ for all $x$, equivalently $\sigma(T) \subseteq [0,\infty)$), the functional calculus defines the positive square root $T^{1/2} = \int \sqrt{\lambda}\,dE(\lambda)$. This is the unique positive self-adjoint operator with $(T^{1/2})^2 = T$.

For general $T = T^*$, the absolute value $|T| = (T^2)^{1/2} = \int |\lambda|\,dE(\lambda)$ is defined. One then gets the polar decomposition $T = U|T|$ where $U$ is a partial isometry.
