2.1 Ordinary Generating Functions

The ordinary generating function (OGF) of a sequence $(a_n)_{n \ge 0}$ is the formal power series $A(x) = \sum_{n \ge 0} a_n x^n$. We treat $x$ as a formal variable — convergence is secondary to algebraic structure.

Definition — OGF

Given a sequence $(a_n)_{n \ge 0}$, its ordinary generating function is: $$A(x) = \sum_{n=0}^{\infty} a_n x^n.$$ We write $[x^n] A(x) = a_n$ for the coefficient extraction operator.

Theorem — Operations on OGFs

If $A(x) = \sum a_n x^n$ and $B(x) = \sum b_n x^n$, then:

  • Sum: $A(x) + B(x) = \sum (a_n + b_n)x^n$.
  • Shift: $x A(x) = \sum_{n \ge 1} a_{n-1} x^n$ (right-shift by 1).
  • Product (convolution): $A(x)B(x) = \sum_{n \ge 0}\left(\sum_{k=0}^n a_k b_{n-k}\right)x^n$.
  • Derivative: $A'(x) = \sum_{n \ge 1} n a_n x^{n-1}$.
Theorem — Key OGFs

$$\frac{1}{1-x} = \sum_{n \ge 0} x^n, \qquad \frac{1}{1-x^k} = \sum_{n \ge 0} x^{kn}.$$ $$\frac{1}{(1-x)^k} = \sum_{n \ge 0} \binom{n+k-1}{k-1} x^n \quad (k \ge 1).$$ $$(1+x)^n = \sum_{k=0}^n \binom{n}{k} x^k.$$

Example — Fibonacci Numbers via OGF

The Fibonacci sequence satisfies $F_0=0$, $F_1=1$, $F_n = F_{n-1}+F_{n-2}$ for $n \ge 2$. Let $F(x) = \sum F_n x^n$. Then: $$F(x) - xF(x) - x^2F(x) = x \implies F(x) = \frac{x}{1-x-x^2}.$$ Using partial fractions with roots $\phi = (1+\sqrt5)/2$ and $\hat\phi = (1-\sqrt5)/2$: $$F_n = \frac{\phi^n - \hat\phi^n}{\sqrt{5}} \quad \text{(Binet's formula)}.$$

Example — Number of Coin Compositions

The number of ways to write $n$ as an ordered sum of positive integers (compositions) is $[x^n]$ in: $$\sum_{k \ge 0}\left(\frac{x}{1-x}\right)^k = \frac{1}{1-\frac{x}{1-x}} = \frac{1-x}{1-2x} = 1 + \sum_{n \ge 1} 2^{n-1} x^n.$$ So there are $2^{n-1}$ compositions of $n$ for $n \ge 1$.

Remark — Formal vs Analytic

We work with formal power series: algebraic manipulations are valid regardless of convergence. However, when the generating function converges in a neighbourhood of 0, analytic techniques (contour integration, singularity analysis) yield asymptotic estimates for $a_n$.
