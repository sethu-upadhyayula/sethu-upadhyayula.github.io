4.2 Ramsey Theory

Ramsey theory studies the conditions under which a sufficiently large combinatorial structure must contain a highly ordered substructure, regardless of how the structure is arranged. As Ramsey's theorem says: complete disorder is impossible.

Definition - Ramsey Number $R(s,t)$

The Ramsey number $R(s,t)$ is the least integer $n$ such that any 2-colouring (red/blue) of the edges of the complete graph $K_n$ contains either a red $K_s$ or a blue $K_t$.

Theorem - $R(3,3) = 6$

Sufficiency ($n=6$ works): Fix any vertex $v$. It has 5 edges; by PHP, at least 3 have the same colour, say red, going to $\{u_1, u_2, u_3\}$. If any edge among $u_1, u_2, u_3$ is red, we have a red triangle with $v$. If all three edges among $u_1,u_2,u_3$ are blue, they form a blue triangle.

Necessity ($n=5$ fails): Colour $K_5$ as the Petersen graph (cycle $C_5$ red, all diagonals blue); one can verify there is no monochromatic triangle.

Theorem - Upper Bound via Pascal Recursion

$$R(s,t) \le R(s-1,t) + R(s,t-1).$$ Consequently, $R(s,t) \le \binom{s+t-2}{s-1}$.

Proof Sketch

In $K_{R(s-1,t)+R(s,t-1)}$, fix any vertex $v$. It has red or blue degree $\ge R(s-1,t)$ or $\ge R(s,t-1)$ respectively (by PHP). In the first case, the red neighbourhood contains a red $K_{s-1}$ (by definition of $R(s-1,t)$) giving a red $K_s$ with $v$, or a blue $K_t$. Similarly for the second case. $\square$

Known Ramsey Numbers

Computing exact Ramsey numbers is extremely hard. Known values include:

  • $R(3,3) = 6$
  • $R(3,4) = 9$
  • $R(3,5) = 14$
  • $R(4,4) = 18$
  • $R(4,5) = 25$

$R(5,5)$ is unknown; only $43 \le R(5,5) \le 48$ is established.

Theorem - Van der Waerden's Theorem

For any $k, r \ge 1$, there exists $W(k,r)$ such that any $k$-colouring of $\{1, 2, \ldots, W(k,r)\}$ contains a monochromatic arithmetic progression of length $r$.

Remark - The Ramsey Paradigm

Ramsey theory encapsulates the idea that in any sufficiently large system, complete randomness cannot be sustained - some pattern must emerge. This principle appears in graph theory (Ramsey numbers), number theory (Van der Waerden, Hales-Jewett), and geometry (Happy Ending problem: any 5 points in general position contain the vertices of a convex quadrilateral).
