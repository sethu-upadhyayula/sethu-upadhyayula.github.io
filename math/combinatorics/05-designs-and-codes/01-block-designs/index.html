5.1 Block Designs

A block design is a collection of subsets (called blocks) of a finite set of points, arranged so that every pair (or $t$-subset) of points appears together in exactly $\lambda$ blocks. These combinatorial structures arise in experimental design, where one wishes to compare treatments in a balanced way.

Definition — BIBD $(v,b,r,k,\lambda)$

A balanced incomplete block design (BIBD) with parameters $(v,b,r,k,\lambda)$ consists of:

  • A set $V$ of $v$ points.
  • A collection $\mathcal{B}$ of $b$ blocks, each a $k$-element subset of $V$.

such that every point lies in exactly $r$ blocks, and every pair of distinct points lies in exactly $\lambda$ blocks. We require $k < v$ (incomplete) and $\lambda \ge 1$.

Necessary Conditions

Any BIBD $(v,b,r,k,\lambda)$ satisfies:

$$bk = vr \qquad \text{(count incidences)}$$ $$\lambda(v-1) = r(k-1) \qquad \text{(count pairs through a fixed point)}$$

These follow by double-counting the number of (point, block) incidence pairs, and the number of pairs (point, block containing a fixed point and one other).

Proof of Necessary Conditions

For the first: count pairs $(p, B)$ with $p \in B$. Each block contributes $k$ such pairs, giving $bk$ total. Each point lies in $r$ blocks, giving $vr$ total. Hence $bk = vr$.

For the second: fix a point $p$. Count pairs $(q, B)$ with $q \ne p$ and $\{p,q\} \subseteq B$. Each of the $v-1$ choices of $q$ contributes $\lambda$ pairs (since $\{p,q\}$ lies in $\lambda$ blocks), giving $\lambda(v-1)$. Each of the $r$ blocks containing $p$ contributes $k-1$ choices of $q$, giving $r(k-1)$. Hence $\lambda(v-1) = r(k-1)$.

Theorem — Fisher's Inequality

For any BIBD with $\lambda \ge 1$, we have $b \ge v$.

Proof Sketch

Consider the $v \times b$ incidence matrix $M$ where $M_{ij} = 1$ if point $i$ lies in block $j$. Then $MM^T = (r-\lambda)I + \lambda J$, where $J$ is the all-ones matrix. The eigenvalues of $MM^T$ are $rk$ (with eigenvector $\mathbf{1}$) and $r - \lambda$ (with multiplicity $v-1$). Since $r > \lambda$ (which follows from $k < v$), both eigenvalues are positive, so $MM^T$ has rank $v$. Since $\mathrm{rank}(MM^T) \le \mathrm{rank}(M) \le \min(v,b)$, we get $b \ge v$.

Definition — Symmetric Design

A BIBD is called symmetric (or a $(v,k,\lambda)$-design) if $b = v$ (equivalently $r = k$). In a symmetric design, any two distinct blocks intersect in exactly $\lambda$ points.

Example — The Fano Plane

The Fano plane is the unique symmetric $(7,3,1)$-design. It has $v = 7$ points (labeled $1,\ldots,7$) and $b = 7$ blocks (lines), each containing 3 points, with every pair of points on exactly 1 line. The blocks are:

$$\{1,2,4\},\{2,3,5\},\{3,4,6\},\{4,5,7\},\{5,6,1\},\{6,7,2\},\{7,1,3\}$$

One may verify: $bk = 7\cdot 3 = 21 = vr = 7\cdot 3$, and $\lambda(v-1) = 1\cdot 6 = 6 = r(k-1) = 3\cdot 2$.

The Fano plane is the projective plane $PG(2,2)$ over $\mathbb{F}_2$.

Definition — Steiner Triple System

A Steiner triple system $S(2,3,v)$ is a BIBD with $k=3$ and $\lambda=1$: every pair of points occurs in exactly one triple. The necessary conditions give $v \equiv 1$ or $3 \pmod{6}$. Kirkman (1847) showed these are also sufficient.

Example — $S(2,3,9)$: the Affine Plane $AG(2,3)$

Take points $\mathbb{Z}_3 \times \mathbb{Z}_3$ (a $3\times 3$ grid). The $12$ triples are the $12$ lines of the affine plane over $\mathbb{F}_3$: each row $\{(i,0),(i,1),(i,2)\}$, each column, and the four "slope" families. Parameters: $v=9, b=12, r=4, k=3, \lambda=1$.

Remark — Existence

Fisher's inequality gives a necessary condition, but not sufficient. Existence of designs with given parameters is a deep combinatorial problem. For example:

  • A $(22,8,4)$-design exists (related to the Golay code).
  • A $(43,7,1)$-design (i.e., a projective plane of order 6) does not exist (Bruck–Ryser theorem).
  • The existence of a projective plane of order 10 required a massive computer search (Lam, 1989) to rule out.
Theorem — Bruck–Ryser–Chowla

Let $D$ be a symmetric $(v,k,\lambda)$-design. If $v$ is even, then $k-\lambda$ must be a perfect square. If $v$ is odd, then the Diophantine equation

$$z^2 = (k-\lambda)x^2 + (-1)^{(v-1)/2}\lambda y^2$$

must have a nontrivial integer solution.
