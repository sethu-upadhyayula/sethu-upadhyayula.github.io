1.1 Addition and Multiplication Principles

The two most fundamental counting tools are the addition principle (for disjoint choices) and the multiplication principle (for sequential independent choices). Every counting argument ultimately rests on one or both of these.

Definition - Addition Principle

If $A$ and $B$ are disjoint finite sets (i.e., $A \cap B = \emptyset$), then $$|A \cup B| = |A| + |B|.$$ More generally, for pairwise disjoint sets $A_1, A_2, \ldots, A_k$: $$\left|\bigsqcup_{i=1}^k A_i\right| = \sum_{i=1}^k |A_i|.$$

Definition - Multiplication Principle

If a task consists of $k$ sequential steps, where step $i$ can be performed in $n_i$ ways independently of the choices at other steps, then the total number of ways to complete the task is $$n_1 \times n_2 \times \cdots \times n_k = \prod_{i=1}^k n_i.$$

Example - Functions Between Sets

Let $A$ and $B$ be finite sets with $|A| = m$ and $|B| = n$. How many functions $f : A \to B$ are there?

For each of the $m$ elements of $A$, we independently choose one of $n$ elements of $B$ as its image. By the multiplication principle:

$$|B^A| = n^m = |B|^{|A|}.$$
Example - Power Set

How many subsets does a set $A$ with $|A| = n$ have?

Each element independently either belongs to the subset (1) or does not (0). By the multiplication principle with $n$ binary choices:

$$|\mathcal{P}(A)| = 2^n.$$
Example - Binary Strings

The number of binary strings of length $n$ is $2^n$ (each position is independently 0 or 1).

Among these, how many contain at least one 1? By the addition principle, this is $2^n - 1$ (all strings minus the all-zeros string).

Example - Subsets Containing a Fixed Element

How many subsets of $\{1, 2, \ldots, n\}$ contain element 1?

Element 1 is fixed in the subset; each of the remaining $n-1$ elements independently belongs or not. So the count is $2^{n-1}$.

Alternatively, by the addition principle: subsets containing 1 and subsets not containing 1 partition all $2^n$ subsets equally, giving $2^n / 2 = 2^{n-1}$ each.

Theorem - Complement Counting

To count the elements of $A$ satisfying property $P$, it is often easier to count those not satisfying $P$ and subtract: $$|\{x \in A : P(x)\}| = |A| - |\{x \in A : \lnot P(x)\}|.$$

Remark

The multiplication principle applies when choices are independent. If later choices depend on earlier ones (e.g., sampling without replacement), the principle still applies but the number of options at each step must be re-evaluated. For example, choosing an ordered pair of distinct elements from $\{1,\ldots,n\}$ gives $n(n-1)$ options, not $n^2$.
