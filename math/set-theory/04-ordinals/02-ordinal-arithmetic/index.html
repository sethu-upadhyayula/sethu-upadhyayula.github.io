4.2 Ordinal Arithmetic

Arithmetic on ordinals is defined by transfinite recursion. The key surprise is that addition and multiplication are not commutative - the order of operands matters.

Addition

Definition 4.2.1 - Ordinal Addition

$\alpha + \beta$ is the order type of the well-ordering formed by placing a copy of $\beta$ after a copy of $\alpha$. Formally by recursion on $\beta$:

  • $\alpha + 0 = \alpha$
  • $\alpha + (\beta + 1) = (\alpha + \beta) + 1$
  • $\alpha + \lambda = \sup_{\beta < \lambda} (\alpha + \beta)$ for limit $\lambda$.

Example 4.2.2 - Non-Commutativity

$1 + \omega$: Place a single element before $\omega = \{0,1,2,\ldots\}$. The resulting order $\{*, 0, 1, 2, \ldots\}$ has order type $\omega$ (rename $* \mapsto 0$, $0 \mapsto 1$, etc.).

$\omega + 1$: Place a single element after $\omega$. The result is $\{0, 1, 2, \ldots, \omega\}$, which has a largest element - this is a strictly larger ordinal than $\omega$.

Therefore $1 + \omega = \omega \neq \omega + 1$. Ordinal addition is not commutative!

Multiplication

Definition 4.2.3 - Ordinal Multiplication

$\alpha \cdot \beta$ is the order type of $\beta$ many copies of $\alpha$ laid end-to-end. For example, $\omega \cdot 2 = \omega + \omega$ (two copies of $\omega$), while $2 \cdot \omega = \omega$ (countably many copies of $2$, each of size $2$, giving $\{0,1,2,3,\ldots\}$ again).

Cantor Normal Form

Theorem 4.2.4 - Cantor Normal Form

Every ordinal $\alpha > 0$ has a unique representation $$\alpha = \omega^{\beta_1} k_1 + \omega^{\beta_2} k_2 + \cdots + \omega^{\beta_n} k_n$$ where $\beta_1 > \beta_2 > \cdots > \beta_n \geq 0$ are ordinals and $k_1, \ldots, k_n \in \mathbb{N}_{>0}$. This is the Cantor normal form of $\alpha$ in base $\omega$.

Example 4.2.5

$\omega^2 \cdot 3 + \omega \cdot 5 + 7$ is in Cantor normal form. The ordinals $\omega^2$, $\omega$, and $1 = \omega^0$ appear with coefficients $3$, $5$, and $7$. This ordinal is countable (it is $< \omega^3$).
