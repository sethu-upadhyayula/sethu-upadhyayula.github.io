4.1 Well-Ordering

A well-ordering extends a total order with a crucial finiteness property: every nonempty subset has a smallest element. This guarantees that processes which "follow the order" must eventually terminate — or more precisely, that induction can proceed along the entire ordering.

Definition 4.1.1 — Well-Ordering

A total order $(A, \leq)$ is a well-ordering if every nonempty subset $S \subseteq A$ has a minimum element: $\exists m \in S,\, \forall s \in S,\, m \leq s$.

Example 4.1.2

  • $(\mathbb{N}, \leq)$ is well-ordered. Every nonempty set of natural numbers has a minimum.
  • $(\mathbb{Z}, \leq)$ is not well-ordered: $\mathbb{Z}$ itself has no minimum.
  • $(\mathbb{Q} \cap [0,1], \leq)$ is not well-ordered: the open interval $(0,1) \cap \mathbb{Q}$ has no minimum.
  • Any finite total order is a well-ordering.

Von Neumann Ordinals

John von Neumann's idea: represent each ordinal as the set of all smaller ordinals. This makes ordinals concrete sets and $\in$ the ordering relation.

Definition 4.1.3 — Von Neumann Ordinals

The first few ordinals are: $$0 = \emptyset, \quad 1 = \{0\} = \{\emptyset\}, \quad 2 = \{0,1\} = \{\emptyset, \{\emptyset\}\}, \quad \ldots$$ $$n+1 = n \cup \{n\} \quad \text{(successor of } n\text{)}.$$

The first infinite ordinal is $\omega = \{0, 1, 2, 3, \ldots\} = \mathbb{N}$.

In general, an ordinal $\alpha$ is a transitive set ($x \in \alpha \Rightarrow x \subseteq \alpha$) that is well-ordered by $\in$. The ordering is simply: $\alpha < \beta \iff \alpha \in \beta$.

Theorem 4.1.4 — Every Well-Ordered Set is Isomorphic to a Unique Ordinal

For every well-ordered set $(A, \leq)$, there exists a unique ordinal $\alpha$ and a unique order-isomorphism $f: (A, \leq) \to (\alpha, \in)$. This $\alpha$ is the order type of $(A, \leq)$.

Definition 4.1.5 — Successor and Limit Ordinals

An ordinal $\alpha$ is:

  • A successor ordinal if $\alpha = \beta + 1 = \beta \cup \{\beta\}$ for some ordinal $\beta$. Examples: $1, 2, 3, \ldots, \omega+1$.
  • A limit ordinal if $\alpha \neq 0$ and $\alpha$ is not a successor. The smallest is $\omega = \sup\{0,1,2,\ldots\}$.

Every ordinal is $0$, a successor, or a limit. This trichotomy drives the three-case structure of transfinite induction.
