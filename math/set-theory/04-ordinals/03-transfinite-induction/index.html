4.3 Transfinite Induction

Transfinite induction generalises mathematical induction from $\mathbb{N}$ to all ordinals. Because every ordinal is $0$, a successor, or a limit, the inductive scheme splits into three cases.

Theorem 4.3.1 - Transfinite Induction

Let $P(\alpha)$ be a property of ordinals. Suppose:

  1. Base: $P(0)$ holds.
  2. Successor step: $P(\beta) \Rightarrow P(\beta + 1)$ for all ordinals $\beta$.
  3. Limit step: If $\lambda$ is a limit ordinal and $P(\beta)$ holds for all $\beta < \lambda$, then $P(\lambda)$ holds.

Then $P(\alpha)$ holds for all ordinals $\alpha$.

Proof

Let $S = \{\alpha : \neg P(\alpha)\}$. If $S \neq \emptyset$, it has a minimum $\mu$ (well-ordering of ordinals). $\mu \neq 0$ (base case). $\mu$ cannot be a successor $\beta + 1$ (successor step). $\mu$ cannot be a limit (limit step). Contradiction - so $S = \emptyset$. $\square$

Transfinite Recursion

Theorem 4.3.2 - Transfinite Recursion

Given functions $G_0$ (base), $G_S$ (successor step), and $G_L$ (limit step), there is a unique function $F$ on all ordinals satisfying:

  • $F(0) = G_0$
  • $F(\alpha + 1) = G_S(F(\alpha))$
  • $F(\lambda) = G_L(\{F(\beta) : \beta < \lambda\})$ for limit $\lambda$.

This is the standard tool for defining arithmetic on ordinals, the cumulative hierarchy $V_\alpha$, and many other set-theoretic constructions.

The Von Neumann Hierarchy

Definition 4.3.3 - Von Neumann Hierarchy $V_\alpha$

Define by transfinite recursion: $$V_0 = \emptyset, \quad V_{\alpha+1} = \mathcal{P}(V_\alpha), \quad V_\lambda = \bigcup_{\beta < \lambda} V_\beta \text{ (for limit } \lambda\text{)}.$$

The first few levels:

  • $V_0 = \emptyset$
  • $V_1 = \{\emptyset\}$
  • $V_2 = \{\emptyset, \{\emptyset\}\}$
  • $V_\omega = \bigcup_{n \in \mathbb{N}} V_n$ = all hereditarily finite sets

The cumulative hierarchy satisfies $V = \bigcup_{\alpha \in \mathrm{Ord}} V_\alpha$: every set appears at some level. This is the content of the Axiom of Foundation (see Chapter 7).

Remark

The rank of a set $x$ is the least $\alpha$ with $x \in V_{\alpha+1}$, i.e., the least $\alpha$ with $x \subseteq V_\alpha$. The Axiom of Foundation is equivalent to: every set has a rank.
