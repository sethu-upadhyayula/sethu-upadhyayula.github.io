2.3 Equivalence Relations

An equivalence relation captures the idea of "being the same in some respect." It partitions a set into disjoint classes of mutually equivalent elements, and the collection of these classes is the quotient set.

Definition 2.3.1 - Equivalence Relation

A relation $\sim$ on $A$ is an equivalence relation if it is:

  • Reflexive: $a \sim a$ for all $a \in A$.
  • Symmetric: $a \sim b \Rightarrow b \sim a$.
  • Transitive: $a \sim b$ and $b \sim c$ imply $a \sim c$.

Definition 2.3.2 - Equivalence Class

The equivalence class of $a \in A$ under $\sim$ is $$[a] = [a]_\sim = \{x \in A : x \sim a\}.$$ Every element $b \in [a]$ is called a representative of the class.

Theorem 2.3.3 - Classes are Equal or Disjoint

For an equivalence relation $\sim$ on $A$: either $[a] = [b]$ or $[a] \cap [b] = \emptyset$. Equivalently, $a \sim b \iff [a] = [b]$.

Proof

Suppose $[a] \cap [b] \neq \emptyset$; let $c \in [a] \cap [b]$, so $c \sim a$ and $c \sim b$. By symmetry $a \sim c$, and by transitivity $a \sim b$.

Now take any $x \in [a]$: $x \sim a \sim b$, so $x \in [b]$. Hence $[a] \subseteq [b]$. By symmetry $[b] \subseteq [a]$, so $[a] = [b]$. $\square$

Partitions and Quotient Sets

Definition 2.3.4 - Partition

A partition of $A$ is a collection $\mathcal{P}$ of nonempty, pairwise disjoint subsets of $A$ whose union is $A$: $$\bigcup_{C \in \mathcal{P}} C = A, \quad C \cap D = \emptyset \text{ for } C \neq D \in \mathcal{P}.$$

Theorem 2.3.5 - Equivalence Relations $\leftrightarrow$ Partitions

Every equivalence relation on $A$ induces a partition of $A$ (the equivalence classes), and every partition of $A$ induces an equivalence relation ($a \sim b \iff$ they belong to the same block). These correspondences are mutually inverse.

Definition 2.3.6 - Quotient Set

The quotient set of $A$ by $\sim$ is $$A/{\sim} = \{[a] : a \in A\},$$ the set of all equivalence classes. The map $\pi: A \to A/{\sim}$, $a \mapsto [a]$, is the natural projection.

Example 2.3.7 - Integers Modulo $n$

Define $a \equiv b \pmod{n}$ iff $n \mid (a - b)$. This is an equivalence relation on $\mathbb{Z}$. The equivalence classes are the residue classes $[0], [1], \ldots, [n-1]$, and the quotient set is $\mathbb{Z}/n\mathbb{Z}$.

For $n = 3$: $[0] = \{\ldots, -3, 0, 3, 6, \ldots\}$, $[1] = \{\ldots,-2,1,4,7,\ldots\}$, $[2] = \{\ldots,-1,2,5,8,\ldots\}$.
