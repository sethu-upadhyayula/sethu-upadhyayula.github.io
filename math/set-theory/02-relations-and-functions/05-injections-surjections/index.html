2.5 Injections and Surjections

The classification of functions by how they relate domain to codomain is fundamental to cardinality theory. A bijection between two sets is the set-theoretic notion of "the same size."

Definition 2.5.1 — Injective, Surjective, Bijective

Let $f: A \to B$.

  • Injective (one-to-one): $f(a_1) = f(a_2) \Rightarrow a_1 = a_2$. Equivalently, distinct inputs give distinct outputs.
  • Surjective (onto): $\forall b \in B,\, \exists a \in A,\, f(a) = b$. Every element of $B$ is hit.
  • Bijective: both injective and surjective.

Definition 2.5.2 — Left and Right Inverses

$f: A \to B$ has:

  • A left inverse $g: B \to A$ with $g \circ f = \mathrm{id}_A$ iff $f$ is injective.
  • A right inverse $h: B \to A$ with $f \circ h = \mathrm{id}_B$ iff $f$ is surjective (requires Axiom of Choice in general).
  • A two-sided inverse iff $f$ is bijective; in this case $g = h = f^{-1}$.

Example 2.5.3

On $\mathbb{R}$:

  • $f(x) = 2x$ is injective and surjective (bijective).
  • $f(x) = x^2$ is neither (not injective: $f(-1) = f(1)$; not surjective: $-1 \notin f(\mathbb{R})$).
  • $f: \mathbb{R} \to [0,\infty)$, $f(x) = x^2$ is surjective but not injective.
  • $f: [0,\infty) \to \mathbb{R}$, $f(x) = x^2$ is injective but not surjective.

Schröder–Bernstein Theorem

Theorem 2.5.4 — Schröder–Bernstein

If there exist injections $f: A \hookrightarrow B$ and $g: B \hookrightarrow A$, then there exists a bijection $h: A \to B$. In other words, $|A| \leq |B|$ and $|B| \leq |A|$ imply $|A| = |B|$.

Proof sketch

Define $A_0 = A$, $A_{n+1} = g(f(A_n))$ and let $C = \bigcap_{n=0}^\infty A_n$. Set $$h(a) = \begin{cases} f(a) & \text{if } a \in A_n \setminus A_{n+1} \text{ for some } n, \text{ or } a \in C, \\ g^{-1}(a) & \text{otherwise.} \end{cases}$$ One verifies $h$ is well-defined and bijective. $\square$

Example 2.5.5 — Application

The function $f: (0,1) \to \mathbb{R}$, $f(x) = \tan(\pi(x - 1/2))$ is a bijection, so $|(0,1)| = |\mathbb{R}|$. Alternatively: inject $(0,1) \hookrightarrow \mathbb{R}$ by inclusion and inject $\mathbb{R} \hookrightarrow (0,1)$ by any homeomorphism; the CBS theorem then gives a bijection.
