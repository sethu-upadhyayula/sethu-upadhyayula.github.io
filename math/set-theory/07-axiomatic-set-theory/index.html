Chapter 7: Axiomatic Set Theory (ZFC)

The Zermelo-Fraenkel axioms with the Axiom of Choice - ZFC - form the standard foundation for modern mathematics. They were developed to avoid the paradoxes of naive set theory while preserving all the mathematics one actually uses. Every standard mathematical object (number systems, functions, topological spaces, groups) can be constructed as a set within ZFC.

The Ten Axioms of ZFC

Axiom 1 - Extensionality

Two sets are equal iff they have the same elements: $$\forall x\, \forall y\, [\forall z\, (z \in x \leftrightarrow z \in y) \Rightarrow x = y].$$

Axiom 2 - Empty Set

There exists a set with no elements: $$\exists x\, \forall y\, (y \notin x).$$ By extensionality this set is unique; it is $\emptyset$.

Axiom 3 - Pairing

For any two sets $x, y$, there exists the set $\{x, y\}$: $$\forall x\, \forall y\, \exists z\, \forall w\, (w \in z \leftrightarrow w = x \vee w = y).$$

Axiom 4 - Union

For any set $\mathcal{F}$, there exists the set $\bigcup \mathcal{F}$ of all elements of elements of $\mathcal{F}$: $$\forall \mathcal{F}\, \exists A\, \forall x\, (x \in A \leftrightarrow \exists B\, (B \in \mathcal{F} \wedge x \in B)).$$

Axiom 5 - Power Set

For any set $x$, there exists $\mathcal{P}(x)$, the set of all subsets of $x$: $$\forall x\, \exists y\, \forall z\, (z \in y \leftrightarrow z \subseteq x).$$

Axiom 6 - Infinity

There exists an infinite set - specifically, a set containing $\emptyset$ and closed under the successor operation $x \mapsto x \cup \{x\}$: $$\exists I\, (\emptyset \in I \wedge \forall x\, (x \in I \Rightarrow x \cup \{x\} \in I)).$$

Axiom Schema 7 - Separation (Aussonderung)

For any set $A$ and formula $\phi(x)$, there exists $\{x \in A : \phi(x)\}$. (A schema: one axiom for each formula $\phi$.) This replaces unrestricted comprehension and resolves Russell's paradox.

Axiom Schema 8 - Replacement

If $F(x,y)$ is a functional formula (for each $x$ there is at most one $y$ with $F(x,y)$), then for any set $A$, $\{y : \exists x \in A,\, F(x,y)\}$ is a set. This allows defining sets by applying class functions to existing sets.

Axiom 9 - Foundation (Regularity)

Every nonempty set has an $\in$-minimal element: $$\forall x\, (x \neq \emptyset \Rightarrow \exists y\, (y \in x \wedge y \cap x = \emptyset)).$$ This rules out sets like $x = \{x\}$ (a set containing itself) and guarantees the cumulative hierarchy $V = \bigcup_\alpha V_\alpha$ covers all sets.

Axiom 10 - Choice

For any family $\mathcal{F}$ of nonempty sets, there is a choice function $f$ with $f(A) \in A$ for all $A \in \mathcal{F}$. (See Chapter 6 for detailed discussion.)

Resolution of Russell's Paradox

How ZFC Avoids Russell's Paradox

Russell's set $R = \{x : x \notin x\}$ cannot be formed in ZFC because there is no universal set from which to apply Separation. Applying Separation to any set $A$, we can form $R_A = \{x \in A : x \notin x\}$. The argument then shows $R_A \notin A$, which is perfectly consistent - $R_A$ is a legitimate set, just not a member of $A$.

The absence of a "set of all sets" in ZFC (shown by Separation + Cantor's paradox) is not a bug but a feature.

The Von Neumann Universe

Theorem 7.1 - Cumulative Hierarchy

The von Neumann universe $V = \bigcup_{\alpha \in \mathrm{Ord}} V_\alpha$ (where $V_0 = \emptyset$, $V_{\alpha+1} = \mathcal{P}(V_\alpha)$, $V_\lambda = \bigcup_{\beta<\lambda} V_\beta$) satisfies ZFC and contains every set. The Axiom of Foundation is equivalent to "$V$ is all there is."

$V_\omega$ contains all hereditarily finite sets, which already forms a model of ZFC minus Infinity. $V_{\omega+\omega}$ is a model of ZFC (assuming large cardinals exist).

Independence Results and Large Cardinals

What ZFC Cannot Decide

By Gödel's Second Incompleteness Theorem, ZFC cannot prove its own consistency. Many natural mathematical statements are independent of ZFC:

  • The Continuum Hypothesis (Chapter 5).
  • The existence of measurable cardinals, Woodin cardinals, supercompact cardinals, etc.
  • Suslin's Hypothesis (every dense linear order without endpoints satisfying the countable chain condition is isomorphic to $\mathbb{R}$).

Large cardinal axioms form a natural hierarchy extending ZFC and resolve many of these questions, but at the cost of adding axioms whose consistency we cannot prove.

Alternative Foundations

NBG Set Theory

Von Neumann-Bernays-Gödel (NBG) set theory extends ZFC by allowing proper classes (like the class of all sets, or the class of all ordinals) as explicit objects. NBG is a conservative extension of ZFC: every statement about sets provable in NBG is provable in ZFC.

Homotopy Type Theory (HoTT)

Homotopy Type Theory provides an alternative foundation based on type theory and homotopy theory rather than set theory. Types correspond to spaces, terms to points, equalities to paths, and the Univalence Axiom (due to Voevodsky) identifies equivalent structures. HoTT is particularly suited for formalising mathematics in proof assistants like Coq, Agda, and Lean.
