5.2 Cardinal Arithmetic

Cardinal arithmetic is defined set-theoretically, and its laws for infinite cardinals are dramatically simpler than for finite numbers: infinite cardinal addition and multiplication both collapse to the maximum.

Definition 5.2.1 — Cardinal Operations

For cardinals $\kappa$ and $\lambda$:

  • Sum: $\kappa + \lambda = |\kappa \sqcup \lambda|$ where $\kappa \sqcup \lambda = (\kappa \times \{0\}) \cup (\lambda \times \{1\})$ is the disjoint union.
  • Product: $\kappa \cdot \lambda = |\kappa \times \lambda|$.
  • Power: $\kappa^\lambda = |\{f : \lambda \to \kappa\}| = |\kappa^\lambda|$ (the set of all functions from $\lambda$ to $\kappa$).

Theorem 5.2.2 — Infinite Cardinal Arithmetic

For infinite cardinals $\kappa \leq \lambda$: $$\kappa + \lambda = \lambda, \qquad \kappa \cdot \lambda = \lambda.$$ In particular, $\aleph_0 + \aleph_0 = \aleph_0$ and $\aleph_0 \cdot \aleph_0 = \aleph_0$.

Proof sketch (for $\kappa \cdot \kappa = \kappa$)

For $\kappa = \aleph_0$: $\mathbb{N} \times \mathbb{N}$ is countable (diagonal enumeration), so $\aleph_0 \cdot \aleph_0 = \aleph_0$. For uncountable $\kappa$: use transfinite induction — at each stage $\alpha$, one shows $|\alpha \times \alpha| = |\alpha|$ using the well-order on pairs. This is Hessenberg's theorem. $\square$

Cardinal Exponentiation

Remark — Cardinal Exponentiation is Non-Trivial

Unlike addition and multiplication, cardinal exponentiation does not simplify to the maximum for infinite cardinals. Key facts:

  • $2^{\aleph_0} = |\mathcal{P}(\mathbb{N})| = |\mathbb{R}| = \mathfrak{c} > \aleph_0$ (by Cantor's theorem).
  • $\aleph_0^{\aleph_0} = 2^{\aleph_0} = \mathfrak{c}$.
  • $\aleph_1^{\aleph_0}$ can equal $\aleph_1$ or $2^{\aleph_0}$ depending on which large cardinal assumptions (or the truth value of GCH) hold.

Theorem 5.2.3 — König's Theorem

If $\kappa_i < \lambda_i$ for all $i \in I$, then $\sum_{i \in I} \kappa_i < \prod_{i \in I} \lambda_i$. As a corollary, $\kappa < \mathrm{cf}(\kappa^\lambda)$ for any infinite $\kappa, \lambda$ with $\lambda < \kappa$. In particular, $\mathrm{cf}(2^{\aleph_0}) > \aleph_0$, so $2^{\aleph_0} \neq \aleph_\omega$ (whose cofinality is $\aleph_0$).
