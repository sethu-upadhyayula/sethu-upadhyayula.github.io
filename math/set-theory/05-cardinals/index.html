Chapter 5: Cardinal Numbers

While ordinals encode order type, cardinals encode size. Using the Axiom of Choice, every set can be well-ordered, and so every cardinality is represented by a unique initial ordinal (an ordinal that does not biject with any smaller ordinal). These special ordinals are the cardinals, denoted $\aleph_0, \aleph_1, \aleph_2, \ldots$

This chapter defines cardinal numbers precisely, develops cardinal arithmetic, and culminates in the famous Continuum Hypothesis — a statement whose truth value is independent of the ZFC axioms.

Chapter Overview

5.1 — Definition

$|A|$ as the least ordinal bijecting with $A$ (requires AC). The aleph sequence $\aleph_0, \aleph_1, \aleph_2, \ldots$ The cardinality $|\mathbb{R}| = \mathfrak{c} = 2^{\aleph_0}$.

5.2 — Cardinal Arithmetic

$\kappa + \lambda = |\kappa \sqcup \lambda|$ (disjoint union), $\kappa \cdot \lambda = |\kappa \times \lambda|$, $\kappa^\lambda = |\kappa^\lambda|$ (function sets). For infinite cardinals: $\kappa + \lambda = \kappa \cdot \lambda = \max(\kappa, \lambda)$. König's theorem: $\kappa < \mathrm{cf}(\kappa^\lambda)$.

5.3 — Continuum Hypothesis

CH: $2^{\aleph_0} = \aleph_1$. Gödel (1938): CH is consistent with ZFC. Cohen (1963): $\neg$CH is consistent with ZFC. Therefore CH is independent of ZFC.
