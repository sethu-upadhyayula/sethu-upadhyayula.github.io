Chapter 3: Cardinality

Cardinality is Cantor's measure of the "size" of a set, defined purely in terms of bijections. Two sets have the same cardinality iff there exists a bijection between them. This deceptively simple definition reveals that not all infinite sets are equally large: there are infinitely many distinct infinite cardinalities.

The chapter proceeds from finite sets and the pigeonhole principle through the countability of $\mathbb{Q}$, the uncountability of $\mathbb{R}$ via Cantor's diagonal argument, Cantor's theorem $|A| < |\mathcal{P}(A)|$, and the Schröder-Bernstein theorem that makes the comparison of sizes coherent.

Chapter Overview

3.1 - Finite Sets

A set is finite if it bijects with $\{1,\ldots,n\}$ for some $n \in \mathbb{N}$. The Pigeonhole Principle and the impossibility of a finite set bijecting with a proper subset.

3.2 - Countable Sets

A set is countably infinite if it bijects with $\mathbb{N}$. $\mathbb{Z}$ and $\mathbb{Q}$ are countable via diagonal enumeration. A countable union of countable sets is countable.

3.3 - Uncountable Sets

Cantor's diagonal argument shows $[0,1]$ and $\mathbb{R}$ are uncountable. The cardinality of the continuum $\mathfrak{c} = 2^{\aleph_0}$.

3.4 - Cantor's Theorem

$|A| < |\mathcal{P}(A)|$ for every set $A$, proved by a diagonal argument. This produces a hierarchy $\aleph_0 < 2^{\aleph_0} < 2^{2^{\aleph_0}} < \cdots$ of distinct infinite cardinalities.

3.5 - Schröder-Bernstein

If $|A| \leq |B|$ and $|B| \leq |A|$, then $|A| = |B|$. Proof and application: $|(0,1)| = |\mathbb{R}|$.
