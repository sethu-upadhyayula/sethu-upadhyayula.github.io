3.5 Schröder-Bernstein Theorem

The Cantor-Schröder-Bernstein (CBS) theorem (also called the Schröder-Bernstein theorem) establishes that the cardinality comparison $\leq$ is antisymmetric: if each of two sets injects into the other, they have the same cardinality. This makes the comparison of infinite sets well-behaved.

Theorem 3.5.1 — Cantor-Schröder-Bernstein

Let $f: A \to B$ and $g: B \to A$ be injections. Then there exists a bijection $h: A \to B$.

Proof

Define the "chain" starting from $A$: $$A_0 = A, \quad A_{n+1} = g(f(A_n)), \quad C = \bigcap_{n=0}^\infty A_n.$$ Note $A_0 \supseteq A_1 \supseteq A_2 \supseteq \cdots$ Define: $$h(a) = \begin{cases} f(a) & \text{if } a \in A_n \setminus A_{n+1} \text{ for some } n \geq 0, \text{ or } a \in C \\ g^{-1}(a) & \text{if } a \in A_{n+1} \setminus A_{n+2} \text{ for some } n \geq 0 \text{ but } a \notin C. \end{cases}$$ One verifies that every $a \in A$ falls into exactly one case, that $h$ is well-defined (since $g$ restricted to its range has an inverse), and that $h$ is a bijection. $\square$

Remark

The CBS theorem does not require the Axiom of Choice. It is provable in ZF alone. This distinguishes it from the Well-Ordering Theorem, which requires full AC.

Applications

Example 3.5.2 — $|(0,1)| = |\mathbb{R}|$

We need injections in both directions:

  • $f: (0,1) \hookrightarrow \mathbb{R}$ is the inclusion map.
  • $g: \mathbb{R} \hookrightarrow (0,1)$ can be $g(x) = \frac{1}{2} + \frac{1}{\pi}\arctan(x)$, which maps $\mathbb{R}$ bijectively onto $(0,1)$ (so in particular it is an injection).

By CBS, $|(0,1)| = |\mathbb{R}| = \mathfrak{c}$.

Example 3.5.3 — $|\mathbb{R}^2| = |\mathbb{R}|$

Inject $\mathbb{R} \hookrightarrow \mathbb{R}^2$ by $x \mapsto (x,0)$. Inject $\mathbb{R}^2 \hookrightarrow \mathbb{R}$ by interleaving decimal expansions of the two coordinates. By CBS, $|\mathbb{R}^2| = |\mathbb{R}|$. More generally $|\mathbb{R}^n| = |\mathbb{R}|$ for all finite $n$.
