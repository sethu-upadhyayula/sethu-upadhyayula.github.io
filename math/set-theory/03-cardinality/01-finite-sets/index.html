3.1 Finite Sets

The finite sets are those that can be completely enumerated. Cardinality for finite sets simply counts elements, but even here the set-theoretic definition via bijections is illuminating and generalises cleanly to the infinite case.

Definition 3.1.1 — Finite Set and Cardinality

A set $A$ is finite if either $A = \emptyset$ or there exists $n \in \mathbb{N}_{>0}$ and a bijection $f: \{1, 2, \ldots, n\} \to A$. In this case, the cardinality of $A$ is $|A| = n$ (and $|\emptyset| = 0$).

A set that is not finite is infinite.

Theorem 3.1.2 — Cardinality is Well-Defined

If $f: \{1,\ldots,m\} \to A$ and $g: \{1,\ldots,n\} \to A$ are both bijections, then $m = n$. (Proved by induction on $m$.)

Pigeonhole Principle

Theorem 3.1.3 — Pigeonhole Principle

If $|A| > |B|$ and $f: A \to B$, then $f$ is not injective: some element of $B$ has at least two preimages. Equivalently, there is no injection from a set with more elements into one with fewer.

Example 3.1.4

Among any 13 people, at least two share a birth month. (13 people, 12 months — by pigeonhole, some month contains two people.)

Theorem 3.1.5 — No Bijection with a Proper Subset

A finite set $A$ is not in bijection with any proper subset $B \subsetneq A$. (This property fails for infinite sets — Dedekind used this as the definition of infinite.)

Dedekind Infinite Sets

Definition 3.1.6 — Dedekind Infinite

A set $A$ is Dedekind infinite if there exists a bijection between $A$ and a proper subset of $A$. For example, $f: \mathbb{N} \to \mathbb{N} \setminus \{0\}$, $f(n) = n+1$ shows $\mathbb{N}$ is Dedekind infinite. Under ZFC (which includes the Axiom of Choice), Dedekind infinite is equivalent to infinite.
