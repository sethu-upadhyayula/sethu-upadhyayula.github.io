3.3 Uncountable Sets

Cantor's 1874 and 1891 proofs that $\mathbb{R}$ is uncountable were revolutionary: they showed for the first time that not all infinite sets are the same size. The second proof - the diagonal argument - is one of the most elegant and influential arguments in mathematics.

Theorem 3.3.1 - $[0,1]$ is Uncountable (Cantor, 1891)

There is no surjection $f: \mathbb{N} \to [0,1]$, so $|[0,1]| > \aleph_0$.

Proof - Diagonal Argument

Suppose for contradiction that $f: \mathbb{N} \to [0,1]$ is a surjection. Write each $f(n)$ in decimal: $$f(0) = 0.d_{00}d_{01}d_{02}\cdots$$ $$f(1) = 0.d_{10}d_{11}d_{12}\cdots$$ $$f(2) = 0.d_{20}d_{21}d_{22}\cdots$$ Define $x = 0.x_0 x_1 x_2 \cdots$ where $$x_n = \begin{cases} 5 & \text{if } d_{nn} \neq 5 \\ 6 & \text{if } d_{nn} = 5. \end{cases}$$ Then $x \in [0,1]$ but $x \neq f(n)$ for every $n$ (they differ in the $n$th decimal place). This contradicts surjectivity. $\square$

Remark on the Proof

The choices $5$ and $6$ avoid the non-uniqueness of decimal expansions (e.g., $0.999\ldots = 1.000\ldots$). Any two digits both different from $0$ and $9$ work equally well.

Theorem 3.3.2 - $\mathbb{R}$ is Uncountable

$|\mathbb{R}| = \mathfrak{c} = 2^{\aleph_0} > \aleph_0$. Since $[0,1] \subseteq \mathbb{R}$ and $[0,1]$ is uncountable, $\mathbb{R}$ is uncountable. A bijection $[0,1] \to \mathbb{R}$ (e.g., $x \mapsto \tan(\pi(x-1/2))$) shows $|\mathbb{R}| = |[0,1]|$.

Theorem 3.3.3 - $|\mathbb{R}| = 2^{\aleph_0}$

There is a bijection between $\mathbb{R}$ and $\mathcal{P}(\mathbb{N})$ (or equivalently, between $\mathbb{R}$ and $\{0,1\}^{\mathbb{N}}$). Map each real number in $[0,1]$ to its binary expansion (resolving ambiguities canonically) to get $|[0,1]| = 2^{\aleph_0}$. The cardinality of the continuum is denoted $\mathfrak{c} = 2^{\aleph_0}$.
