3.4 Cantor's Theorem

Cantor's theorem is the fundamental engine of the cardinality hierarchy. It shows that the power set is always strictly larger than the original set — there is no largest cardinality, and there are infinitely many distinct infinite cardinalities.

Theorem 3.4.1 — Cantor's Theorem

For any set $A$, $|A| < |\mathcal{P}(A)|$. That is, there is an injection $A \hookrightarrow \mathcal{P}(A)$ but no surjection $A \twoheadrightarrow \mathcal{P}(A)$.

Proof

Injection: $a \mapsto \{a\}$ is an injection $A \to \mathcal{P}(A)$.

No surjection: Suppose $f: A \to \mathcal{P}(A)$ is any function. Define $$D = \{x \in A : x \notin f(x)\} \in \mathcal{P}(A).$$ Claim: $D \notin \mathrm{ran}(f)$. If $D = f(a)$ for some $a \in A$, ask whether $a \in D$:

  • If $a \in D$, then by definition of $D$, $a \notin f(a) = D$. Contradiction.
  • If $a \notin D$, then $a \notin f(a)$, so $a \in D$. Contradiction.
In both cases we have a contradiction, so $D$ is not in the range of $f$. Therefore $f$ is not surjective. $\square$

Remark — Connection to Russell's Paradox

The set $D$ in the proof is essentially the Russell set applied to the map $f$. This is the diagonal argument in its most abstract form.

The Cardinality Hierarchy

Corollary 3.4.2 — Infinite Hierarchy

Repeated application of Cantor's theorem gives a strict hierarchy of infinite cardinalities: $$\aleph_0 = |\mathbb{N}| < 2^{\aleph_0} = |\mathcal{P}(\mathbb{N})| < 2^{2^{\aleph_0}} < 2^{2^{2^{\aleph_0}}} < \cdots$$ There is no largest cardinality.

Example 3.4.3

$|\mathbb{R}| = 2^{\aleph_0}$ and $|\mathcal{P}(\mathbb{R})| = 2^{2^{\aleph_0}}$. By Cantor's theorem $|\mathbb{R}| < |\mathcal{P}(\mathbb{R})|$, confirming there are cardinalities strictly between $\aleph_0$ (if the Continuum Hypothesis fails) and arbitrarily large ones above $|\mathbb{R}|$.
