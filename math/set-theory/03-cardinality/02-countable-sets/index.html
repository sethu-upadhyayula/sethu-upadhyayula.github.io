3.2 Countable Sets

The smallest infinite cardinality is $\aleph_0 = |\mathbb{N}|$. A set is countably infinite if it can be listed in a sequence $a_0, a_1, a_2, \ldots$ - that is, if it bijects with $\mathbb{N}$.

Definition 3.2.1 - Countable

A set $A$ is:

  • Countably infinite if $|A| = |\mathbb{N}| = \aleph_0$ (there is a bijection $\mathbb{N} \to A$).
  • Countable if it is finite or countably infinite.
  • Uncountable if it is not countable.

Theorem 3.2.2 - $\mathbb{Z}$ is Countably Infinite

The integers $\mathbb{Z}$ are countably infinite. A bijection is given by $f: \mathbb{N} \to \mathbb{Z}$ defined by $f(2n) = n$ and $f(2n+1) = -n-1$ for $n \geq 0$, listing $0, -1, 1, -2, 2, -3, 3, \ldots$

Theorem 3.2.3 - $\mathbb{Q}$ is Countably Infinite

The rational numbers $\mathbb{Q}$ are countably infinite, despite being dense in $\mathbb{R}$.

Proof (diagonal enumeration)

Arrange all fractions $p/q$ ($p \in \mathbb{Z}$, $q \in \mathbb{N}_{>0}$, $\gcd(|p|,q)=1$) in an infinite grid. Follow the diagonals: $0/1,\, 1/1,\, -1/1,\, 1/2,\, -1/2,\, 2/1,\, \ldots$ This enumerates all rationals without repetition. $\square$

Theorem 3.2.4 - Countable Union of Countable Sets

A countable union of countable sets is countable: if each $A_n$ is countable, then $\bigcup_{n \in \mathbb{N}} A_n$ is countable. (Requires a weak form of the Axiom of Choice.)

Proof sketch

Enumerate $A_n = \{a_{n,0}, a_{n,1}, a_{n,2}, \ldots\}$ for each $n$. Arrange the elements $a_{n,k}$ in an $\mathbb{N} \times \mathbb{N}$ grid and enumerate diagonally. Since $\mathbb{N} \times \mathbb{N}$ is countable (bijection $(m,n) \mapsto 2^m(2n+1)-1$), the union is countable. $\square$

Remark

The algebraic numbers (roots of polynomials with integer coefficients) form a countable set, since there are only countably many polynomials and each has finitely many roots. Thus "most" real numbers are transcendental, even though constructing explicit transcendentals ($\pi$, $e$, Liouville numbers) requires work.
