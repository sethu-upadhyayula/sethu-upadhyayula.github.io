1.1 Sets and Membership

A set is a collection of distinct objects, called elements or members. The fundamental relation is membership: we write $x \in A$ to mean "$x$ is an element of $A$", and $x \notin A$ to mean "$x$ is not an element of $A$".

Definition 1.1.1 — Membership

For any object $x$ and set $A$, exactly one of $x \in A$ or $x \notin A$ holds. The symbol $\in$ is the membership relation.

Specifying Sets

There are two standard ways to describe a set:

Definition 1.1.2 — Roster Notation

A set is listed by explicitly writing all its elements between braces: $$A = \{1, 2, 3\}, \quad B = \{a, b, c, d\}.$$ For infinite sets we use an ellipsis: $\mathbb{N} = \{0, 1, 2, 3, \ldots\}$.

Order does not matter: $\{1,2,3\} = \{3,1,2\}$. Repetition does not matter: $\{1,1,2\} = \{1,2\}$.

Definition 1.1.3 — Set-Builder Notation

A set is described by a property $P(x)$ satisfied by its elements: $$A = \{x : P(x)\} \quad \text{or} \quad A = \{x \in B : P(x)\}.$$ For example, $\{n \in \mathbb{Z} : n > 0\} = \mathbb{N} \setminus \{0\}$ and $\{x \in \mathbb{R} : x^2 = 1\} = \{-1, 1\}$.

The Empty Set

Definition 1.1.4 — Empty Set

The empty set $\emptyset$ (also written $\{\}$) is the unique set containing no elements: for every $x$, $x \notin \emptyset$.

Theorem 1.1.5 — Uniqueness of $\emptyset$

There is exactly one empty set. If $A$ and $B$ both contain no elements, then $A = B$.

Proof

Suppose $A$ and $B$ are both empty. For every $x$: $x \in A \Rightarrow x \in B$ (vacuously true, since $x \in A$ is always false), and symmetrically $x \in B \Rightarrow x \in A$. By extensionality, $A = B$. $\square$

Axiom of Extensionality

Axiom 1.1.6 — Extensionality

Two sets are equal if and only if they have the same elements: $$A = B \iff \forall x\,(x \in A \leftrightarrow x \in B).$$ A set is completely determined by its elements, not by any description or order.

Standard Number Sets

Example 1.1.7 — Number Sets

The standard sets of numbers are:

  • $\mathbb{N} = \{0, 1, 2, 3, \ldots\}$ — natural numbers (including $0$)
  • $\mathbb{Z} = \{\ldots, -2, -1, 0, 1, 2, \ldots\}$ — integers
  • $\mathbb{Q} = \{p/q : p \in \mathbb{Z},\, q \in \mathbb{Z} \setminus \{0\}\}$ — rationals
  • $\mathbb{R}$ — real numbers
  • $\mathbb{C} = \{a + bi : a, b \in \mathbb{R}\}$ — complex numbers

We have the chain of inclusions $\mathbb{N} \subset \mathbb{Z} \subset \mathbb{Q} \subset \mathbb{R} \subset \mathbb{C}$.

Remark

Whether $0 \in \mathbb{N}$ is a convention. In logic and set theory, $0$ is typically included; in number theory, $\mathbb{N}$ often starts at $1$. We use the convention $0 \in \mathbb{N}$ throughout these notes.
