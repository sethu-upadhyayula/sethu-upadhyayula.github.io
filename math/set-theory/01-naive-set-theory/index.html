Chapter 1: Naive Set Theory

Naive set theory, developed by Georg Cantor in the late nineteenth century, treats sets as collections of objects satisfying some property, without worrying about the logical foundations. This intuitive approach proved extraordinarily fruitful, giving rise to the modern theory of infinite cardinalities, but it also harboured a fatal flaw: Russell's paradox. This chapter develops the intuitive material and ends with the crisis that forced the axiomatic reform of Chapter 7.

We work informally with sets, membership, subsets, and the standard set operations. All the results here remain valid in the axiomatic setting of ZFC; the difference is merely that we do not yet justify which collections are legitimately called sets.

Chapter Overview

1.1 - Sets and Membership

We introduce the membership relation $x \in A$, roster and set-builder notation, the empty set $\emptyset$, the axiom of extensionality, and the standard number sets $\mathbb{N}, \mathbb{Z}, \mathbb{Q}, \mathbb{R}, \mathbb{C}$.

1.2 - Subsets

The subset relation $A \subseteq B$, the power set $\mathcal{P}(A)$, the cardinality formula $|\mathcal{P}(A)| = 2^{|A|}$ for finite sets, and the partial order structure on the power set under inclusion.

1.3 - Operations

Union $\cup$, intersection $\cap$, difference $\setminus$, symmetric difference $\triangle$, complement, and Cartesian product $A \times B$. De Morgan's laws and the distributive laws.

1.4 - Russell's Paradox

The set $R = \{x : x \notin x\}$ leads to a contradiction in naive set theory. We discuss the resolution via the Separation (Aussonderung) axiom and related paradoxes of Burali-Forti and Cantor.
