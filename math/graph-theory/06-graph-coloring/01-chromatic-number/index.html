6.1 Chromatic Number

A proper colouring of a graph assigns a colour to each vertex so that no two adjacent vertices share the same colour. The chromatic number measures the minimum number of colours required.

Definition 6.1.1 — Proper Colouring and Chromatic Number

A proper $k$-colouring of $G$ is a function $c: V(G) \to \{1,\ldots,k\}$ such that $c(u) \neq c(v)$ whenever $\{u,v\} \in E(G)$. The chromatic number $\chi(G)$ is the smallest $k$ for which a proper $k$-colouring exists.

Example 6.1.2 — Basic Chromatic Numbers

$\chi(K_n) = n$; $\chi(C_{2k}) = 2$ (even cycle); $\chi(C_{2k+1}) = 3$ (odd cycle); $\chi(\text{bipartite, non-empty}) = 2$; $\chi(\text{empty graph}) = 1$.

$\chi(\text{Petersen graph}) = 3$; $\chi(\text{planar graph}) \le 4$ (Four Colour Theorem).

Bounds on the Chromatic Number

Theorem 6.1.3 — Greedy Bound

For any graph $G$, $\chi(G) \le \Delta(G) + 1$, where $\Delta(G)$ is the maximum degree. The greedy algorithm colours vertices one by one; each vertex has at most $\Delta$ already-coloured neighbours, so at least one of the $\Delta+1$ colours is always available.

Theorem 6.1.4 — Brooks' Theorem (1941)

If $G$ is a connected graph that is neither a complete graph $K_n$ nor an odd cycle, then $\chi(G) \le \Delta(G)$.

Chromatic Polynomial

Definition 6.1.5 — Chromatic Polynomial

The chromatic polynomial $P(G, k)$ counts the number of proper $k$-colourings of $G$. It satisfies the deletion–contraction recurrence: $$P(G, k) = P(G - e, k) - P(G/e, k)$$ where $G - e$ removes edge $e$ and $G/e$ contracts it. For the complete graph, $P(K_n, k) = k(k-1)(k-2)\cdots(k-n+1)$. For a tree $T_n$ on $n$ vertices, $P(T_n, k) = k(k-1)^{n-1}$.

Remark

The smallest positive integer $k$ for which $P(G,k) > 0$ equals $\chi(G)$. The chromatic polynomial always factors over $\mathbb{Q}$ into linear factors of the form $(k - a)$ for non-negative integers $a$.

Clique Number and Perfect Graphs

Theorem 6.1.6 — Clique Lower Bound

$\chi(G) \ge \omega(G)$, where $\omega(G)$ is the clique number (size of the largest complete subgraph). This bound is not always tight: there exist triangle-free graphs with arbitrarily large chromatic number (Mycielski's construction).

Definition 6.1.7 — Perfect Graph

$G$ is perfect if $\chi(H) = \omega(H)$ for every induced subgraph $H$. By the Strong Perfect Graph Theorem (Chudnovsky et al. 2006), $G$ is perfect if and only if it contains no odd hole (odd induced cycle of length $\ge 5$) and no odd antihole as an induced subgraph. Bipartite graphs, chordal graphs, and interval graphs are all perfect.
