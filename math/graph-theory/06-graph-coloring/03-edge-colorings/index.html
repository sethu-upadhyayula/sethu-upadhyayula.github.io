6.3 Edge Colorings

An edge colouring assigns colours to edges so that no two edges sharing a vertex (i.e., incident edges) receive the same colour. The minimum number of colours required is the chromatic index.

Definition 6.3.1 — Edge Colouring and Chromatic Index

A proper edge $k$-colouring assigns colours from $\{1, \ldots, k\}$ to edges so that no two adjacent edges (sharing a vertex) have the same colour. The chromatic index $\chi'(G)$ is the minimum such $k$. A colour class in an edge colouring is a matching.

Theorem 6.3.2 — Vizing's Theorem (1964)

For any simple graph $G$, $\Delta(G) \le \chi'(G) \le \Delta(G) + 1$. Graphs with $\chi'(G) = \Delta(G)$ are called Class 1; those with $\chi'(G) = \Delta(G) + 1$ are called Class 2.

Proof Sketch (Vizing's Theorem)

The lower bound $\chi' \ge \Delta$ is clear: any vertex of degree $\Delta$ requires $\Delta$ distinct colours on its incident edges.

For the upper bound, use the fan argument: given a $(\Delta+1)$-edge-colouring of $G - e$ for an edge $e = \{u, v\}$, construct a fan of edges at $u$ and use Kempe chain recolouring to free a colour at both $u$ and $v$ simultaneously, thereby colouring $e$.

König's Theorem for Bipartite Graphs

Theorem 6.3.3 — König's Edge Colouring Theorem (1916)

Every bipartite graph $G$ satisfies $\chi'(G) = \Delta(G)$; that is, all bipartite graphs are Class 1. The proof uses the fact that every regular bipartite graph has a perfect matching (Hall's theorem), which can be iteratively removed to produce a $\Delta$-edge-colouring.

Class 1 and Class 2 Examples

Example 6.3.4 — Class 1 and Class 2 Graphs

Class 1: All bipartite graphs; $K_n$ for even $n$; planar graphs with $\Delta \ge 7$ (Vizing's planar graph conjecture, proved for $\Delta \ge 7$).

Class 2: $K_n$ for odd $n \ge 3$ (since $K_n$ has $n$ vertices each of odd degree $n-1$, a perfect matching cannot be removed cleanly); odd cycles ($\chi' = 3 = \Delta + 1$); the Petersen graph ($\Delta = 3$, $\chi' = 4$).

Deciding whether a graph is Class 1 or Class 2 is NP-complete in general.

Example 6.3.5 — Edge Chromatic Index of $K_n$

$\chi'(K_n) = n - 1$ if $n$ is even, and $\chi'(K_n) = n$ if $n$ is odd. For even $n$, $K_n$ decomposes into $n-1$ perfect matchings (each a round of a round-robin tournament); for odd $n$, $K_n$ is Class 2.
