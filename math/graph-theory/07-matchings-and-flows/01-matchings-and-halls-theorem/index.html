7.1 Matchings and Hall's Theorem

A matching selects a set of edges with no shared vertices. The fundamental questions are: how large can a matching be, and when does a perfect matching exist?

Definition 7.1.1 - Matching

A matching $M$ in $G$ is a set of pairwise non-adjacent edges. A vertex is $M$-saturated (or matched) if it is an endpoint of some edge in $M$; otherwise it is $M$-exposed (unmatched). A perfect matching saturates every vertex; it exists only when $|V|$ is even.

Definition 7.1.2 - Augmenting Path

An augmenting path with respect to $M$ is a path whose edges alternate between non-matching and matching edges, starting and ending at $M$-exposed vertices. Augmenting along such a path (swapping matched/unmatched edges) increases $|M|$ by 1.

Theorem 7.1.3 - Berge's Theorem (1957)

A matching $M$ is maximum if and only if there is no $M$-augmenting path in $G$.

Hall's Marriage Theorem

Theorem 7.1.4 - Hall's Theorem (1935)

Let $G = (A \cup B, E)$ be a bipartite graph. Then $G$ has a matching that saturates every vertex of $A$ if and only if $$|N(S)| \ge |S| \quad \text{for every } S \subseteq A,$$ where $N(S)$ denotes the set of neighbours of $S$ in $B$. This condition is called Hall's condition or the marriage condition.

The necessity ($\Rightarrow$) is immediate: any $|S|$ vertices in $A$ matched to $B$ need $|S|$ distinct neighbours. Sufficiency ($\Leftarrow$) follows by induction or via the max-flow min-cut theorem.

Example 7.1.5 - Marriage Analogy

Think of $A$ as a set of $n$ men and $B$ as a set of women, with edges representing compatibility. Hall's condition says: for every subset $S$ of men, the women they collectively find compatible number at least $|S|$. When this holds, a stable assignment exists.

König's Theorem

Theorem 7.1.6 - König's Theorem (1931)

In a bipartite graph, the maximum matching size equals the minimum vertex cover size: $$\nu(G) = \tau(G),$$ where $\nu$ is the maximum matching and $\tau$ is the minimum vertex cover. (A vertex cover is a set of vertices touching every edge.) This is a special case of LP duality and follows from the max-flow min-cut theorem.

Remark - Deficiency Version of Hall's Theorem

The maximum matching in a bipartite graph $G = (A \cup B, E)$ has size $|A| - \max_{S \subseteq A}(|S| - |N(S)|)$. The term $\max_{S}(|S| - |N(S)|)$ is the deficiency.
