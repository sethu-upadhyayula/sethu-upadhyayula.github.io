7.2 Network Flows and Max-Flow Min-Cut

A network flow model routes commodities through a directed graph subject to edge capacities. The max-flow min-cut theorem is one of the deepest and most useful results in combinatorial optimisation.

Definition 7.2.1 - Flow Network

A flow network $(G, c, s, t)$ consists of a directed graph $G = (V, E)$, a capacity function $c: E \to \mathbb{R}_{\ge 0}$, a source $s$, and a sink $t$. A flow is a function $f: E \to \mathbb{R}_{\ge 0}$ satisfying (1) $f(e) \le c(e)$ for all $e$ (capacity constraint) and (2) for every $v \neq s, t$: $\sum_{e\text{ into }v} f(e) = \sum_{e\text{ out of }v} f(e)$ (conservation). The value of $f$ is $|f| = \sum_{e\text{ out of }s} f(e)$.

Definition 7.2.2 - $s$-$t$ Cut

An $s$-$t$ cut is a partition $(S, T)$ of $V$ with $s \in S$ and $t \in T$. Its capacity is $\text{cap}(S,T) = \sum_{u \in S,\, v \in T,\, (u,v)\in E} c(u,v)$. Weak duality: $|f| \le \text{cap}(S,T)$ for every flow $f$ and every cut $(S,T)$.

Theorem 7.2.3 - Max-Flow Min-Cut Theorem (Ford-Fulkerson 1956)

In any flow network, the maximum flow value equals the minimum cut capacity: $$\max_{f} |f| = \min_{(S,T) \text{ cut}} \mathrm{cap}(S,T).$$

Algorithms

Definition 7.2.4 - Residual Graph

Given a flow $f$, the residual graph $G_f$ has the same vertex set; for each edge $(u,v)$ with $f(u,v) < c(u,v)$, add a forward edge with residual capacity $c(u,v) - f(u,v)$; for each edge with $f(u,v) > 0$, add a backward edge $(v,u)$ with residual capacity $f(u,v)$. An $s$-$t$ path in $G_f$ is an augmenting path.

Theorem 7.2.5 - Ford-Fulkerson Method

Repeatedly find an $s$-$t$ augmenting path in the residual graph and push as much flow as possible. When no augmenting path exists, the current flow is maximum. For integer capacities this terminates in $O(E \cdot |f^*|)$ time where $|f^*|$ is the optimal flow value.

Theorem 7.2.6 - Edmonds-Karp Algorithm

Using BFS to always find a shortest augmenting path (fewest edges), the algorithm runs in $O(VE^2)$ time regardless of edge capacities. The number of augmenting path iterations is $O(VE)$; each augmenting path takes $O(E)$ via BFS.

Applications

Example 7.2.7 - Bipartite Matching via Flow

Given bipartite graph $G = (A \cup B, E)$, add a super-source $s$ with unit-capacity edges to all $a \in A$, and a super-sink $t$ with unit-capacity edges from all $b \in B$. Direct edges $A \to B$ with unit capacity. The maximum integer flow equals the maximum matching size.

Example 7.2.8 - Edge-Disjoint Paths

The maximum number of edge-disjoint $s$-$t$ paths in a directed graph equals the minimum $s$-$t$ edge cut (Menger's theorem for directed graphs), obtained by setting all capacities to 1 and computing max flow.
