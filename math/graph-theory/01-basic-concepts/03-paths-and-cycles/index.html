1.3 Paths and Cycles

Paths and cycles capture how we can traverse a graph. They underlie notions of distance, connectivity, and the fundamental dichotomy between bipartite and non-bipartite graphs.

Walks, Trails, and Paths

Definition 1.3.1 — Walk, Trail, Path

A walk in $G$ is a sequence $v_0, v_1, \ldots, v_k$ of vertices such that $\{v_{i-1}, v_i\} \in E$ for each $i = 1, \ldots, k$. Its length is $k$ (the number of edges traversed).

A trail is a walk in which no edge is repeated.

A path $P$ is a walk $v_0, v_1, \ldots, v_k$ in which all vertices are distinct. We call it a $v_0$-$v_k$ path and write $P = v_0 v_1 \cdots v_k$.

Definition 1.3.2 — Cycle

A cycle $C$ is a closed walk $v_0, v_1, \ldots, v_k, v_0$ (returning to start) where $k \ge 3$, all $v_0, \ldots, v_{k-1}$ are distinct, and all edges are distinct. The length of the cycle is $k$. A cycle of length $k$ is denoted $C_k$.

Distance and Graph Metrics

Definition 1.3.3 — Distance, Diameter, Girth

The distance $d_G(u, v)$ between vertices $u$ and $v$ is the length of a shortest $u$-$v$ path; if no path exists, $d_G(u,v) = \infty$.

The diameter $\text{diam}(G) = \max_{u,v \in V} d(u,v)$ is the largest distance in $G$.

The girth $g(G)$ is the length of a shortest cycle in $G$; if $G$ is acyclic, we set $g(G) = \infty$.

Connectivity

Definition 1.3.4 — Connected Graph and Components

A graph $G$ is connected if $d(u, v) < \infty$ for every pair $u, v \in V(G)$. A connected component is a maximal connected subgraph of $G$.

$G$ is connected if and only if it has exactly one connected component.

BFS and Shortest Paths

Theorem 1.3.5 — BFS Computes All Distances

Breadth-first search (BFS) from a source vertex $s$ computes $d(s, v)$ for all $v \in V(G)$ in time $O(|V| + |E|)$. BFS explores vertices layer by layer: layer 0 is $\{s\}$, layer $i+1$ consists of all unvisited neighbours of layer $i$ vertices.

Bipartite Graphs and Odd Cycles

Theorem 1.3.6 — Bipartite Characterisation

A graph $G$ is bipartite if and only if it contains no odd cycle.

Proof

Necessity. Suppose $G$ is bipartite with parts $A$ and $B$. Any walk alternates between $A$ and $B$, so a closed walk must have even length. Hence every cycle is even.

Sufficiency. Assume $G$ has no odd cycle; WLOG $G$ is connected. Fix $s \in V$ and run BFS. Set $A = \{v : d(s,v) \text{ even}\}$ and $B = \{v : d(s,v) \text{ odd}\}$. If any edge $\{u,v\}$ had both endpoints in $A$ (or both in $B$), then together with the BFS paths from $s$ to $u$ and $v$ this would produce an odd closed walk containing an odd cycle — a contradiction. So all edges go between $A$ and $B$, and $G$ is bipartite. $\square$

Example 1.3.7

$C_4$ is bipartite (no odd cycles). $C_5$ is not bipartite: it contains an odd cycle of length 5. The complete graph $K_n$ for $n \ge 3$ contains triangles, so it is bipartite only when $n \le 2$.
