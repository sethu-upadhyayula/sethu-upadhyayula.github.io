1.1 Graphs and Digraphs

A graph is a mathematical structure encoding pairwise relationships. Graph theory began with Euler's 1736 solution to the Königsberg bridge problem and today underpins computer science, optimisation, and combinatorics.

Graphs

Definition 1.1.1 — Graph

A graph $G = (V, E)$ consists of a finite non-empty set $V$ of vertices (or nodes) and a set $E \subseteq \binom{V}{2}$ of edges, where $\binom{V}{2}$ denotes the set of all 2-element subsets of $V$. The order of $G$ is $|V| = n$ and the size is $|E| = m$. A graph is simple if it has no loops (edges from a vertex to itself) and no multiple edges between the same pair of vertices.

Remark

Because edges are unordered pairs, the graph is undirected: the edge $\{u,v\}$ is the same as $\{v,u\}$. We write $uv$ as shorthand for $\{u,v\} \in E$. All graphs in this chapter are simple unless stated otherwise.

Special Graph Families

Definition 1.1.2 — Complete Graph $K_n$

The complete graph $K_n$ on $n$ vertices has every pair of distinct vertices connected by an edge. Its size is $|E(K_n)| = \binom{n}{2} = \frac{n(n-1)}{2}$.

Definition 1.1.3 — Bipartite Graph

A graph $G = (A \cup B, E)$ is bipartite with parts $A$ and $B$ if $E \subseteq \{\{a,b\} : a \in A,\, b \in B\}$, i.e., every edge goes between the two parts.

The complete bipartite graph $K_{m,n}$ has $|A| = m$, $|B| = n$, and every vertex in $A$ adjacent to every vertex in $B$; it has $mn$ edges.

Example 1.1.4

$K_4$ has 4 vertices and $\binom{4}{2} = 6$ edges. $K_{2,3}$ has 5 vertices and $2 \cdot 3 = 6$ edges. The cycle $C_6$ is bipartite (parts: alternating vertices), while $C_5$ is not.

Directed Graphs

Definition 1.1.5 — Digraph

A directed graph (or digraph) $D = (V, A)$ has vertex set $V$ and arc set $A \subseteq V \times V$ of ordered pairs. An arc $(u, v)$ goes from $u$ to $v$; $u$ is the tail and $v$ the head. The in-degree $\deg^-(v)$ counts arcs entering $v$; the out-degree $\deg^+(v)$ counts arcs leaving $v$.

Graph Operations

Definition 1.1.6 — Complement

The complement $\bar{G}$ of a simple graph $G$ on vertex set $V$ has the same vertex set and edge set $E(\bar{G}) = \binom{V}{2} \setminus E(G)$. Thus $\{u,v\} \in E(\bar{G})$ if and only if $\{u,v\} \notin E(G)$.

Definition 1.1.7 — Subgraph and Induced Subgraph

A graph $H$ is a subgraph of $G$ if $V(H) \subseteq V(G)$ and $E(H) \subseteq E(G)$. It is a spanning subgraph if additionally $V(H) = V(G)$.

For $S \subseteq V(G)$, the induced subgraph $G[S]$ has vertex set $S$ and edge set $\{\{u,v\} \in E(G) : u, v \in S\}$ — all edges of $G$ with both endpoints in $S$.

Graph Isomorphism

Definition 1.1.8 — Isomorphism

Graphs $G$ and $H$ are isomorphic ($G \cong H$) if there exists a bijection $\varphi : V(G) \to V(H)$ such that $\{u,v\} \in E(G)$ if and only if $\{\varphi(u), \varphi(v)\} \in E(H)$. Such a $\varphi$ is called an isomorphism. Isomorphic graphs are structurally identical; they differ only in the labelling of vertices.

Remark — Necessary Conditions for Isomorphism

Isomorphic graphs must have the same order $n$, size $m$, and degree sequence. However, these conditions are not sufficient: there exist non-isomorphic graphs sharing all three invariants. Graph isomorphism is not known to be solvable in polynomial time in general (the GI problem).
