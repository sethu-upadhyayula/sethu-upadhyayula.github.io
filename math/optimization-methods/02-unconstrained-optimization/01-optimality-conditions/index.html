2.1 Optimality Conditions

Optimality conditions characterise what a minimiser $x^*$ must satisfy, and when it is guaranteed to be one. For smooth unconstrained problems, these reduce to conditions on the gradient and Hessian at $x^*$.

First-Order Conditions

Theorem 2.1.1 — First-Order Necessary Condition

Let $f:\mathbb{R}^n\to\mathbb{R}$ be continuously differentiable. If $x^*$ is a local minimum of $f$, then: $$\nabla f(x^*) = 0.$$

A point satisfying $\nabla f(x^*)=0$ is called a stationary point. Stationary points include local minima, local maxima, and saddle points — the gradient condition alone cannot distinguish them.

Second-Order Conditions

Theorem 2.1.2 — Second-Order Necessary Condition

If $x^*$ is a local minimum and $f$ is twice continuously differentiable, then $\nabla f(x^*)=0$ and $\nabla^2f(x^*)\succeq0$ (positive semidefinite). A negative definite Hessian certifies a local maximum; indefinite Hessian at a stationary point indicates a saddle.

Theorem 2.1.3 — Second-Order Sufficient Condition

If $\nabla f(x^*)=0$ and $\nabla^2f(x^*)\succ0$ (positive definite), then $x^*$ is a strict local minimum.

Note the gap: the necessary condition requires $\nabla^2f\succeq0$ while the sufficient condition requires $\nabla^2f\succ0$. The intermediate case $\nabla^2f(x^*)\succeq0$ (with some zero eigenvalues) requires higher-order analysis.

Convex Functions

Theorem 2.1.4 — Global Optimality for Convex Functions

Let $f$ be convex and differentiable. Then $x^*$ is a global minimum if and only if $\nabla f(x^*)=0$.

Proof

By the first-order characterisation of convexity: $f(y)\ge f(x^*)+\nabla f(x^*)^T(y-x^*)=f(x^*)$ for all $y$. Hence $x^*$ is a global minimiser. $\square$

For strictly convex $f$, the global minimiser is unique.

Existence of Minimisers

Theorem 2.1.5 — Coercivity and Existence

A continuous function $f:\mathbb{R}^n\to\mathbb{R}$ is coercive if $f(x)\to+\infty$ as $\|x\|\to\infty$. If $f$ is continuous and coercive, then it attains its infimum — a global minimiser exists.

For strongly convex $f$ (with constant $m>0$), coercivity holds automatically since $f(x)\ge f(x_0)+\nabla f(x_0)^T(x-x_0)+\frac{m}{2}\|x-x_0\|^2$ grows quadratically. Hence strongly convex functions always have a unique global minimiser.

Example — Quadratic Functions

For $f(x)=\frac{1}{2}x^TQx+c^Tx+d$ with $Q\succ0$: the gradient condition gives $Qx^*+c=0$, so $x^*=-Q^{-1}c$ is the unique global minimiser. The Hessian $\nabla^2f=Q\succ0$ satisfies the sufficient condition. This example underpins all quadratic programming.

If $Q\succeq0$ (but not positive definite): minimisers may not be unique, and existence requires $c\in\operatorname{range}(Q)$.
