2.2 Gradient Descent

Gradient descent is the simplest and most widely used first-order optimisation algorithm. It exploits the fact that $-\nabla f(x)$ is the direction of steepest decrease of $f$ at $x$, iteratively moving in this direction by a step size $\alpha_k$.

The Algorithm

Definition 2.2.1 — Gradient Descent

Given a starting point $x_0\in\mathbb{R}^n$ and step sizes $\{\alpha_k\}$, the gradient descent iteration is: $$x_{k+1} = x_k - \alpha_k \nabla f(x_k), \quad k=0,1,2,\ldots$$

The sequence $\{x_k\}$ converges to a stationary point under mild conditions on $f$ and $\{\alpha_k\}$.

Convergence Rates

Theorem 2.2.2 — Convergence for $L$-Smooth Convex Functions

Suppose $f$ is convex and $L$-smooth (i.e., $\|\nabla f(x)-\nabla f(y)\|\le L\|x-y\|$ for all $x,y$). With constant step $\alpha=1/L$: $$f(x_k) - f(x^*) \le \frac{L\|x_0-x^*\|^2}{2k}.$$

This is an $O(1/k)$ rate. The condition $\alpha<2/L$ is necessary for convergence; the optimal fixed step is $\alpha=1/L$. $L$-smoothness is equivalent to $\nabla^2f\preceq LI$ for smooth $f$.

Theorem 2.2.3 — Linear Convergence for Strongly Convex Functions

If $f$ is $m$-strongly convex and $L$-smooth, with $\kappa=L/m$ (the condition number), then with step $\alpha=1/L$: $$\|x_k - x^*\|^2 \le \left(1 - \frac{1}{\kappa}\right)^k \|x_0-x^*\|^2.$$

This is a linear (geometric) rate. The convergence speed is $O(\kappa\log(1/\varepsilon))$ iterations to reach $\varepsilon$ accuracy. Large condition number $\kappa$ leads to slow convergence — the notorious zigzag behaviour of steepest descent.

Step Size Selection

Definition 2.2.4 — Backtracking Armijo Line Search

At each step, start with a trial step $\alpha=\alpha_0$ and reduce by factor $\beta\in(0,1)$ until the Armijo condition is satisfied: $$f(x_k - \alpha\nabla f(x_k)) \le f(x_k) - c\alpha\|\nabla f(x_k)\|^2$$ for a constant $c\in(0,1)$ (typically $c=10^{-4}$).

Backtracking line search is adaptive — it uses large steps when curvature is small and small steps near rapid changes, without requiring knowledge of $L$.

Nesterov's Accelerated Method

Theorem 2.2.5 — Nesterov Acceleration

Nesterov's accelerated gradient method uses a momentum term: $$y_{k+1} = x_k - \frac{1}{L}\nabla f(x_k), \qquad x_{k+1} = y_{k+1} + \frac{k-1}{k+2}(y_{k+1}-y_k).$$

This achieves the optimal rate for first-order methods on convex $L$-smooth functions: $$f(x_k)-f(x^*) \le \frac{2L\|x_0-x^*\|^2}{(k+1)^2} = O(1/k^2).$$

The improvement from $O(1/k)$ to $O(1/k^2)$ is optimal — no first-order method can achieve a better worst-case rate for this class.

Remark — Limitations of Gradient Descent

Gradient descent uses only first-order information (gradient). It can be very slow for ill-conditioned problems ($\kappa\gg1$) due to zigzag behaviour: steps in nearly orthogonal directions cancel progress. Newton's method addresses this by incorporating curvature (second-order) information, at the cost of computing and inverting the Hessian.
