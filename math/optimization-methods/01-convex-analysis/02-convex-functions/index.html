1.2 Convex Functions

A convex function is one whose graph "curves upward" — the chord between any two points on the graph lies above the graph. This guarantees that any local minimum is a global minimum, a crucial property for optimisation.

Definition

Definition 1.2.1 — Convex Function

A function $f:C\to\mathbb{R}$ on a convex set $C\subseteq\mathbb{R}^n$ is convex if for all $x,y\in C$ and $\lambda\in[0,1]$: $$f(\lambda x + (1-\lambda)y) \le \lambda f(x) + (1-\lambda)f(y).$$

$f$ is strictly convex if the inequality is strict for $x\ne y$ and $\lambda\in(0,1)$.

Examples: $e^x$, $x^2$, $\|x\|$, $-\log x$ on $(0,\infty)$, $\max(x_1,\ldots,x_n)$, any affine function.

Equivalent Characterisations

Theorem 1.2.2 — Epigraph Criterion

$f$ is convex if and only if its epigraph $\operatorname{epi}(f)=\{(x,t):f(x)\le t\}$ is a convex set in $\mathbb{R}^{n+1}$. This connects the theory of convex functions entirely to that of convex sets.

Theorem 1.2.3 — First-Order Condition

If $f$ is differentiable on an open convex set $C$, then $f$ is convex if and only if for all $x,y\in C$: $$f(y) \ge f(x) + \nabla f(x)^T(y-x).$$

The tangent hyperplane at any point is a global under-estimate of $f$. This is the key property exploited by gradient-based algorithms: gradient information gives a globally valid lower bound.

Theorem 1.2.4 — Second-Order Condition

If $f$ is twice continuously differentiable, then $f$ is convex on $C$ if and only if $\nabla^2f(x)\succeq0$ (positive semidefinite) for all $x\in C$. Strict convexity follows from $\nabla^2f(x)\succ0$ (positive definite), though this is not necessary.

Strong Convexity

Definition 1.2.5 — Strong Convexity

$f$ is $m$-strongly convex ($m>0$) if for all $x,y\in C$ and $\lambda\in[0,1]$: $$f(\lambda x+(1-\lambda)y) \le \lambda f(x)+(1-\lambda)f(y) - \frac{m}{2}\lambda(1-\lambda)\|y-x\|^2.$$

Equivalently (for smooth $f$): $\nabla^2f(x)\succeq mI$ for all $x$. Strong convexity implies the quadratic lower bound $f(y)\ge f(x)+\nabla f(x)^T(y-x)+\frac{m}{2}\|y-x\|^2$, and guarantees a unique minimiser. It is the condition that yields linear (geometric) convergence rates for gradient descent.

Jensen's Inequality and the Fenchel Conjugate

Theorem 1.2.6 — Jensen's Inequality

For a convex function $f$ and any random variable $X$ with $\mathbb{E}[X]\in\operatorname{dom}(f)$: $$f(\mathbb{E}[X]) \le \mathbb{E}[f(X)].$$ This follows by applying the definition of convexity to the convex combination defining the expectation.

Definition 1.2.7 — Fenchel Conjugate

The Fenchel (convex) conjugate of $f:\mathbb{R}^n\to\mathbb{R}\cup\{+\infty\}$ is: $$f^*(y) = \sup_{x\in\operatorname{dom}(f)}\bigl(y^Tx - f(x)\bigr).$$

$f^*$ is always convex (as a supremum of affine functions). Key results: $(f^*)^*=f$ for closed convex $f$ (Fenchel–Moreau); if $f(x)=\frac{1}{2}\|x\|^2$ then $f^*(y)=\frac{1}{2}\|y\|^2$; the conjugate of the log-sum-exp is the negative entropy.

The conjugate arises naturally in duality theory and in the analysis of proximal algorithms.
