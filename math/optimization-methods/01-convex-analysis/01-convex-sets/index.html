1.1 Convex Sets

Convex sets are the fundamental geometric objects of optimisation theory. A set is convex if the straight line between any two of its points lies entirely within the set — a simple condition with surprisingly rich algebraic and geometric consequences.

Definition and Basic Examples

Definition 1.1.1 — Convex Set

A set $C\subseteq\mathbb{R}^n$ is convex if for all $x,y\in C$ and all $\lambda\in[0,1]$, $$\lambda x + (1-\lambda)y \in C.$$

Geometrically: the line segment connecting any two points in $C$ stays in $C$. Examples: the empty set, all of $\mathbb{R}^n$, halfspaces $\{x:a^Tx\le b\}$, balls $\{x:\|x-c\|\le r\}$, hyperplanes, and convex cones.

Example — Polyhedra

A polyhedron is a set of the form $\{x\in\mathbb{R}^n : Ax\le b\}$ for a matrix $A\in\mathbb{R}^{m\times n}$ and vector $b\in\mathbb{R}^m$. Polyhedra are convex because they are finite intersections of halfspaces, and each halfspace is convex.

A bounded polyhedron is called a polytope. The feasible region of a linear program is a polyhedron.

Operations Preserving Convexity

Theorem 1.1.2 — Convexity-Preserving Operations

The following operations preserve convexity:

  • Intersection: if $C_\alpha$ is convex for each $\alpha$, then $\bigcap_\alpha C_\alpha$ is convex.
  • Affine image: if $C$ is convex and $f(x)=Ax+b$, then $f(C)=\{Ax+b:x\in C\}$ is convex.
  • Affine preimage: if $D$ is convex, then $\{x:Ax+b\in D\}$ is convex.
  • Minkowski sum: $C_1+C_2=\{x_1+x_2:x_1\in C_1,x_2\in C_2\}$ is convex if $C_1,C_2$ are convex.

Notably, the union of convex sets need not be convex.

Convex Hull and Extreme Points

Definition 1.1.3 — Convex Hull

The convex hull of a set $S\subseteq\mathbb{R}^n$ is the smallest convex set containing $S$: $$\operatorname{conv}(S) = \left\{\sum_{i=1}^k \lambda_i x_i : k\ge1,\, x_i\in S,\, \lambda_i\ge0,\, \sum_i\lambda_i=1\right\}.$$

The convex hull of finitely many points is a polytope.

Definition 1.1.4 — Extreme Point

A point $x\in C$ is an extreme point (vertex) of a convex set $C$ if it cannot be written as $x=\lambda y+(1-\lambda)z$ with $y,z\in C$, $y\ne z$, and $\lambda\in(0,1)$. Extreme points cannot be expressed as non-trivial convex combinations of other points in $C$.

Minkowski–Weyl Theorem: every compact convex set is the convex hull of its extreme points. For polytopes, the extreme points are exactly the vertices of the polyhedron.

Recession Cone

Definition 1.1.5 — Recession Cone

The recession cone of a convex set $C$ is $$\operatorname{rec}(C) = \{d\in\mathbb{R}^n : x+td\in C \text{ for all } x\in C,\, t\ge0\}.$$

It captures the directions in which $C$ is unbounded. A convex set is bounded if and only if $\operatorname{rec}(C)=\{0\}$. For the polyhedron $\{x:Ax\le b\}$, the recession cone is $\{d:Ad\le0\}$.

Remark — Convex Cones

A set $K$ is a convex cone if it is convex and closed under non-negative scaling: $x\in K$ and $\lambda\ge0$ imply $\lambda x\in K$. Important examples: the non-negative orthant $\mathbb{R}^n_+$, the second-order cone $\{(x,t):\|x\|\le t\}$, and the cone of positive semidefinite matrices $\mathbb{S}^n_+$.
