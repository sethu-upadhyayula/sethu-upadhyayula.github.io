4.2 The Simplex Method

The simplex method, due to Dantzig (1947), solves LP by moving along edges of the feasible polyhedron from one basic feasible solution to an adjacent one, decreasing the objective at each step. The algorithm exploits the algebraic characterisation of BFS developed in Section 4.1.

Reduced Costs

Definition 4.2.1 — Reduced Costs

Given a basis $B$ with $x_B = A_B^{-1}b$ and $x_N = 0$, the objective value is $c^\top x = c_B^\top x_B = c_B^\top A_B^{-1} b$. Define $y = A_B^{-\top} c_B$ (the dual variables / simplex multipliers). The reduced cost of variable $j$ is $$\bar{c}_j = c_j - c_B^\top A_B^{-1} A_j = c_j - y^\top A_j.$$

For basic indices $j \in B$, $\bar{c}_j = 0$ by construction. The reduced cost measures how much the objective changes per unit increase in $x_j$ from zero, holding the basis fixed.

Theorem 4.2.2 — Optimality Condition

A BFS $x$ with basis $B$ is optimal if and only if $\bar{c}_j \ge 0$ for all non-basic $j \in N$. If some $\bar{c}_j < 0$, then increasing $x_j$ from zero will decrease the objective, so $x$ is not optimal.

Pivot Rules

Definition 4.2.3 — Entering and Leaving Variables

Entering variable (most-negative rule): Choose the non-basic index $q$ with the most negative reduced cost: $$q = \arg\min_{j \in N}\; \bar{c}_j.$$

Let $d = A_B^{-1} A_q$ (the direction of change in basic variables when $x_q$ increases). Minimum ratio test (leaving variable): the maximum step $\theta^*$ before some basic variable hits zero is $$\theta^* = \min_{i : d_i > 0}\; \frac{(x_B)_i}{d_i}.$$ The index $p$ achieving the minimum is the leaving variable. After the pivot, $q$ enters $B$ and $p$ leaves $B$.

If $d \le 0$ component-wise (no positive entries), the problem is unbounded: we can increase $x_q$ to $+\infty$ while maintaining feasibility.

Two-Phase Method for Initial BFS

Definition 4.2.4 — Two-Phase Simplex

When no initial BFS is available, the two-phase method proceeds as follows.

Phase I: Introduce artificial variables $a_i \ge 0$ and solve $$\min\; \sum_{i=1}^m a_i \quad\text{s.t.}\quad Ax + a = b,\; x, a \ge 0.$$ The initial BFS uses $a$ as the basis ($a = b \ge 0$, assuming $b \ge 0$ after row negation). If the Phase I optimum equals zero, we have a feasible BFS for the original problem. If it is positive, the original LP is infeasible.

Phase II: Use the BFS obtained at the end of Phase I as the starting point for the simplex method on the original objective.

Bland's Rule and Cycling Prevention

Theorem 4.2.5 — Bland's Rule

Bland's rule: Among all eligible entering variables ($\bar{c}_j < 0$), choose the one with the smallest index; among eligible leaving variables (ties in the ratio test), choose the one with the smallest index.

Under Bland's rule, the simplex method is guaranteed to terminate in finitely many steps, even for degenerate problems (where $\theta^* = 0$ and the BFS does not change). The proof shows that under Bland's rule, no basis can be repeated, so the finite number of bases implies termination.

Complexity

Remark — Worst-Case and Average-Case Complexity

Klee and Minty (1972) constructed an $n$-variable LP for which the simplex method (with the most-negative reduced cost rule) visits all $2^n$ vertices of a deformed hypercube, giving worst-case exponential complexity.

However, empirically and under probabilistic models of random LPs, the simplex method runs in expected polynomial time (Borgwardt, Spielman–Teng smoothed analysis). In practice it is highly efficient, solving problems with millions of variables in seconds.

Example — One Simplex Iteration

Consider $\min\; -2x_1 - 3x_2$ s.t. $x_1 + x_2 + s_1 = 4$, $2x_1 + x_2 + s_2 = 6$, $x_1, x_2, s_1, s_2 \ge 0$. Initial BFS: $B = \{s_1, s_2\}$, $x_B = (4, 6)$, $\bar{c}_{x_1} = -2$, $\bar{c}_{x_2} = -3$.

Most negative: $x_2$ enters. Direction $d = A_B^{-1} A_{x_2} = (1, 1)^\top$. Ratio test: $\min(4/1, 6/1) = 4$, so $s_1$ leaves. New basis $B = \{x_2, s_2\}$: $x_2 = 4$, $s_2 = 2$, objective $= -12$.

After next iteration with $x_1$ entering, one finds the optimal BFS $(x_1, x_2) = (2, 2)$ with objective $-10$. (Verify: $\bar{c}_j \ge 0$ for all non-basic variables.)
