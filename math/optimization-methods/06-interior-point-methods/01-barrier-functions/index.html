6.1 Barrier Functions and the Central Path

Interior point methods (IPMs) traverse the interior of the feasible set rather than its boundary. The log-barrier approach reformulates the constrained problem as a sequence of unconstrained ones, each solved by Newton's method. The solution path — the central path — converges to the optimal solution.

Log-Barrier Problem

Definition 6.1.1 — Log-Barrier Formulation

Consider the problem $\min f(x)$ s.t. $g_i(x) \le 0$, $i = 1,\ldots,m$, with $f, g_i$ convex and the feasible set having non-empty interior. The log-barrier function is $$\phi(x) = -\sum_{i=1}^m \log(-g_i(x)), \quad \text{dom}\,\phi = \{x : g_i(x) < 0\;\forall i\}.$$ The barrier subproblem with parameter $t > 0$ is the unconstrained problem $$x^*(t) = \arg\min_x\; B(x, t) := t\,f(x) + \phi(x).$$

As $x$ approaches the boundary of the feasible set, $\phi(x) \to +\infty$, preventing the iterates from leaving the feasible interior. For fixed $t$, $B(\cdot, t)$ is strictly convex (under mild assumptions) and has a unique minimiser $x^*(t)$.

The Central Path

Definition 6.1.2 — Central Path

The central path is the curve $\{x^*(t) : t > 0\}$ in the interior of the feasible set. As $t \to \infty$, the barrier term $\phi/t \to 0$ and $x^*(t) \to x^*$ (the constrained minimiser).

The first-order condition for $x^*(t)$ is $t\,\nabla f + \nabla\phi = 0$, i.e., $$t\,\nabla f(x^*(t)) - \sum_i \frac{\nabla g_i(x^*(t))}{-g_i(x^*(t))} = 0.$$ Setting $\lambda_i^*(t) = 1/(-t\,g_i(x^*(t))) > 0$, this becomes $\nabla f + \sum_i \lambda_i^* \nabla g_i = 0$ — the KKT stationarity condition with multipliers $\lambda_i^*$. Thus $x^*(t)$ is the KKT point of the problem with complementary slackness perturbed to $\lambda_i^* g_i = -1/t$.

Theorem 6.1.3 — Duality Gap Along the Central Path

At the central path point $x^*(t)$, the duality gap is exactly $m/t$: $$f(x^*(t)) - f(x^*) \le \frac{m}{t}.$$

Proof sketch. The dual variables $\lambda_i^*(t) = 1/(-t\,g_i(x^*(t)))$ satisfy $-\sum_i \lambda_i^* g_i = m/t$. By duality, the duality gap $f(x^*(t)) - d^*(\lambda^*(t)) = -\sum_i \lambda_i^* g_i = m/t$. $\square$

Barrier Method Algorithm

Definition 6.1.4 — Barrier Method

The barrier method solves the original problem to tolerance $\varepsilon$ by the following procedure:

  1. Choose initial interior point $x_0$, initial $t_0 > 0$, scaling factor $\mu > 1$.
  2. For $k = 0, 1, 2, \ldots$: compute $x^*(t_k)$ by minimising $B(x, t_k)$ starting from $x_k$, typically using Newton's method (Section 2.3). Set $x_{k+1} = x^*(t_k)$, $t_{k+1} = \mu t_k$.
  3. Stop when $m/t_k \le \varepsilon$ (duality gap $\le \varepsilon$).

The number of outer iterations is $O(\log(m/(t_0\varepsilon)) / \log \mu)$. Each inner Newton solve requires $O(m)$ iterations (quadratic convergence). The total iteration count for LP is $O(\sqrt{n}\log(1/\varepsilon))$.

Analytic Centre

Definition 6.1.5 — Analytic Centre

The analytic centre of the feasible set $\{x : g_i(x) \le 0\}$ is the minimiser of $\phi(x) = -\sum_i \log(-g_i(x))$: $$x_{\mathrm{ac}} = \arg\min_x\; \phi(x).$$

This is the central path point $x^*(0^+)$ (formal limit as $t \to 0$). For an LP polytope $\{x : Ax \le b\}$, the analytic centre is the point that maximises the product $\prod_i (b_i - A_i x)$. It lies deep in the interior and is the natural starting point for interior point methods.

Remark — Newton Step for the Barrier Subproblem

The barrier subproblem $\min B(x,t) = tf(x) - \sum_i \log(-g_i(x))$ is solved by Newton's method. The Hessian of $B$ is $$\nabla^2 B = t\,\nabla^2 f + \sum_i \frac{\nabla g_i \nabla g_i^\top}{g_i^2} - \sum_i \frac{\nabla^2 g_i}{g_i}.$$

For LP, $f$ is linear and $g_i$ are linear, so $\nabla^2 B = \sum_i \frac{a_i a_i^\top}{s_i^2}$ where $s_i = -g_i(x)$ are the slack variables. This is the scaled normal equation system underlying primal-dual methods (Section 6.2).
