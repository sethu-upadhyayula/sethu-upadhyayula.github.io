4.2 Huffman Codes

Huffman coding constructs an optimal variable-length prefix-free code for a given set of symbols and their frequencies. It is used in GZIP, JPEG, MP3, and many other compression formats.

Definition 4.2.1 - Prefix-Free Code

A prefix-free (or prefix) code is a binary code in which no codeword is a prefix of another. Prefix-free codes correspond bijectively to full binary tries: leaves are symbols, and the path from root to a leaf gives the codeword. The expected code length is $\bar{\ell} = \sum_i f_i \ell_i$ where $f_i$ is the frequency and $\ell_i$ is the code length of symbol $i$.

Algorithm 4.2.2 - Huffman's Algorithm

Given symbols $s_1, \ldots, s_n$ with frequencies $f_1, \ldots, f_n$:

  1. Insert all symbols as leaf nodes into a min-priority queue, keyed by frequency.
  2. Repeat $n-1$ times: extract the two nodes $x, y$ with minimum frequencies; create a new internal node $z$ with $f_z = f_x + f_y$ and children $x, y$; insert $z$ into the queue.
  3. The remaining node is the root of the Huffman tree.

Time: $O(n \log n)$ using a binary heap.

Proof of Optimality Sketch

Lemma: In an optimal code, the two least-frequent symbols $x$ and $y$ have the longest codeword lengths and can always be made siblings (same depth). (Exchange argument: swapping the deepest leaf with $x$ or $y$ does not increase $\bar{\ell}$.)

Induction: Replacing $x$ and $y$ with a meta-symbol $z$ of frequency $f_x + f_y$ reduces to a subproblem. By induction, Huffman solves the subproblem optimally, giving an optimal code for the original problem. $\square$

Information-Theoretic Bounds

Theorem 4.2.3 - Shannon Entropy Bound

Let $H(p) = -\sum_i f_i \log_2 f_i$ be the Shannon entropy of the frequency distribution. Then the expected code length $\bar{\ell}$ of any prefix-free code satisfies $H(p) \le \bar{\ell} < H(p) + 1$. The Huffman code achieves $\bar{\ell} < H(p) + 1$, and there is no prefix-free code with $\bar{\ell} < H(p)$. Thus Huffman coding is at most 1 bit per symbol above the information-theoretic optimum.
