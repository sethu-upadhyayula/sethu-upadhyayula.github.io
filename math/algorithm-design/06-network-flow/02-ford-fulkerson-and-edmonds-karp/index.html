6.2 Ford–Fulkerson and Edmonds–Karp

The Ford-Fulkerson method iteratively augments flow along $s$-$t$ paths in the residual graph. The choice of augmenting path determines termination and running time; Edmonds-Karp's choice of BFS (shortest path) yields the strongest polynomial guarantee.

Definition 6.2.1 — Residual Graph

Given flow network $G$ and flow $f$, the residual graph $G_f$ has the same vertex set. For each edge $(u,v) \in E$:

  • Forward residual edge $(u,v)$ with capacity $c_f(u,v) = c(u,v) - f(u,v)$ (remaining capacity).
  • Backward residual edge $(v,u)$ with capacity $c_f(v,u) = f(u,v)$ (flow that can be cancelled).

Only edges with positive residual capacity are included. An augmenting path is any $s$-$t$ path in $G_f$. Its bottleneck is $\Delta = \min_{e \in P} c_f(e)$. Augmenting by $\Delta$ increases $|f|$ by $\Delta$.

Theorem 6.2.2 — Ford-Fulkerson Method

Algorithm: While there exists an augmenting path $P$ in $G_f$: augment by the bottleneck of $P$. When no augmenting path exists, $f$ is a maximum flow (by the max-flow min-cut theorem).

Termination (integer capacities): Each augmentation increases $|f|$ by at least 1. Since the max flow is at most $\sum_{(s,v)} c(s,v) < \infty$, the algorithm terminates. Total time: $O(E \cdot |f^*|)$ where $f^*$ is the maximum flow.

Caution: With irrational capacities, Ford-Fulkerson may not terminate and may not converge to the maximum flow.

Theorem 6.2.3 — Edmonds-Karp Algorithm

Algorithm: Ford-Fulkerson using BFS to find the shortest augmenting path (fewest edges) at each step.

Key property: The BFS distance $d_f(s,v)$ in $G_f$ never decreases as augmentations proceed. Each edge $(u,v)$ can become a bottleneck at most $O(V)$ times (because after becoming a bottleneck, $d_f(s,u)$ must increase by 2 before it can be a bottleneck again). Total augmentations: $O(VE)$. Each augmentation: $O(E)$.

Total time: $O(VE^2)$.

Remark 6.2.4 — Dinic's Algorithm

Dinic's algorithm improves on Edmonds-Karp by augmenting along all shortest paths simultaneously using a blocking flow in the level graph (BFS layers). After each blocking flow, the BFS distance from $s$ to $t$ strictly increases. Since $d_f(s,t) \le V-1$, there are at most $V-1$ phases. Each phase computes a blocking flow in $O(VE)$.

Total time: $O(V^2 E)$. For unit-capacity networks: $O(E\sqrt{V})$.

Example 6.2.5 — Ford-Fulkerson Pathological Case

With irrational capacities (or adversarial rational ones), Ford-Fulkerson using DFS augmenting paths may take exponentially many iterations. Example: a graph designed so alternating augmenting paths each increase flow by $1/\phi^k$ (where $\phi$ is the golden ratio) gives a sequence that converges to a non-maximum flow. Edmonds-Karp avoids this by always using shortest paths.
