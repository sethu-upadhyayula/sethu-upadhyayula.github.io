1.2 Recurrences and the Master Theorem

Divide-and-conquer algorithms naturally produce recurrences of the form $T(n) = aT(n/b) + f(n)$. The Master Theorem gives the asymptotic solution in three clean cases determined by comparing $f(n)$ with the "watershed" function $n^{\log_b a}$.

Theorem 1.2.1 - Master Theorem

Let $T(n) = aT(n/b) + f(n)$ where $a \ge 1$, $b > 1$ are constants, and set $c^* = \log_b a$.

Case 1: If $f(n) = O(n^{c^* - \varepsilon})$ for some $\varepsilon > 0$, then $T(n) = \Theta(n^{c^*})$. (Subproblems dominate.)

Case 2: If $f(n) = \Theta(n^{c^*} \log^k n)$ for some $k \ge 0$, then $T(n) = \Theta(n^{c^*} \log^{k+1} n)$. (Even split.)

Case 3: If $f(n) = \Omega(n^{c^* + \varepsilon})$ for some $\varepsilon > 0$, and $af(n/b) \le \delta f(n)$ for some $\delta < 1$ and large $n$, then $T(n) = \Theta(f(n))$. (Combination step dominates.)

Key Examples

Example 1.2.2 - Merge Sort

$T(n) = 2T(n/2) + \Theta(n)$. Here $a = 2$, $b = 2$, $c^* = \log_2 2 = 1$, $f(n) = \Theta(n) = \Theta(n^1 \log^0 n)$. Case 2 with $k = 0$: $T(n) = \Theta(n \log n)$.

Example 1.2.3 - Binary Search

$T(n) = T(n/2) + O(1)$. Here $a = 1$, $b = 2$, $c^* = 0$, $f(n) = O(1) = \Theta(n^0)$. Case 2 with $k = 0$: $T(n) = \Theta(\log n)$.

Example 1.2.4 - Strassen's Algorithm

$T(n) = 7T(n/2) + \Theta(n^2)$. Here $c^* = \log_2 7 \approx 2.807$, $f(n) = \Theta(n^2) = O(n^{2.807 - 0.8})$. Case 1: $T(n) = \Theta(n^{\log_2 7}) \approx O(n^{2.807})$.

Example 1.2.5 - Karatsuba Multiplication

$T(n) = 3T(n/2) + \Theta(n)$. Here $c^* = \log_2 3 \approx 1.585$, $f(n) = \Theta(n) = O(n^{1.585 - 0.5})$. Case 1: $T(n) = \Theta(n^{\log_2 3}) \approx \Theta(n^{1.585})$, beating the $O(n^2)$ naive algorithm.

Beyond the Master Theorem

Remark - Akra-Bazzi Theorem

The Akra-Bazzi theorem generalises the Master Theorem to recurrences $T(n) = \sum_{i=1}^k a_i T(b_i n) + f(n)$ with possibly different ratios $b_i$. The solution is $T(n) = \Theta\!\left(n^p\!\left(1 + \int_1^n \frac{f(u)}{u^{p+1}} du\right)\right)$ where $p$ satisfies $\sum a_i b_i^p = 1$.
