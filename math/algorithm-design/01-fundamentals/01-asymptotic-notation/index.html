1.1 Asymptotic Notation

Asymptotic notation provides a precise, machine-independent language for comparing algorithm running times. We discard constants and lower-order terms, focusing on how running time scales with input size $n$.

Definition 1.1.1 — Big-$O$ Notation

$f(n) = O(g(n))$ if there exist constants $c > 0$ and $n_0 \ge 1$ such that $f(n) \le c \cdot g(n)$ for all $n \ge n_0$. Informally, $f$ grows no faster than $g$.

Definition 1.1.2 — $\Omega$, $\Theta$, $o$, $\omega$

$f = \Omega(g)$: $f(n) \ge c \cdot g(n)$ eventually (lower bound). $f = \Theta(g)$: both $f = O(g)$ and $f = \Omega(g)$ (tight bound).

$f = o(g)$: $f(n)/g(n) \to 0$ as $n \to \infty$ (strictly slower). $f = \omega(g)$: $f(n)/g(n) \to \infty$ (strictly faster).

Relationship: $f = \Theta(g) \iff f = O(g) \text{ and } f = \Omega(g) \iff g = \Theta(f)$.

Standard Complexity Hierarchy

Theorem 1.1.3 — Growth Rate Hierarchy

The following functions are listed in strictly increasing order of asymptotic growth: $$1 \ll \log \log n \ll \log n \ll \sqrt{n} \ll n \ll n \log n \ll n^2 \ll n^3 \ll 2^n \ll n! \ll n^n.$$ That is, each function is $o$ of the next.

Example 1.1.4 — Common Algorithm Complexities

$O(1)$: array access, hash table lookup. $O(\log n)$: binary search, balanced BST. $O(n)$: linear scan, BFS/DFS. $O(n \log n)$: merge sort, heap sort, FFT.

$O(n^2)$: bubble/insertion sort, brute-force matrix multiply. $O(n^3)$: Floyd–Warshall, naive matrix multiply. $O(2^n)$: subset enumeration. $O(n!)$: brute-force TSP.

Arithmetic Rules

Theorem 1.1.5 — Rules for Asymptotic Notation

$O(f) + O(g) = O(\max(f, g))$ (dominated by the larger term).

$O(f) \cdot O(g) = O(f \cdot g)$ (multiply running times in nested loops).

$O(c \cdot f) = O(f)$ for any constant $c > 0$ (constants don't matter).

$\log_a n = \Theta(\log n)$ for any base $a > 1$ (log base doesn't matter for $\Theta$).

$\log n^k = k \log n = O(\log n)$ for constant $k$ (polynomial powers of $\log$ stay $O(\log n)$); but $n^{0.01} = \omega(\log n)$ (any polynomial dominates any logarithm).

Remark — Limits and Asymptotic Comparison

If $\lim_{n\to\infty} f(n)/g(n) = L$ with $0 < L < \infty$, then $f = \Theta(g)$. If $L = 0$ then $f = o(g)$; if $L = \infty$ then $f = \omega(g)$. L'Hôpital's rule and Stirling's approximation ($n! \sim \sqrt{2\pi n}(n/e)^n$) are often useful here.
