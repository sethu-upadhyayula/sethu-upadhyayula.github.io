1.1 Definition and Examples

Definition - Vector Space

A vector space over $\mathbb{R}$ is a set $V$ together with two operations - addition $+: V\times V\to V$ and scalar multiplication $\cdot:\mathbb{R}\times V\to V$ - satisfying the following axioms for all $u,v,w\in V$ and $c,d\in\mathbb{R}$:

  1. Closure under addition: $u+v\in V$.
  2. Commutativity: $u+v = v+u$.
  3. Associativity of addition: $(u+v)+w = u+(v+w)$.
  4. Additive identity: There exists $\mathbf{0}\in V$ such that $v+\mathbf{0}=v$ for all $v$.
  5. Additive inverses: For each $v$ there exists $-v\in V$ with $v+(-v)=\mathbf{0}$.
  6. Closure under scalar multiplication: $cv\in V$.
  7. Distributivity over vectors: $c(u+v)=cu+cv$.
  8. Distributivity over scalars: $(c+d)v=cv+dv$.
  9. Associativity of scalar multiplication: $c(dv)=(cd)v$.
  10. Scalar identity: $1\cdot v=v$.

Elements of $V$ are called vectors and elements of $\mathbb{R}$ are called scalars.

Example - $\mathbb{R}^n$

The prototypical vector space is $\mathbb{R}^n = \{(x_1,\ldots,x_n) : x_i\in\mathbb{R}\}$ with componentwise operations: $$(x_1,\ldots,x_n)+(y_1,\ldots,y_n) = (x_1+y_1,\ldots,x_n+y_n),\qquad c(x_1,\ldots,x_n)=(cx_1,\ldots,cx_n).$$ The zero vector is $\mathbf{0}=(0,\ldots,0)$ and $-(x_1,\ldots,x_n)=(-x_1,\ldots,-x_n)$. All ten axioms are inherited directly from the field $\mathbb{R}$.

Example - Matrices $\mathbb{R}^{m\times n}$

The set $\mathbb{R}^{m\times n}$ of all $m\times n$ real matrices forms a vector space under entrywise addition and scalar multiplication: $$(A+B)_{ij} = A_{ij}+B_{ij}, \qquad (cA)_{ij} = cA_{ij}.$$ The zero element is the $m\times n$ zero matrix $0_{m\times n}$. This space has dimension $mn$ (see §1.4).

Example - Polynomials $\mathcal{P}_n$

Let $\mathcal{P}_n$ denote the set of all polynomials of degree at most $n$ with real coefficients: $$\mathcal{P}_n = \{a_0 + a_1 t + a_2 t^2 + \cdots + a_n t^n : a_i\in\mathbb{R}\}.$$ Addition and scalar multiplication are the usual polynomial operations. The zero polynomial serves as $\mathbf{0}$. Note $\mathcal{P}_n$ is a subspace of $\mathcal{P}$ (all polynomials), which is itself an infinite-dimensional vector space.

Example - Continuous Functions $C[a,b]$

The set $C[a,b]$ of all continuous real-valued functions on a closed interval $[a,b]$ is a vector space under pointwise operations: $$(f+g)(t)=f(t)+g(t), \qquad (cf)(t)=c\,f(t).$$ The zero vector is the identically-zero function $\mathbf{0}(t)=0$. More generally, $C^k[a,b]$ ($k$-times continuously differentiable) and $C^\infty[a,b]$ (smooth functions) are also vector spaces, all infinite-dimensional.

Non-Example - $\mathbb{R}^2$ with non-standard addition

Define addition on $\mathbb{R}^2$ by $(x_1,y_1)\oplus(x_2,y_2)=(x_1 x_2,\, y_1 y_2)$ (componentwise multiplication). This fails the additive identity axiom: a candidate identity $(e_1,e_2)$ must satisfy $(x,y)\oplus(e_1,e_2)=(x,y)$ for all $(x,y)$, forcing $e_1=1$, $e_2=1$. But then $(0,1)\oplus(-e_1,-e_2)$ cannot equal $(0,0)$, so the pair $(0,1)$ has no additive inverse. Hence this is not a vector space.

Theorem - Elementary Consequences of the Axioms

Let $V$ be a vector space. Then:

  • The zero vector $\mathbf{0}$ is unique.
  • The additive inverse $-v$ is unique for each $v\in V$.
  • $0\cdot v = \mathbf{0}$ for all $v\in V$.
  • $c\cdot\mathbf{0} = \mathbf{0}$ for all $c\in\mathbb{R}$.
  • $(-1)v = -v$ for all $v\in V$.
Proof - $0\cdot v = \mathbf{0}$

By distributivity over scalars (axiom 8): $$0\cdot v = (0+0)\cdot v = 0\cdot v + 0\cdot v.$$ Adding $-(0\cdot v)$ to both sides: $$\mathbf{0} = 0\cdot v.$$

$\square$
