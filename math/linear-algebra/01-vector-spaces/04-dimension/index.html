1.4 Dimension

Theorem — All Bases Have the Same Size

Let $V$ be a finite-dimensional vector space. If $\mathcal{B}=(b_1,\ldots,m)$ and $\mathcal{B}'=(b_1',\ldots,b_k')$ are both bases for $V$, then $|\mathcal{B}|=|\mathcal{B}'|$. This common cardinality is called the dimension of $V$, denoted $\dim V$.

Proof Sketch

Suppose $\mathcal{B}$ has $n$ elements and $\mathcal{B}'$ has $k$ elements. Since $\mathcal{B}$ spans $V$, every vector in $\mathcal{B}'$ is a linear combination of $\mathcal{B}$. By a standard exchange/replacement argument, any linearly independent set has at most as many elements as any spanning set. Hence $k\le n$. By symmetry $n\le k$, so $n=k$.

$\square$

Example — Dimensions of Common Spaces

  • $\dim(\mathbb{R}^n) = n$: the standard basis has $n$ elements.
  • $\dim(\mathcal{P}_n) = n+1$: the monomial basis $\{1,t,\ldots,t^n\}$ has $n+1$ elements.
  • $\dim(\mathbb{R}^{m\times n}) = mn$: the $mn$ elementary matrices $E_{ij}$ form a basis.
  • $\dim(\{\mathbf{0}\}) = 0$: the empty set is a basis for the trivial space.
  • $\dim(C[a,b]) = \infty$: for any $n$, the functions $1,t,t^2,\ldots,t^n$ are $n+1$ linearly independent vectors in $C[a,b]$, so no finite basis exists.

Remark — Dimension and Subspaces

If $W$ is a subspace of a finite-dimensional vector space $V$, then $\dim W \le \dim V$, with equality if and only if $W=V$. This gives a quick test: a subspace of the right dimension is the whole space.

Theorem — Rank-Nullity

Let $T:V\to W$ be a linear transformation with $V$ finite-dimensional. Define: $$\operatorname{nullity}(T) = \dim(\ker T), \qquad \operatorname{rank}(T) = \dim(\operatorname{im}\, T).$$ Then: $$\operatorname{nullity}(T) + \operatorname{rank}(T) = \dim V.$$

Proof

Let $\{u_1,\ldots,u_k\}$ be a basis for $\ker T$. By the basis extension theorem, extend it to a basis $\{u_1,\ldots,u_k,v_1,\ldots,v_r\}$ for $V$, so $k+r=\dim V$.

Claim: $\{T(v_1),\ldots,T(v_r)\}$ is a basis for $\operatorname{im}\, T$.

Spanning: for any $T(v)\in\operatorname{im}\, T$, write $v=\sum c_i u_i + \sum d_j v_j$. Then $T(v)=\sum c_i T(u_i)+\sum d_j T(v_j)=\sum d_j T(v_j)$, since $T(u_i)=\mathbf{0}$.

Independence: if $\sum d_j T(v_j)=\mathbf{0}$, then $T(\sum d_j v_j)=\mathbf{0}$, so $\sum d_j v_j\in\ker T$. Write $\sum d_j v_j = \sum e_i u_i$. Then $\sum d_j v_j - \sum e_i u_i=\mathbf{0}$; by independence of the whole basis, all $d_j=0$.

Hence $\dim(\operatorname{im}\, T)=r$ and the theorem follows: $k+r=\dim V$.

$\square$

Example — Rank-Nullity for Differentiation

Let $D:\mathcal{P}_n\to\mathcal{P}_{n-1}$ be the differentiation operator $D(p)=p'$. Then $\ker D = \{$constant polynomials$\}$, so $\operatorname{nullity}(D)=1$. By rank-nullity, $\operatorname{rank}(D)=(n+1)-1=n$, meaning $D$ is surjective (every polynomial of degree $\le n-1$ is the derivative of some polynomial of degree $\le n$).
