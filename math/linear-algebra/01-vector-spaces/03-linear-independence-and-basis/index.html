1.3 Linear Independence and Basis

Definition - Linear Independence

A list of vectors $v_1,\ldots,v_k\in V$ is linearly independent if the only scalars $c_1,\ldots,c_k\in\mathbb{R}$ satisfying $$c_1 v_1 + c_2 v_2 + \cdots + c_k v_k = \mathbf{0}$$ are $c_1=c_2=\cdots=c_k=0$. Otherwise the list is linearly dependent, meaning there exists a non-trivial solution, and consequently some $v_j$ is a linear combination of the others.

Example - Standard Basis Vectors in $\mathbb{R}^n$

The vectors $e_1=(1,0,\ldots,0),\, e_2=(0,1,\ldots,0),\,\ldots,\, e_n=(0,\ldots,0,1)$ are linearly independent: if $\sum c_i e_i = \mathbf{0}$ then $(c_1,\ldots,c_n)=(0,\ldots,0)$, so each $c_i=0$.

Example - Dependent Set

The vectors $v_1=(1,2)$, $v_2=(2,4)$ in $\mathbb{R}^2$ are linearly dependent: $2v_1 - v_2 = (2,4)-(2,4)=\mathbf{0}$, so $c_1=2$, $c_2=-1$ is a non-trivial solution.

Example - Monomials Are Independent

The list $1, t, t^2, \ldots, t^n$ in $\mathcal{P}_n$ is linearly independent: if $a_0 + a_1 t + \cdots + a_n t^n = 0$ (the zero polynomial, meaning it vanishes for all $t$), then differentiating repeatedly and evaluating at $t=0$ gives $a_0=a_1=\cdots=a_n=0$.

Definition - Basis

A basis for a vector space $V$ is an ordered list $\mathcal{B}=(b_1,\ldots,b_n)$ of vectors that is simultaneously linearly independent and spanning ($\operatorname{span}\mathcal{B}=V$).

Theorem - Unique Representation

If $\mathcal{B}=(b_1,\ldots,b_n)$ is a basis for $V$, then every vector $v\in V$ has a unique representation $$v = c_1 b_1 + c_2 b_2 + \cdots + c_n b_n.$$ The scalars $(c_1,\ldots,c_n)$ are called the coordinates of $v$ with respect to $\mathcal{B}$, denoted $[v]_{\mathcal{B}} = (c_1,\ldots,c_n)^T\in\mathbb{R}^n$.

Proof

Existence: since $\mathcal{B}$ spans $V$, every $v$ is a linear combination $v=\sum c_i b_i$.

Uniqueness: suppose also $v=\sum d_i b_i$. Then $\sum(c_i-d_i)b_i=\mathbf{0}$. By linear independence, $c_i-d_i=0$ for all $i$.

$\square$

Example - Standard Bases

  • $\mathbb{R}^n$: standard basis $\mathcal{E}=(e_1,\ldots,e_n)$; coordinates are just the components: $[v]_{\mathcal{E}}=v$.
  • $\mathcal{P}_n$: monomial basis $(1,t,t^2,\ldots,t^n)$ of size $n+1$; coordinates of $\sum a_i t^i$ are $(a_0,\ldots,a_n)^T$.
  • $\mathbb{R}^{m\times n}$: elementary matrices $E_{ij}$ (with a single $1$ at position $(i,j)$) form a basis of size $mn$.

Theorem - Basis Extension and Compression

Let $V$ be finite-dimensional. Then:

  • Any linearly independent set in $V$ can be extended to a basis of $V$.
  • Any spanning set for $V$ can be reduced (by removing vectors) to a basis.

In particular, every finite-dimensional vector space has a basis.
