6.1 Symmetric Matrices

Definition — Symmetric Matrix

A matrix $A\in M_n(\mathbb{R})$ is symmetric if $A=A^T$, i.e., $a_{ij}=a_{ji}$ for all $i,j$. Symmetric matrices arise naturally as covariance matrices, Laplacians of graphs, and Hessians in optimisation.

Theorem — Real Eigenvalues

Every eigenvalue of a real symmetric matrix is real.

Proof — Real Eigenvalues

Let $Av=\lambda v$ with $v\neq\mathbf{0}$ (over $\mathbb{C}$). Take the inner product (over $\mathbb{C}$) with $v$:

$$\lambda\|v\|^2=\langle Av,v\rangle=v^*(Av)=(Av)^*v=(v^*A^T)\overline{v}.$$

Since $A=A^T$ and $A$ is real: $\langle Av,v\rangle=v^*Av$. Also $\langle v,Av\rangle=\bar\lambda\|v\|^2$. But $\langle Av,v\rangle=\overline{\langle v,Av\rangle}$, so $\lambda\|v\|^2=\bar\lambda\|v\|^2$. Since $\|v\|^2>0$, we get $\lambda=\bar\lambda\in\mathbb{R}$.

$\square$

Theorem — Orthogonality of Eigenvectors

Eigenvectors of a real symmetric matrix corresponding to distinct eigenvalues are orthogonal.

Proof — Orthogonality

Let $Av_1=\lambda_1 v_1$ and $Av_2=\lambda_2 v_2$ with $\lambda_1\neq\lambda_2$. Then:

$$\lambda_1\langle v_1,v_2\rangle=\langle Av_1,v_2\rangle=\langle v_1,A^Tv_2\rangle=\langle v_1,Av_2\rangle=\lambda_2\langle v_1,v_2\rangle.$$

So $(\lambda_1-\lambda_2)\langle v_1,v_2\rangle=0$. Since $\lambda_1\neq\lambda_2$, we conclude $\langle v_1,v_2\rangle=0$.

$\square$

Definition — Positive Definite Matrix

A symmetric matrix $A=A^T$ is:

  • Positive definite (PD) if $x^TAx>0$ for all $x\neq\mathbf{0}$; equivalently, all eigenvalues $\lambda_i>0$.
  • Positive semi-definite (PSD) if $x^TAx\ge0$ for all $x$; equivalently, all $\lambda_i\ge0$.
  • Negative definite / semi-definite: analogous with signs reversed.
  • Indefinite: has both positive and negative eigenvalues.

Cholesky decomposition: $A$ is symmetric positive definite if and only if there exists a unique lower triangular matrix $L$ with positive diagonal entries such that $A=LL^T$. This is the most efficient way to solve $Ax=b$ when $A$ is PD.

Example — Testing Positive Definiteness

Let $A=\begin{pmatrix}4&2\\2&3\end{pmatrix}$. Eigenvalues: $p(\lambda)=(\lambda-4)(\lambda-3)-4=\lambda^2-7\lambda+8$; roots $\lambda=\frac{7\pm\sqrt{17}}{2}$, both positive. Hence $A$ is positive definite.

Cholesky: $A=LL^T$ where $L=\begin{pmatrix}2&0\\1&\sqrt{2}\end{pmatrix}$. Check: $LL^T=\begin{pmatrix}4&2\\2&3\end{pmatrix}=A$. $\checkmark$
