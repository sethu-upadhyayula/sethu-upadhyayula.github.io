6.3 Singular Value Decomposition

Theorem — Singular Value Decomposition (SVD)

Every matrix $A\in M_{m\times n}(\mathbb{R})$ of rank $r$ admits the singular value decomposition:

$$A = U\Sigma V^T$$

where:

  • $U\in M_{m\times m}(\mathbb{R})$ is orthogonal ($U^TU=I_m$); its columns $u_1,\ldots,u_m$ are the left singular vectors.
  • $V\in M_{n\times n}(\mathbb{R})$ is orthogonal ($V^TV=I_n$); its columns $v_1,\ldots,v_n$ are the right singular vectors.
  • $\Sigma\in M_{m\times n}(\mathbb{R})$ has singular values $\sigma_1\ge\sigma_2\ge\cdots\ge\sigma_r>0=\sigma_{r+1}=\cdots$ on the main diagonal, and zeros elsewhere.
Construction — Existence of SVD

The matrix $A^TA\in M_n(\mathbb{R})$ is symmetric positive semi-definite. By the spectral theorem, $A^TA=V\Lambda V^T$ with eigenvalues $\lambda_1\ge\cdots\ge\lambda_n\ge0$. Define:

  • Singular values: $\sigma_i=\sqrt{\lambda_i}$ for $i=1,\ldots,r$ (where $r=\operatorname{rank}(A)$).
  • Right singular vectors: columns $v_i$ of $V$ (eigenvectors of $A^TA$).
  • Left singular vectors: $u_i=Av_i/\sigma_i$ for $i=1,\ldots,r$; extend to an orthonormal basis of $\mathbb{R}^m$.

One verifies $Av_i=\sigma_i u_i$ for $i\le r$ and $Av_i=0$ for $i>r$, which gives $A=U\Sigma V^T$.

Remark — Geometric Interpretation

The SVD factorises $A$ as three operations: $V^T$ (rotation/reflection in the input space), $\Sigma$ (scaling along the coordinate axes), $U$ (rotation/reflection in the output space). The linear map $x\mapsto Ax$ sends the unit sphere $\|x\|=1$ in $\mathbb{R}^n$ to an ellipsoid in $\mathbb{R}^m$ with semi-axes of lengths $\sigma_1,\ldots,\sigma_r$ in the directions $u_1,\ldots,u_r$.

Theorem — Eckart–Young Low-Rank Approximation

Given the SVD $A=U\Sigma V^T=\sum_{i=1}^r\sigma_i u_iv_i^T$, define the rank-$k$ truncation ($k\le r$):

$$A_k=\sum_{i=1}^k\sigma_i u_iv_i^T.$$

Then $A_k$ is the best rank-$k$ approximation to $A$ in both the spectral norm and the Frobenius norm:

$$\|A-A_k\|_2=\sigma_{k+1},\qquad\|A-A_k\|_F=\sqrt{\sigma_{k+1}^2+\cdots+\sigma_r^2}.$$

No rank-$k$ matrix achieves a smaller error in either of these norms.

Definition — Moore–Penrose Pseudo-Inverse

Given $A=U\Sigma V^T$, the pseudo-inverse is: $$A^+=V\Sigma^+U^T,$$ where $\Sigma^+$ is obtained from $\Sigma^T$ by replacing each nonzero singular value $\sigma_i$ by $1/\sigma_i$.

Properties: $AA^+A=A$; $A^+AA^+=A^+$; $(AA^+)^T=AA^+$; $(A^+A)^T=A^+A$. When $A$ has full column rank: $A^+=(A^TA)^{-1}A^T$ (recovers the least-squares formula from §5.3).

Applications of SVD
  • PCA (Principal Component Analysis): The right singular vectors $v_1,\ldots,v_k$ of a data matrix give the principal components; $\sigma_i^2$ is proportional to the variance explained by the $i$-th component.
  • Image compression: Store only the first $k$ terms $\sigma_iu_iv_i^T$; the number of values stored is $k(m+n)$ instead of $mn$. For small $k$ this is a significant compression.
  • Recommender systems: Collaborative filtering uses the low-rank SVD approximation to fill in missing entries of a user-item rating matrix.
  • Numerical linear algebra: The condition number $\kappa(A)=\sigma_1/\sigma_r$ measures sensitivity of $Ax=b$ to perturbations; small $\sigma_r$ signals near-singularity.
