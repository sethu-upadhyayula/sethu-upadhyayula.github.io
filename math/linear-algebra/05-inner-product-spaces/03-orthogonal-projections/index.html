5.3 Orthogonal Projections

Theorem - Projection onto a Subspace

Let $W$ be a subspace of an inner product space $V$ with orthonormal basis $\{e_1,\ldots,e_k\}$. The orthogonal projection of $v\in V$ onto $W$ is:

$$\operatorname{proj}_W(v)=\sum_{i=1}^k\langle v,e_i\rangle e_i.$$

This is the unique element of $W$ that minimises $\|v-w\|$ over all $w\in W$: the error $v-\operatorname{proj}_W(v)$ is orthogonal to every vector in $W$.

Definition - Orthogonal Complement

The orthogonal complement of a subspace $W\subseteq V$ is: $$W^\perp=\{x\in V:\langle x,w\rangle=0\text{ for all }w\in W\}.$$

$W^\perp$ is always a subspace of $V$. If $V$ is finite-dimensional: $V=W\oplus W^\perp$ (every $v$ decomposes uniquely as $v=w+w^\perp$ with $w\in W$, $w^\perp\in W^\perp$), and $\dim W+\dim W^\perp=\dim V$.

Theorem - Projection Matrix

If $Q\in M_{m\times k}(\mathbb{R})$ has orthonormal columns spanning $W$, the projection matrix onto $W$ is:

$$P=QQ^T.$$

Key properties of a projection matrix:

  • $P^2=P$ (idempotent).
  • $P^T=P$ (symmetric).
  • $\operatorname{rank}(P)=\dim W=k$.
  • Eigenvalues are $0$ and $1$ only.

In general (without assuming orthonormal columns): if $A$ has linearly independent columns spanning $W$, then $P=A(A^TA)^{-1}A^T$.

Theorem - Least Squares

Given $A\in M_{m\times n}(\mathbb{R})$ (with $m>n$) and $\mathbf{b}\in\mathbb{R}^m$, when the system $Ax=\mathbf{b}$ has no exact solution, the least-squares solution minimises $\|Ax-\mathbf{b}\|^2$. It satisfies the normal equations:

$$A^TAx=A^T\mathbf{b}.$$

If $A$ has full column rank (columns linearly independent), then $A^TA$ is invertible and: $$\hat x=(A^TA)^{-1}A^T\mathbf{b}.$$

The vector $A\hat x=P\mathbf{b}$ is the projection of $\mathbf{b}$ onto $\operatorname{col}(A)$.

Example - Fitting a Line to Data

Find the best-fit line $y=a+bx$ to the data points $(0,1),(1,2),(2,4)$.

Set up $Ax=\mathbf{b}$: $$A=\begin{pmatrix}1&0\\1&1\\1&2\end{pmatrix},\quad\mathbf{b}=\begin{pmatrix}1\\2\\4\end{pmatrix},\quad x=\begin{pmatrix}a\\b\end{pmatrix}.$$

$A^TA=\begin{pmatrix}3&3\\3&5\end{pmatrix}$, $A^T\mathbf{b}=\begin{pmatrix}7\\10\end{pmatrix}$. Solving the normal equations: $$\begin{pmatrix}3&3\\3&5\end{pmatrix}\begin{pmatrix}a\\b\end{pmatrix}=\begin{pmatrix}7\\10\end{pmatrix}\implies a=\frac{5}{6},\;b=\frac{3}{2}.$$ Best-fit line: $y=\frac{5}{6}+\frac{3}{2}x$.
