3.2 Properties

Theorem — Multiplicativity

For any $n\times n$ matrices $A$ and $B$:

$$\det(AB) = \det A \cdot \det B.$$

Consequently: $\det(A^k)=(\det A)^k$, and if $A$ is invertible then $\det(A^{-1})=(\det A)^{-1}$.

Theorem — Transpose

For any square matrix $A$: $\det(A^T)=\det A$. This implies that every property of the determinant stated for rows holds equally for columns.

Theorem — Invertibility Criterion

A matrix $A\in M_n(\mathbb{R})$ is invertible if and only if $\det A\neq 0$. Equivalently, the rows (or columns) of $A$ are linearly independent if and only if $\det A\neq 0$.

Theorem — Effect of Row Operations

Elementary row operations change $\det$ as follows:

  • Row swap ($R_i \leftrightarrow R_j$, $i\neq j$): $\det\mapsto -\det A$.
  • Row scaling ($R_i \mapsto cR_i$): $\det\mapsto c\cdot\det A$.
  • Row addition ($R_i \mapsto R_i+\lambda R_j$, $j\neq i$): $\det$ is unchanged.

Therefore: $\det(\lambda A)=\lambda^n\det A$ for an $n\times n$ matrix $A$ (scaling every row by $\lambda$ multiplies $\det$ by $\lambda$ exactly $n$ times).

Example — Computing a $3\times 3$ Determinant via Row Reduction

Let $A=\begin{pmatrix}2&1&3\\0&-1&2\\4&0&5\end{pmatrix}$. We row-reduce to upper triangular form, tracking the effect on $\det$:

$R_3\leftarrow R_3-2R_1$: no sign change.

$$\begin{pmatrix}2&1&3\\0&-1&2\\0&-2&-1\end{pmatrix}$$

$R_3\leftarrow R_3-2R_2$: no sign change.

$$\begin{pmatrix}2&1&3\\0&-1&2\\0&0&-5\end{pmatrix}$$

The determinant of an upper triangular matrix is the product of its diagonal entries:

$$\det A = 2\cdot(-1)\cdot(-5) = 10.$$
Theorem — Cramer's Rule

Let $A$ be an invertible $n\times n$ matrix and $\mathbf{b}\in\mathbb{R}^n$. The unique solution $x$ to $Ax=\mathbf{b}$ has components:

$$x_i = \frac{\det(A_i)}{\det A},\qquad i=1,\ldots,n,$$

where $A_i$ is the matrix obtained from $A$ by replacing its $i$-th column with $\mathbf{b}$. Cramer's rule is theoretically important but computationally expensive for large $n$; Gaussian elimination is preferred in practice.

Example — Cramer's Rule for $2\times2$

Solve $\begin{pmatrix}3&1\\2&5\end{pmatrix}\begin{pmatrix}x\\y\end{pmatrix}=\begin{pmatrix}7\\4\end{pmatrix}$.

$\det A=15-2=13$. Then: $$x_1=\frac{\det\begin{pmatrix}7&1\\4&5\end{pmatrix}}{13}=\frac{35-4}{13}=\frac{31}{13},\qquad x_2=\frac{\det\begin{pmatrix}3&7\\2&4\end{pmatrix}}{13}=\frac{12-14}{13}=\frac{-2}{13}.$$
