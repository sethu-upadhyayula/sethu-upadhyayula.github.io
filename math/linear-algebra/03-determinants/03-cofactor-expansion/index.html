3.3 Cofactor Expansion

Definition - Minor and Cofactor

Let $A$ be an $n\times n$ matrix. The $(i,j)$-minor $M_{ij}$ is the determinant of the $(n-1)\times(n-1)$ submatrix obtained by deleting row $i$ and column $j$ from $A$.

The $(i,j)$-cofactor is: $$C_{ij} = (-1)^{i+j}M_{ij}.$$ The sign pattern $(-1)^{i+j}$ is the checkerboard pattern $\begin{pmatrix}+&-&+&\cdots\\-&+&-&\cdots\\\vdots&&\ddots\end{pmatrix}$.

Theorem - Cofactor Expansion

For any $i\in\{1,\ldots,n\}$ (expansion along row $i$):

$$\det A = \sum_{j=1}^{n} a_{ij}C_{ij} = \sum_{j=1}^{n}(-1)^{i+j}a_{ij}M_{ij}.$$

Equivalently, for any $j\in\{1,\ldots,n\}$ (expansion along column $j$):

$$\det A = \sum_{i=1}^{n} a_{ij}C_{ij}.$$

All $2n$ such expansions (one per row or column) yield the same value. The choice of row or column is strategic: pick the one with the most zeros to minimise computation.

Definition - Adjugate (Classical Adjoint)

The adjugate (or classical adjoint) of $A$ is the transpose of the cofactor matrix:

$$\operatorname{adj}(A) = [C_{ji}]_{i,j} = \begin{pmatrix}C_{11}&C_{21}&\cdots&C_{n1}\\C_{12}&C_{22}&\cdots&C_{n2}\\\vdots&&\ddots&\vdots\\C_{1n}&C_{2n}&\cdots&C_{nn}\end{pmatrix}.$$

Note the transposition: the $(i,j)$ entry of $\operatorname{adj}(A)$ is $C_{ji}$, not $C_{ij}$.

Theorem - Adjugate Formula for the Inverse

For any $n\times n$ matrix $A$:

$$A\cdot\operatorname{adj}(A) = \operatorname{adj}(A)\cdot A = \det(A)\cdot I_n.$$

Therefore, if $\det A\neq 0$, then $A$ is invertible and: $$A^{-1} = \frac{1}{\det A}\operatorname{adj}(A).$$

Example - Cofactor Expansion of a $3\times3$ Matrix

Let $A=\begin{pmatrix}1&2&3\\0&4&5\\1&0&6\end{pmatrix}$. Expand along row 1:

$C_{11}=(-1)^2\det\begin{pmatrix}4&5\\0&6\end{pmatrix}=24$.

$C_{12}=(-1)^3\det\begin{pmatrix}0&5\\1&6\end{pmatrix}=-(0-5)=5$.

$C_{13}=(-1)^4\det\begin{pmatrix}0&4\\1&0\end{pmatrix}=(0-4)=-4$.

$$\det A = 1\cdot24 + 2\cdot5 + 3\cdot(-4) = 24+10-12 = 22.$$

Verification via row reduction: $R_3\leftarrow R_3-R_1$: $\begin{pmatrix}1&2&3\\0&4&5\\0&-2&3\end{pmatrix}$. Then $R_3\leftarrow R_3+\frac{1}{2}R_2$: $\begin{pmatrix}1&2&3\\0&4&5\\0&0&\frac{11}{2}\end{pmatrix}$. Product of diagonals: $1\cdot4\cdot\frac{11}{2}=22$. $\checkmark$

Computing $A^{-1}$: The full cofactor matrix is: $$[C_{ij}]=\begin{pmatrix}24&5&-4\\-12&3&2\\-2&-5&4\end{pmatrix}$$ so $\operatorname{adj}(A)=[C_{ij}]^T=\begin{pmatrix}24&-12&-2\\5&3&-5\\-4&2&4\end{pmatrix}$ and $A^{-1}=\frac{1}{22}\operatorname{adj}(A)$.
