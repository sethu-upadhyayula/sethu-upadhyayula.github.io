2.3 Matrix Representations

Definition — Matrix of a Linear Transformation

Let $T:V\to W$ be linear, $\mathcal{B}=(b_1,\ldots,b_n)$ a basis for $V$, and $\mathcal{C}=(c_1,\ldots,c_m)$ a basis for $W$. The matrix of $T$ with respect to $\mathcal{B}$ and $\mathcal{C}$ is the $m\times n$ matrix $$[T]_{\mathcal{B}}^{\mathcal{C}}$$ whose $j$-th column is $[T(b_j)]_{\mathcal{C}}$, the coordinate vector of $T(b_j)$ in basis $\mathcal{C}$.

The fundamental property is: for any $v\in V$, $$[T(v)]_{\mathcal{C}} = [T]_{\mathcal{B}}^{\mathcal{C}}\,[v]_{\mathcal{B}}.$$ That is, applying $T$ corresponds to matrix-vector multiplication in coordinates.

Example — Differentiation Matrix

Take $D:\mathcal{P}_2\to\mathcal{P}_1$ with $\mathcal{B}=(1,t,t^2)$ and $\mathcal{C}=(1,t)$. Then: $$D(1)=0=0\cdot1+0\cdot t, \quad D(t)=1=1\cdot1+0\cdot t, \quad D(t^2)=2t=0\cdot1+2\cdot t.$$ So $[D]_{\mathcal{B}}^{\mathcal{C}} = \begin{pmatrix}0&1&0\\0&0&2\end{pmatrix}$. Check: $[D(3+5t-2t^2)]_{\mathcal{C}} = \begin{pmatrix}0&1&0\\0&0&2\end{pmatrix}\begin{pmatrix}3\\5\\-2\end{pmatrix}=\begin{pmatrix}5\\-4\end{pmatrix}$, matching $D(3+5t-2t^2)=5-4t$.

Theorem — Composition = Matrix Multiplication

If $T:U\to V$ and $S:V\to W$ are linear with bases $\mathcal{A},\mathcal{B},\mathcal{C}$ for $U,V,W$ respectively, then: $$[S\circ T]_{\mathcal{A}}^{\mathcal{C}} = [S]_{\mathcal{B}}^{\mathcal{C}}\,[T]_{\mathcal{A}}^{\mathcal{B}}.$$ This is why matrix multiplication is defined the way it is: it encodes function composition.

Remark — Change of Representation

The matrix $[T]_{\mathcal{B}}^{\mathcal{C}}$ depends on the choice of bases. If we change from $\mathcal{B}$ to $\mathcal{B}'$ and from $\mathcal{C}$ to $\mathcal{C}'$, the matrix changes by: $$[T]_{\mathcal{B}'}^{\mathcal{C}'} = [I_W]_{\mathcal{C}}^{\mathcal{C}'}[T]_{\mathcal{B}}^{\mathcal{C}}[I_V]_{\mathcal{B}'}^{\mathcal{B}}.$$ For the special case $V=W$ and $\mathcal{B}=\mathcal{C}$, this becomes $[T]_{\mathcal{B}'} = P^{-1}[T]_{\mathcal{B}}P$ (similar matrices), the topic of §2.4.
