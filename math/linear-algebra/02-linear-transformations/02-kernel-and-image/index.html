2.2 Kernel and Image

Definition - Kernel and Image

Let $T:V\to W$ be a linear transformation. The kernel (or null space) of $T$ is $$\ker T = \{v\in V : T(v)=\mathbf{0}_W\},$$ and the image (or range) of $T$ is $$\operatorname{im} T = T(V) = \{T(v) : v\in V\}.$$

Theorem - Kernel and Image are Subspaces

$\ker T$ is a subspace of $V$ and $\operatorname{im} T$ is a subspace of $W$.

Proof

Kernel: $T(\mathbf{0})=\mathbf{0}$ so $\mathbf{0}\in\ker T$. If $T(u)=T(v)=\mathbf{0}$ then $T(u+v)=\mathbf{0}+\mathbf{0}=\mathbf{0}$ and $T(cv)=cT(v)=\mathbf{0}$. The three subspace conditions hold.

Image: $T(\mathbf{0})=\mathbf{0}\in\operatorname{im}T$. If $w_1=T(v_1),w_2=T(v_2)\in\operatorname{im}T$ then $w_1+w_2=T(v_1+v_2)\in\operatorname{im}T$ and $cw_1=T(cv_1)\in\operatorname{im}T$.

$\square$

Theorem - Injectivity and the Kernel

$T$ is injective (one-to-one) if and only if $\ker T=\{\mathbf{0}\}$.

Proof

($\Rightarrow$) If $T$ is injective and $T(v)=\mathbf{0}=T(\mathbf{0})$, then $v=\mathbf{0}$.

($\Leftarrow$) Suppose $\ker T=\{\mathbf{0}\}$ and $T(u)=T(v)$. Then $T(u-v)=\mathbf{0}$, so $u-v\in\ker T=\{\mathbf{0}\}$, giving $u=v$.

$\square$

Remark - Surjectivity

$T$ is surjective (onto) if and only if $\operatorname{im} T=W$, i.e., $\operatorname{rank}(T)=\dim W$.

Theorem - Rank-Nullity

For any linear $T:V\to W$ with $\dim V < \infty$: $$\operatorname{nullity}(T) + \operatorname{rank}(T) = \dim V,$$ where $\operatorname{nullity}(T)=\dim(\ker T)$ and $\operatorname{rank}(T)=\dim(\operatorname{im}T)$.

Example - Matrix Kernel and Image

For $T_A:\mathbb{R}^n\to\mathbb{R}^m$ defined by $T_A(x)=Ax$: $\ker T_A=\{x: Ax=0\}$ is the null space of $A$, and $\operatorname{im} T_A = \operatorname{col}(A)$ is the column space of $A$. Rank-nullity says $\operatorname{nullity}(A)+\operatorname{rank}(A)=n$, a fundamental result in linear systems.
