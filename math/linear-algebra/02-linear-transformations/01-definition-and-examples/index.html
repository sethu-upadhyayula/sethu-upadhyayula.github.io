2.1 Definition and Examples

Definition - Linear Transformation

Let $V$ and $W$ be vector spaces over $\mathbb{R}$. A function $T:V\to W$ is a linear transformation (or linear map) if for all $u,v\in V$ and $c\in\mathbb{R}$:

  1. Additivity: $T(u+v) = T(u)+T(v)$.
  2. Homogeneity: $T(cv) = cT(v)$.

Equivalently (combining both conditions): $T(cu+dv)=cT(u)+dT(v)$ for all $u,v\in V$ and $c,d\in\mathbb{R}$.

Theorem - $T$ Maps Zero to Zero

If $T:V\to W$ is linear, then $T(\mathbf{0}_V)=\mathbf{0}_W$.

Proof

$T(\mathbf{0}_V)=T(0\cdot\mathbf{0}_V)=0\cdot T(\mathbf{0}_V)=\mathbf{0}_W$ by homogeneity.

$\square$

Example - Zero Map and Identity

The zero map $0:V\to W$ defined by $0(v)=\mathbf{0}_W$ is linear. The identity $I:V\to V$, $I(v)=v$, is linear. Both are trivial but important reference examples.

Example - Differentiation $D:\mathcal{P}_n\to\mathcal{P}_{n-1}$

Define $D(p)=p'$ (the derivative). Then: $$D(p+q)=(p+q)'=p'+q'=D(p)+D(q), \quad D(cp)=(cp)'=cp'=cD(p).$$ So $D$ is linear. Its kernel is the set of constant polynomials (degree 0), and its image is all of $\mathcal{P}_{n-1}$ (see §2.2).

Example - Integration $I:\mathcal{P}_n\to\mathcal{P}_{n+1}$

Define $I(p)(t)=\int_0^t p(s)\,ds$. Linearity follows from linearity of integration. This map is injective (its kernel is $\{0\}$) but not surjective (no polynomial in $\mathcal{P}_{n+1}$ maps onto a non-zero constant).

Example - Rotation in $\mathbb{R}^2$

Counterclockwise rotation by angle $\theta$: $$R_\theta\begin{pmatrix}x\\y\end{pmatrix} = \begin{pmatrix}\cos\theta & -\sin\theta\\\sin\theta & \cos\theta\end{pmatrix}\begin{pmatrix}x\\y\end{pmatrix}.$$ Any matrix-vector multiplication $x\mapsto Ax$ defines a linear map $T_A:\mathbb{R}^n\to\mathbb{R}^m$, and conversely every linear map $\mathbb{R}^n\to\mathbb{R}^m$ is of this form (the matrix $A$ has columns $T(e_1),\ldots,T(e_n)$).

Example - Orthogonal Projection

Let $\hat{u}$ be a unit vector in $\mathbb{R}^n$. The map $P(v)=(v\cdot\hat{u})\hat{u}$ (projection onto the line spanned by $\hat{u}$) is linear. More generally, orthogonal projection onto any subspace is linear (see §5.3).

Remark - Linearity Determined by Values on a Basis

A linear transformation $T:V\to W$ is completely determined by its values $T(b_1),\ldots,T(b_n)$ on a basis $\mathcal{B}=(b_1,\ldots,b_n)$ of $V$. Moreover, for any vectors $w_1,\ldots,w_n\in W$, there exists a unique linear $T:V\to W$ with $T(b_i)=w_i$. This is the key to the matrix representation in §2.3.
