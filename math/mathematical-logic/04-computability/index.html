Chapter 4: Computability

Computability theory asks the fundamental question: which functions and problems can be solved by an algorithm? Alan Turing answered this in 1936 by introducing the Turing machine - an idealised abstract model of computation that has since been shown equivalent to every other reasonable model of computation.

The surprising and important result is that there exist well-defined mathematical problems for which no algorithm can exist. The Halting Problem is the most famous example. These limits on computation have deep connections to Gödel's Incompleteness Theorems.

Chapter Overview

4.1 - Turing Machines

A Turing machine has a finite set of states, an infinite tape, and a transition function governing head movement and symbol writing. The Church-Turing thesis asserts that this model captures all effective computation. We also discuss the Universal Turing Machine $U$, which can simulate any TM given its encoding.

4.2 - Decidability

A language is decidable if a TM always halts and correctly classifies every input. The Halting Problem is undecidable - proven via Cantor's diagonalisation argument. Rice's Theorem generalises this: no non-trivial property of TM-recognisable languages is decidable. We also introduce many-one reductions ($\leq_m$) for comparing problem difficulty.
