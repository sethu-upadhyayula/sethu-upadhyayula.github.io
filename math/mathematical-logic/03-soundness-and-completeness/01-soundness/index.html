3.1 Soundness

Theorem — Soundness

For any set of formulas $\Gamma$ and formula $\phi$: $$\text{If } \Gamma \vdash \phi \text{ then } \Gamma \models \phi.$$ In words: every provable sentence is a logical consequence of its hypotheses.

Proof Sketch — Induction on Proof Length

Let $\phi_1, \phi_2, \ldots, \phi_n = \phi$ be a proof of $\phi$ from $\Gamma$. We show by induction on $i$ that $\Gamma \models \phi_i$ for each $i$.

Base cases:

  • If $\phi_i \in \Gamma$: every model of $\Gamma$ satisfies $\phi_i$ by definition. $\checkmark$
  • If $\phi_i$ is an axiom schema instance: all axiom schemas are logically valid (tautologies), verified by semantic analysis of each schema. $\checkmark$

Inductive step (Modus Ponens): Suppose $\phi_i$ follows from earlier $\phi_j$ and $\phi_k = \phi_j \Rightarrow \phi_i$ by MP. By induction hypothesis, $\Gamma \models \phi_j$ and $\Gamma \models \phi_j \Rightarrow \phi_i$. Let $\mathcal{M} \models \Gamma$. Then $\mathcal{M} \models \phi_j$ and $\mathcal{M} \models \phi_j \Rightarrow \phi_i$. By the semantics of $\Rightarrow$, $\mathcal{M} \models \phi_i$. $\checkmark$

Since the last line $\phi_n = \phi$ satisfies $\Gamma \models \phi_n$, we have $\Gamma \models \phi$. $\square$

Consequences

Corollary — No False Theorems

If $\Gamma$ is a consistent set of true sentences (i.e., $\Gamma$ has a model), then no formal proof from $\Gamma$ can derive a false sentence. In particular, a sound proof system never proves $\bot$ from consistent premises.

Corollary — Theorems are Valid

If $\vdash \phi$ (i.e., $\phi$ is provable from the empty set of hypotheses), then $\models \phi$ ($\phi$ is logically valid). Every theorem of the proof system is a tautology of first-order logic.

Remark — Soundness is Easy, Completeness is Hard

Soundness is a relatively straightforward property to verify for any well-designed proof system: one need only check that the axioms are valid and that each inference rule preserves validity. Completeness — showing the converse — is a much deeper result requiring the construction of a model.
