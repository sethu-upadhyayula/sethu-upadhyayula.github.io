3.2 Completeness

Theorem - Gödel's Completeness Theorem (1930)

For any set of sentences $\Gamma$ and sentence $\phi$: $$\text{If } \Gamma \models \phi \text{ then } \Gamma \vdash \phi.$$ Equivalently: if $\Gamma$ is consistent, then $\Gamma$ has a model.

Henkin's Proof

Proof Sketch - Henkin's Method

It suffices to show: every consistent theory $\Gamma$ has a model. We construct one in three steps.

Step 1 - Extend to a maximal consistent set. Enumerate all sentences $\phi_0, \phi_1, \phi_2, \ldots$ of the language. Define $\Gamma_0 = \Gamma$ and inductively set $\Gamma_{i+1} = \Gamma_i \cup \{\phi_i\}$ if this is consistent, otherwise $\Gamma_{i+1} = \Gamma_i$. Let $\Gamma^* = \bigcup_i \Gamma_i$. Then $\Gamma^*$ is a maximal consistent set: consistent, and for every sentence $\phi$, either $\phi \in \Gamma^*$ or $\neg\phi \in \Gamma^*$.

Step 2 - Add Henkin witnesses. For every existential sentence $\exists x\,\phi(x) \in \Gamma^*$, introduce a fresh constant $c_{\phi}$ (a Henkin constant) and add $\phi(c_{\phi})$ to the language and theory. After repeating $\omega$ times, obtain a Henkin theory $\Gamma^+$ with witnesses for all existential statements.

Step 3 - Canonical model. Define the canonical structure $\mathcal{M}^*$: its domain is the set of all closed terms (constants) of the extended language, with equality interpreted as provable identity. Then verify by induction on formula complexity that $$\mathcal{M}^* \models \phi \iff \phi \in \Gamma^+.$$ In particular $\mathcal{M}^* \models \Gamma^*$, so $\mathcal{M}^* \models \Gamma$. $\square$

Note: The existence of the maximal consistent extension uses Zorn's Lemma (or equivalently, the Axiom of Choice) for uncountable languages.

Important Corollaries

Corollary - Compactness Theorem

A set of sentences $\Gamma$ is satisfiable if and only if every finite subset of $\Gamma$ is satisfiable. This is immediate from completeness: $\Gamma \models \bot$ iff $\Gamma \vdash \bot$, and any proof uses only finitely many hypotheses.

Corollary - Löwenheim-Skolem Theorem

Any satisfiable first-order theory has a countable model. (The Henkin construction always produces a model whose domain is a countable set of constant terms.)

Remark - Completeness vs Incompleteness

Gödel's Completeness Theorem (Chapter 3) and his Incompleteness Theorems (Chapter 5) are complementary, not contradictory. Completeness says: every logically valid sentence is provable. Incompleteness says: no consistent, sufficiently expressive formal system can prove all arithmetical truths. The difference is between logical validity (true in all models) and truth in a specific intended model (e.g., $\mathbb{N}$).
