2.1 Predicates and Quantifiers

Definition — Predicate

A predicate $P(x_1, \ldots, x_n)$ is an $n$-ary expression that becomes a proposition once each variable is assigned a value from a domain $D$. For example, $\text{Even}(x)$ is a predicate on $\mathbb{Z}$: $\text{Even}(4)$ is true, $\text{Even}(3)$ is false.

Quantifiers

Universal Quantifier $\forall$

$\forall x\, P(x)$ is true (in a structure with domain $D$) if and only if $P(d)$ is true for every $d \in D$.

Read: "for all $x$, $P(x)$" or "every $x$ has property $P$".

Existential Quantifier $\exists$

$\exists x\, P(x)$ is true if and only if $P(d)$ is true for at least one $d \in D$.

Read: "there exists an $x$ such that $P(x)$".

Negation of Quantifiers

Theorem — Quantifier Negation

$$\neg \forall x\, P(x) \equiv \exists x\, \neg P(x)$$

$$\neg \exists x\, P(x) \equiv \forall x\, \neg P(x)$$

Example

"Not every integer is even" $\equiv$ "There exists an integer that is not even."

"There is no largest prime" $\equiv$ "For all $n$, $n$ is not the largest prime."

Free and Bound Variables

Definition — Scope, Free, Bound

In $\forall x\, \phi$, the variable $x$ is bound by the quantifier; the formula $\phi$ is the scope. An occurrence of $x$ not within any quantifier binding it is free.

Example: In $\forall x\,(P(x) \wedge Q(x,y))$, the variable $x$ is bound and $y$ is free. The formula is not a sentence (it has a free variable $y$).

Nested Quantifiers

Remark — Order Matters

$\forall x\, \exists y\, (x < y)$ says "for every $x$ there is a larger $y$" — true in $\mathbb{N}$.

$\exists y\, \forall x\, (x < y)$ says "there is a $y$ larger than every $x$" — false in $\mathbb{N}$.

These two sentences are not logically equivalent. The order of mixed quantifiers is semantically significant.

Examples in Arithmetic and Set Theory
  • $\forall x\, \forall y\, (x + y = y + x)$: commutativity of addition (sentence in the language of arithmetic).
  • $\exists x\, (x \cdot x = 2)$: true in $\mathbb{R}$, false in $\mathbb{Q}$.
  • $\forall A\, \exists B\, \forall x\, (x \in B \Leftrightarrow x \notin A)$: every set has a complement (in set theory).
