2.2 Formal Language

Definition - Signature (Language)

A signature (or first-order language) $\mathcal{L}$ consists of:

  • Constant symbols: $c_1, c_2, \ldots$ (denoting fixed domain elements).
  • Function symbols: each $f$ has an arity $n \geq 1$; $f(t_1,\ldots,t_n)$ denotes an $n$-ary function.
  • Predicate (relation) symbols: each $P$ has an arity $n \geq 1$; $P(t_1,\ldots,t_n)$ is an atomic formula.

Additionally, equality $=$ is usually included as a binary predicate symbol.

Terms

Definition - Terms

The set of terms of $\mathcal{L}$ is defined inductively:

  1. Every variable $x, y, z, \ldots$ is a term.
  2. Every constant symbol $c$ is a term.
  3. If $f$ is an $n$-ary function symbol and $t_1, \ldots, t_n$ are terms, then $f(t_1, \ldots, t_n)$ is a term.

Terms denote objects in the domain. Formulas denote truth values.

Well-Formed Formulas

Definition - Atomic Formulas and WFFs

Atomic formulas: $P(t_1,\ldots,t_n)$ for predicate $P$ of arity $n$, or $t_1 = t_2$ for terms $t_1, t_2$.

The set of well-formed formulas (wffs) is defined inductively:

  1. Every atomic formula is a wff.
  2. If $\phi$ is a wff, then $\neg\phi$ is a wff.
  3. If $\phi$ and $\psi$ are wffs, then $(\phi \wedge \psi)$, $(\phi \vee \psi)$, $(\phi \Rightarrow \psi)$, $(\phi \Leftrightarrow \psi)$ are wffs.
  4. If $\phi$ is a wff and $x$ is a variable, then $\forall x\,\phi$ and $\exists x\,\phi$ are wffs.

A sentence is a wff with no free variables.

Examples of Languages

Example - Language of Ordered Rings

Signature: constants $0, 1$; binary functions $+, \cdot$; binary predicate $\leq$.

Terms include: $0$, $1$, $x$, $x + 1$, $x \cdot x$, $(x + y) \cdot z$, etc.

Sentences include: $\forall x\, (x + 0 = x)$ (additive identity), $\forall x\,\exists y\,(x + y = 0)$ (additive inverse).

Example - Language of Graphs

Signature: one binary predicate $E(x,y)$ (meaning "there is an edge from $x$ to $y$").

The sentence $\forall x\, \forall y\,(E(x,y) \Rightarrow E(y,x))$ expresses that the graph is undirected. The sentence $\forall x\,\neg E(x,x)$ says there are no self-loops.
