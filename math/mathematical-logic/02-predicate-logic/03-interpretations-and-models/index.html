2.3 Interpretations and Models

Definition — Structure (Interpretation)

A structure (or interpretation, or $\mathcal{L}$-structure) $\mathcal{M}$ for a signature $\mathcal{L}$ consists of:

  • A nonempty set $M$ called the domain (or universe).
  • For each constant symbol $c$: an element $c^{\mathcal{M}} \in M$.
  • For each $n$-ary function symbol $f$: a function $f^{\mathcal{M}} : M^n \to M$.
  • For each $n$-ary predicate symbol $P$: a relation $P^{\mathcal{M}} \subseteq M^n$.

We write $\mathcal{M} = (M, \ldots)$ specifying the interpretations of all non-logical symbols.

Satisfaction

Definition — Satisfaction Relation

A variable assignment $s: \text{Var} \to M$ maps each variable to a domain element. The satisfaction relation $\mathcal{M} \models \phi[s]$ ("$\mathcal{M}$ satisfies $\phi$ under $s$") is defined inductively:

  • $\mathcal{M} \models P(t_1,\ldots,t_n)[s]$ iff $(t_1^s, \ldots, t_n^s) \in P^{\mathcal{M}}$, where $t^s$ denotes the value of term $t$ under $s$.
  • $\mathcal{M} \models \neg\phi[s]$ iff $\mathcal{M} \not\models \phi[s]$.
  • $\mathcal{M} \models (\phi \wedge \psi)[s]$ iff $\mathcal{M} \models \phi[s]$ and $\mathcal{M} \models \psi[s]$.
  • $\mathcal{M} \models \forall x\,\phi[s]$ iff for all $d \in M$, $\mathcal{M} \models \phi[s(x \mapsto d)]$.
  • $\mathcal{M} \models \exists x\,\phi[s]$ iff there exists $d \in M$ with $\mathcal{M} \models \phi[s(x \mapsto d)]$.

For a sentence $\phi$ (no free variables), the satisfaction relation is independent of the assignment: we simply write $\mathcal{M} \models \phi$.

Models of Theories

Definition — Theory and Model

A theory $T$ is a set of sentences. A structure $\mathcal{M}$ is a model of $T$, written $\mathcal{M} \models T$, if $\mathcal{M} \models \phi$ for every $\phi \in T$.

$T$ is satisfiable if it has at least one model; unsatisfiable (or inconsistent) otherwise.

Definition — Logical Consequence and Validity

$\phi$ is a logical consequence of $T$, written $T \models \phi$, if every model of $T$ also satisfies $\phi$.

$\phi$ is logically valid, written $\models \phi$, if $\mathcal{M} \models \phi$ for every structure $\mathcal{M}$ (i.e., $\phi$ is true in all interpretations). Valid first-order sentences are called tautologies of FOL.

Example

In the language with one binary predicate $<$, the sentence $\forall x\, \neg(x < x)$ (irreflexivity) holds in $(\mathbb{N}, <)$ and $(\mathbb{R}, <)$ but fails in $(\mathbb{N}, \leq)$.

The sentence $\forall x\, \forall y\, \forall z\,((x < y \wedge y < z) \Rightarrow x < z)$ (transitivity) holds in all these structures.
