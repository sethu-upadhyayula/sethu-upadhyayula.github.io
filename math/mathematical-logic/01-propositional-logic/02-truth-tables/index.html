1.2 Truth Tables

Definition - Truth Table

A truth table for a formula with $n$ distinct propositional variables lists all $2^n$ possible assignments of truth values and computes the formula's value for each. Columns are built left to right, from smallest sub-formulas to the whole formula.

Tables for the Basic Connectives

Truth Tables - Two Variables

For $P$ and $Q$ (four rows each):

$P$$Q$$\neg P$$P\wedge Q$$P\vee Q$$P\Rightarrow Q$$P\Leftrightarrow Q$
TTFTTTT
TFFFTFF
FTTFTTF
FFTFFTT

Note: $P \Rightarrow Q$ is false only in row 2 (true hypothesis, false conclusion).

De Morgan's Laws

Theorem - De Morgan's Laws

For any propositions $P$ and $Q$:

$$\neg(P \wedge Q) \equiv \neg P \vee \neg Q$$

$$\neg(P \vee Q) \equiv \neg P \wedge \neg Q$$

Proof - De Morgan via Truth Table
$P$$Q$$\neg P$$\neg Q$$P\wedge Q$$\neg(P\wedge Q)$$\neg P\vee\neg Q$
TTFFTFF
TFFTFTT
FTTFFTT
FFTTFTT

Columns 6 and 7 are identical, confirming $\neg(P\wedge Q)\equiv\neg P\vee\neg Q$. The second law is verified analogously.

Logical Equivalence

Definition - Logical Equivalence

Two formulas $A$ and $B$ are logically equivalent, written $A \equiv B$, if they have identical truth-value columns in their combined truth table (i.e., they agree on every possible assignment of truth values to their variables).

Example - $P \Rightarrow Q \equiv \neg P \vee Q$
$P$$Q$$\neg P$$P\Rightarrow Q$$\neg P\vee Q$
TTFTT
TFFFF
FTTTT
FFTTT

The last two columns are equal, so $P\Rightarrow Q\equiv\neg P\vee Q$.
