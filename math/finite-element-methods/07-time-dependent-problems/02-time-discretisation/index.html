7.2 Time Discretisation

To complete the numerical scheme, we discretise the semi-discrete ODE system $M\dot{\mathbf{u}} + K\mathbf{u} = \mathbf{f}(t)$ in time. A uniform partition $0 = t_0 < t_1 < \cdots < t_n = T$ with step $\Delta t = T/n$ is used. We denote by $\mathbf{u}^m$ the approximation to $\mathbf{u}(t_m)$.

The $\theta$-Scheme

The $\theta$-scheme is a one-parameter family of one-step methods that unifies the three most commonly used time-steppers. Replace the time derivative by a backward difference and evaluate the spatial operator at a weighted average of the current and next time levels:

Definition — $\theta$-Scheme

Given $\mathbf{u}^n$, find $\mathbf{u}^{n+1}$ such that $$M\,\frac{\mathbf{u}^{n+1} - \mathbf{u}^n}{\Delta t} + K\bigl[\theta\,\mathbf{u}^{n+1} + (1-\theta)\,\mathbf{u}^n\bigr] = \theta\,\mathbf{f}^{n+1} + (1-\theta)\,\mathbf{f}^n,$$ where $\mathbf{f}^m = \mathbf{f}(t_m)$ and $\theta \in [0,1]$. Rearranging, this is the linear system $$(M + \theta\Delta t\, K)\,\mathbf{u}^{n+1} = (M - (1-\theta)\Delta t\, K)\,\mathbf{u}^n + \Delta t\,(\theta\,\mathbf{f}^{n+1} + (1-\theta)\,\mathbf{f}^n).$$

Special Cases

Special Cases of the $\theta$-Scheme
$\theta$ Scheme System to solve Stability Temporal order
$\theta = 0$ Explicit Euler $M\mathbf{u}^{n+1} = (M - \Delta t\,K)\mathbf{u}^n + \Delta t\,\mathbf{f}^n$ Conditionally stable: $\Delta t \leq Ch^2$ $O(\Delta t)$
$\theta = 1$ Implicit Euler $(M + \Delta t\,K)\mathbf{u}^{n+1} = M\mathbf{u}^n + \Delta t\,\mathbf{f}^{n+1}$ Unconditionally stable $O(\Delta t)$
$\theta = \tfrac{1}{2}$ Crank–Nicolson $(M + \tfrac{\Delta t}{2}K)\mathbf{u}^{n+1} = (M - \tfrac{\Delta t}{2}K)\mathbf{u}^n + \tfrac{\Delta t}{2}(\mathbf{f}^{n+1}+\mathbf{f}^n)$ Unconditionally stable $O(\Delta t^2)$

Stability Analysis via Eigenvalues

Consider the homogeneous case $\mathbf{f} = 0$. Since $M$ and $K$ are simultaneously diagonalisable (via the generalised eigenproblem $K\mathbf{v} = \lambda M\mathbf{v}$, which has positive eigenvalues $0 < \lambda_1 \leq \lambda_2 \leq \cdots$), we can analyse stability mode by mode. For eigenvalue $\lambda_j$, the $\theta$-scheme reduces to the scalar recurrence $$u^{n+1} = \frac{1 - (1-\theta)\Delta t\,\lambda_j}{1 + \theta\Delta t\,\lambda_j}\,u^n =: g_j\, u^n.$$ The amplification factor is $g_j = \frac{1-(1-\theta)\mu}{1+\theta\mu}$ where $\mu = \Delta t\,\lambda_j > 0$. For stability we need $|g_j| \leq 1$.

Theorem — Stability of the $\theta$-Scheme

  • For $\theta \geq 1/2$: $|g_j| < 1$ for all $\mu > 0$, so the scheme is unconditionally stable.
  • For $0 \leq \theta < 1/2$: $|g_j| \leq 1$ requires $\mu \leq \frac{2}{1-2\theta}$, i.e., $\Delta t\,\lambda_{\max} \leq \frac{2}{1-2\theta}$. Since the largest eigenvalue satisfies $\lambda_{\max} \sim Ch^{-2}$ for FEM on quasi-uniform meshes, this gives the CFL condition $\Delta t \leq C'h^2$.

Fully Discrete Error Estimates

Theorem — Fully Discrete Error

Let $u(x,t_n)$ be the exact solution and $u_h^n$ the fully discrete approximation from the $\theta$-scheme with $P_1$ FEM. Under sufficient regularity of $u$:

  • Implicit Euler ($\theta=1$): $\|u(t_n) - u_h^n\|_{L^2} \leq C\bigl(h^2 + \Delta t\bigr)$.
  • Crank–Nicolson ($\theta=\tfrac{1}{2}$): $\|u(t_n) - u_h^n\|_{L^2} \leq C\bigl(h^2 + \Delta t^2\bigr)$.
To balance the spatial and temporal errors for $P_1$ elements, choose $\Delta t \sim h^2$ for implicit Euler (first-order accuracy in $\Delta t$ matches $h^2$ only if $\Delta t \sim h^2$), and $\Delta t \sim h$ for Crank–Nicolson (second-order in time matches $h^2$ if $\Delta t \sim h$).

Higher-Order Time Integration: Runge–Kutta Methods

The semi-discrete system $M\dot{\mathbf{u}} = -K\mathbf{u} + \mathbf{f}(t)$ can be integrated with any Runge–Kutta (RK) method. The combination of FEM in space with a $p$-stage RK method in time is called a method of lines discretisation. The accuracy in time is $O(\Delta t^p)$ for a $p$th-order RK scheme.

Remark — Stiffness and Implicit RK

For fine meshes, the eigenvalues $\lambda_j$ of $M^{-1}K$ span a wide range up to $O(h^{-2})$, making the semi-discrete system stiff. Explicit RK methods require $\Delta t = O(h^2)$ for stability (same as explicit Euler), which is prohibitively small. Implicit or IMEX (implicit-explicit) RK methods are preferred: they allow larger time steps while treating the stiff diffusion term implicitly and any non-stiff terms (e.g., nonlinear reactions) explicitly.
