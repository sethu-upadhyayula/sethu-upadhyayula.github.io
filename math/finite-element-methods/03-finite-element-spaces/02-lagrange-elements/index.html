3.2 Lagrange Elements

Lagrange elements are the most widely used finite elements. The key idea is the nodal basis: basis functions are uniquely determined by the requirement that each function equals 1 at one node and 0 at all other nodes - a natural generalisation of Lagrange interpolation to two and three dimensions.

$P_1$ Linear Elements

Definition - $P_1$ Basis Functions (Hat Functions)

On a triangulation $\mathcal{T}_h$, number the vertices (nodes) globally as $x_1, \ldots, x_N$. The $P_1$ nodal basis function $\varphi_i$ associated with node $x_i$ is the unique continuous, piecewise linear function satisfying $$\varphi_i(x_j) = \delta_{ij} = \begin{cases} 1 & i = j, \\ 0 & i \neq j. \end{cases}$$ On each triangle $K$, $\varphi_i|_K$ is the unique linear polynomial with these values at the three vertices of $K$. These are sometimes called tent functions or hat functions because of their shape in 1D.

Proposition - Support and Global Continuity

The support of $\varphi_i$ is the union of all triangles sharing node $x_i$: $$\operatorname{supp}(\varphi_i) = \bigcup_{\substack{K \in \mathcal{T}_h \\ x_i \in K}} K.$$ The compactness of this support is essential for the sparsity of the stiffness matrix: $K_{ij} = a(\varphi_j, \varphi_i) \neq 0$ only when $\operatorname{supp}(\varphi_i) \cap \operatorname{supp}(\varphi_j) \neq \emptyset$, which happens only when $x_i$ and $x_j$ are connected by an edge in $\mathcal{T}_h$.

Barycentric Coordinates

Definition - Barycentric Coordinates

For a triangle $K$ with vertices $a_1, a_2, a_3$, the barycentric coordinates $\lambda_1, \lambda_2, \lambda_3$ of a point $x \in K$ are the unique non-negative reals satisfying $$x = \lambda_1 a_1 + \lambda_2 a_2 + \lambda_3 a_3, \qquad \lambda_1 + \lambda_2 + \lambda_3 = 1.$$ The local $P_1$ basis functions on $K$ are simply $\hat\varphi_i = \lambda_i$. These are linear, satisfy the Kronecker property, and have unit sum, i.e.\ $\sum_i \lambda_i = 1$ (partition of unity).

$P_2$ Quadratic Elements

Definition - $P_2$ Nodes and Basis

The $P_2$ space on a triangle $K$ has $\dim P_2 = 6$. The 6 nodes are:

  • the 3 vertex nodes $a_1, a_2, a_3$, and
  • the 3 edge midpoint nodes $m_{12} = \frac{a_1+a_2}{2}$, $m_{23} = \frac{a_2+a_3}{2}$, $m_{31} = \frac{a_3+a_1}{2}$.
In terms of barycentric coordinates, the 6 local basis functions are: $$\hat\varphi_1 = \lambda_1(2\lambda_1 - 1),\quad \hat\varphi_2 = \lambda_2(2\lambda_2 - 1),\quad \hat\varphi_3 = \lambda_3(2\lambda_3 - 1),$$ $$\hat\varphi_{12} = 4\lambda_1\lambda_2,\quad \hat\varphi_{23} = 4\lambda_2\lambda_3,\quad \hat\varphi_{31} = 4\lambda_3\lambda_1.$$ Each function equals 1 at its node and 0 at all others (Kronecker property).

Nodal Interpolation and Interpolation Error

Definition - Nodal Interpolant

Given $u \in C^0(\bar\Omega)$ (or $u \in H^2(\Omega) \hookrightarrow C^0(\bar\Omega)$ by Sobolev embedding in 2D), the nodal interpolant $I_h u \in V_h$ is defined by $$I_h u = \sum_{i=1}^N u(x_i)\,\varphi_i,$$ where the $x_i$ are the nodes of the space. It satisfies $(I_h u)(x_i) = u(x_i)$ at every node.

Theorem - Interpolation Error Estimate

Let $K$ be a shape-regular triangle with diameter $h_K$. For $u \in H^{k+1}(K)$ and $0 \leq m \leq k+1$, the local $P_k$ nodal interpolant satisfies $$|u - I_h u|_{H^m(K)} \leq C\, h_K^{k+1-m}\, |u|_{H^{k+1}(K)},$$ where $C > 0$ depends on $k$, $m$, and the shape-regularity constant $\sigma$, but not on $h_K$ or $u$.

For $P_1$ elements ($k=1$), this gives: $\|u - I_h u\|_{L^2(K)} \leq Ch_K^2|u|_{H^2(K)}$ and $|u - I_h u|_{H^1(K)} \leq Ch_K|u|_{H^2(K)}$.

Remark - Global Interpolation Error

Summing element contributions over the mesh: $\|u - I_h u\|_{H^m(\Omega)}^2 = \sum_K |u - I_h u|_{H^m(K)}^2 \leq C^2 h^{2(k+1-m)} |u|_{H^{k+1}(\Omega)}^2$. This gives the global estimate $\|u - I_h u\|_{H^m(\Omega)} \leq Ch^{k+1-m}\|u\|_{H^{k+1}(\Omega)}$, which feeds directly into the a priori error analysis of Chapter 5.

Example - Global Assembly

To assemble the global stiffness matrix $K$ for $P_1$ elements: loop over each triangle $K$, compute the $3 \times 3$ element stiffness matrix $K^e$ using the local gradients $\nabla\varphi_i|_K$, then scatter entries to global positions using the element-to-global DOF map. The global $\varphi_i$ is the sum over all element contributions from triangles sharing node $i$. See Section 4.1 for the assembly algorithm in detail.
