Chapter 3: Finite Element Spaces

To apply the Galerkin method in practice we must choose a concrete finite-dimensional subspace $V_h \subset H^1_0(\Omega)$. The finite element method constructs $V_h$ by partitioning the domain $\Omega$ into simple geometric shapes — elements — and prescribing polynomial functions on each element that are glued together to form globally continuous (or piecewise smooth) functions.

This chapter develops the three pillars of finite element space construction: the triangulation of the domain and its regularity properties, the Lagrange nodal basis functions defined by interpolation conditions at element nodes, and the reference element machinery that standardises integration and differentiation across all physical elements via an affine (or isoparametric) mapping.
