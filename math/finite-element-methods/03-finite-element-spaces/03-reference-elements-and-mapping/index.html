3.3 Reference Elements and Mapping

Computing integrals over each physical triangle $K$ separately would require bespoke quadrature rules and gradient formulae for each element. The reference element approach standardises all computations: one computes everything on a single fixed reference triangle $\hat{K}$, then pulls back to the physical element via an affine mapping.

The Reference Triangle and Affine Map

Definition — Reference Triangle

The reference triangle is $$\hat{K} = \{(\hat{x}_1, \hat{x}_2) \in \mathbb{R}^2 : \hat{x}_1 \geq 0,\; \hat{x}_2 \geq 0,\; \hat{x}_1 + \hat{x}_2 \leq 1\},$$ with vertices $\hat{a}_1 = (0,0)$, $\hat{a}_2 = (1,0)$, $\hat{a}_3 = (0,1)$. The reference barycentric coordinates are $\hat\lambda_1 = 1 - \hat{x}_1 - \hat{x}_2$, $\hat\lambda_2 = \hat{x}_1$, $\hat\lambda_3 = \hat{x}_2$.

Definition — Affine Map

For a physical triangle $K$ with vertices $a_1, a_2, a_3 \in \mathbb{R}^2$, the affine element map $F_K: \hat{K} \to K$ is defined by $$x = F_K(\hat{x}) = B_K \hat{x} + b_K,$$ where $$B_K = [a_2 - a_1 \;|\; a_3 - a_1] \in \mathbb{R}^{2\times 2}, \qquad b_K = a_1.$$ $F_K$ is invertible (since $K$ is non-degenerate) with $|\det B_K| = 2|K|$, where $|K|$ denotes the area of $K$.

Change of Variables in Integrals

Theorem — Integral Transformation

For any integrable function $f: K \to \mathbb{R}$: $$\int_K f(x)\,dx = \int_{\hat{K}} f(F_K(\hat{x}))\,|\det B_K|\,d\hat{x} = 2|K|\int_{\hat{K}} \hat{f}(\hat{x})\,d\hat{x},$$ where $\hat{f} = f \circ F_K$. This reduces every element integral to a single reference domain integral, which can be evaluated by a fixed quadrature rule on $\hat{K}$.

Gradient Transformation

Proposition — Chain Rule for Gradients

Let $\varphi: K \to \mathbb{R}$ and $\hat\varphi = \varphi \circ F_K : \hat{K} \to \mathbb{R}$. By the chain rule: $$\nabla_{\hat{x}} \hat\varphi = B_K^\top \nabla_x \varphi \circ F_K.$$ Solving for the physical gradient: $$\nabla_x \varphi = B_K^{-\top} \nabla_{\hat{x}} \hat\varphi.$$ The element stiffness integral transforms as: $$\int_K \nabla\varphi_j \cdot \nabla\varphi_i\,dx = |\det B_K| \int_{\hat{K}} (B_K^{-\top}\nabla_{\hat{x}}\hat\varphi_j) \cdot (B_K^{-\top}\nabla_{\hat{x}}\hat\varphi_i)\,d\hat{x}.$$

Numerical Quadrature on the Reference Element

Definition — Gauss Quadrature on $\hat{K}$

A quadrature rule on $\hat{K}$ approximates $\int_{\hat{K}} \hat{f}\,d\hat{x} \approx \sum_{q=1}^{n_q} w_q \hat{f}(\hat{x}_q)$, where $\hat{x}_q$ are quadrature points and $w_q > 0$ are weights with $\sum_q w_q = |\hat{K}| = 1/2$. Common rules:

  • 1-point: centroid $\hat{x} = (1/3, 1/3)$, $w = 1/2$; exact for $P_1$.
  • 3-point: midpoints of edges; exact for $P_2$.
  • 4-point Dunavant: exact for $P_3$.
For the $P_1$ stiffness matrix (integrands in $P_0$), the 1-point rule suffices; for $P_2$ elements one typically uses a 3- or 4-point rule.

Isoparametric Elements

Definition — Isoparametric Mapping

For curved domains or higher-order accuracy, the isoparametric approach uses the same $P_k$ polynomial basis to parametrise both the element geometry and the solution. For $P_2$ isoparametric elements, the map $F_K: \hat{K} \to K$ is a degree-2 polynomial (using edge midpoints as geometric nodes), allowing accurate representation of curved boundaries. The Jacobian $B_K(\hat{x}) = \partial F_K / \partial\hat{x}$ is then position-dependent, and $|\det B_K(\hat{x})|$ must be evaluated at each quadrature point.

Remark — Affine vs. Isoparametric

For polygonal domains and $P_k$ elements on triangles, the affine map suffices and $B_K$ is constant on $K$, so the Jacobian factor $|\det B_K| = 2|K|$ can be precomputed. Isoparametric elements are needed when the domain boundary is curved (e.g.\ a disc) and one wishes to approximate the boundary geometry to order $h^{k+1}$ as well.

Example — $P_1$ Gradient Computation

For $P_1$ elements, the reference basis gradients are constant: $\nabla_{\hat{x}}\hat\lambda_1 = (-1,-1)^\top$, $\nabla_{\hat{x}}\hat\lambda_2 = (1,0)^\top$, $\nabla_{\hat{x}}\hat\lambda_3 = (0,1)^\top$. The physical gradients are $\nabla_x\lambda_i = B_K^{-\top}\nabla_{\hat{x}}\hat\lambda_i$, constant on $K$. The element stiffness entries are then $K^e_{ij} = |K|\,(\nabla_x\lambda_j)\cdot(\nabla_x\lambda_i)$, computable with a single matrix-vector product per element.
