Chapter 5: Error Analysis

Having constructed finite element spaces and assembled the discrete system, the central question becomes: how close is the FEM solution $u_h$ to the exact solution $u$? Error analysis provides rigorous, quantitative answers in terms of the mesh size $h$ and the regularity of the exact solution.

The fundamental tool is Céa's lemma, which reduces the problem of bounding the discretisation error $\|u - u_h\|$ to the purely geometric problem of finding the best approximation in the finite element space $V_h$. Combining this with interpolation error estimates — specifically the Bramble–Hilbert lemma — yields the celebrated a priori error bounds: $\|u - u_h\|_{H^1} \leq Ch\|u\|_{H^2}$ for $P_1$ elements. An additional duality argument (Aubin–Nitsche) then upgrades the $H^1$ estimate to an $L^2$ estimate with one extra power of $h$.
