4.1 Stiffness Matrix and Load Vector

The global stiffness matrix $K$ and load vector $\mathbf{f}$ are assembled by summing element contributions over all triangles in $\mathcal{T}_h$. This element-by-element strategy decouples the computation: each element is processed independently using the reference mapping, and results are scattered into global positions determined by the DOF numbering.

Element Stiffness Matrix

Definition - Element Stiffness Matrix

For triangle $K$ with local DOFs $\{\varphi_1^K, \varphi_2^K, \varphi_3^K\}$ (indexed locally $i = 1,2,3$), the element stiffness matrix $K^e \in \mathbb{R}^{3\times 3}$ has entries $$K^e_{ij} = \int_K \nabla\varphi_j^K \cdot \nabla\varphi_i^K\,dx, \quad i,j = 1,2,3.$$ For $P_1$ elements, $\nabla\varphi_i^K$ is constant on $K$, so: $$K^e_{ij} = |K|\,(\nabla\varphi_j^K)\cdot(\nabla\varphi_i^K).$$ Using the reference mapping $F_K$ with Jacobian $B_K$, and setting $\mathbf{g}_i = B_K^{-\top}\nabla_{\hat{x}}\hat\lambda_i$: $$K^e_{ij} = |K|\,\mathbf{g}_j \cdot \mathbf{g}_i = |K|\,(B_K^{-\top}\nabla_{\hat{x}}\hat\lambda_j)^\top (B_K^{-\top}\nabla_{\hat{x}}\hat\lambda_i).$$

Proposition - Explicit Formula for $P_1$ Stiffness

Let $K$ have vertices $a_1, a_2, a_3$ and let $B_K = [a_2-a_1 \;|\; a_3-a_1]$. Define $C_K = B_K^{-\top}$. Then $K^e = |K|\,G^\top G$, where $G = [C_K\nabla_{\hat{x}}\hat\lambda_1 \;|\; C_K\nabla_{\hat{x}}\hat\lambda_2 \;|\; C_K\nabla_{\hat{x}}\hat\lambda_3] \in \mathbb{R}^{2\times 3}$. The reference gradients are $\nabla_{\hat{x}}\hat\lambda_1 = (-1,-1)^\top$, $\nabla_{\hat{x}}\hat\lambda_2 = (1,0)^\top$, $\nabla_{\hat{x}}\hat\lambda_3 = (0,1)^\top$, so $K^e$ is computable with $O(1)$ flops per element.

Element Load Vector

Definition - Element Load Vector

The element load vector $\mathbf{f}^e \in \mathbb{R}^3$ has components $$f^e_i = \int_K f\,\varphi_i^K\,dx \approx |K|\sum_{q=1}^{n_q} w_q\, f(F_K(\hat{x}_q))\,\hat\lambda_i(\hat{x}_q),$$ where the quadrature weights and points are on $\hat{K}$. For a constant $f$ and $P_1$ elements, using the 1-point centroid rule: $f^e_i = \frac{|K|}{3}f(\text{centroid})$ for each $i = 1,2,3$.

Global Assembly

Definition - DOF Map and Assembly

Let $\mathrm{dof}(K, i)$ denote the global DOF index corresponding to local DOF $i$ on element $K$. The global stiffness matrix and load vector are: $$K = \sum_{K \in \mathcal{T}_h} \mathcal{A}_K K^e, \qquad \mathbf{f} = \sum_{K \in \mathcal{T}_h} \mathcal{A}_K \mathbf{f}^e,$$ where $\mathcal{A}_K$ is the assembly operator that maps local entries to global positions. Explicitly, the assembly pseudocode is: 

Initialise K = 0 (N×N sparse), f = 0 (N-vector)
For each element K in T_h:
    Compute B_K, |K|
    Compute K^e (3×3) and f^e (3-vector)
    For local i = 1,2,3:
        I = dof(K, i)
        f[I] += f^e[i]
        For local j = 1,2,3:
            J = dof(K, j)
            K[I,J] += K^e[i,j]

Proposition - Sparsity of $K$

The global stiffness matrix $K$ has at most $O(N)$ nonzero entries for a shape-regular, quasi-uniform mesh in $\mathbb{R}^2$. Specifically, $K_{IJ} \neq 0$ only when global nodes $I$ and $J$ belong to a common element, i.e.\ are connected by an edge in $\mathcal{T}_h$. The average vertex valence (number of neighbouring edges) is bounded independently of $h$, so the number of nonzeros per row is $O(1)$, giving $O(N)$ nonzeros globally.

Remark - Sparse Storage (CSR Format)

In practice $K$ is stored in Compressed Sparse Row (CSR) or Compressed Sparse Column (CSC) format, which stores only the nonzero entries and their column/row indices. This enables matrix-vector products in $O(N)$ time (vs. $O(N^2)$ for dense storage) and is essential for iterative solvers such as Conjugate Gradient.

Example - $P_1$ on Unit Right Triangle

Let $K$ be the triangle with $a_1=(0,0)$, $a_2=(1,0)$, $a_3=(0,1)$; then $|K|=1/2$ and $B_K = I$, so $C_K = I$. The physical gradients are $\nabla\lambda_1=(-1,-1)^\top$, $\nabla\lambda_2=(1,0)^\top$, $\nabla\lambda_3=(0,1)^\top$. The element stiffness is: $$K^e = \frac{1}{2}\begin{pmatrix} 2 & -1 & -1 \\ -1 & 1 & 0 \\ -1 & 0 & 1 \end{pmatrix}.$$ This is symmetric, positive semi-definite (with one zero eigenvalue from the constant), and sums to zero along each row/column - reflecting that the Poisson bilinear form annihilates constants.
