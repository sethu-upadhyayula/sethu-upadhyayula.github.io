4.2 Boundary Conditions

After global assembly, the unconstrained stiffness matrix $K$ is positive semi-definite - it has a non-trivial null space spanned by the constant function. Boundary conditions are required to eliminate this degeneracy and produce a well-posed system. Different types of boundary conditions require different treatment in the discrete system.

Essential (Dirichlet) Boundary Conditions

A Dirichlet condition $u = g$ on $\partial\Omega_D$ is called essential because it must be built into the trial space. For homogeneous Dirichlet ($g = 0$), the assembly restricts to interior DOFs from the start. For non-homogeneous Dirichlet data, there are two standard implementation strategies.

Definition - Elimination Method

Let $\mathcal{I}$ denote the set of interior (free) DOF indices and $\mathcal{B}$ the set of boundary DOF indices. The assembled system $K\mathbf{u} = \mathbf{f}$ partitions as: $$\begin{pmatrix} K_{\mathcal{I}\mathcal{I}} & K_{\mathcal{I}\mathcal{B}} \\ K_{\mathcal{B}\mathcal{I}} & K_{\mathcal{B}\mathcal{B}} \end{pmatrix} \begin{pmatrix} \mathbf{u}_\mathcal{I} \\ \mathbf{g}_\mathcal{B} \end{pmatrix} = \begin{pmatrix} \mathbf{f}_\mathcal{I} \\ \mathbf{f}_\mathcal{B} \end{pmatrix},$$ where $\mathbf{g}_\mathcal{B}$ is the vector of prescribed boundary values $g(x_i)$ at boundary nodes $i \in \mathcal{B}$. The interior unknowns satisfy the reduced system: $$K_{\mathcal{I}\mathcal{I}}\,\mathbf{u}_\mathcal{I} = \mathbf{f}_\mathcal{I} - K_{\mathcal{I}\mathcal{B}}\,\mathbf{g}_\mathcal{B}.$$ The matrix $K_{\mathcal{I}\mathcal{I}}$ is SPD (for a coercive symmetric form) and the system is uniquely solvable.

Definition - Penalty Method

The penalty method keeps the full $N \times N$ system and enforces Dirichlet conditions approximately by adding a large penalty $M \gg \|K\|$: for each boundary DOF $i \in \mathcal{B}$, replace $$K_{ii} \leftarrow K_{ii} + M, \qquad f_i \leftarrow f_i + M\cdot g(x_i).$$ The solution satisfies $u_i \approx g(x_i)$ with error $O(\|K\|/M)$. Advantage: simple to implement without reindexing. Disadvantage: introduces ill-conditioning ($\kappa(K) \sim M$), requiring careful choice of $M$ (typically $M \sim h^{-d}\|K\|$ for optimal accuracy).

Natural (Neumann) Boundary Conditions

Definition - Neumann Condition in the Load Vector

A Neumann condition $\partial u/\partial n = g$ on $\partial\Omega_N$ is natural: it arises from integration by parts in the weak form and enters the load vector as a boundary integral. For a boundary edge $e \subset \partial\Omega_N$ shared by element $K$, the contribution to $\mathbf{f}$ is: $$f_i^{\partial} = \int_e g\,\varphi_i\,dS,$$ assembled via a 1D quadrature rule on the edge $e$. For $P_1$ elements and constant $g$: $f_i^{\partial} = g\cdot|e|/2$ for each endpoint node of $e$. No modification to $K$ is needed; the condition is automatically incorporated.

Robin Boundary Conditions

Definition - Robin Condition

A Robin (or mixed) condition $\alpha u + \partial u/\partial n = g$ on $\partial\Omega_R$ modifies both the bilinear form and the load vector. The weak contribution is: $$a_R(u,v) = \int_{\partial\Omega_R} \alpha\,u\,v\,dS, \qquad \ell_R(v) = \int_{\partial\Omega_R} g\,v\,dS.$$ The total bilinear form becomes $a(u,v) + a_R(u,v)$, which is still continuous and coercive (for $\alpha > 0$). This adds a boundary mass matrix $M_{\partial}$ to $K$: $K \leftarrow K + \alpha M_{\partial}$, with $M_{\partial}$ assembled from boundary edge integrals $\int_e \varphi_j\varphi_i\,dS$.

Mixed Boundary Conditions

Remark - Mixed BCs in Practice

When $\partial\Omega = \Gamma_D \cup \Gamma_N \cup \Gamma_R$ (disjoint parts), apply each condition to the corresponding boundary DOFs and edges:

  • Nodes on $\Gamma_D$: apply Dirichlet (elimination or penalty).
  • Edges on $\Gamma_N$: add Neumann flux to load vector.
  • Edges on $\Gamma_R$: add Robin mass matrix to $K$ and Robin term to $\mathbf{f}$.
The assembly loop can handle all cases by checking, for each boundary edge, which type of condition is imposed. The final system remains SPD (or at least non-singular) provided $|\Gamma_D| > 0$ or the Robin coefficient $\alpha > 0$.

Example - Enforcing Homogeneous Dirichlet by Exclusion

The simplest and most efficient strategy for homogeneous Dirichlet is to never assemble DOFs on $\partial\Omega$. The global assembly loop simply maps boundary local DOFs to a sentinel (e.g.\ index $-1$) and skips contributions to/from those DOFs. The resulting global matrix is already $K_{\mathcal{I}\mathcal{I}}$, SPD by construction, with no post-processing required. This is the standard approach in most FEM codes (FEniCS, deal.II, etc.).
