5.2 Urysohn's Lemma

Urysohn's lemma is a fundamental theorem of general topology, sometimes called "the first deep theorem of the subject." It transforms the purely set-theoretic condition of normality into a rich supply of continuous real-valued functions, bridging the gap between topology and analysis.

Theorem — Urysohn's Lemma

Let $X$ be a topological space. Then $X$ is normal if and only if for every pair of disjoint closed sets $A, B \subseteq X$, there exists a continuous function $f: X \to [0,1]$ satisfying

$$f(a) = 0 \quad \forall\, a \in A, \qquad f(b) = 1 \quad \forall\, b \in B.$$

Such a function $f$ is called a Urysohn function for the pair $(A, B)$.

Proof ($\Leftarrow$)

If such $f$ always exists, then for disjoint closed $A, B$, the sets $U = f^{-1}([0, \tfrac{1}{2}))$ and $V = f^{-1}((\tfrac{1}{2}, 1])$ are disjoint open sets containing $A$ and $B$ respectively. Since $X$ is $T_1$ (implied by the function separating points from singletons), $X$ is normal.

Proof ($\Rightarrow$) — Construction of the Urysohn Function

Assume $X$ is normal and $A, B$ are disjoint closed sets. We construct $f$ via a dyadic rational indexing.

Step 1 — Dyadic interpolation. Let $D = \{k/2^n : k = 0, 1, \ldots, 2^n,\; n \in \mathbb{N}\} \cap [0,1]$ be the dyadic rationals in $[0,1]$. By normality and induction, for each $q \in D$ we find open sets $U_q$ such that:

  • $A \subseteq U_0$ and $\overline{U_1} \subseteq X \setminus B$,
  • $\overline{U_q} \subseteq U_r$ whenever $q < r$ in $D$.

Step 2 — Define $f$. Set

$$f(x) = \inf\{q \in D : x \in U_q\},$$

with the convention $\inf \emptyset = 1$.

Step 3 — Verify $f|_A = 0$ and $f|_B = 1$. If $x \in A$ then $x \in U_q$ for all $q \in D$ (since $A \subseteq U_0 \subseteq U_q$), so $f(x) = 0$. If $x \in B$ then $x \notin U_q$ for any $q \in D$ (since $\overline{U_q} \subseteq X\setminus B$), so $f(x) = 1$.

Step 4 — Verify continuity. The sets $\{f < t\} = \bigcup_{q < t} U_q$ and $\{f > t\} = \bigcup_{q > t}(X \setminus \overline{U_q})$ are open for each $t \in \mathbb{R}$. Since these generate the topology of $\mathbb{R}$, $f$ is continuous.

This completes the construction. $\square$

Remark — Richness of Continuous Functions

Urysohn's lemma shows that normal spaces carry a rich supply of continuous real-valued functions. In particular, the $C(X)$-algebra of continuous functions on a compact Hausdorff space $X$ completely determines $X$ (Gelfand duality), a cornerstone of functional analysis.

Theorem — Urysohn Metrisation Theorem

Every second-countable $T_3$ (regular) space $X$ is metrisable. In fact $X$ embeds homeomorphically into the Hilbert cube $[0,1]^{\mathbb{N}}$.

Proof idea: A second-countable $T_3$ space is normal (shown by a countable-basis argument). Urysohn's lemma then supplies a countable family of continuous functions $\{f_n\}$ separating points, and the map $x \mapsto (f_1(x), f_2(x), \ldots)$ embeds $X$ into $[0,1]^\mathbb{N}$, which carries the metric $d((x_n),(y_n)) = \sum_n 2^{-n}|x_n - y_n|$.

Example — Explicit Urysohn Function on $\mathbb{R}$

Let $X = \mathbb{R}$, $A = (-\infty, 0]$, $B = [1, \infty)$. Then

$$f(x) = \begin{cases} 0 & x \le 0 \\ x & 0 \le x \le 1 \\ 1 & x \ge 1 \end{cases}$$

is a Urysohn function for $(A, B)$. More generally, for $A = (-\infty, a]$ and $B = [b, \infty)$ with $a < b$, one uses $f(x) = \max(0, \min(1, (x-a)/(b-a)))$.

Corollary — Completely Regular Characterisation

A $T_1$ space $X$ is completely regular ($T_{3\frac{1}{2}}$) if and only if its topology is initial with respect to the family $C(X, [0,1])$ of continuous functions into $[0,1]$. Equivalently, $X$ embeds into a product $[0,1]^I$ for some index set $I$ (it is a subspace of a compact Hausdorff space — a Tychonoff space).
