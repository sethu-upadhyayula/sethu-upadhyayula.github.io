3.2 Path Connectedness

Path connectedness is a stronger, more geometric notion of connectedness: a space is path-connected if any two points can be joined by a continuous curve.

Definition — Path

A path in $X$ from $x$ to $y$ is a continuous map $\gamma: [0,1] \to X$ with $\gamma(0) = x$ and $\gamma(1) = y$.

$X$ is path-connected if for every $x, y \in X$ there exists a path from $x$ to $y$.

Theorem — Path-Connected Implies Connected

Proof. Suppose $X$ is path-connected. If $X = U \cup V$ is a separation, pick $x \in U$ and $y \in V$. Let $\gamma: [0,1] \to X$ be a path from $x$ to $y$. Then $[0,1] = \gamma^{-1}(U) \cup \gamma^{-1}(V)$ is a separation of the connected space $[0,1]$, a contradiction. $\square$

The converse is false: a connected space need not be path-connected.

Example — Topologist's Sine Curve

The topologist's sine curve is the subspace of $\mathbb{R}^2$: $$S = \overline{\left\{\left(x, \sin\frac{1}{x}\right) : x > 0\right\}} = \left\{\left(x, \sin\frac{1}{x}\right) : x > 0\right\} \cup (\{0\} \times [-1,1]).$$ $S$ is connected (the closure of a connected set is connected). However, $S$ is not path-connected: any path $\gamma: [0,1] \to S$ ending at a point $(0, y_0)$ on the vertical segment must oscillate infinitely often in finite time near $t=0$, contradicting uniform continuity of $\gamma$ on $[0,1]$. Specifically, the $x$-coordinate $\gamma_1(t) \to 0$ forces the path to traverse arbitrarily many oscillations of $\sin(1/x)$, which is impossible for a continuous map.

This example shows that connected $\not\Rightarrow$ path-connected.

Path Components

Definition — Path Component

The path component of a point $x \in X$ is the set of all $y \in X$ for which there exists a path from $x$ to $y$. Path components partition $X$ into disjoint classes (since path-connectedness is an equivalence relation: reflexive via constant paths, symmetric via reversed paths $\bar{\gamma}(t) = \gamma(1-t)$, and transitive via concatenation).

$\mathbb{R}^n \setminus \{0\}$

Proposition — Path-Connectedness of $\mathbb{R}^n \setminus \{0\}$

For $n \geq 2$: $\mathbb{R}^n \setminus \{0\}$ is path-connected. Given $x, y \in \mathbb{R}^n \setminus \{0\}$, if the straight-line segment $[x,y]$ does not pass through $0$, use it directly. Otherwise, choose an intermediate point $z \notin \{0\}$ not collinear with $x, y, 0$ (possible since $n \geq 2$), and concatenate the segments $[x,z]$ and $[z,y]$.

For $n = 1$: $\mathbb{R} \setminus \{0\} = (-\infty, 0) \cup (0, \infty)$ is not path-connected (it is disconnected); its two path components are $(-\infty, 0)$ and $(0, \infty)$.
