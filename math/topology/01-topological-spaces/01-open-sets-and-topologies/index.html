1.1 Open Sets and Topologies

A topology on a set formalises the intuitive notion of "nearness" without reference to any metric. By declaring which subsets are open, we obtain a framework in which continuity, convergence, and all of point-set topology can be developed axiomatically.

Definition — Topological Space

A topological space is a pair $(X, \tau)$ where $X$ is a set and $\tau \subseteq \mathcal{P}(X)$ is a collection of subsets of $X$, called the open sets, satisfying:

  1. $\emptyset \in \tau$ and $X \in \tau$.
  2. If $\{U_\alpha\}_{\alpha \in A} \subseteq \tau$, then $\bigcup_{\alpha \in A} U_\alpha \in \tau$ (arbitrary unions).
  3. If $U_1, \ldots, U_n \in \tau$, then $U_1 \cap \cdots \cap U_n \in \tau$ (finite intersections).

The collection $\tau$ is called a topology on $X$. The finiteness restriction on intersections is essential: an infinite intersection of open sets need not be open (e.g., $\bigcap_{n=1}^\infty (-1/n, 1/n) = \{0\}$, which is not open in $\mathbb{R}$).

Example — Discrete Topology

The discrete topology on $X$ is $\tau = \mathcal{P}(X)$, i.e., every subset of $X$ is open. This is the finest possible topology on $X$: it has the most open sets. Every function $f: X \to Y$ from a discrete space is continuous.

Example — Indiscrete Topology

The indiscrete topology (or trivial topology) on $X$ is $\tau = \{\emptyset, X\}$, the coarsest possible topology. A function $f: Y \to X$ into an indiscrete space is always continuous. An indiscrete space with more than one point is not $T_1$.

Example — Standard Topology on $\mathbb{R}$

The standard topology on $\mathbb{R}$ is generated by the open intervals $(a, b)$, $a < b$. A set $U \subseteq \mathbb{R}$ is open if and only if for every $x \in U$ there exists $\varepsilon > 0$ such that $(x - \varepsilon, x + \varepsilon) \subseteq U$.

More generally, the standard topology on $\mathbb{R}^n$ is generated by open balls $B(x, r) = \{y \in \mathbb{R}^n : \|y - x\| < r\}$. A set is open if and only if it is a neighbourhood of each of its points.

Example — Cofinite Topology

The cofinite topology on an infinite set $X$ declares $U \in \tau$ if and only if $U = \emptyset$ or $X \setminus U$ is finite. One verifies the axioms: $X \setminus X = \emptyset$ and $X \setminus \emptyset = X$, and finite intersections of cofinite sets are cofinite (the complement of $U_1 \cap \cdots \cap U_n$ is $(X \setminus U_1) \cup \cdots \cup (X \setminus U_n)$, a finite union of finite sets). The cofinite topology on $\mathbb{R}$ is strictly coarser than the standard topology.

Closed Sets

Definition — Closed Sets

A subset $C \subseteq X$ is closed if its complement $X \setminus C$ is open. By De Morgan's laws, the closed sets satisfy: $\emptyset$ and $X$ are closed; arbitrary intersections of closed sets are closed; finite unions of closed sets are closed. Note that "closed" and "open" are not mutually exclusive: $\emptyset$ and $X$ are both open and closed (clopen).

Closure, Interior, and Boundary

Definition — Interior

The interior of $A \subseteq X$ is $$\operatorname{Int}(A) = \bigcup \{ U \in \tau : U \subseteq A \},$$ the largest open set contained in $A$. A point $x$ belongs to $\operatorname{Int}(A)$ if and only if some open neighbourhood of $x$ is entirely contained in $A$.

Definition — Closure

The closure of $A \subseteq X$ is $$\overline{A} = \bigcap \{ C \subseteq X : C \text{ closed},\, A \subseteq C \},$$ the smallest closed set containing $A$. A point $x \in \overline{A}$ if and only if every open neighbourhood of $x$ meets $A$.

Definition — Boundary

The boundary of $A$ is $\partial A = \overline{A} \setminus \operatorname{Int}(A)$. A point $x \in \partial A$ if and only if every open neighbourhood of $x$ meets both $A$ and $X \setminus A$. We have $\overline{A} = \operatorname{Int}(A) \sqcup \partial A$ (disjoint union).

Limit Points and Dense Sets

Definition — Limit Point

A point $x \in X$ is a limit point (or accumulation point) of $A \subseteq X$ if every open neighbourhood of $x$ contains a point of $A$ other than $x$ itself. The set of all limit points of $A$ is the derived set $A'$. Then $\overline{A} = A \cup A'$.

Definition — Dense Subset

A subset $D \subseteq X$ is dense in $X$ if $\overline{D} = X$, equivalently if every non-empty open set meets $D$. A space is separable if it has a countable dense subset.

Example — $\mathbb{Q}$ is Dense in $\mathbb{R}$

The rationals $\mathbb{Q}$ are dense in $\mathbb{R}$ with the standard topology: every open interval $(a, b)$ contains a rational number (by the Archimedean property). Hence $\overline{\mathbb{Q}} = \mathbb{R}$, and $\mathbb{R}$ is separable. By contrast, $\mathbb{Q}$ with the subspace topology is itself dense in $\mathbb{R}$ but has empty interior as a subset of $\mathbb{R}$, since every open interval contains irrationals.

Proposition — Kuratowski Closure Axioms

The closure operator $A \mapsto \overline{A}$ on a topological space satisfies:

  1. $\overline{\emptyset} = \emptyset$.
  2. $A \subseteq \overline{A}$ for all $A$.
  3. $\overline{\overline{A}} = \overline{A}$ (idempotency).
  4. $\overline{A \cup B} = \overline{A} \cup \overline{B}$.

Conversely, any operator satisfying these four axioms defines a unique topology on $X$ for which the operator is the closure. This provides an alternative, equivalent axiomatisation of topological spaces.
