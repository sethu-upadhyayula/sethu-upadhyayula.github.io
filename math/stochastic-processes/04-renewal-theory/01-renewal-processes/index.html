4.1 Renewal Processes

A renewal process is a generalisation of the Poisson process in which the inter-event times are i.i.d. but not necessarily exponential. The Poisson process is the special case of exponential inter-arrival times.

Definition 4.1.1 — Renewal Process

Let $T_1, T_2, \ldots$ be i.i.d. non-negative random variables with distribution $F$ and mean $\mu = E[T_1] \in (0, \infty]$. Define the renewal times $$S_0 = 0, \quad S_n = T_1 + \cdots + T_n, \quad n \ge 1.$$ The renewal counting process is $$N(t) = \max\{n \ge 0 : S_n \le t\} = \sum_{n=1}^\infty \mathbf{1}_{S_n \le t}.$$

$N(t)$ counts the number of renewals in $[0,t]$. Note $N(t) \to \infty$ a.s. as $t \to \infty$.

Definition 4.1.2 — Renewal Function

The renewal function is $m(t) = E[N(t)]$. It satisfies the renewal equation: $$m(t) = F(t) + \int_0^t m(t-s) \, dF(s),$$ where $F(t) = P(T_1 \le t)$. This integral equation is solved via Laplace transforms.

Theorem 4.1.3 — Elementary Renewal Theorem

As $t \to \infty$, $$\frac{m(t)}{t} = \frac{E[N(t)]}{t} \to \frac{1}{\mu}.$$ In other words, the long-run rate of renewals is $1/\mu$ (one renewal per $\mu$ units of time on average). This holds even when $\mu = \infty$, in which case $m(t)/t \to 0$.

Theorem 4.1.4 — Blackwell's Renewal Theorem

For a non-arithmetic renewal process (inter-arrival distribution is non-lattice) with $\mu < \infty$, $$m(t+h) - m(t) \to \frac{h}{\mu} \quad \text{as } t \to \infty,$$ for each fixed $h > 0$. This says the expected number of renewals in any interval of length $h$ converges to $h/\mu$ as the interval is shifted far into the future.

Example — Alternating Renewal Process

A machine alternates between working periods (duration $U_n \sim F_U$, mean $\mu_U$) and repair periods (duration $D_n \sim F_D$, mean $\mu_D$). This is a renewal process with cycle length $T_n = U_n + D_n$. By the renewal reward theorem (Section 4.2), the long-run availability is $$A = \frac{\mu_U}{\mu_U + \mu_D}.$$

This result is independent of the distributions $F_U$ and $F_D$ — only the means matter.

Remark — Renewal Equation

Many quantities of interest (e.g., the probability of being in a renewal at time $t$, the expected age of the current item) satisfy the renewal equation $z(t) = g(t) + \int_0^t z(t-s) \, dF(s)$ for some forcing function $g$. The Key Renewal Theorem gives the long-run limit: $z(t) \to \frac{1}{\mu}\int_0^\infty g(t) \, dt$ as $t \to \infty$ (under non-arithmetic conditions).
