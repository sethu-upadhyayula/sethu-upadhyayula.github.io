1.2 Classification of States

Not all states of a Markov chain behave alike. Some are visited infinitely often; others are eventually abandoned forever. The classification of states — into communicating classes, by period, and by recurrence type — is the key to understanding the long-run behaviour of the chain.

Communicating Classes

Definition 1.2.1 — Accessibility and Communication

State $j$ is accessible from $i$ (written $i \to j$) if $(P^n)_{ij} > 0$ for some $n \ge 0$. States $i$ and $j$ communicate (written $i \leftrightarrow j$) if $i \to j$ and $j \to i$.

Communication is an equivalence relation, and the state space partitions into disjoint communicating classes. A Markov chain is irreducible if all states communicate, i.e., there is only one communicating class.

Period

Definition 1.2.2 — Period

The period of state $i$ is $$d(i) = \gcd\{n \ge 1 : (P^n)_{ii} > 0\}.$$ A state is aperiodic if $d(i) = 1$. Period is a class property: all states in a communicating class have the same period.

Example — Simple Random Walk

On $\mathbb{Z}$, the simple random walk (step $+1$ or $-1$ with equal probability) has $(P^n)_{00} > 0$ only for even $n$, so $d(0) = 2$. The chain is periodic with period 2.

Recurrence and Transience

Definition 1.2.3 — First Return Time

Let $T_i = \min\{n \ge 1 : X_n = i\}$ be the first return time to state $i$. Define the return probability $f_i = P_i(T_i < \infty) = P(T_i < \infty \mid X_0 = i)$.

Definition 1.2.4 — Recurrent and Transient

State $i$ is recurrent if $f_i = 1$ (the chain returns to $i$ with probability 1) and transient if $f_i < 1$ (the chain escapes $i$ forever with positive probability). Recurrence/transience is a class property.

Theorem 1.2.5 — Recurrence Criterion

State $i$ is recurrent if and only if $\sum_{n=0}^{\infty} (P^n)_{ii} = \infty$. It is transient if and only if $\sum_{n=0}^{\infty} (P^n)_{ii} < \infty$.

Sketch of Proof

The expected number of visits to $i$ starting from $i$ is $\sum_n (P^n)_{ii}$. By a renewal argument, if $f_i < 1$, the number of returns is geometrically distributed with finite mean; if $f_i = 1$, the chain returns infinitely often. $\square$

Positive and Null Recurrence

Definition 1.2.6 — Positive and Null Recurrent

A recurrent state $i$ is positive recurrent if $E_i[T_i] < \infty$ (finite mean return time) and null recurrent if $E_i[T_i] = \infty$.

For finite irreducible chains, every state is positive recurrent. For the simple random walk on $\mathbb{Z}$, every state is null recurrent ($E_0[T_0] = \infty$). In $\mathbb{Z}^d$ with $d \ge 3$, the simple random walk is transient.

Remark — Polya's Theorem

The simple symmetric random walk on $\mathbb{Z}^d$ is recurrent for $d = 1, 2$ and transient for $d \ge 3$. This striking dimension-dependence is a fundamental result in probability theory.
