1.1 Transition Matrices

A discrete-time Markov chain (DTMC) is a sequence of random variables $(X_n)_{n \ge 0}$ on a countable state space $S$. The essential feature is that given the present state, the past is irrelevant for predicting the future.

Definition 1.1.1 — Markov Property

A sequence $(X_n)_{n \ge 0}$ is a Markov chain with transition matrix $P$ if for all $n \ge 0$ and all states $i_0, \ldots, i_n, j \in S$, $$P(X_{n+1} = j \mid X_0 = i_0, \ldots, X_n = i_n) = P(X_{n+1} = j \mid X_n = i_n) = P_{i_n j}.$$ The matrix $P$ is called the transition matrix (or stochastic matrix) of the chain.

Definition 1.1.2 — Stochastic Matrix

A matrix $P = (P_{ij})_{i,j \in S}$ is a stochastic matrix if:
(i) $P_{ij} \ge 0$ for all $i, j \in S$;
(ii) $\sum_{j \in S} P_{ij} = 1$ for all $i \in S$ (each row sums to 1).

$n$-Step Transition Probabilities

The probability of going from state $i$ to state $j$ in exactly $n$ steps is the $(i,j)$ entry of the $n$th power of $P$.

Theorem 1.1.3 — $n$-Step Probabilities

For a homogeneous Markov chain with transition matrix $P$, $$(P^n)_{ij} = P(X_n = j \mid X_0 = i).$$ In particular, $(P^1)_{ij} = P_{ij}$ and $(P^0)_{ij} = \mathbf{1}_{i=j}$.

Theorem 1.1.4 — Chapman–Kolmogorov Equations

For all $m, n \ge 0$ and states $i, j$, $$P^{m+n} = P^m P^n, \qquad (P^{m+n})_{ij} = \sum_{k \in S} (P^m)_{ik}(P^n)_{kj}.$$ This is simply the rule for matrix multiplication, reflecting the law of total probability.

Proof

By the Markov property and total probability: $$(P^{m+n})_{ij} = P(X_{m+n}=j \mid X_0=i) = \sum_{k} P(X_{m+n}=j \mid X_m=k) P(X_m=k \mid X_0=i) = \sum_k (P^n)_{kj}(P^m)_{ik}. \quad \square$$

Initial Distribution

Definition 1.1.5 — Initial Distribution

The initial distribution is a probability vector $\mu$ with $\mu_i = P(X_0 = i)$. The marginal distribution at time $n$ is then $\mu P^n$, i.e., $$P(X_n = j) = \sum_i \mu_i (P^n)_{ij} = (\mu P^n)_j.$$

Example — Two-State Chain

Consider $S = \{0,1\}$ with transition matrix $$P = \begin{pmatrix} 1-\alpha & \alpha \\ \beta & 1-\beta \end{pmatrix}, \quad \alpha, \beta \in (0,1).$$ The $n$-step matrix has closed form $(P^n)_{00} = \frac{\beta}{\alpha+\beta} + \frac{\alpha}{\alpha+\beta}(1-\alpha-\beta)^n$, which converges to $\beta/(\alpha+\beta)$ geometrically fast.

This models any system switching between two states, such as a machine alternating between working and broken.

Remark

The Chapman–Kolmogorov equations reduce all questions about finite-time distributions to matrix algebra. For large state spaces, computing $P^n$ directly is expensive; instead one uses eigendecomposition or generating functions.
