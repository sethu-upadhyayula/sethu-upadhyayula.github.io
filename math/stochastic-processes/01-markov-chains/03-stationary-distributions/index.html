1.3 Stationary Distributions

A stationary (or invariant) distribution is a probability distribution that is preserved under the dynamics of the chain. It describes the long-run fraction of time spent in each state and is central to equilibrium analysis.

Definition 1.3.1 — Stationary Distribution

A probability vector $\pi = (\pi_i)_{i \in S}$ is a stationary distribution for the Markov chain with transition matrix $P$ if: $$\pi P = \pi, \quad \pi_i \ge 0 \text{ for all } i, \quad \sum_{i \in S} \pi_i = 1.$$ Equivalently, $\pi_j = \sum_{i} \pi_i P_{ij}$ for all $j$. If $X_0 \sim \pi$, then $X_n \sim \pi$ for all $n$.

Theorem 1.3.2 — Existence and Uniqueness

An irreducible Markov chain has a stationary distribution if and only if it is positive recurrent. In that case, the stationary distribution is unique and satisfies $$\pi_i = \frac{1}{E_i[T_i]},$$ where $T_i = \min\{n \ge 1 : X_n = i\}$ is the first return time to $i$.

Sketch of Proof

Between successive visits to a fixed state $i$, the chain visits each state $j$ an expected number of $E_i[\text{visits to }j\text{ before }T_i]$ times. Defining $\pi_j$ proportional to this quantity and normalising by $E_i[T_i]$ yields the unique stationary distribution. $\square$

Detailed Balance

Definition 1.3.3 — Detailed Balance and Reversibility

A distribution $\pi$ satisfies detailed balance with $P$ if $$\pi_i P_{ij} = \pi_j P_{ji} \quad \text{for all } i, j \in S.$$ A chain satisfying detailed balance is called reversible. Summing over $i$ shows that detailed balance implies $\pi P = \pi$, so detailed balance is sufficient (but not necessary) for $\pi$ to be stationary.

Example — Birth-Death Chains

A birth-death chain on $S = \{0, 1, 2, \ldots\}$ has transitions only to adjacent states: $P_{i,i+1} = p_i$, $P_{i,i-1} = q_i = 1 - p_i$ (with $P_{0,-1} = 0$). Detailed balance gives $$\pi_n = \pi_0 \prod_{k=0}^{n-1} \frac{p_k}{q_{k+1}}.$$ The chain is positive recurrent iff $\sum_n \pi_n < \infty$, which determines the normalising constant.

This framework covers queuing models such as the M/M/1 queue and geometric distributions.

Remark — MCMC

In Markov chain Monte Carlo (MCMC), one constructs a chain whose stationary distribution is a desired target $\pi$ (e.g., a Bayesian posterior). The Metropolis–Hastings algorithm achieves this by enforcing detailed balance with an acceptance-rejection step.
