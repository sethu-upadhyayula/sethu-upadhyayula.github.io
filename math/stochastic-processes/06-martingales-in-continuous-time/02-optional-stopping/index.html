6.2 Optional Stopping

The optional stopping theorem (OST) is one of the most powerful tools derived from martingale theory. It says that under appropriate conditions, the martingale property is preserved when a fixed time is replaced by a random stopping time — you cannot gain an advantage over a fair game by choosing a clever stopping rule.

Stopping Times

Definition 6.2.1 — Stopping Time

A random variable $\tau : \Omega \to [0, \infty]$ is a stopping time with respect to filtration $(\mathcal{F}_t)$ if $$\{\tau \le t\} \in \mathcal{F}_t \quad \text{for all } t \ge 0.$$

Equivalently (for right-continuous filtrations), $\{\tau < t\} \in \mathcal{F}_t$ for all $t$, or $\{\tau = t\} \in \mathcal{F}_t$ for all $t$.

The stopped $\sigma$-algebra is $\mathcal{F}_\tau = \{A \in \mathcal{F} : A \cap \{\tau \le t\} \in \mathcal{F}_t \text{ for all } t\}$. It represents the information available at the (random) time $\tau$.

Example — First Passage Times are Stopping Times

For Brownian motion $(B_t)$ adapted to $(\mathcal{F}_t)$ and $a \in \mathbb{R}$, the first passage time $T_a = \inf\{t \ge 0 : B_t = a\}$ is a stopping time. Indeed, since $B$ has continuous paths, $\{T_a \le t\} = \{\sup_{0 \le s \le t} (a - B_s) \le 0 \text{ if } a > 0\} \in \mathcal{F}_t$. Similarly, the exit time $\tau = \inf\{t : B_t \notin (c, d)\}$ from an interval is a stopping time.

On the other hand, the last time a process is at a given level is generally not a stopping time, since one would need to know the future to determine when the process last visited a level.

The Optional Stopping Theorem

Theorem 6.2.2 — Optional Stopping Theorem

Let $(M_t)_{t \ge 0}$ be a right-continuous martingale and $\tau$ a stopping time. Then $E[M_\tau] = E[M_0]$ under any of the following conditions:

  1. $\tau \le T$ a.s. for some deterministic constant $T < \infty$ (bounded stopping time).
  2. $(M_{t \wedge \tau})_{t \ge 0}$ is uniformly integrable.
  3. $E[\tau] < \infty$ and $|M_t - M_{t-}| \le C$ a.s. for some constant $C$ (bounded increments and finite mean stopping time).

The result also holds in conditional form: if $\sigma \le \tau$ are stopping times satisfying the above conditions, then $E[M_\tau \mid \mathcal{F}_\sigma] = M_\sigma$ a.s.

Remark — Failure of OST without Conditions

Without additional conditions, $E[M_\tau] = E[M_0]$ can fail. Classic counterexample: BM hits level 1 at time $T_1$ with $P(T_1 < \infty) = 1$, so $B_{T_1} = 1$, giving $E[B_{T_1}] = 1 \ne 0 = E[B_0]$. The OST fails because $T_1$ has infinite mean and the stopped process is not uniformly integrable. The bounded stopping time version avoids this: $E[B_{T_1 \wedge t}] = 0$ for all $t$, but taking $t \to \infty$ requires uniform integrability to swap limit and expectation.

Wald's Identities for Brownian Motion

Theorem 6.2.3 — Wald's Identities for BM

Let $\tau$ be a stopping time with $E[\tau] < \infty$. Then:

  1. $E[B_\tau] = 0$ (first Wald identity).
  2. $E[B_\tau^2] = E[\tau]$ (second Wald identity).

Proof. (1) Apply OST to the martingale $B_t$: under $E[\tau] < \infty$ and continuity of BM, the stopped process is uniformly integrable, giving $E[B_\tau] = E[B_0] = 0$.

(2) Apply OST to the martingale $B_t^2 - t$: $E[B_\tau^2 - \tau] = E[B_0^2 - 0] = 0$, so $E[B_\tau^2] = E[\tau]$. $\square$

These identities are the continuous-time analogues of Wald's equations for sums of i.i.d. random variables. They allow us to compute $E[\tau]$ from $E[B_\tau^2]$ — often the latter is easier to determine from the geometry of the stopping region.

Applications

Example — Gambler's Ruin for BM

Let $B_0 = x \in (0, b)$ and $\tau = \inf\{t : B_t \notin (0, b)\} = T_0 \wedge T_b$. Since $B$ has continuous paths, $B_\tau \in \{0, b\}$. By OST applied to the martingale $B_t$: $$x = E[B_0] = E[B_\tau] = b \cdot P(B_\tau = b) + 0 \cdot P(B_\tau = 0),$$ so $P(B_\tau = b) = x/b$ and $P(B_\tau = 0) = (b-x)/b$.

For the exit time, apply OST to $B_t^2 - t$: $$x^2 = E[B_\tau^2] - E[\tau] \cdot (-1) \implies E[\tau] = E[B_\tau^2] - x^2.$$ Computing $E[B_\tau^2] = b^2 \cdot (x/b) + 0 = bx$, so $E[\tau] = bx - x^2 = x(b-x)$.

The expected exit time $x(b-x)$ is maximised at $x = b/2$, giving $E[\tau] = b^2/4$. This matches the solution of the ODE $\frac{1}{2} u''(x) = -1$, $u(0) = u(b) = 0$.

Example — First Passage Time via Exponential Martingale

For $a > 0$, find $E[e^{-\lambda T_a}]$ (the Laplace transform of $T_a$). Consider the exponential martingale $M_t = e^{\theta B_t - \theta^2 t/2}$ with $\theta > 0$. Apply OST at $\tau = T_a \wedge t$: $$1 = E[M_0] = E[M_{T_a \wedge t}] = E[e^{\theta B_{T_a \wedge t} - \theta^2 (T_a \wedge t)/2}].$$ Taking $t \to \infty$ and using dominated convergence (the stopped process is bounded in the appropriate sense): $$1 = E[e^{\theta a - \theta^2 T_a/2}] = e^{\theta a} E[e^{-\theta^2 T_a /2}].$$ Setting $\lambda = \theta^2/2$ (so $\theta = \sqrt{2\lambda}$): $$E[e^{-\lambda T_a}] = e^{-a\sqrt{2\lambda}}.$$

This is the Laplace transform of the inverse Gaussian (Lévy) distribution with density $f(t) = \frac{a}{\sqrt{2\pi t^3}} e^{-a^2/(2t)}$, confirming Section 5.2.

Example — BM Hitting Time for an Interval

More generally, let $\tau = T_{-b} \wedge T_a$ for $a, b > 0$ (BM starts at 0 and exits $(-b, a)$). By the OST for $B_t$: $E[B_\tau] = 0 = a \cdot P(B_\tau = a) - b \cdot P(B_\tau = -b)$. Combined with $P(B_\tau = a) + P(B_\tau = -b) = 1$: $$P(B_\tau = a) = \frac{b}{a+b}, \quad P(B_\tau = -b) = \frac{a}{a+b}.$$ By OST for $B_t^2 - t$: $E[\tau] = E[B_\tau^2] = a^2 \cdot \frac{b}{a+b} + b^2 \cdot \frac{a}{a+b} = ab.$

The expected exit time is $ab$, symmetric in $a$ and $b$. Setting $a = b$ (symmetric exit from $(-a,a)$) gives $E[\tau] = a^2$, consistent with scaling: if $\tau_1$ is the exit time from $(-1,1)$ then the exit time from $(-a,a)$ has the same distribution as $a^2 \tau_1$ by BM scaling.
