6.1 Continuous-Time Martingales

Continuous-time martingales extend the discrete martingale concept to uncountably many time indices. The theory requires a careful setup of filtrations and regularity conditions on paths, but the essential intuition — that a martingale is a "fair game" with no drift — remains the same.

Filtrations and Adapted Processes

Definition 6.1.1 — Filtration and Adapted Process

A filtration $(\mathcal{F}_t)_{t \ge 0}$ is an increasing family of sub-$\sigma$-algebras of $\mathcal{F}$: $\mathcal{F}_s \subseteq \mathcal{F}_t \subseteq \mathcal{F}$ for $s \le t$. Intuitively, $\mathcal{F}_t$ encodes all information available up to time $t$.

A stochastic process $(X_t)_{t \ge 0}$ is adapted to $(\mathcal{F}_t)$ if $X_t$ is $\mathcal{F}_t$-measurable for each $t \ge 0$.

The natural filtration of a process $(X_t)$ is $\mathcal{F}_t^X = \sigma(X_s : s \le t)$. A filtration satisfies the usual conditions if it is right-continuous ($\mathcal{F}_t = \mathcal{F}_{t+} := \bigcap_{s>t} \mathcal{F}_s$) and complete (each $\mathcal{F}_t$ contains all $P$-null sets of $\mathcal{F}$). We always assume the usual conditions hold.

Definition 6.1.2 — Continuous-Time Martingale

An adapted process $(M_t)_{t \ge 0}$ with $E[|M_t|] < \infty$ for all $t$ is a martingale with respect to $(\mathcal{F}_t)$ if $$E[M_t \mid \mathcal{F}_s] = M_s \quad \text{for all } 0 \le s \le t.$$

It is a supermartingale if $E[M_t \mid \mathcal{F}_s] \le M_s$ and a submartingale if $E[M_t \mid \mathcal{F}_s] \ge M_s$.

Equivalently, for a martingale: $E[M_t] = E[M_0]$ for all $t$ (constant mean), and the process has no drift on average. A submartingale tends to increase on average; a supermartingale tends to decrease.

Brownian Martingales

Proposition 6.1.3 — Fundamental Brownian Martingales

Let $(B_t)_{t \ge 0}$ be standard Brownian motion adapted to $(\mathcal{F}_t)$. The following are martingales with respect to $(\mathcal{F}_t)$:

  1. $M_t = B_t$ (since $E[B_t \mid \mathcal{F}_s] = B_s$ by the independent increment property).
  2. $M_t = B_t^2 - t$ (the "compensated square": $B^2$ is a submartingale and $t$ is its compensator).
  3. $M_t = e^{\theta B_t - \theta^2 t/2}$ for any $\theta \in \mathbb{R}$ (the exponential martingale or Doléans-Dade exponential).

Verification for (3): $E[e^{\theta B_t} \mid \mathcal{F}_s] = e^{\theta B_s} \cdot E[e^{\theta(B_t - B_s)}] = e^{\theta B_s} \cdot e^{\theta^2(t-s)/2}$. Multiplying by $e^{-\theta^2 t/2}$ gives $e^{\theta B_s - \theta^2 s/2}$. $\square$

The exponential martingale is crucial in Girsanov's theorem: it provides the Radon–Nikodym derivative for a change of measure that turns a drifted BM into a standard BM.

Path Regularity: Càdlàg Processes

Definition 6.1.4 — Càdlàg Processes

A process $(X_t)_{t \ge 0}$ is càdlàg (from French: continu à droite, limites à gauche) if almost every sample path is right-continuous with left limits: for all $t \ge 0$, $$X_{t+} := \lim_{s \downarrow t} X_s = X_t \quad \text{a.s.,} \qquad X_{t-} := \lim_{s \uparrow t} X_s \text{ exists a.s.}$$

Every martingale with respect to a filtration satisfying the usual conditions has a càdlàg modification (Doob's regularity theorem). For continuous processes (like BM-driven stochastic integrals), càdlàg reduces to continuous. The space of càdlàg functions on $[0,T]$ is denoted $D[0,T]$ and equipped with the Skorokhod topology.

Local Martingales

Definition 6.1.5 — Local Martingale

An adapted càdlàg process $(M_t)_{t \ge 0}$ is a local martingale if there exists an increasing sequence of stopping times $\tau_n \uparrow \infty$ a.s. such that each stopped process $M^{\tau_n}_t := M_{t \wedge \tau_n}$ is a (true) martingale.

Every $L^2$-bounded martingale is a local martingale; the converse fails. Stochastic integrals $\int_0^t H_s\,dB_s$ are local martingales but may fail to be true martingales if $H$ is not square-integrable. The distinction matters: Novikov's condition $E[\exp(\frac{1}{2}\int_0^T H_s^2\,ds)] < \infty$ is a standard sufficient condition for the exponential local martingale to be a true martingale.

Martingale Convergence

Theorem 6.1.6 — Martingale Convergence Theorem

Let $(M_t)_{t \ge 0}$ be a right-continuous martingale. If $\sup_{t \ge 0} E[|M_t|] < \infty$ (i.e., $(M_t)$ is $L^1$-bounded), then $M_\infty := \lim_{t \to \infty} M_t$ exists a.s. and $E[|M_\infty|] < \infty$.

Stronger: if $(M_t)$ is $L^p$-bounded for some $p > 1$, then $M_t \to M_\infty$ in $L^p$ and $M_t = E[M_\infty \mid \mathcal{F}_t]$ for all $t$ (the martingale is uniformly integrable). The $L^1$ case requires an additional uniform integrability assumption for $L^1$ convergence.

Theorem 6.1.7 — Doob's Maximal Inequality

For a non-negative submartingale $(X_t)_{0 \le t \le T}$ and $\lambda > 0$: $$P\!\left(\sup_{0 \le t \le T} X_t \ge \lambda\right) \le \frac{E[X_T]}{\lambda}.$$

For a martingale $(M_t)$ and $p > 1$, Doob's $L^p$ inequality gives: $$E\!\left[\sup_{0 \le t \le T} |M_t|^p\right] \le \left(\frac{p}{p-1}\right)^p E[|M_T|^p].$$

These inequalities control the running maximum and are fundamental for stochastic integration theory. For BM, $M_t = B_t$ is a martingale, so $P(\sup_{0 \le s \le t} |B_s| \ge \lambda) \le E[B_t^2]/\lambda^2 = t/\lambda^2$ by the $L^2$ version.

Example — Lévy's Characterisation of Brownian Motion

A continuous local martingale $(M_t)_{t \ge 0}$ with $M_0 = 0$ and quadratic variation $[M]_t = t$ is a standard Brownian motion. More precisely, if $(M_t)$ is a continuous local martingale with $[M]_t = t$ a.s., then $M$ has the same distribution as standard BM.

This is Lévy's characterisation theorem. It is the converse to the fact that BM is a continuous martingale with $[B]_t = t$. It is used to identify BM in practice: any continuous local martingale with deterministic quadratic variation $t$ must be BM (up to distribution).
