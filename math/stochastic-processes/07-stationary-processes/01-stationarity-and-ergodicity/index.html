7.1 Stationarity and Ergodicity

Stationarity formalises the idea that a process is "in equilibrium": its statistical properties are invariant under time shifts. Two levels of stationarity are commonly distinguished: strict (distributional) and weak (second-order). Ergodicity then asks whether a single realisation of the process contains enough information to recover ensemble averages.

Strict Stationarity

Definition 7.1.1 - Strict Stationarity

A stochastic process $(X_t)_{t \in \mathbb{R}}$ (or $t \in \mathbb{Z}$) is strictly stationary if for all $n \ge 1$, all $t_1, \ldots, t_n$, and all $h$: $$(X_{t_1+h}, X_{t_2+h}, \ldots, X_{t_n+h}) \overset{d}{=} (X_{t_1}, X_{t_2}, \ldots, X_{t_n}).$$

That is, the joint distribution of any finite subcollection is invariant under time shifts. Strict stationarity implies all marginal distributions are identical: $X_t \overset{d}{=} X_0$ for all $t$. If $E[X_t^2] < \infty$, strict stationarity implies weak stationarity. The converse holds for Gaussian processes.

Weak (Wide-Sense) Stationarity

Definition 7.1.2 - Weak Stationarity

A process $(X_t)$ with $E[X_t^2] < \infty$ is weakly stationary (or wide-sense stationary, WSS) if:

  1. $E[X_t] = \mu$ (constant mean, independent of $t$).
  2. $\operatorname{Cov}(X_t, X_s) = R(t - s)$ (the covariance depends only on the lag $\tau = t - s$).

The function $R: \mathbb{R} \to \mathbb{R}$ is called the autocovariance function (ACF). Note $R(0) = \operatorname{Var}(X_t)$ and $R(-\tau) = R(\tau)$ (symmetry).

Weak stationarity is the standard assumption in time series analysis. It allows the use of Fourier methods (Section 7.2) to decompose the process by frequency. Many models (AR, MA, ARMA) are designed to be weakly stationary.

Example - AR(1) Process

The autoregressive process of order 1, AR(1): $X_t = \phi X_{t-1} + \varepsilon_t$ where $\varepsilon_t \sim \mathcal{N}(0, \sigma^2)$ i.i.d. For $|\phi| < 1$, the unique weakly stationary solution is $X_t = \sum_{k=0}^\infty \phi^k \varepsilon_{t-k}$, with:

  • $E[X_t] = 0$
  • $\operatorname{Var}(X_t) = \sigma^2 / (1 - \phi^2)$
  • $R(\tau) = \frac{\sigma^2}{1 - \phi^2} \phi^{|\tau|}$ for $\tau \in \mathbb{Z}$.

The ACF decays geometrically in the lag $|\tau|$. For $\phi > 0$, the process has positive autocorrelation at all lags (persistence). For $\phi < 0$, the ACF alternates in sign (oscillation). At $|\phi| = 1$ (unit root), the process is a random walk and is non-stationary.

The Ergodic Theorem

Definition 7.1.3 - Measure-Preserving Transformation

A measurable map $T: \Omega \to \Omega$ is measure-preserving with respect to $P$ if $P(T^{-1}A) = P(A)$ for all $A \in \mathcal{F}$. For a strictly stationary process $(X_t)_{t \in \mathbb{Z}}$, the shift $T: (x_0, x_1, \ldots) \mapsto (x_1, x_2, \ldots)$ is measure-preserving on the sequence space.

A set $A$ is $T$-invariant if $T^{-1}A = A$. The transformation $T$ is ergodic if every invariant set has probability 0 or 1 - equivalently, the process cannot be decomposed into non-trivial invariant sub-processes.

Theorem 7.1.4 - Birkhoff's Ergodic Theorem

Let $T$ be a measure-preserving transformation and $f \in L^1(P)$. Then the time average converges almost surely: $$\frac{1}{n}\sum_{k=0}^{n-1} f(T^k \omega) \xrightarrow{n \to \infty} E[f \mid \mathcal{I}](\omega) \quad \text{a.s.,}$$ where $\mathcal{I} = \{A \in \mathcal{F} : T^{-1}A = A\}$ is the $\sigma$-algebra of $T$-invariant sets.

If $T$ is ergodic (so $\mathcal{I}$ is trivial), then $E[f \mid \mathcal{I}] = E[f]$ a.s., giving: $$\frac{1}{n}\sum_{k=0}^{n-1} f(T^k \omega) \to E[f] \quad \text{a.s.}$$

For a strictly stationary ergodic process $(X_t)$ and $f(x_0) = x_0$, this gives $\frac{1}{n}\sum_{t=0}^{n-1} X_t \to E[X_0]$ a.s.: time averages converge to ensemble averages. This is the statistical foundation for estimating parameters from time series data.

Theorem 7.1.5 - Continuous-Time Ergodic Theorem

For a strictly stationary ergodic process $(X_t)_{t \ge 0}$ with $E[|X_0|] < \infty$: $$\frac{1}{T}\int_0^T X_t\,dt \to E[X_0] \quad \text{a.s. as } T \to \infty.$$

More generally, for any $f$ with $E[|f(X_0)|] < \infty$: $\frac{1}{T}\int_0^T f(X_t)\,dt \to E[f(X_0)]$ a.s. This theorem justifies the use of time averages to estimate distributional properties, and underpins the ergodic hypothesis in statistical physics.

Mixing Conditions

Definition 7.1.6 - Mixing

A stationary process is strongly mixing (or $\alpha$-mixing) if $$\alpha(h) := \sup_{A \in \mathcal{F}_0^{-\infty},\, B \in \mathcal{F}_h^\infty} |P(A \cap B) - P(A)P(B)| \to 0 \quad \text{as } h \to \infty,$$ where $\mathcal{F}_a^b = \sigma(X_t : a \le t \le b)$.

The process is $m$-dependent if $X_t$ and $X_s$ are independent whenever $|t-s| > m$. Other mixing notions include $\beta$-mixing (absolute regularity) and $\phi$-mixing.

Mixing implies ergodicity. Under mixing conditions, CLT-type results hold for partial sums $\sum_{t=1}^n X_t$: $\frac{1}{\sqrt{n}} \sum_{t=1}^n (X_t - \mu) \xrightarrow{d} \mathcal{N}(0, \sigma^2_\infty)$ where $\sigma^2_\infty = \sum_{h=-\infty}^\infty R(h)$ is the long-run variance. This is the basis for statistical inference about stationary time series.

Example - Ergodicity of AR(1)

The weakly stationary AR(1) process with $|\phi| < 1$ is ergodic. This follows because its innovation sequence $(\varepsilon_t)$ is i.i.d. (hence ergodic), and the AR(1) is a measurable function of this sequence.

Concretely, the ergodic theorem gives $\frac{1}{n}\sum_{t=1}^n X_t \to 0$ a.s. and $\frac{1}{n}\sum_{t=1}^n X_t X_{t+k} \to R(k) = \frac{\sigma^2 \phi^{|k|}}{1-\phi^2}$ a.s. for each fixed $k$. This allows consistent estimation of the ACF from a single observed trajectory.
