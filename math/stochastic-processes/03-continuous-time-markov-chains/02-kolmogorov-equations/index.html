3.2 Kolmogorov Equations

The Kolmogorov equations are ODEs for the transition probability matrix $P(t) = (p_{ij}(t))$ of a CTMC, where $p_{ij}(t) = P(X(t) = j \mid X(0) = i)$. They are the continuous-time analogue of the Chapman–Kolmogorov equations.

Theorem 3.2.1 — Matrix Exponential Solution

For a CTMC with generator $Q$, the transition semigroup is $$P(t) = e^{Qt} = \sum_{n=0}^{\infty} \frac{(Qt)^n}{n!}, \quad t \ge 0.$$ This satisfies $P(0) = I$, $P(s+t) = P(s)P(t)$ (semigroup property), and $P'(0) = Q$.

Theorem 3.2.2 — Kolmogorov Forward Equation

The transition semigroup satisfies the forward (Fokker–Planck) equation: $$\frac{d}{dt} P(t) = P(t) Q, \quad P(0) = I.$$ Entry-wise: $p'_{ij}(t) = \sum_k p_{ik}(t) q_{kj}$. The forward equation describes how probability flows into state $j$ from all states $k$.

Theorem 3.2.3 — Kolmogorov Backward Equation

Equivalently, the transition semigroup satisfies the backward equation: $$\frac{d}{dt} P(t) = Q P(t), \quad P(0) = I.$$ Entry-wise: $p'_{ij}(t) = \sum_k q_{ik} p_{kj}(t)$. The backward equation describes evolution from the initial state $i$.

Computing via Eigendecomposition

Theorem 3.2.4 — Spectral Computation

If $Q$ is diagonalisable with eigenvalues $\lambda_1, \ldots, \lambda_n$ and $Q = V \Lambda V^{-1}$, then $$e^{Qt} = V e^{\Lambda t} V^{-1}, \quad e^{\Lambda t} = \text{diag}(e^{\lambda_1 t}, \ldots, e^{\lambda_n t}).$$ Since $Q$ has non-positive eigenvalues with $\lambda_1 = 0$, each $p_{ij}(t)$ is a sum of decaying exponentials converging to $\pi_j$.

Example — Two-State CTMC

For the two-state generator $Q = \begin{pmatrix} -\alpha & \alpha \\ \beta & -\beta \end{pmatrix}$, the eigenvalues are $0$ and $-(\alpha+\beta)$. The transition probabilities are: $$p_{00}(t) = \frac{\beta}{\alpha+\beta} + \frac{\alpha}{\alpha+\beta} e^{-(\alpha+\beta)t}, \quad p_{01}(t) = \frac{\alpha}{\alpha+\beta}\left(1 - e^{-(\alpha+\beta)t}\right).$$

As $t \to \infty$, these converge to the stationary probabilities $\pi_0 = \beta/(\alpha+\beta)$ and $\pi_1 = \alpha/(\alpha+\beta)$.

Remark — Uniformisation

A useful computational technique is uniformisation: set $\Lambda = \max_i \nu_i$ and write $Q = \Lambda(\tilde{P} - I)$ where $\tilde{P} = I + Q/\Lambda$ is a stochastic matrix. Then $e^{Qt} = e^{-\Lambda t} \sum_{n=0}^\infty \frac{(\Lambda t)^n}{n!} \tilde{P}^n$, a Poisson mixture of discrete-time chains — useful for numerical computation.
