3.3 Stationary Distributions

Stationary distributions for CTMCs are defined through the generator $Q$, not through $P(t)$. They characterise the long-run time-average behaviour of the chain and arise as the null space of $Q^T$.

Definition 3.3.1 — Stationary Distribution for a CTMC

A probability vector $\pi$ is a stationary distribution for the CTMC with generator $Q$ if $$\pi Q = 0 \quad \text{and} \quad \sum_i \pi_i = 1, \quad \pi_i \ge 0.$$ Entry-wise: $\sum_i \pi_i q_{ij} = 0$ for all $j$, i.e., the rate of flow into $j$ equals the rate of flow out of $j$ (global balance).

Definition 3.3.2 — Detailed Balance for CTMCs

A distribution $\pi$ satisfies detailed balance with $Q$ if $$\pi_i q_{ij} = \pi_j q_{ji} \quad \text{for all } i \ne j.$$ Summing over $j$ shows that detailed balance implies $\pi Q = 0$. A CTMC with a detailed balance solution is called reversible in equilibrium.

M/M/1 Queue

Example — M/M/1 Queue Stationary Distribution

The M/M/1 queue is a birth-death CTMC on $\{0, 1, 2, \ldots\}$ with $q_{n,n+1} = \lambda$ and $q_{n,n-1} = \mu$ ($n \ge 1$). Detailed balance gives $$\pi_n \lambda = \pi_{n+1} \mu \implies \pi_{n+1} = \rho \pi_n, \quad \rho = \frac{\lambda}{\mu}.$$ For stability ($\rho < 1$), normalising gives: $$\pi_n = (1-\rho)\rho^n, \quad n = 0, 1, 2, \ldots$$

This is a geometric distribution. The mean queue length is $E[N] = \rho/(1-\rho)$ and the mean waiting time (by Little's law) is $W = 1/(\mu - \lambda)$.

Theorem 3.3.3 — Ergodic Theorem for CTMCs

For an irreducible positive recurrent CTMC with stationary distribution $\pi$, $$\frac{1}{t} \int_0^t \mathbf{1}_{X(s)=j} \, ds \xrightarrow{a.s.} \pi_j \quad \text{as } t \to \infty.$$ That is, $\pi_j$ equals the long-run proportion of time spent in state $j$.

Remark — PASTA Property

For Poisson arrivals to a CTMC system, the PASTA property (Poisson Arrivals See Time Averages) states that arriving customers see the system in state $j$ with probability $\pi_j$. This is because Poisson arrivals are independent of the system history, making each arrival an independent sample from the stationary distribution.
