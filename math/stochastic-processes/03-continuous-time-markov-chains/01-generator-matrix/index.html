3.1 Generator Matrix

The generator matrix (also called the rate matrix or $Q$-matrix) plays the role for CTMCs that the transition matrix $P$ plays for DTMCs. It describes the instantaneous rates of transitions between states.

Definition 3.1.1 — Generator (Rate) Matrix

A matrix $Q = (q_{ij})_{i,j \in S}$ is a generator matrix (or $Q$-matrix) if:
(i) $q_{ij} \ge 0$ for all $i \ne j$ (off-diagonal entries are non-negative transition rates);
(ii) $q_{ii} = -\sum_{j \ne i} q_{ij}$ for all $i$ (rows sum to zero).

The quantity $\nu_i := |q_{ii}| = \sum_{j \ne i} q_{ij}$ is called the total exit rate from state $i$.

Definition 3.1.2 — Holding Times

When a CTMC enters state $i$, it remains there for a holding time $H_i \sim \text{Exp}(\nu_i)$, independent of everything else. The mean holding time is $1/\nu_i$. After the holding time expires, the chain jumps to state $j \ne i$ with probability $q_{ij}/\nu_i$.

Definition 3.1.3 — Embedded Chain

The sequence of states visited by the CTMC forms the embedded (jump) chain, a DTMC with transition matrix $$\tilde{P}_{ij} = \begin{cases} q_{ij}/\nu_i & \text{if } j \ne i \\ 0 & \text{if } j = i. \end{cases}$$ The CTMC is constructed by running the embedded chain, with each state $i$ held for an independent $\text{Exp}(\nu_i)$ time.

Example — Two-State CTMC

Consider $S = \{0,1\}$ with rates $q_{01} = \alpha > 0$ and $q_{10} = \beta > 0$. The generator is $$Q = \begin{pmatrix} -\alpha & \alpha \\ \beta & -\beta \end{pmatrix}.$$ The chain alternates between states 0 and 1, holding in state 0 for $\text{Exp}(\alpha)$ time and in state 1 for $\text{Exp}(\beta)$ time.

This is the simplest CTMC model for a system switching between two modes (on/off, working/broken).

Example — M/M/1 Queue

Customers arrive at rate $\lambda$ (Poisson) and are served at rate $\mu$. The state $X(t)$ = number of customers at time $t$ is a CTMC on $\{0, 1, 2, \ldots\}$ with $q_{n,n+1} = \lambda$, $q_{n,n-1} = \mu \cdot \mathbf{1}_{n \ge 1}$, and $q_{nn} = -(\lambda + \mu \cdot \mathbf{1}_{n \ge 1})$. This is a birth-death CTMC.

Remark — Infinitesimal Interpretation

The generator has an infinitesimal interpretation: for $h \to 0$, $$P(X(t+h) = j \mid X(t) = i) = q_{ij} h + o(h) \quad (i \ne j), \quad P(X(t+h) = i \mid X(t) = i) = 1 + q_{ii} h + o(h).$$ This connects the generator to the derivative of the transition semigroup at $t=0$: $Q = P'(0)$.
